# Часто задаваемые вопросы (FAQ)

## 1. Как выдать ключи игроку?

Используйте консольную команду:
```
cases.give <steamid> keys <количество>
```

Например:
```
cases.give 76561198012345678 keys 10
```

Игрок получит уведомление о получении ключей.

---

## 2. Как создать новый кейс?

Вы можете создавать и редактировать кейсы прямо в игре, если у вас есть право `ultimatecases.edit`. Откройте интерфейс кейсов и используйте режим редактирования для добавления нового кейса, настройки наград и шансов.

---

## 3. Как настроить обмен scrap на ключи?

В секции `"Exchange Settings" -> "Currencies"` добавьте валюту типа `Item`:

```json
{
 "ID": 1,
 "Title": "SCRAP",
 "Type": "Item",
 "ItemShortname": "scrap",
 "ItemSkin": 0,
 "BaseExchangeRate": 200,
 "DailyLimit": 100
}
```

Это позволит обменивать 200 scrap на 1 ключ.

---

## 4. Как настроить скидки для VIP-игроков?

В секции `"PrivilegeSettings"` настройте права и скидки:

```json
{
 "Permission": "ultimatecases.vip1",
 "DiscountPercent": 10,
 "CustomDailyLimit": 0
}
```

Затем выдайте право:
```
oxide.grant group vip ultimatecases.vip1
```

---

## 5. Можно ли ограничить открытие кейсов при комбат-блоке?

Да. Плагин автоматически блокирует открытие кейсов в следующих ситуациях (при наличии соответствующих плагинов):
- Комбат-блок
- Рейд-блок
- Зона чужого шкафа
- Плавание, ранение, дуэль

---

## 6. Как добавить NPC-торговца на монументе?

В секции `"Exchange Settings" -> "MonumentTraders"` настройте бота:

```json
{
 "MonumentFilter": "lighthouse",
 "Prefab": "assets/prefabs/npc/bandit/missionproviders/missionprovider_outpost_b.prefab",
 "PositionOffset": {"x": -7, "y": 14, "z": 0},
 "CurrencyID": 1,
 "DefaultKeysAmount": 1
}
```

Необходим плагин **MonumentFinder**.

---

## 7. Как работает система раритета?

Система раритета автоматически окрашивает фон предмета в зависимости от его шанса выпадения. Настройте диапазоны шансов и соответствующие изображения в секции `"Rarity Settings"`.

---

## 8. Как включить бесплатные кейсы?

Бесплатные кейсы настраиваются для каждого кейса отдельно через режим редактирования. Установите стоимость 0 ключей и настройте таймер для повторного получения.

---

## 9. Игрок не может забрать предмет из хранилища. Почему?

Возможные причины:
1. **Вайп-блок** -- предмет заблокирован после вайпа.
2. **Рейд-блок / комбат-блок** -- активна блокировка.
3. **Зона чужого шкафа** -- игрок в зоне Building Blocked.
4. **Предмет не найден** -- ошибка в конфигурации предмета.

---

## 10. Как работают оптовые скидки?

В секции `"BulkDiscounts"` настройте пороги:
```json
[
 {"MinKeys": 10, "DiscountPercent": 5},
 {"MinKeys": 25, "DiscountPercent": 10},
 {"MinKeys": 50, "DiscountPercent": 15}
]
```

При покупке 10+ ключей -- скидка 5%, 25+ -- 10% и т.д. Скидки накладываются на базовый курс обмена.

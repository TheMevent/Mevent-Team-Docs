# Конфигурация Servers

Файл конфигурации расположен по пути: `oxide/config/Servers.json`

## Основные параметры

| Параметр | По умолчанию | Описание |
|---|---|---|
| Permission | `servers.use` | Право доступа для использования |
| ServerPanel Template | `V1` | Шаблон отображения: `V1` или `V2` |
| Interval updates servers info | `300` | Интервал обновления информации (секунды, минимум 30) |
| Open UI Commands | `["servers"]` | Чат-команды для открытия |
| Default server banner | URL | Стандартный баннер (если у сервера нет своего) |

## Настройки серверов

Массив `"Servers"` содержит список серверов:

| Параметр | Описание |
|---|---|
| IP | IP-адрес или домен сервера |
| Port | Порт подключения |
| BmID | ID на BattleMetrics (пустая строка -- отключить) |
| ScrollDescriptionHeight | Высота прокрутки описания (пиксели) |

### Пример конфигурации сервера

```json
{
 "IP": "64.40.9.56",
 "Port": 28014,
 "BmID": "9255283",
 "ScrollDescriptionHeight": 186.0
}
```

## Источники информации

### Без BattleMetrics (BmID = "")

Плагин запрашивает информацию напрямую через Steam Query API. Доступна базовая информация: название, онлайн.

### С BattleMetrics (BmID указан)

Плагин запрашивает расширенную информацию через BattleMetrics API:
- Название сервера
- Описание
- Текущий и максимальный онлайн
- Размер карты
- Тип карты
- Баннер

:::tip Совет
Для получения BmID найдите ваш сервер на [battlemetrics.com](https://www.battlemetrics.com) и скопируйте ID из URL.
:::

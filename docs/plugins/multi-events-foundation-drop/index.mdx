# Foundation Drop

## Описание

**Foundation Drop** -- это режим расширения для плагина MultiEvents, представляющий собой мини-игру на выживание. Игроки спавнятся на платформе из фундаментов, которые начинают разрушаться один за другим. Игроки должны прыгать между оставшимися фундаментами, избегая падения. Последний выживший игрок побеждает и получает награду. Когда остаётся один фундамент, начинается финальная фаза с радиацией и выдачей оружия.

## Ключевые возможности

- Мини-игра на выживание с разрушающимися фундаментами
- Настраиваемый размер арены (матрица NxN)
- Фаза голосования для регистрации участников
- Финальная битва с радиацией и оружием на последнем фундаменте
- Настраиваемая выдача кита или предмета для финальной фазы
- Поддержка нескольких победителей
- Блокировка определённых команд во время ивента
- Интеграция с NoEscape, NTeleportation и Vanish
- Интеграция с MultiEventsInventory для выдачи наград
- Интеграция с MultiEventsLeaderboard для глобальной статистики
- Поддержка Notify / UINotify для уведомлений
- Минимальное количество игроков для старта (настраивается)

## Требования

| Зависимость | Обязательность | Описание |
|---|---|---|
| **MultiEvents** | Обязательно | Основной плагин системы событий |
| **ImageLibrary** | Обязательно | Загрузка и отображение изображений в интерфейсе |
| MultiEventsInventory | Опционально | Для выдачи наград через инвентарь событий |
| MultiEventsLeaderboard | Опционально | Для глобальной таблицы лидеров |
| Kits | Опционально | Для выдачи наборов в финальной фазе |
| Vanish | Опционально | Для обнаружения невидимых игроков |
| NTeleportation | Опционально | Для блокировки телепортации во время ивента |
| NoEscape | Опционально | Для блокировки выхода из зоны ивента |
| Notify / UINotify | Опционально | Для визуальных уведомлений |

## Быстрый старт

1. Убедитесь, что плагины **MultiEvents** и **ImageLibrary** установлены и работают.
2. Скопируйте файл `MultiEventsFoundationDrop.cs` в папку `oxide/plugins/` вашего сервера.
3. Плагин автоматически зарегистрируется в системе MultiEvents.
4. Настройте минимальное количество игроков и размер арены в конфигурации.
5. Запустите ивент через MultiEvents или командой:
 ```
 foundationdrop.start
 ```

# Команды

## Команды для игроков

| Команда | Описание |
|---|---|
| `/events` | Открыть панель серверных ивентов |

:::note Настраиваемые команды
Команда по умолчанию — `/events`. Вы можете изменить или добавить дополнительные команды в конфигурации (параметр "Info Commands").
:::

### Использование панели игроков

После открытия панели игрок может:

- Просматривать список всех серверных ивентов
- Видеть информацию о каждом ивенте (описание, баннер)
- Открывать подробную информацию об ивенте
- Просматривать таблицу лидеров (если модуль включён)
- Открывать инвентарь наград (если модуль включён)
- Скрывать уведомления о конкретных ивентах

## Команды для администраторов

| Команда | Право | Описание |
|---|---|---|
| `/ShowUiEditor` | `multievents.edit` | Открыть визуальный редактор интерфейса |

### Визуальный редактор

Редактор позволяет администраторам:

- **Редактировать элементы UI** — изменять позицию, размер, цвет и прозрачность любого элемента
- **Выбирать якоря** — настраивать привязку элементов (верх, низ, центр, углы)
- **Редактировать цвета** — интерфейс выбора цветов для любого элемента
- **Редактировать текст** — изменять тексты, шрифты и размеры
- **Запускать и останавливать ивенты** — кнопки запуска/остановки прямо из панели

### Управление ивентами через панель

Администраторы с правом `multievents.edit` могут:

| Действие | Описание |
|---|---|
| Запустить ивент | Кнопка "ЗАПУСТИТЬ" на странице ивента |
| Остановить ивент | Кнопка "ОСТАНОВИТЬ" для активного ивента |
| Случайный ивент | Кнопка "РАНДОМ" для запуска случайного события |
| Редактировать UI | Кнопка "UI РЕДАКТОР" для открытия визуального редактора |

## Права доступа

| Право | Описание |
|---|---|
| `multievents.edit` | Доступ к визуальному редактору и управлению ивентами |

Дополнительные права (настраиваемые в конфигурации):

| Параметр | Описание |
|---|---|
| Право для инвентаря | Доступ к инвентарю наград (если задано) |
| Право для таблицы лидеров | Доступ к таблице лидеров (если задано) |

## Интерфейс уведомлений

При запуске ивентов игроки получают всплывающие уведомления. Игроки могут:

- Скрыть все уведомления об ивентах
- Скрыть уведомления для конкретного ивента

Настройки уведомлений сохраняются для каждого игрока.

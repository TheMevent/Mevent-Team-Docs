# Установка Car Sharing

## Системные требования

- Сервер Rust с Oxide/uMod
- Плагин **ImageLibrary** (обязателен)
- Плагин **SpawnModularCar** (обязателен)
- Один из экономических плагинов (Economics, ServerRewards, BankSystem) или использование внутриигровых предметов

## Пошаговая установка

### Шаг 1: Загрузка и установка зависимостей

1. Установите плагин **ImageLibrary** в папку `oxide/plugins`.
2. Установите плагин **SpawnModularCar** в папку `oxide/plugins`.
3. (Опционально) Установите один из экономических плагинов.
4. (Опционально) Установите **Notify** или **UINotify** для уведомлений.
5. (Опционально) Установите **HumanNPC** для NPC-ботов.

### Шаг 2: Установка плагина

1. Загрузите файл `CarSharing.cs`.
2. Поместите его в папку `oxide/plugins`.
3. Перезагрузите сервер или выполните команду в консоли:
 ```
 oxide.reload CarSharing
 ```

### Шаг 3: Первоначальная настройка

После первого запуска плагин создаст конфигурационный файл:
```
oxide/config/CarSharing.json
```

Откройте его и настройте:

1. **Команды** -- задайте чат-команды для открытия меню (по умолчанию: `rent`, `carsharing`).
2. **Экономика** -- укажите тип валюты (плагин или предмет), название плагина и хуки.
3. **Предмет вызова** -- настройте предмет, который игрок получает для вызова арендованной машины.
4. **Машины** -- добавьте автомобили с указанием названия, изображения, стоимости, класса и настроек модулей.
5. **NPC-боты** -- привяжите NPC-ботов к определённым классам машин.

### Шаг 4: Настройка прав доступа

Назначьте права доступа игрокам или группам через консоль:

```
oxide.grant user <SteamID> <право>
oxide.grant group <группа> <право>
```

### Шаг 5: Настройка NPC (опционально)

1. Создайте NPC-бота с помощью плагина **HumanNPC**.
2. Скопируйте его ID.
3. Добавьте ID бота в секцию `Bots` конфигурации, указав доступные классы машин.

## Проверка установки

1. Зайдите на сервер.
2. Введите `/rent` в чат.
3. Должно открыться меню каршеринга с доступными машинами.
4. Выберите машину и нажмите кнопку аренды.
5. Убедитесь, что стоимость списывается и машина появляется.

## Обновление плагина

При обновлении плагина просто замените файл `CarSharing.cs` в папке `oxide/plugins`. Конфигурация сохранится. Если в новой версии появились новые параметры, они будут добавлены автоматически.

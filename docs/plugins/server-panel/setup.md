# Установка ServerPanel

Это руководство проведёт вас через весь процесс установки информационного меню сервера.

## Шаг 1: Установите зависимости

### ImageLibrary (обязательно)

1. Скачайте **ImageLibrary** с [umod.org](https://umod.org/plugins/image-library).
2. Поместите файл `ImageLibrary.cs` в папку `oxide/plugins/`.
3. Дождитесь загрузки плагина.

:::warning Важно
Без ImageLibrary плагин ServerPanel **не будет корректно работать**. Все изображения в меню загружаются через эту библиотеку.
:::

### Дополнительные зависимости (рекомендуется)

| Плагин | Файл | Назначение |
|---|---|---|
| ServerPanel Installer | `ServerPanelInstaller.cs` | Установка готовых шаблонов меню |
| ServerPanel PopUps | `ServerPanelPopUps.cs` | Всплывающие окна в меню |
| ServerPanel Migrations | `ServerPanelMigrations.cs` | Миграция данных при обновлении |
| Notify | `Notify.cs` | Красивые уведомления вместо чата |
| NoEscape | -- | Блокировка меню при рейд-блоке |

## Шаг 2: Установите основной плагин

1. Скопируйте следующие файлы в папку `oxide/plugins/`:
 - `ServerPanel.cs` (основной плагин)
 - `ServerPanelInstaller.cs` (установщик шаблонов)
 - `ServerPanelPopUps.cs` (всплывающие окна)
 - `ServerPanelMigrations.cs` (миграции)

2. Дождитесь загрузки всех плагинов.

:::info Порядок загрузки
Плагины-расширения (Installer, PopUps, Migrations) зависят от основного ServerPanel и загрузятся автоматически после него.
:::

## Шаг 3: Первоначальная настройка с помощью установщика

1. Зайдите на сервер как администратор.
2. Выдайте себе право на использование установщика:
 ```
 oxide.grant user ВашSteamID serverpanelinstaller.admin
 ```
3. Откройте установщик:
 ```
 serverpanelinstaller.open
 ```
4. Выберите один из доступных шаблонов меню.
5. Нажмите "Установить".

:::tip Совет
Шаблон создаст начальную структуру меню с категориями, страницами и элементами интерфейса. Вы сможете полностью настроить его под свой сервер через встроенный визуальный редактор.
:::

## Шаг 4: Настройте права доступа

### Право на редактирование меню
```
oxide.grant user ВашSteamID serverpanel.edit
```

### Право на редактирование всплывающих окон
```
oxide.grant user ВашSteamID serverpanelpopups.edit
```

:::warning Внимание
Выдавайте право на редактирование только доверенным администраторам. Редактор позволяет полностью менять содержимое и структуру меню.
:::

## Шаг 5: Настройте команды открытия

Команды для открытия меню задаются для каждой категории. По умолчанию шаблон уже содержит команды (например, `/menu`). Чтобы изменить или добавить команды:

1. Зайдите в режим редактирования (кнопка в интерфейсе).
2. Откройте настройки нужной категории.
3. Измените команды.
4. Сохраните изменения.

## Шаг 6: Настройте конфигурацию

Конфигурационный файл расположен по пути: `oxide/config/ServerPanel.json`

Основные параметры для настройки:
- **Интеграция с Notify** -- использовать ли красивые уведомления
- **Автоматическое открытие** -- показывать ли меню при входе игрока
- **Блокировки** -- запрет открытия при рейд-блоке, билд-блоке, боевом блоке
- **Экономика в шапке** -- показ баланса Economics, ServerRewards, BankSystem

Подробнее читайте в разделе [Конфигурация](./configuration).

## Шаг 7: Настройте содержимое меню

Войдите в режим редактирования на сервере:

1. Откройте меню.
2. Нажмите кнопку редактирования в верхней части.
3. Добавляйте элементы: текст, изображения, кнопки.
4. Настраивайте позицию, размер, шрифт и цвет каждого элемента.
5. Создавайте новые страницы и категории.

:::info Примечание
Все изменения через редактор сохраняются автоматически. Перезагрузка плагина не требуется.
:::

## Обновление плагина

При обновлении:

1. Замените все файлы плагина новыми версиями.
2. Если появляется сообщение о необходимости миграции, установите `ServerPanelMigrations.cs`.
3. Миграция запустится автоматически или может быть запущена вручную:
 ```
 sp.migrations run all
 ```

:::warning Внимание
Обязательно создайте резервную копию папки `oxide/data/ServerPanel/` перед обновлением. Там хранятся все данные меню: категории, шаблоны, элементы, локализация.
:::

## Структура данных

После установки плагин создаёт следующие файлы данных:

```
oxide/data/ServerPanel/
 Categories.json -- категории меню
 Template.json -- шаблон интерфейса
 Localization.json -- настройки локализации
 HeaderFields.json -- поля шапки меню
 Players.json -- данные игроков
```

Конфигурация:
```
oxide/config/ServerPanel.json
```

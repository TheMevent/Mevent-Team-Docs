# Wipe Schedule

## Что делает этот плагин?

**Wipe Schedule** позволяет вашему серверу отображать красивый интерактивный календарь с расписанием вайпов, ивентов и других событий. Администраторы могут настраивать повторяющиеся события (глобальные вайпы, вайпы карты, блокировки после вайпа, пользовательские события) через удобный внутриигровой интерфейс. Игроки всегда будут в курсе, когда произойдёт следующий вайп или событие на сервере.

## Ключевые возможности

- Полноценный интерактивный календарь с месячным видом и подсветкой текущего дня
- Поддержка различных типов событий: глобальный вайп, вайп карты, блокировка после вайпа, пользовательские события
- Гибкая настройка повторяющихся событий (по дням, неделям, месяцам, годам)
- Визуальный редактор событий прямо в игре для администраторов
- Настраиваемые иконки и цвета для каждого типа событий
- Поддержка часовых поясов
- Интеграция с ServerPanel, Notify и UINotify
- Поддержка ImageLibrary для пользовательских иконок
- Возможность начинать неделю с воскресенья или понедельника

## Требования

- Сервер Rust с Oxide/uMod
- **ImageLibrary** (обязательно) -- для отображения иконок событий
- Рекомендуемые плагины (опционально): Notify или UINotify, ServerPanel

## Быстрый старт

1. Установите ImageLibrary, если он ещё не установлен.
2. Скопируйте файл плагина в папку `oxide/plugins`.
3. Загрузите плагин и дождитесь загрузки изображений.
4. Введите в чат `/wipe` для просмотра календаря.
5. Для настройки событий используйте кнопку "Изменить ивенты" в интерфейсе (требуется право администратора).

## Права доступа

| Право | Описание |
|---|---|
| `wipeschedule.admin` | Доступ к редактированию событий и расписания через интерфейс |

## Команды игроков

| Команда | Описание |
|---|---|
| `/wipe` | Открывает календарь событий сервера |
| `/wipedata` | Альтернативная команда для открытия календаря |

:::tip Пример
Введите `/wipe` в чате, чтобы увидеть календарь с отмеченными датами вайпов и событий.
:::

## Команды администратора

| Команда | Описание |
|---|---|
| `wipeschedule.time` | Консольная команда для просмотра текущего времени сервера с учётом часового пояса |

Основная настройка событий выполняется через визуальный интерфейс в игре (кнопка "Изменить ивенты" в календаре).

:::tip Пример
Для создания нового события: откройте календарь командой `/wipe`, нажмите "Изменить ивенты", затем "Добавить событие". Выберите тип, укажите название и настройте расписание повтора.
:::

## Устранение неполадок

| Проблема | Решение |
|---|---|
| Иконки не отображаются | Убедитесь, что ImageLibrary установлен и загружен. Перезагрузите плагин. |
| Кнопка "Изменить ивенты" не видна | Проверьте, что у вас есть право `wipeschedule.admin`. |
| Неправильное время в календаре | Настройте параметр "Time Zone" в конфигурации под ваш часовой пояс. |
| Календарь пустой | Добавьте события через интерфейс администратора или проверьте файл данных расписания. |
| Плагин не загружается | Убедитесь, что установлен ImageLibrary. Проверьте консоль на наличие ошибок. |
| Пользовательские события не отображаются | Проверьте, что в конфигурации заданы ключи перевода для пользовательских событий. |

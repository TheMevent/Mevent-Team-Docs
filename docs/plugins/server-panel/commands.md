# Команды ServerPanel

## Обзор

Команды ServerPanel делятся на три группы: пользовательские команды для открытия меню (настраиваемые), административные команды для управления, и внутренние команды установщика.

## Команды игроков

Команды для открытия меню задаются администратором для каждой категории. При создании категории вы указываете одну или несколько команд, по которым игроки могут открыть меню.

### Использование

```
/(команда)
```

Игрок вводит команду в чат, и у него открывается меню на соответствующей категории.

**Типичные примеры:**

| Команда | Что открывается |
|---|---|
| `/menu` | Главная страница меню сервера |
| `/info` | Информация о сервере |
| `/rules` | Правила сервера |
| `/shop` | Магазин (если интегрирован) |
| `/donate` | Информация о донате |

:::info Примечание
Это примерные команды. Конкретные команды зависят от того, как администратор настроил категории меню. Каждая категория может иметь несколько команд.
:::

### Сообщения об ограничениях

Игроки могут получить следующие сообщения:

| Сообщение | Причина |
|---|---|
| "Вы не можете открыть меню: вы находитесь в строительной зоне!" | Игрок в зоне блокировки строительства |
| "Вы не можете открыть меню: рейд-блок!" | Игрок под рейд-блоком |
| "Вы не можете открыть меню: боевой блок!" | Игрок под боевым блоком |
| "У вас нет доступа!" | У игрока нет права на открытие данной категории |

## Команды администратора

### Управление установщиком шаблонов

```
serverpanelinstaller.open
```

**Требуемое право:** `serverpanelinstaller.admin`

Открывает интерфейс установщика шаблонов. Позволяет выбрать и установить готовый дизайн меню.

---

### Миграция данных

```
sp.migrations list
```

Выводит список всех доступных миграций. Выполняется из серверной (RCON) консоли.

```
sp.migrations run [имя]
```

Запускает указанную миграцию данных. Используется при обновлении плагина, когда формат данных изменился.

**Пример:**
```
sp.migrations run all
```

:::warning Внимание
Миграции выполняются только из серверной консоли (RCON). Рекомендуется создать резервную копию данных перед миграцией.
:::

## Режим редактирования (встроенный визуальный редактор)

**Требуемое право:** `serverpanel.edit`

Администраторы с правом редактирования видят специальные кнопки в интерфейсе меню.

### Вход в режим редактирования

1. Откройте меню на сервере.
2. Нажмите кнопку режима редактирования в области категорий.
3. Интерфейс переключится в режим администрирования.

### Доступные действия в режиме редактирования

#### Работа с категориями

| Действие | Описание |
|---|---|
| Добавить категорию | Создать новую категорию в меню |
| Удалить категорию | Убрать категорию (нельзя удалить последнюю) |
| Изменить порядок | Переместить категорию |
| Настроить | Изменить название, команды, права, видимость |

#### Работа со страницами

| Действие | Описание |
|---|---|
| NEW PAGE | Добавить новую страницу в текущую категорию |
| DELETE PAGE | Удалить текущую страницу (нельзя удалить последнюю) |
| CLONE PAGE | Создать копию текущей страницы |
| MOVE PAGE | Переместить страницу |

#### Работа с элементами

| Действие | Описание |
|---|---|
| Добавить элемент | Разместить текст, изображение или кнопку на странице |
| Редактировать элемент | Изменить содержимое, позицию, размер, шрифт, цвет |
| Удалить элемент | Убрать элемент со страницы |
| Перемещение | Перетаскивание элемента на нужную позицию |

#### Работа с шапкой

| Действие | Описание |
|---|---|
| Редактировать шапку | Изменить элементы в верхней части меню |
| Редактировать всплывающие окна | Открыть редактор Pop-Ups (требуется плагин ServerPanelPopUps) |

### Типы страниц

| Тип | Описание |
|---|---|
| UI (интерфейс) | Страница с визуальными элементами, которые вы размещаете через редактор |
| Plugin (плагин) | Страница, содержимое которой генерируется другим плагином |

:::tip Совет
Тип "Plugin" позволяет интегрировать другие плагины прямо в меню ServerPanel. Указанный плагин будет вызван для генерации содержимого страницы.
:::

## Внутренние консольные команды

Эти команды используются интерфейсом автоматически и не предназначены для ручного ввода:

| Команда | Описание |
|---|---|
| `UI_ServerPanel` | Основная команда обработки действий в интерфейсе |
| `UI_ServerPanel_Close` | Закрытие меню |
| `UI_ServerPanel_Send_Command` | Выполнение команды из интерфейса |
| `serverpanel_broadcastvideo` | Воспроизведение видео из интерфейса |

:::warning Внимание
Не используйте внутренние команды вручную -- они предназначены для работы интерфейса и могут вызвать непредвиденное поведение.
:::

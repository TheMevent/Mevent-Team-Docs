# Конфигурация Skip Night

Файл конфигурации расположен по пути: `oxide/config/SkipNight.json`

## Основные параметры

| Параметр | По умолчанию | Описание |
|---|---|---|
| Enable Voting System | `true` | Включить систему голосования |

## Настройки экономики (Economy)

| Параметр | Описание |
|---|---|
| Type | Тип оплаты: `Plugin` или `Item` |
| Plugin name | Название плагина (Economics и др.) |
| Balance add/remove/show hook | Функции для работы с балансом |
| ShortName | ShortName предмета (для типа Item) |
| Skin | Скин предмета |

## Настройки времени (Time Settings)

| Параметр | По умолчанию | Описание |
|---|---|---|
| DayStart | `08:00` | Время начала дня |
| NightStart | `20:00` | Время начала ночи |
| TimeStart | `20:00` | Начало окна голосования |
| TimeEnd | `21:00` | Конец окна голосования |
| TimeSet | `08:00` | Время после пропуска ночи |
| TimeVote | `60` | Длительность голосования (секунды) |
| ForceSkip | `true` | Автоматический пропуск при пороге |
| LengthNight | `5` | Длительность ночи (минуты) |
| LengthFastNight | `2` | Длительность ускоренной ночи |
| LengthDay | `45` | Длительность дня (минуты) |
| FullMoon | `true` | Включить полнолуние |
| MissedNights | `0` | Ночей до автоскипа (0 -- выключено) |
| FullMoonDates | Массив дат | Даты полнолуний |

## Настройки голосования (Votes Settings)

| Параметр | По умолчанию | Описание |
|---|---|---|
| VoteCommands | `["voteday", "vt"]` | Команды для голосования |
| OnStartBroadcasting | `false` | Объявлять в чат о начале голосования |
| VotesCount | `5` | Минимальное количество голосов |
| UsePercent | `true` | Использовать процент от онлайна |
| Percent | `30` | Процент голосов для пропуска |
| EnabledPay | `false` | Включить оплату за голосование |
| SkipCost | `100` | Стоимость голосования |
| RefundWhenCanceled | `false` | Возврат оплаты при неудаче |
| EnableCommandsAfterVoting | `false` | Выполнять команды после результата |
| CommandsAfterSuccessfulVoting | `""` | Команды после успеха (разделитель `\|`) |
| CommandsAfterLostVoting | `""` | Команды после неудачи |

## Настройки интерфейса (UI Settings)

| Параметр | Описание |
|---|---|
| InitialDisplay | Показывать интерфейс при начале голосования |
| DestroyTime | Время до автоматического скрытия (секунды) |
| ShowImage | Показывать иконку |
| Image | URL иконки |
| Colors | Цвета интерфейса (4 настраиваемых цвета) |
| VotingButton | Размеры кнопки голосования |
| ProgressBar | Размеры прогресс-бара |
| HideBtn / CloseBtn | Кнопки скрытия и закрытия |
| CloseAfterVoting | Закрывать интерфейс после голосования |

# Clans

## Что делает этот плагин?

**Clans** позволяет вашему серверу предоставить игрокам полноценную клановую систему с графическим интерфейсом. Игроки могут создавать и управлять кланами, приглашать участников, назначать модераторов, формировать альянсы с другими кланами и отслеживать рейтинги. Плагин интегрируется с игровыми командами Rust, поддерживает клановый чат, систему дружественного огня, скины для снаряжения и детальную статистику участников.

## Ключевые возможности

- **Создание и управление кланами** -- создание клана с уникальным тегом, назначение модераторов, приглашение и исключение участников
- **Система альянсов** -- объединение кланов в союзы с общим чатом и управлением дружественным огнём
- **Рейтинги и топы** -- таблицы лучших игроков и кланов с настраиваемой системой очков за убийства, добычу и лут
- **Клановый чат** -- отдельный чат для общения внутри клана и между союзниками
- **Дружественный огонь** -- гибкая настройка дружественного огня для клана и альянсов, включая турели
- **Скины снаряжения** -- единый стиль клана с настраиваемыми скинами для оружия и брони
- **Интеграция с игровыми командами** -- работа через стандартную систему команд Rust (приглашения, кик, выход из команды)
- **Платные функции** -- опциональная плата за создание клана, вступление, исключение и другие действия

## Требования

| Зависимость | Обязательная? | Описание |
|---|---|---|
| ImageLibrary | Да | Загрузка и отображение изображений в интерфейсе |
| BetterChat / IQChat | Нет | Отображение клан-тегов в чате |
| Notify / UINotify | Нет | Уведомления для игроков |
| ZoneManager | Нет | Управление дружественным огнём в определённых зонах |
| Economics / ServerRewards | Нет | Платные функции (создание клана, вступление и т.д.) |

## Быстрый старт

1. Установите плагин **ImageLibrary** (обязательная зависимость)
2. Поместите файл плагина в папку `oxide/plugins` вашего сервера
3. Перезагрузите сервер или выполните команду `o.reload Clans` в консоли
4. Откройте сгенерированный файл конфигурации `oxide/config/Clans.json` и настройте параметры под ваш сервер
5. Игроки могут открыть интерфейс кланов командой `/clan` в чате

## Права доступа

| Право | Кому нужно | Что открывает |
|---|---|---|
| `clans.admin` | Администраторам | Полное управление всеми кланами через консольные команды |
| `clans.cancreate` | Игрокам (опционально) | Разрешение на создание клана (если включено в настройках) |
| `clans.canjoin` | Игрокам (опционально) | Разрешение на вступление в клан (если включено в настройках) |
| `clans.canleave` | Игрокам (опционально) | Разрешение на выход из клана (если включено в настройках) |
| `clans.cankick` | Модераторам (опционально) | Разрешение на исключение участников (если включено в настройках) |
| `clans.candisband` | Лидерам (опционально) | Разрешение на расформирование клана (если включено в настройках) |
| `clans.canskins` | Игрокам (опционально) | Доступ к системе скинов клана (если включено в настройках) |

:::info
Все права, кроме `clans.admin`, являются опциональными. Они активируются только если включена соответствующая настройка в секции "Permission Settings" конфигурации.
:::

## Команды игроков

| Команда | Описание | Пример |
|---|---|---|
| `/clan` | Открыть интерфейс клана | `/clan` |
| `/clan create <тег>` | Создать клан с указанным тегом | `/clan create ALPHA` |
| `/clan leave` | Покинуть текущий клан | `/clan leave` |
| `/clan invite <игрок>` | Пригласить игрока в клан | `/clan invite PlayerName` |
| `/clan withdraw <игрок>` | Отозвать приглашение | `/clan withdraw PlayerName` |
| `/clan kick <игрок>` | Исключить участника из клана (для модераторов) | `/clan kick PlayerName` |
| `/clan promote <игрок>` | Повысить участника до модератора (для лидера) | `/clan promote PlayerName` |
| `/clan demote <игрок>` | Понизить модератора до участника (для лидера) | `/clan demote PlayerName` |
| `/clan disband` | Расформировать клан (для лидера) | `/clan disband` |
| `/clan description <текст>` | Задать описание клана (для лидера) | `/clan description Лучший клан!` |
| `/clan tagcolor <цвет>` | Установить цвет тега клана (HEX) | `/clan tagcolor FF5500` |
| `/clan join` | Открыть интерфейс поиска клана | `/clan join` |
| `/clan ff` | Переключить дружественный огонь по соклановцам | `/clan ff` |
| `/clan allyff` | Переключить дружественный огонь по союзникам | `/clan allyff` |
| `/clan allyinvite <тег>` | Пригласить другой клан в альянс | `/clan allyinvite BRAVO` |
| `/clan allywithdraw <тег>` | Отозвать приглашение в альянс | `/clan allywithdraw BRAVO` |
| `/clan allyaccept <тег>` | Принять приглашение в альянс | `/clan allyaccept BRAVO` |
| `/clan allycancel <тег>` | Отклонить приглашение в альянс | `/clan allycancel BRAVO` |
| `/clan allyrevoke <тег>` | Разорвать альянс с кланом | `/clan allyrevoke BRAVO` |
| `/cff` | Быстрое переключение дружественного огня клана | `/cff` |
| `/aff` | Быстрое переключение дружественного огня альянса | `/aff` |
| `/c <сообщение>` | Отправить сообщение в клановый чат | `/c Привет всем!` |
| `/a <сообщение>` | Отправить сообщение в чат альянса | `/a Собираемся на рейд` |
| `/cinfo <тег>` | Посмотреть информацию о клане | `/cinfo ALPHA` |

## Устранение неполадок

### Интерфейс кланов не открывается
**Симптомы:** Игрок вводит `/clan`, но ничего не происходит.
**Причина:** Не установлен плагин ImageLibrary.
**Решение:** Установите плагин ImageLibrary и перезагрузите сервер. Убедитесь, что ImageLibrary успешно загрузился.

### Клан-теги не отображаются в чате
**Симптомы:** Игроки состоят в кланах, но теги не видны в чате.
**Причина:** Не настроена интеграция с чат-плагином.
**Решение:** Убедитесь, что в конфигурации включена опция "Enable clan tags in chat?" и установлен один из совместимых плагинов (BetterChat или IQChat). Проверьте формат тега в настройке "Tag format".

### Дружественный огонь не работает
**Симптомы:** Участники клана всё ещё наносят урон друг другу.
**Причина:** Дружественный огонь не включён или используется общий режим.
**Решение:** Проверьте настройку "Use Friendly Fire?" в конфигурации. Если используется общий дружественный огонь, его должен включить лидер клана. Также проверьте, что игроки не находятся в зонах из белого списка ZoneManager.

### Ошибка при создании клана: запрещённые символы
**Симптомы:** Игрок получает сообщение о запрещённых символах при создании клана.
**Причина:** Тег содержит символы, не входящие в список разрешённых, или содержит заблокированные слова.
**Решение:** Проверьте настройки "Clan Tag Settings" в конфигурации: список заблокированных слов и параметр "Allowed Characters". Убедитесь, что длина тега соответствует ограничениям (по умолчанию 2-6 символов).

### Платные функции не списывают баланс
**Симптомы:** Создание клана бесплатно, хотя настроена плата.
**Причина:** Не установлен или неправильно настроен плагин экономики.
**Решение:** Убедитесь, что указанный в конфигурации плагин экономики (например, Economics) загружен. Проверьте правильность названий хуков в секции "Economy".

### Данные кланов исчезают после вайпа
**Симптомы:** После вайпа карты все кланы пропадают.
**Причина:** Включена опция очистки данных при смене карты.
**Решение:** Проверьте настройки "Purge Settings" -- параметры "Wipe clans on new map save" и "Wipe players on new map save". Отключите их, если хотите сохранять кланы между вайпами.

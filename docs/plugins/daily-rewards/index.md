# Daily Rewards

## Что делает этот плагин?

**Daily Rewards** -- это система ежедневных наград, которая мотивирует игроков каждый день заходить на сервер. За каждый день входа игрок получает уникальную награду: предметы, ресурсы или пользовательские команды. Плагин отслеживает время нахождения игрока на сервере и автоматически отмечает дни посещения.

## Ключевые возможности

- Настраиваемые награды за каждый день (предметы, команды, валюта)
- Система кулдауна: игроку нужно провести определённое время на сервере для получения отметки
- VIP-кулдауны: разные группы могут иметь разное время ожидания
- Встроенный инвентарь для хранения невостребованных наград
- Специальные дни с уникальными наградами
- Уведомления через Notify при получении наград и окончании кулдауна
- Поддержка автоматического показа интерфейса при подключении игрока
- Вайп данных при новом сохранении карты (опционально)
- Настройка часового пояса и времени сброса ежедневного счётчика
- Режим редактирования наград прямо из интерфейса игры
- Поддержка нескольких языков (EN, RU, ZH-CN)
- Защита от получения наград при рейд-блоке, комбат-блоке и в зоне чужого шкафа
- Интеграция с AFKAPI, WipeBlock, NoEscape

## Требования

| Зависимость | Обязательность | Описание |
|---|---|---|
| **ImageLibrary** | Обязательно | Необходима для загрузки и отображения изображений в интерфейсе |
| Notify | Опционально | Для отправки графических уведомлений |
| WipeBlock | Опционально | Для блокировки получения наград после вайпа |
| NoEscape | Опционально | Для блокировки наград во время рейда/комбата |
| AFKAPI | Опционально | Для проверки AFK-статуса игрока |
| LangAPI | Опционально | Для поддержки мультиязычности |
| Oxide / uMod / Carbon | Обязательно | Совместим с Oxide, uMod и Carbon |

## Быстрый старт

1. Установите плагин **ImageLibrary**, если он ещё не установлен.
2. Скопируйте файл `DailyRewards.cs` в папку `oxide/plugins/` вашего сервера.
3. Перезагрузите сервер или выполните команду `oxide.reload DailyRewards`.
4. Выдайте право на использование плагина:
 ```
 oxide.grant group default dailyrewards.use
 ```
5. Зайдите на сервер и введите в чат `/daily` или `/rewards`.

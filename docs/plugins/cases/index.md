# Cases

## Что делает этот плагин?

**Cases** позволяет вашему серверу добавить систему кейсов (сундуков) с анимированной рулеткой. Игроки покупают кейсы за внутриигровую валюту или получают их за время, проведённое на сервере. Каждый кейс содержит набор предметов с разными шансами выпадения — от обычных ресурсов до редкого оружия.

## Ключевые возможности

- Неограниченное количество кейсов с индивидуальными настройками
- Анимированная рулетка с прокруткой предметов
- Покупка кейсов за внутриигровую валюту (Economics, ServerRewards или предметы)
- Индивидуальная валюта для каждого кейса
- Бонусный кейс с кулдауном (ежедневный бесплатный кейс)
- Система начисления кейсов за время игры (PlaytimeTracker)
- Кейсы по таймеру — автоматическое начисление через определённые интервалы
- Настраиваемые шансы выпадения и цветовая система редкости
- Система прав доступа для каждого кейса
- Типы наград: предметы, команды, плагины
- Полноэкранный и встроенный интерфейс
- Поддержка Oxide и Carbon

## Требования

| Зависимость | Обязательная? | Описание |
|---|---|---|
| **ImageLibrary** | Да | Требуется для отображения изображений |
| PlaytimeTracker | Нет | Начисление кейсов за время игры |
| Notify / UINotify | Нет | Красивые уведомления |
| ServerPanel | Нет | Интеграция с серверным меню |

## Быстрый старт

1. Установите **ImageLibrary**
2. Поместите файл плагина в папку `oxide/plugins`
3. Перезагрузите сервер или выполните `o.reload Cases`
4. Откройте конфигурационный файл и настройте кейсы, экономику и шансы
5. Выдайте необходимые права доступа игрокам

## Права доступа

| Право | Описание |
|---|---|
| `Cases.edit` | Право на редактирование кейсов через интерфейс |
| Индивидуальные права для кейсов | Задаются в конфигурации каждого кейса (например, `cases.vip`) |

## Команды игроков

| Команда | Описание |
|---|---|
| `/cases` | Открывает главное меню кейсов |
| `/opencase` | Аналог команды `/cases` |

:::tip Пример
```
/cases
```
Откроется интерфейс со списком всех доступных кейсов. Выберите кейс и нажмите кнопку для открытия.
:::

## Команды администратора

| Команда | Описание |
|---|---|
| `givecase <steamid> <id_кейса> <кол-во>` | Выдать кейс(ы) игроку |
| `givecase all <id_кейса> <кол-во>` | Выдать кейс(ы) всем игрокам онлайн |

:::tip Примеры
```
givecase 76561198000000000 1 5
givecase all 2 1
```
:::

## Устранение неполадок

### Интерфейс не открывается
Убедитесь, что **ImageLibrary** установлен и загружен. Проверьте консоль на наличие ошибок загрузки изображений.

### Игрок не может открыть кейс
Проверьте, что у игрока достаточно валюты для покупки кейса, или что у него есть необходимое право доступа, если для кейса задано ограничение.

### Бонусный кейс недоступен
Бонусный кейс имеет кулдаун (по умолчанию 24 часа). Игрок увидит таймер до следующего доступного открытия.

### Кейсы за время игры не начисляются
Убедитесь, что плагин **PlaytimeTracker** установлен, и что настройки трекера времени включены в конфигурации.

### Рулетка не прокручивается
Проверьте параметр "Enable item scrolling?" в конфигурации. Он должен быть установлен в `true`.

### Предметы с нулевым шансом не выпадают
Шанс выпадения определяет вероятность появления предмета в рулетке. Убедитесь, что суммарные шансы настроены корректно и не все предметы имеют минимальный шанс.

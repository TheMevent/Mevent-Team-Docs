# Установка Patrol Helicopter Hit

## Предварительные требования

Перед установкой убедитесь, что на сервере установлены:

- **Multi Events** — основной плагин (обязательно)
- **ImageLibrary** — для отображения UI-элементов (обязательно)

### Рекомендуемые плагины

- **Multi Events Inventory** — для хранения наград в инвентаре ивентов
- **Multi Events Leaderboard** — для расширенной таблицы лидеров
- **Notify** или **UINotify** — для красивых внутриигровых уведомлений

## Пошаговая установка

### Шаг 1. Загрузка

Загрузите файл режима Patrol Helicopter Hit из комплекта Expansion Pack 3.

### Шаг 2. Размещение файла

Скопируйте файл плагина в папку:

```
oxide/plugins/
```

### Шаг 3. Загрузка плагина

Плагин загрузится автоматически. Если сервер уже запущен, выполните в консоли:

```
oxide.reload MultiEventsPatrolHelicopterHit
```

### Шаг 4. Проверка

После загрузки плагина:

1. В папке `oxide/config/` появится файл конфигурации
2. Режим автоматически зарегистрируется в Multi Events
3. Ивент станет доступен для запуска

### Шаг 5. Настройка

Откройте файл конфигурации и настройте:

- Позицию и высоту появления вертолёта
- Здоровье вертолёта
- Длительность ивента
- Награды для победителей

После изменения конфигурации перезагрузите плагин.

## Проверка работоспособности

Запустите ивент вручную командой в консоли сервера:

```
patrolhelicopterhit.start
```

Убедитесь, что:
- Патрульный вертолёт появляется на карте
- UI-виджет отображается у всех игроков
- Попадания по вертолёту начисляют очки
- Таблица лидеров обновляется

Для остановки ивента используйте:

```
patrolhelicopterhit.stop
```

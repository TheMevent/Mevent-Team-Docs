# Часто задаваемые вопросы (FAQ)

## 1. Бот не подключается. Что делать?

Проверьте следующее:
1. Discord-бот (LinkBot) настроен и запущен.
2. Бот имеет доступ к RCON вашего сервера.
3. RCON включён в настройках сервера.
4. В консоли сервера нет ошибок подключения.

При успешном подключении в консоли появится: `LinkBot is connected`.

---

## 2. Как изменить группу для привязанных игроков?

В конфигурации измените параметр:
```json
"What group for linked players": "discord_vip"
```

Создайте группу и настройте для неё привилегии:
```
oxide.group add discord_vip
```

---

## 3. Код генерируется, но привязка не подтверждается. Почему?

Возможные причины:
1. **Бот не активен** -- проверьте статус бота командой `/link status`.
2. **Неправильный канал** -- игрок вводит код не в тот Discord-канал.
3. **Код истёк** -- сгенерируйте новый код.
4. **Ошибка RCON** -- проверьте подключение бота к RCON.

---

## 4. Как часто можно генерировать новый код?

По умолчанию кулдаун составляет 60 секунд. Измените в конфигурации:
```json
"Code generation delay (sec.)": 30
```

---

## 5. Можно ли отключить интерфейс и использовать только команды?

Да. Установите в конфигурации:
```json
"Use UI ?": false
```

В этом случае игроки будут управлять привязкой через команды:
- `/link status` -- проверить статус
- `/link code` -- получить код
- `/link remove` -- удалить привязку

---

## 6. Как настроить бонусы за привязку?

Бонусы за привязку настраиваются через группу Oxide. Создайте группу:
```
oxide.group add discord
```

Затем настройте нужные привилегии для этой группы через другие плагины. Элементы в секции `"Rewards UI elements"` -- это только визуальное отображение бонусов в интерфейсе.

---

## 7. Игрок хочет привязать другой Discord-аккаунт. Как?

1. Сначала отключите текущую привязку: `/link remove` или через интерфейс.
2. Затем сгенерируйте новый код и привяжите другой аккаунт.

---

## 8. Как показать предмет в качестве награды в интерфейсе?

Используйте параметр `"Element itemId"` с ID предмета:

```json
{
 "Element image (assets or url)": "",
 "Element itemId": -852563019,
 "Element skinId": 0,
 "Element text": "",
 "Element text Anchor": "LowerRight"
}
```

Если указан `itemId`, изображение предмета будет отображаться автоматически.

---

## 9. Поддерживает ли плагин несколько серверов?

Плагин работает на каждом сервере отдельно. Если у вас несколько серверов, установите плагин и бота на каждый сервер. Привязка будет работать независимо для каждого сервера.

---

## 10. Как проверить, привязан ли аккаунт конкретного игрока?

Игрок может проверить свой статус командой:
```
/link status
```

Администратор может проверить данные привязки через файлы данных плагина в папке `oxide/data/LinkingSystem/`.

# Часто задаваемые вопросы (FAQ)

## 1. Как изменить валюту ставок?

Откройте файл конфигурации `oxide/config/CoinFlipping.json` и измените секцию `Economy`. Вы можете использовать виртуальную валюту через плагин экономики или игровые предметы. Для предметов установите `"Type (Plugin/Item)"` в `"Item"` и укажите `ShortName` нужного предмета.

---

## 2. Как изменить минимальную и максимальную ставку?

В конфигурации измените параметры:
- `"Min Bet for Bot Game"` -- минимальная ставка для игры с ботом
- `"Max Bet for Bot Game (0 - disable)"` -- максимальная ставка для игры с ботом (0 -- без ограничения)
- `"Min Bet for Player Game"` -- минимальная ставка для игры с игроками
- `"Max Bet for Player Game (0 - disable)"` -- максимальная ставка для игры с игроками

---

## 3. Можно ли изменить команду открытия интерфейса?

Да. В конфигурации измените массив `"Commands"`. Например:
```json
"Commands": ["flip", "coin", "coinflip"]
```
Все указанные команды будут работать для открытия интерфейса.

---

## 4. Как изменить изображения монеты?

В секции `"Interface"` конфигурации измените URL-адреса изображений:
- `"Coin Image"` -- изображение монеты в заголовке
- `"Heads Image"` -- изображение стороны "Орёл"
- `"Tails Image"` -- изображение стороны "Решка"

Рекомендуется использовать изображения в формате PNG.

---

## 5. Что происходит, если игрок отключается во время игры?

Если один из игроков отключается во время игры против другого игрока, игра автоматически отменяется, а ставки возвращаются обоим участникам. Игрок получит уведомление: "Игра была отменена из-за отключения. Ставки возвращены."

---

## 6. Интерфейс не открывается. Что делать?

**Возможные причины:**
1. У игрока нет права `coinflipping.use`. Решение: `oxide.grant group default coinflipping.use`
2. Плагин ImageLibrary не установлен. Решение: установите ImageLibrary.
3. Плагин загрузился с ошибками. Решение: проверьте консоль сервера на наличие ошибок.

---

## 7. У игрока недостаточно средств, но он пытается сделать ставку.

Плагин автоматически проверяет баланс игрока перед началом игры. Если средств недостаточно, игроку будет показано сообщение об ошибке, и игра не начнётся.

---

## 8. Можно ли разрешить игру с ботом всем, а игру с игроками только VIP?

Да. Выдайте права следующим образом:
```
oxide.grant group default coinflipping.use
oxide.grant group default coinflipping.npcs
oxide.grant group vip coinflipping.players
oxide.grant group vip coinflipping.createroom
```

---

## 9. Можно ли использовать scrap как валюту для ставок?

Да. Настройте секцию экономики на использование предметов:
```json
{
 "Economy": {
 "Type (Plugin/Item)": "Item",
 "ShortName": "scrap",
 "Display Name (empty - default)": "",
 "Skin": 0
 }
}
```

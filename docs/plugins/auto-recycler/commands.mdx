# Команды и права доступа

## Права доступа (permissions)

| Право | Описание |
| ---------------------------------- | ----------------------------------------------------------- |
| `autorecycler.admin` | Полный доступ ко всем функциям плагина |
| `autorecycler.autorec` | Доступ к интерфейсу настройки автопереработки |
| `autorecycler.recvirtual` | Доступ к виртуальному переработчику по команде |
| `autorecycler.recyclercratetest` | Доступ к тестированию ящика-переработчика |
| `autorecycler.characteristics` | Доступ к включению/выключению переработки предметов (админ) |
| `autorecycler.recboxbuy` | Право на покупку ящика-переработчика |

### Выдача прав

Для конкретного игрока:
```
oxide.grant user <steamid> autorecycler.recvirtual
oxide.grant user <steamid> autorecycler.autorec
```

Для группы:
```
oxide.grant group default autorecycler.recvirtual
oxide.grant group default autorecycler.autorec
oxide.grant group vip autorecycler.recboxbuy
oxide.grant group admin autorecycler.admin
```

## Чат-команды игроков

### Виртуальный переработчик

```
/vrec
/virtualrec
/recvirtual
```

Открывает виртуальный переработчик. Работает так же, как обычный переработчик, но доступен из любого места. Требуется право `autorecycler.recvirtual`.

### Интерфейс автопереработки

```
/autorec
/ar
/autorecycler
```

Открывает графическую панель, в которой можно выбрать предметы для автоматической переработки. Требуется право `autorecycler.autorec`.

**Как пользоваться:**
1. Откройте панель командой `/autorec`.
2. Предметы сгруппированы по категориям (справа).
3. Нажмите на предмет, чтобы включить/выключить его автопереработку.
4. Зелёная подсветка = автопереработка включена, красная = выключена.

### Переключение автопереработки

```
/autorectoggle
```

Быстро включает или выключает всю автопереработку. Статус отображается в чате: зелёным -- включено, красным -- выключено.

### Информация о ящике-переработчике

```
/recbox
```

Показывает текущую стоимость покупки ящика-переработчика.

### Покупка ящика-переработчика

```
/recboxbuy
```

Покупает ящик-переработчик за ресурсы, указанные в конфигурации. Требуется право `autorecycler.recboxbuy`.

## Команды администратора

### Чат-команды

| Команда | Описание |
| --------------- | ----------------------------------------------------- |
| `/recboxtest` | Выдаёт 1000 ящиков-переработчиков (только для админов)|

### Консольные команды

| Команда | Описание |
| ------------------------------------- | ----------------------------------------------- |
| `recyclercrate.give` | Выдаёт ящик себе (если выполняется из игры) |
| `recyclercrate.give [имя_игрока]` | Выдаёт ящик-переработчик указанному игроку |

**Примеры:**

```
recyclercrate.give
recyclercrate.give PlayerName
recyclercrate.give 76561198012345678
```

## Графический интерфейс автопереработки

### Навигация

- **Категории** (правая панель): переключение между категориями предметов (Все, Оружие, Одежда, Ресурсы и т.д.)
- **Страницы** (нижняя панель): навигация по страницам с предметами
- **Поиск**: кнопка поиска для быстрого нахождения предметов

### Управление предметами

- Нажмите на предмет для переключения его состояния автопереработки
- Зелёный фон и галочка -- автопереработка включена
- Красный фон и крестик -- автопереработка выключена

### Административные функции

Администраторы с правом `autorecycler.characteristics` видят дополнительную кнопку "ENABLED/DISABLED" для глобального включения/выключения переработки предмета для всех игроков.

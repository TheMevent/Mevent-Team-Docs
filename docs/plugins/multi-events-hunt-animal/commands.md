# Команды и права

## Команды

### Команды администратора

| Команда | Описание | Право |
|---|---|---|
| `huntanimal.start` | Запустить ивент Hunt Animal | `MultiEventsHuntAnimal.start` |
| `huntanimal.stop` | Остановить текущий ивент | `MultiEventsHuntAnimal.stop` |

:::info Примечание
Ивент также может быть запущен автоматически через систему MultiEvents по расписанию.
:::

### Консольные команды

| Команда | Описание |
|---|---|
| `UI_HuntAnimals hide` | Скрыть панель ивента с экрана (вызывается через UI) |
| `me.hunt.animals.list` | Вывести в серверную консоль список доступных животных (только из консоли сервера) |

## Права (Permissions)

| Право | Описание |
|---|---|
| `MultiEventsHuntAnimal.start` | Право на запуск ивента |
| `MultiEventsHuntAnimal.stop` | Право на остановку ивента |

### Настройка прав

```
oxide.grant group admin MultiEventsHuntAnimal.start
oxide.grant group admin MultiEventsHuntAnimal.stop
```

## Команды игроков

У обычных игроков нет специальных команд. Участие в ивенте автоматическое -- достаточно убивать животных во время активного ивента.

## Механика начисления очков

Очки начисляются за убийство животных. Количество очков зависит от типа животного:

| Животное | Очки |
|---|---|
| Курица (chicken) | 1 |
| Зомби (zombie) | 2 |
| Волк (wolf2) | 4 |
| Кабан (boar) | 4 |
| Лошадь (horse) | 4 |
| Олень (stag) | 4 |
| Медведь (bear) | 10 |
| Полярный медведь (polarbear) | 10 |
| Тигр (tiger) | 10 |
| Крокодил (crocodile) | 10 |
| Пантера (panther) | 10 |
| Змея (snake / snake.entity) | 10 |
| Олень (deer) | 10 |

:::tip Совет
Используйте серверную консольную команду `me.hunt.animals.list` для просмотра всех доступных типов животных и добавления новых в конфигурацию.
:::

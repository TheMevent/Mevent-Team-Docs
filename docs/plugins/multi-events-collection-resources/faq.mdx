# Часто задаваемые вопросы

## Общие вопросы

### Какие виды добычи засчитываются?

Засчитывается вся добыча ресурсов: рубка деревьев, разбивание камней и руды, сбор растений, а также подбор предметов с земли. Система отслеживает все стандартные способы сбора ресурсов в игре.

### Как работает система множителей?

Итоговое количество очков рассчитывается путём последовательного применения трёх множителей:
1. **Множитель инструмента** — зависит от того, каким инструментом игрок добывает ресурс
2. **Множитель предмета** — зависит от типа добываемого ресурса
3. **Множитель права** — зависит от привилегий игрока

Например, если базовое количество очков = 10, множитель инструмента = 0.5, множитель предмета = 1, множитель права = 2, то итоговые очки = 10 * 0.5 * 1 * 2 = 10.

### Нужно ли игрокам регистрироваться?

Нет. Все игроки на сервере автоматически участвуют в ивенте.

## Настройка

### Как сбалансировать ивент для серверов с высокими рейтами?

Уменьшите множители для мощных инструментов (например, `jackhammer: 0.3`) и увеличьте для слабых (`rock: 2.0`). Также можно уменьшить множители для часто добываемых ресурсов.

### Как настроить VIP-множители?

Добавьте нужные права и множители в раздел `Permissions Multiplier` конфигурации, затем выдайте эти права нужным группам через систему прав сервера.

### Можно ли указать множители для предметов с определённым скином?

Да. В разделе `Items Multiplier` каждый предмет имеет параметр `Skin`. Установите нужный ID скина для точного совпадения. Значение `0` означает стандартный предмет без скина.

## Технические вопросы

### Плагин совместим с Carbon?

Да, плагин полностью совместим как с Oxide, так и с Carbon.

### Что произойдёт, если у игрока несколько прав множителей?

Будет использован наибольший множитель из всех прав, которые есть у игрока. Множители прав не складываются.

### Как работает право на редактирование?

Игроки с соответствующим правом на редактирование могут изменять настройки ивента через внутриигровой графический интерфейс, не заходя в конфигурационный файл на сервере.

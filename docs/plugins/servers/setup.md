# Установка Servers

## Шаг 1: Установите плагин

1. Скопируйте файл `Servers.cs` в папку `oxide/plugins/`.
2. Плагин загрузится автоматически.

Файл конфигурации:
```
oxide/config/Servers.json
```

## Шаг 2: Установите зависимости (по желанию)

| Плагин | Назначение |
|---|---|
| **ImageLibrary** | Загрузка баннеров серверов |
| **ServerPanel** | Интеграция с панелью сервера |

## Шаг 3: Настройте список серверов

Откройте `oxide/config/Servers.json` и настройте массив `"Servers"`:

```json
"Servers": [
 {
 "IP": "64.40.9.56",
 "Port": 28014,
 "BmID": "",
 "ScrollDescriptionHeight": 186.0
 }
]
```

| Параметр | Описание |
|---|---|
| IP | IP-адрес или домен сервера |
| Port | Порт подключения |
| BmID | ID на BattleMetrics (для получения информации) |
| ScrollDescriptionHeight | Высота прокрутки описания |

### Использование BattleMetrics

Если указан `BmID`, плагин автоматически получит:
- Название сервера
- Описание
- Текущий онлайн
- Максимальный онлайн
- Размер карты
- Баннер

## Шаг 4: Настройте права доступа

```
oxide.grant group default servers.use
```

## Шаг 5: Проверьте работу

1. Зайдите на сервер с правом `servers.use`.
2. Введите `/servers` в чат.
3. Откроется список серверов с информацией и кнопками подключения.

## Шаг 6: Перезагрузите плагин

```
oxide.reload Servers
```

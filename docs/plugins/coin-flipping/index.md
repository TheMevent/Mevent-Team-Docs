# Coin Flipping

## Что делает этот плагин?

**Coin Flipping** добавляет на сервер мини-игру с подбрасыванием монетки. Игроки могут делать ставки и выбирать сторону монеты (орёл или решка). Плагин поддерживает два режима игры: против бота и против другого игрока. Победитель получает выигрыш, а проигравший теряет ставку.

## Ключевые возможности

- Игра против бота с настраиваемыми суммами ставок
- Игра против других игроков в созданных комнатах
- Поддержка экономики через плагины (Economics, ServerRewards и др.) или предметы
- Визуальный интерфейс с анимацией подбрасывания монеты
- Настраиваемые изображения монеты (орёл и решка)
- Звуковые и визуальные эффекты при подбрасывании, выигрыше и проигрыше
- Настраиваемые минимальные и максимальные ставки для обоих режимов
- Интеграция с плагином Notify для отправки уведомлений

## Требования

| Зависимость | Обязательность | Описание |
|---|---|---|
| **ImageLibrary** | Обязательно | Необходима для загрузки и отображения изображений в интерфейсе |
| **Экономика** (Economics, ServerRewards и др.) | По выбору | Необходима при использовании валюты плагина. Можно использовать предметы вместо плагина |
| Notify | Опционально | Для отправки графических уведомлений |
| Oxide / uMod / Carbon | Обязательно | Совместим с Oxide, uMod и Carbon |

## Быстрый старт

1. Установите плагин **ImageLibrary**, если он ещё не установлен.
2. Скопируйте файл `CoinFlipping.cs` в папку `oxide/plugins/` вашего сервера.
3. Перезагрузите сервер или выполните команду `oxide.reload CoinFlipping`.
4. Выдайте права на использование плагина:
 ```
 oxide.grant group default coinflipping.use
 oxide.grant group default coinflipping.npcs
 oxide.grant group default coinflipping.players
 oxide.grant group default coinflipping.createroom
 ```
5. Откройте игру, введите в чат `/flip` и проверьте работу интерфейса.

## Права доступа

| Право | Кому нужно | Что открывает |
|---|---|---|
| `coinflipping.use` | Всем игрокам | Открытие интерфейса плагина |
| `coinflipping.npcs` | Всем игрокам | Разрешение играть против бота |
| `coinflipping.players` | Всем игрокам | Разрешение играть против других игроков |
| `coinflipping.createroom` | Всем игрокам | Разрешение создавать комнаты для игры с другими игроками |

# Команды и права доступа

## Права доступа (permissions)

| Право | Описание |
| ---------------- | --------------------------------------------------------- |
| `stacks.admin` | Доступ к панели управления и всем командам плагина |

### Выдача прав

Для конкретного игрока:
```
oxide.grant user <steamid> stacks.admin
```

Для группы:
```
oxide.grant group admin stacks.admin
```

## Чат-команды

Все чат-команды требуют право `stacks.admin`.

### Открытие панели управления

```
/stacks
```

Открывает графическую панель управления с двумя вкладками: "Предметы" и "Контейнеры". Панель поддерживает поиск, фильтрацию по категориям и постраничную навигацию.

### Установка стака для предмета в руке

```
/stacks sethandstack [размер]
```

Устанавливает новый размер стака для предмета, который вы держите в руке.

**Пример:**
```
/stacks sethandstack 5000
```

Если вы держите в руке дерево (wood), его максимальный стак станет 5000.

### Установка стака для предмета по названию

```
/stacks setstack [shortname] [размер]
```

Устанавливает новый размер стака для предмета с указанным коротким именем.

**Примеры:**
```
/stacks setstack wood 50000
/stacks setstack metal.ore 20000
/stacks setstack sulfur 30000
```

## Консольные команды

Консольные команды выполняются из серверной консоли (RCON).

### Управление стаками

```
stacks.manage
```

Выводит справку по доступным командам управления.

### Установка размера стака

```
stacks.manage stack set [shortname] [размер]
```

**Пример:**
```
stacks.manage stack set wood 50000
```

### Получение текущего размера стака

```
stacks.manage stack get [shortname]
```

**Пример:**
```
stacks.manage stack get wood
```

Ответ: `Stack size: 50000 (0 -- default)`

### Получение всех размеров стаков

```
stacks.manage stack get all
```

Выводит полный список всех предметов и их текущих размеров стаков.

## Графический интерфейс

### Вкладка "Предметы"

- **Категории**: фильтрация предметов по игровым категориям
- **Поиск**: поиск предмета по названию или shortname
- **Список предметов**: отображает название, стандартный стак и текущий стак
- **Настройки**: кнопка для изменения размера стака конкретного предмета

### Вкладка "Контейнеры"

- **Поиск**: поиск контейнера по имени
- **Список контейнеров**: отображает изображение контейнера, стандартный множитель и текущий множитель
- **Настройки**: кнопка для изменения множителя конкретного контейнера

### Изменение через киянку

Если в конфигурации включена опция "Changing multiplies in containers using a hammer", администратор может ударить по контейнеру киянкой, чтобы открыть окно настройки множителя для этого контейнера.

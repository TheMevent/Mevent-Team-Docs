# Конфигурация Hit Markers

Файл конфигурации расположен по пути: `oxide/config/HitMarkers.json`

## Основные параметры

| Параметр | По умолчанию | Описание |
|---|---|---|
| Commands | `["marker", "hits"]` | Чат-команды для открытия настроек |
| Permission | `""` | Право доступа (пустое -- для всех) |
| Work with Notify? | `true` | Использовать Notify для уведомлений |
| Show damage to NPC | `true` | Отображать урон по NPC |
| Show damage to animals | `false` | Отображать урон по животным |
| Marker removal time | `0.25` | Время отображения маркера попадания (секунды) |

## Настройки шрифтов (Fonts)

Доступные шрифты для числового урона:

| ID | Шрифт | По умолчанию |
|---|---|---|
| 0 | `robotocondensed-bold.ttf` | Жирный |
| 1 | `robotocondensed-regular.ttf` | Обычный |
| 2 | `permanentmarker.ttf` | Маркер |
| 3 | `droidsansmono.ttf` | Моноширинный |

Каждый шрифт может иметь собственное право доступа.

## Ограничения размера шрифта

| Параметр | По умолчанию | Описание |
|---|---|---|
| Min Font Size | `8` | Минимальный размер шрифта |
| Max Font Size | `18` | Максимальный размер шрифта |

## Режимы отображения (Buttons)

| Режим | Описание | По умолчанию |
|---|---|---|
| Text | Числовое отображение урона | Включён |
| Icon | Иконка попадания с цветом хедшота | Включён |
| Health Line | Полоска здоровья противника | Включён |
| Buildings | Урон по строениям | Включён |

Каждый режим может иметь своё право доступа.

## Значения по умолчанию (Default Values)

| Параметр | По умолчанию | Описание |
|---|---|---|
| Font ID | `0` | Шрифт по умолчанию |
| Font Size | `14` | Размер шрифта по умолчанию |
| Text | `true` | Числовой урон включён |
| Icon | `false` | Иконка попадания отключена |
| Health Line | `false` | Полоска здоровья отключена |
| Buildings | `false` | Урон по строениям отключён |

## Настройки полоски (Line Settings)

| Параметр | По умолчанию | Описание |
|---|---|---|
| Show Line? | `true` | Показывать полоску здоровья |
| Show Text? | `false` | Показывать текстовое значение здоровья |

# Установка и настройка

## Системные требования

- Сервер Rust с Oxide/uMod
- Нет обязательных зависимостей

## Пошаговая установка

### Шаг 1: Установите плагин

Поместите файл плагина в папку `oxide/plugins` на вашем сервере.

### Шаг 2: Перезапуск или перезагрузка

Перезапустите сервер или выполните команду в консоли:

```
oxide.reload AutoRecycler
```

### Шаг 3: Настройте права доступа

Выдайте игрокам необходимые права в зависимости от того, какие функции вы хотите предоставить:

```
oxide.grant group default autorecycler.autorec
oxide.grant group default autorecycler.recvirtual
oxide.grant group vip autorecycler.recboxbuy
oxide.grant group admin autorecycler.admin
```

### Шаг 4: Настройте конфигурацию

Откройте файл `oxide/config/AutoRecycler.json` и настройте команды, стоимость ящика-переработчика и параметры переработки.

### Шаг 5: Проверьте работу

1. Зайдите на сервер и введите `/vrec` для проверки виртуального переработчика.
2. Введите `/autorec` для проверки интерфейса автопереработки.
3. Введите `/recbox` для проверки информации о ящике-переработчике.

## Настройка ящика-переработчика

Ящик-переработчик -- это специальный предмет (большой деревянный ящик с уникальным скином), который автоматически перерабатывает помещённые в него предметы.

### Выдача ящика игрокам

1. Через консоль: `recyclercrate.give <имя_игрока>`
2. Через покупку: игрок с правом `autorecycler.recboxbuy` вводит `/recboxbuy`
3. Для тестирования (только админы): `/recboxtest`

### Использование ящика

1. Разместите ящик-переработчик на земле как обычный ящик.
2. Поместите предметы внутрь -- они будут автоматически переработаны.
3. Результаты переработки появятся в ящике.

## Проверка работоспособности

| Функция | Команда для проверки | Ожидаемый результат |
| ------- | -------------------- | ------------------- |
| Виртуальный переработчик | `/vrec` | Открывается интерфейс переработчика |
| Автопереработка | `/autorec` | Открывается панель выбора предметов |
| Переключение автопереработки | `/autorectoggle` | Сообщение о включении/выключении |
| Информация о ящике | `/recbox` | Отображается стоимость покупки |

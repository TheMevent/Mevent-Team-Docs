# Установка Cases

## Шаг 1: Установите зависимости

Обязательный плагин:

- **ImageLibrary** — необходим для отображения изображений кейсов и предметов в интерфейсе.

Опциональные плагины:

| Плагин | Для чего нужен |
|---|---|
| PlaytimeTracker | Начисление кейсов за проведённое на сервере время |
| Notify / UINotify | Красивые всплывающие уведомления |
| ServerPanel | Интеграция с серверным меню |
| Economics / ServerRewards | Экономика для покупки кейсов |

## Шаг 2: Загрузите плагин

1. Скопируйте файл плагина в папку:
 - **Oxide**: `server/НАЗВАНИЕ_СЕРВЕРА/oxide/plugins/`
 - **Carbon**: `server/НАЗВАНИЕ_СЕРВЕРА/carbon/plugins/`

2. Плагин загрузится автоматически. Если сервер работает:
 ```
 o.reload Cases
 ```

3. Конфигурационный файл создастся автоматически:
 - `oxide/config/Cases.json`

## Шаг 3: Настройте кейсы

Откройте `Cases.json` и настройте:

### Основные команды
```json
"Commands": ["cases", "opencase"]
```

### Экономика
Выберите тип валюты и плагин для оплаты:
```json
"Economy": {
 "Type (Plugin/Item)": "Plugin",
 "Plugin name": "Economics",
 "Balance add hook": "Deposit",
 "Balance remove hook": "Withdraw",
 "Balance show hook": "Balance"
}
```

### Кейсы
Каждый кейс содержит:
- Уникальный ID
- Название и изображение
- Стоимость
- Право доступа (опционально)
- Список предметов с шансами выпадения

## Шаг 4: Настройте бонусный кейс

Бонусный кейс позволяет игрокам бесплатно открывать кейс раз в определённый период:

- `Enabled` — включить/выключить
- `CooldownTime` — время между открытиями (в секундах, по умолчанию 86400 = 24 часа)
- `Permission` — право доступа (пустая строка = доступен всем)

## Шаг 5: Настройте кейсы за время игры

Если установлен PlaytimeTracker, настройте автоматическое начисление кейсов:

```json
"Playtime Tracker Settings": {
 "Enabled": true,
 "Cases": {
 "3600": 1,
 "14400": 2,
 "28800": 3,
 "86400": 4
 }
}
```

В этом примере: после 1 часа игры — кейс ID 1, после 4 часов — кейс ID 2 и т.д.

## Шаг 6: Выдайте права

```
oxide.grant group default Cases.edit
oxide.grant user <steamid> cases.vip
```

## Шаг 7: Проверка

1. Зайдите в игру и введите `/cases`
2. Убедитесь, что кейсы отображаются
3. Попробуйте открыть кейс
4. Проверьте работу бонусного кейса

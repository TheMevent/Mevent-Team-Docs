# Установка Battlepass

## Требования перед установкой

Перед установкой убедитесь, что на вашем сервере установлены:

- **Oxide/uMod** или **Carbon** — фреймворк для плагинов
- **ImageLibrary** — обязательная зависимость для отображения изображений

## Шаг 1: Загрузка файлов

Скачайте файл `Battlepass.cs` и поместите его в папку плагинов:

```
oxide/plugins/Battlepass.cs
```

## Шаг 2: Первый запуск

Перезагрузите сервер или выполните команду:

```
o.reload Battlepass
```

При первом запуске плагин автоматически создаст:
- Файл конфигурации: `oxide/config/Battlepass.json`
- Папку данных: `oxide/data/Battlepass/`
- Файл с миссиями: `oxide/data/Battlepass/Missions.json`

## Шаг 3: Настройка прав доступа

Выдайте базовое право всем игрокам:

```
o.grant group default battlepass.use
```

Для VIP-игроков (доступ к привилегированным кейсам и повышенный множитель):

```
o.grant group vip battlepass.vip
```

Для премиум-игроков (максимальный множитель заработка):

```
o.grant group premium battlepass.premium
```

## Шаг 4: Настройка экономики

### Встроенная система валюты (по умолчанию)

По умолчанию плагин использует встроенную систему двух валют. Никаких дополнительных плагинов не требуется.

### Внешний экономический плагин

Если вы хотите использовать Economics или ServerRewards, измените в конфигурации раздел «Currency 1»:

1. Установите `"Use embedded system?"` в `false`
2. Укажите имя плагина в `"Plugin name"` (например, `"Economics"`)
3. Настройте хуки для добавления, удаления и проверки баланса

## Шаг 5: Настройка миссий

Откройте `oxide/config/Battlepass.json` и отредактируйте раздел `"Settings shared missions"`. Каждая миссия имеет:

- **Тип задания** — определяет какое действие должен совершить игрок
- **Описание** — текст, который увидит игрок
- **Количество** — сколько раз нужно выполнить действие
- **Награда** — количество валюты за выполнение

:::info Доступные типы миссий
Добыча ресурсов, убийства, крафт, лут предметов, строительство, улучшение построек, рыбалка, лут ящиков, использование карточек, рейдовые базы, переработка, взлом контейнеров, покупки у NPC, а также участие в различных ивентах (Arctic Base, Gas Station, Sputnik, Shipwreck, Harbor, Junkyard, Sat Dish, Water, Air, Power Plant, Armored Train, Convoy, Survival Arena, Kill Boss).
:::

## Шаг 6: Настройка кейсов

В разделе `"Cases"` конфигурации настройте кейсы, которые игроки смогут покупать за заработанную валюту. Для каждого кейса укажите:

- Название и изображение
- Право доступа (оставьте пустым для доступа всем)
- Стоимость в первой и второй валюте
- Список предметов с шансами выпадения

## Шаг 7: Проверка

1. Зайдите на сервер
2. Введите `/pass` в чат
3. Убедитесь, что интерфейс открывается и отображает миссии
4. Выполните одну из миссий и проверьте начисление награды

## Интеграция с другими плагинами

| Плагин | Что даёт |
|---|---|
| **Notify / UINotify** | Красивые всплывающие уведомления о выполнении миссий |
| **NoEscape** | Блокировка действий в инвентаре Battlepass во время рейда или боя |
| **PlayerDatabase** | Альтернативное хранение данных игроков |

Для активации интеграции с Notify, убедитесь что в конфигурации параметр `"Work with Notify?"` установлен в `true`.

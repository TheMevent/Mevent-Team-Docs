# Ultimate Trade

## Что делает этот плагин?

**Ultimate Trade** позволяет вашему серверу добавить систему безопасного обмена предметами между игроками. Игроки отправляют запрос на обмен через чат-команду, после принятия обоих сторон открывается интерфейс торговой витрины, где каждый участник размещает свои предметы. Когда оба подтвердят сделку, предметы автоматически меняются. Это исключает мошенничество при обмене.

## Ключевые возможности

- Игроки могут безопасно обмениваться предметами через торговую витрину
- Игроки могут отправлять и принимать запросы на обмен через чат-команды
- Администраторы могут настраивать количество слотов для обмена
- Администраторы могут задавать кулдауны и ограничения
- Система прав доступа с индивидуальными настройками для каждой группы
- Блокировка обмена в опасных ситуациях (плавание, полёт, ранение, зона строительства)
- Ограничение обмена по дистанции между игроками
- Требование нахождения в одной команде (опционально)
- Поддержка плагина Duel (блокировка обмена во время дуэлей)
- Кулдаун после обмена для предотвращения злоупотреблений

## Требования

- **Oxide** или **Carbon** последней версии
- **Notify** или **UINotify** -- рекомендуется для уведомлений (необязательно)
- **Duel** -- для блокировки обмена во время дуэлей (необязательно)
- **SkinBox** -- для совместимости с кастомизацией предметов (необязательно)

## Быстрый старт

1. Поместите файл `UltimateTrade.cs` в папку `oxide/plugins`.
2. Перезагрузите сервер или выполните `o.reload UltimateTrade`.
3. Отредактируйте конфигурацию `oxide/config/UltimateTrade.json`.
4. Выдайте право `ultimatetrade.use` игрокам: `oxide.grant group default ultimatetrade.use`.
5. Игроки могут использовать `/trade ник` для начала обмена.

## Права доступа

| Право | Кому нужно | Что открывает |
|---|---|---|
| `ultimatetrade.use` | Игрокам | Возможность использовать систему обмена |
| Пользовательские права | VIP/Premium | Увеличенное количество слотов и сниженный кулдаун |

:::tip Примечание
Пользовательские права настраиваются в разделе `Permission Settings` конфигурации. Каждому праву можно задать индивидуальное количество слотов и кулдаун.
:::

## Команды игроков

| Команда | Описание | Пример |
|---|---|---|
| `/trade [ник]` | Отправить запрос на обмен | `/trade PlayerName` |
| `/trade accept` или `/trade yes` | Принять входящий запрос на обмен | `/trade accept` |
| `/trade cancel` или `/trade no` | Отклонить входящий запрос на обмен | `/trade no` |
| `/t [ник]` | Короткая команда для обмена | `/t PlayerName` |

## Команды администратора

Плагин не имеет специальных административных команд. Управление осуществляется через конфигурационный файл и систему прав доступа.

## Устранение неполадок

### Обмен недоступен -- "You don't have permission"
**Симптом:** Игрок не может отправить запрос на обмен.
**Причина:** У игрока нет права `ultimatetrade.use`.
**Решение:** Выдайте право: `oxide.grant group default ultimatetrade.use`.

### Обмен отменён -- "timeout expired"
**Симптом:** Запрос на обмен автоматически отклоняется.
**Причина:** Получатель не ответил в установленное время (по умолчанию 15 секунд).
**Решение:** Попросите получателя быстрее ввести `/trade accept`. Или увеличьте время ответа в конфиге (`Time to answer trade requests`).

### Обмен недоступен в зоне строительства
**Симптом:** Сообщение о блокировке в зоне Building Blocked.
**Причина:** Включена блокировка обмена в зоне чужого шкафа.
**Решение:** Переместитесь в безопасную зону. Для отключения этого ограничения измените `Block sending requests in BuildingBlock` на `false`.

### Не хватает слотов для обмена
**Симптом:** Игрок не может положить больше предметов в окно обмена.
**Причина:** Количество слотов ограничено настройками (по умолчанию 8).
**Решение:** Увеличьте значение `Number of active slots for trade` в конфиге или выдайте игроку VIP-право с увеличенным количеством слотов.

### Кулдаун после обмена слишком долгий
**Симптом:** Игрок получает сообщение "подождите" после недавнего обмена.
**Причина:** Действует кулдаун (по умолчанию 60 секунд).
**Решение:** Уменьшите значение `Trade cooldown` в конфиге. Для VIP-игроков можно задать пониженный кулдаун в разделе `Permission Settings`.

### Обмен заблокирован -- "Blocked by another plugin"
**Симптом:** Обмен недоступен, причина -- "заблокировано другим плагином".
**Причина:** Другой плагин заблокировал обмен через хук `CanTrade`.
**Решение:** Проверьте, какие плагины на вашем сервере используют хук `CanTrade`, и настройте их соответственно.

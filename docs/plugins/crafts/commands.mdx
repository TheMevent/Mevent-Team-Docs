# Команды и права

## Команды чата

| Команда | Описание | Право |
|---|---|---|
| `/craft` или `/crafts` | Открыть меню крафта | Настраиваемое (по умолчанию для всех) |

:::info Примечание
Команды настраиваются в конфигурации в параметре `"Commands"`. По умолчанию: `["craft", "crafts"]`.
:::

## Консольные команды

| Команда | Описание | Право |
|---|---|---|
| `crafts.setwb` | Установить кастомный верстак в точку, на которую смотрит игрок | `crafts.setworkbench` |

## Команды выдачи

Каждый рецепт крафта может иметь индивидуальную команду выдачи, которая задаётся в конфигурации (поле `CmdToGive`). Например:

| Команда | Описание |
|---|---|
| `givecopter` | Выдать миникоптер |
| `giveattackhelicopter` | Выдать боевой вертолёт |
| `giverowboat` | Выдать лодку |

## Права (Permissions)

### Системные права

| Право | Описание |
|---|---|
| `crafts.edit` | Позволяет редактировать крафты через интерфейс |
| `crafts.setworkbench` | Позволяет устанавливать кастомные верстаки |
| `crafts.bypass` | Обходит все проверки прав и верстака |

### Настраиваемые права

| Тип | Где задаётся | Описание |
|---|---|---|
| Право на использование меню | Конфигурация: `"Permission"` | Общее право для открытия меню крафта |
| Право категории | Каждая категория: `"Permission"` | Доступ к конкретной категории |
| Право крафта | Каждый рецепт: `"Permission"` | Доступ к конкретному рецепту |
| Права кастомного верстака | Конфигурация верстака | Доступ к уровням кастомного верстака |

### Пример настройки прав

```
oxide.grant group admin crafts.edit
oxide.grant group default crafts.all
oxide.grant group vip crafts.vip
```

:::tip Совет
Если поле `Permission` оставить пустым, элемент будет доступен всем игрокам без ограничений.
:::

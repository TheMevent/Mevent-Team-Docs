# Bank System

## Что делает этот плагин?

**Bank System** позволяет вашему серверу добавить полноценную банковскую систему с визуальным интерфейсом. Игроки получают банковские карты, пополняют и снимают средства через банкоматы, переводят деньги другим игрокам, а администраторы управляют экономикой сервера. Плагин также отслеживает добычу ресурсов и начисляет за это валюту.

## Ключевые возможности

- Игроки могут создать банковскую карту и использовать банкомат для пополнения, снятия и перевода средств
- Игроки могут получать валюту за добычу ресурсов, открытие ящиков и убийство NPC
- Администраторы могут размещать банкоматы на монументах (Outpost, Bandit Camp) автоматически
- Администраторы могут управлять балансами игроков через консольные команды
- Владельцы банкоматов могут устанавливать свои комиссии за операции
- Банкоматы изнашиваются при использовании и требуют ремонта
- Валюта может выпадать из ящиков и бочек с настраиваемым шансом
- Поддержка NPC-банкиров для открытия интерфейса банка
- Интеграция с Economics, ServerRewards и другими экономическими плагинами
- История транзакций и добычи ресурсов в интерфейсе

## Требования

- **Oxide** или **Carbon** последней версии
- **ImageLibrary** -- обязательно для отображения интерфейса
- **Notify** или **UINotify** -- рекомендуется для уведомлений (необязательно)
- **LangAPI** -- для мультиязычной поддержки (необязательно)
- **Economics** -- если вы хотите использовать внешнюю экономику вместо встроенной (необязательно)

## Быстрый старт

1. Установите **ImageLibrary** на сервер, если ещё не установлен.
2. Поместите файл `BankSystem.cs` в папку `oxide/plugins` (или `carbon/plugins`).
3. Перезагрузите сервер или выполните команду `o.reload BankSystem`.
4. Отредактируйте конфигурационный файл `oxide/config/BankSystem.json` под нужды вашего сервера.
5. Игроки могут использовать команду `/bank` для открытия интерфейса банка.

## Права доступа

| Право | Кому нужно | Что открывает |
|---|---|---|
| `banksystem.use` (настраивается) | Игрокам | Доступ к интерфейсу банка (если задано в конфиге) |
| `banksystem.openatm` | Игрокам | Открытие меню банкомата через команду |
| `banksystem.vip` | VIP-игрокам | Сниженный процент износа банкомата (0.7% вместо 1%) |
| `banksystem.premium` | Premium-игрокам | Ещё более сниженный износ банкомата (0.5%) |

## Команды игроков

| Команда | Описание | Пример |
|---|---|---|
| `/bank` | Открыть интерфейс банка | `/bank` |

:::tip Примечание
Команда для открытия банка настраивается в конфиге. По умолчанию это `/bank`.
:::

## Команды администратора

| Команда | Описание | Пример |
|---|---|---|
| `bank.givenote [steamId] [количество]` | Выдать банкноты игроку | `bank.givenote 76561198000000000 100` |
| `bank.giveatm [steamId]` | Выдать банкомат игроку | `bank.giveatm 76561198000000000` |
| `bank.setbalance [steamId] [сумма]` | Установить баланс игрока | `bank.setbalance 76561198000000000 5000` |
| `bank.deposit [steamId] [сумма]` | Пополнить баланс игрока | `bank.deposit 76561198000000000 1000` |
| `bank.withdraw [steamId] [сумма]` | Снять средства с баланса игрока | `bank.withdraw 76561198000000000 500` |
| `bank.transfer [отправитель] [получатель] [сумма]` | Перевести средства между игроками | `bank.transfer 7656...1 7656...2 300` |
| `bank.wipe` | Очистить все данные банкоматов | `bank.wipe` |
| `bank.convert.economics` | Конвертировать данные из Economics | `bank.convert.economics` |
| `bank.convert.serverrewards` | Конвертировать данные из ServerRewards | `bank.convert.serverrewards` |

:::caution Внимание
Все административные команды доступны только администраторам сервера или из серверной консоли.
:::

## Устранение неполадок

### Интерфейс банка не открывается
**Симптом:** При вводе `/bank` ничего не происходит.
**Причина:** Не установлен плагин ImageLibrary.
**Решение:** Установите ImageLibrary и перезагрузите сервер.

### Игрок не может использовать банкомат
**Симптом:** При взаимодействии с банкоматом появляется сообщение об ошибке.
**Причина:** У игрока нет банковской карты.
**Решение:** Игрок должен открыть интерфейс банка (`/bank`) и создать карту.

### Банкомат сломан
**Симптом:** Банкомат показывает сообщение о поломке.
**Причина:** Состояние банкомата упало до критического уровня из-за использования.
**Решение:** Владелец банкомата должен отремонтировать его, используя необходимые ресурсы (scrap, metal pipe, metal fragments).

### Баланс не отображается
**Симптом:** Баланс в интерфейсе показывает 0.
**Причина:** Используется внешний плагин экономики (Economics), но он не установлен.
**Решение:** Установите Economics или переключите настройку `Use own economic system?` на `true` в конфиге.

### Карта просрочена
**Симптом:** Игрок не может совершать операции, хотя карта есть.
**Причина:** Срок действия карты истёк (по умолчанию 7 дней).
**Решение:** Игроку нужно создать новую карту. Можно отключить срок действия карты в конфиге (`Use card expiration date?`: `false`).

### Валюта не выпадает из ящиков
**Симптом:** В ящиках и бочках нет банкнот.
**Причина:** Дроп отключён или шанс выпадения слишком низкий.
**Решение:** Проверьте секцию `Drop Settings` в конфигурации и убедитесь, что `Enabled` установлен в `true` и шанс выпадения адекватный.

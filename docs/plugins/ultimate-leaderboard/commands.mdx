# Команды и права

## Команды чата

| Команда | Описание | Право |
|---|---|---|
| `/leaderboard` | Открыть таблицу лидеров | Настраивается |
| `/stats` | Открыть таблицу лидеров (альтернативная) | Настраивается |
| `/leaderboard.hide` | Скрыть/показать себя в таблице лидеров | `ultimateleaderboard.hide.self` |

:::tip Совет
Команды настраиваются в конфигурации в параметре `Commands`. По умолчанию: `["leaderboard", "stats"]`.
:::

## Консольные команды

| Команда | Описание | Кто может использовать |
|---|---|---|
| `leaderboard.wipe` | Сбросить все данные таблицы лидеров | Только серверная консоль |
| `leaderboard.migrate` | Миграция данных между хранилищами | Только серверная консоль |
| `leaderboard.find.short.prefab` | Найти ShortPrefabName для сущности | Только серверная консоль |
| `leaderboard.manage` | Управление данными игроков | Только серверная консоль |

### Примеры использования

Сброс всех данных:
```
leaderboard.wipe
```

:::warning Внимание
Команда `leaderboard.wipe` безвозвратно удаляет все данные статистики. Рекомендуется сделать резервную копию перед выполнением.
:::

## Права (Permissions)

| Право | Описание | По умолчанию |
|---|---|---|
| Настраивается | Доступ к плагину | Пустое (доступно всем) |
| `ultimateleaderboard.profile` | Просмотр профилей других игроков | Не выдано |
| `ultimateleaderboard.hide.self` | Скрытие из таблицы лидеров | Не выдано |
| `ultimateleaderboard.ignore.titles` | Игнорирование титулов | Не выдано |

### Настройка прав

```
oxide.grant group default ultimateleaderboard.profile
oxide.grant group admin ultimateleaderboard.hide.self
```

## Отслеживаемая статистика

### Основные показатели

- Убийства (игроки, NPC, животные)
- Смерти
- K/D-соотношение
- Время игры (текущее и общее)
- Самая длинная дистанция убийства
- Общее количество добытых ресурсов
- Любимый ресурс
- Количество скрафченных предметов
- Построенные структуры
- Выполненные улучшения

### Попадания (Hit Rate)

- Статистика попаданий по частям тела (голова, грудь, руки, ноги)

### Строительство (Building)

- Построенные конструкции с разбивкой по типам и уровням

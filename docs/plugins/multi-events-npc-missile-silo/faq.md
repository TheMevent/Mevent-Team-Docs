# Часто задаваемые вопросы

## Общие вопросы

### Нужен ли Multi Events для работы этого режима?

Да, **Multi Events** является обязательной зависимостью. NPC Missile Silo — это режим расширения из Expansion Pack 1.

### Нужна ли на карте ракетная шахта?

Да, для полноценной работы режима на карте должен присутствовать монумент «Ракетная шахта» (Missile Silo) с NPC-учёными.

### Сколько длится ивент?

По умолчанию 10 минут (600 секунд). Значение настраивается через параметр `Event Duration`.

## Настройка

### Как настроить награды?

В разделе `Awards Settings` конфигурации задайте награды для каждого призового места. Поддерживаются предметы, команды, наборы Kit и вызовы сторонних плагинов.

### Можно ли добавить больше призовых мест?

Да, добавьте новые записи с номерами мест (3, 4, 5 и т.д.) в раздел `Awards Settings`.

## Проблемы

### Очки не начисляются за убийство учёных

Убедитесь, что ивент активен и на карте есть ракетная шахта с NPC. Очки начисляются только за учёных, находящихся в зоне ракетной шахты.

### UI не отображается

Проверьте корректную работу ImageLibrary и его наличие на сервере.

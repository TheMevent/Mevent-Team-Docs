# Установка Skills

## Требования перед установкой

- **Oxide/uMod** или **Carbon** — фреймворк для плагинов
- **ImageLibrary** — обязательная зависимость для отображения изображений

## Шаг 1: Загрузка файлов

Поместите файл `Skills.cs` в папку плагинов:

```
oxide/plugins/Skills.cs
```

## Шаг 2: Первый запуск

Перезагрузите сервер или выполните команду:

```
o.reload Skills
```

Плагин автоматически создаст файл конфигурации: `oxide/config/Skills.json`

## Шаг 3: Настройка прав доступа

По умолчанию плагин доступен всем игрокам (поле Permission пустое). Если вы хотите ограничить доступ:

1. В конфигурации укажите право, например: `"skills.use"`
2. Выдайте его нужным группам:

```
o.grant group default skills.use
```

Для VIP-множителей XP:

```
o.grant group vip skills.xprate.vip
o.grant group premium skills.xprate.premium
```

Для доступа к высоким уровням навыков:

```
o.grant group vip skills.gathering.level6
```

Для доступа к редактору интерфейса (только администраторам):

```
o.grant group admin skills.edit
```

## Шаг 4: Настройка категорий и навыков

Откройте `oxide/config/Skills.json` и настройте:

### Категории
Каждая категория содержит группу навыков. По умолчанию настроена категория «Mining» (Добыча) с навыками для разных руд.

### Навыки
Каждый навык имеет:
- **Тип** — определяет, что навык делает
- **Уровни** — каждый уровень имеет эффект (в процентах) и стоимость в XP
- **Право доступа** (опционально) — для ограничения высоких уровней

## Шаг 5: Настройка XP-системы

В разделе «XP Rewards Settings» настройте, за какие действия игроки получают очки опыта:

- Убийства игроков и NPC
- Смерти (можно сделать штраф)
- Лут ящиков (supply drop, crate_normal, crate_elite, bradley_crate, heli_crate)
- Уничтожение бочек, деревьев, руды
- Добыча ресурсов

## Шаг 6: Проверка

1. Зайдите на сервер
2. Введите `/skills` в чат
3. Убедитесь, что отображаются категории и навыки
4. Выполните действие (добыча руды, убийство) и проверьте получение XP
5. Попробуйте прокачать навык

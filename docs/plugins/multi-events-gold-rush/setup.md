# Установка и настройка

Это руководство проведёт вас через процесс установки режима Gold Rush для системы MultiEvents.

## Системные требования

- Сервер Rust с Oxide / uMod / Carbon
- Установленный плагин **MultiEvents** (обязательная зависимость)
- Установленный плагин **ImageLibrary** (обязательная зависимость)

## Шаг 1: Установите зависимости

### MultiEvents (обязательно)

Плагин Gold Rush является режимом расширения и требует основной плагин MultiEvents.

1. Установите **MultiEvents** согласно его документации.
2. Убедитесь, что плагин загружен без ошибок.

### ImageLibrary (обязательно)

1. Скачайте **ImageLibrary** с [umod.org](https://umod.org/plugins/image-library).
2. Поместите файл `ImageLibrary.cs` в папку `oxide/plugins/`.

:::warning Важно
Без MultiEvents и ImageLibrary плагин Gold Rush **не будет работать**.
:::

## Шаг 2: Установите плагин

1. Скопируйте файл `MultiEventsGoldRush.cs` в папку `oxide/plugins/` вашего сервера.
2. Плагин загрузится автоматически и зарегистрируется в MultiEvents.
3. Проверьте консоль на наличие ошибок.

Файл конфигурации будет создан автоматически:
```
oxide/config/MultiEventsGoldRush.json
```

## Шаг 3: Настройте права доступа

Выдайте права администраторам:

```
oxide.grant group admin MultiEventsGoldRush.start
oxide.grant group admin MultiEventsGoldRush.stop
```

Для VIP-множителя шансов (по желанию):

```
oxide.grant group vip MultiEventsGoldRush.vip
```

## Шаг 4: Проверьте работу

Запустите ивент:

```
goldrush.start
```

На экране игроков появится панель ивента. Начните добывать руду или деревья -- при каждой добыче есть шанс получить дополнительные предметы.

## Шаг 5: Настройте конфигурацию (по желанию)

Откройте файл `oxide/config/MultiEventsGoldRush.json` для настройки:

- **Длительность ивента** -- время проведения события
- **Предметы за руду** -- список предметов и шансы при добыче руды
- **Предметы за деревья** -- список предметов и шансы при добыче деревьев
- **VIP-множители** -- множители шансов для игроков с определёнными правами

Подробнее в разделе [Конфигурация](./configuration).

## Шаг 6: Перезагрузите плагин

После изменения конфигурации:

```
oxide.reload MultiEventsGoldRush
```

:::tip Совет
Настройте шансы выпадения так, чтобы бонусные предметы были достаточно редкими для мотивации, но не слишком редкими, чтобы игроки чувствовали прогресс.
:::

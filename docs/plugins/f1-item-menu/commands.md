# Команды и права

## Команды чата

| Команда | Описание | Право |
|---|---|---|
| `/f1menu` | Открыть визуальное меню предметов | `f1itemmenu.f1` |

## Права (Permissions)

| Право | Описание |
|---|---|
| `f1itemmenu.f1` | Разрешает использование команды `/f1menu` для открытия меню выдачи предметов |

:::warning Внимание
Это право следует выдавать только доверенным администраторам, так как оно позволяет выдавать любые предметы в неограниченном количестве.
:::

## Управление в интерфейсе

| Действие | Описание |
|---|---|
| Клик по предмету | Открывает меню выбора количества |
| Поле поиска | Фильтрует предметы по названию |
| Кнопки `<` / `>` | Навигация по страницам |
| Кнопка `Close` | Закрыть меню |

:::info Примечание
Предметы, добавленные в чёрный список (например, транспорт), отображаются красным цветом и недоступны для выдачи.
:::

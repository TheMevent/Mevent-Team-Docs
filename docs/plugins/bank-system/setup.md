# Установка Bank System

## Предварительные требования

Перед установкой убедитесь, что у вас есть:

- Сервер Rust с **Oxide** или **Carbon**
- Плагин **ImageLibrary** (обязательно)
- Доступ к файловой системе сервера (FTP или панель управления)

:::danger Важно
Без плагина **ImageLibrary** банковская система не будет работать. Интерфейс не откроется, и игроки увидят сообщение об ошибке.
:::

## Шаг 1: Установка зависимостей

1. Скачайте плагин **ImageLibrary** и поместите его в папку `oxide/plugins`.
2. (Рекомендуется) Установите плагин **Notify** или **UINotify** для красивых уведомлений.
3. (Опционально) Установите **LangAPI** для мультиязычной поддержки.

## Шаг 2: Установка плагина

1. Поместите файл `BankSystem.cs` в папку `oxide/plugins` (для Oxide) или `carbon/plugins` (для Carbon).
2. Плагин автоматически скомпилируется и создаст конфигурационный файл.

:::tip Совет
Если вы используете Carbon, плагин будет работать с модулем ImageDatabase вместо ImageLibrary.
:::

## Шаг 3: Базовая настройка конфигурации

После первого запуска найдите файл конфигурации:
- **Oxide:** `oxide/config/BankSystem.json`
- **Carbon:** `carbon/configs/BankSystem.json`

### Настройка валюты

Задайте параметры банкнот, которые будут использоваться как физическая валюта:

```json
"Currency Settings": {
 "Enable item split control?": true,
 "Display Name": "RUSTNote",
 "Short Name": "sticks",
 "Skin": 2536195910
}
```

:::info Информация
`Short Name` -- это базовый предмет Rust, на основе которого создаётся валюта. `Skin` определяет внешний вид предмета. Не меняйте эти значения после запуска сервера, чтобы избежать потери валюты у игроков.
:::

### Настройка стартового баланса

```json
"Starting balance": 100
```

Каждый новый игрок получит указанную сумму при создании банковской карты.

### Настройка экономики

Если вы хотите использовать встроенную экономику:

```json
"Economy Settings": {
 "Use own economic system?": true
}
```

Если вы хотите использовать внешний плагин (например Economics):

```json
"Economy Settings": {
 "Use own economic system?": false,
 "Plugin name": "Economics",
 "Balance add hook": "Deposit",
 "Balance remove hook": "Withdraw",
 "Balance show hook": "Balance"
}
```

## Шаг 4: Настройка банкоматов

### Автоматический спавн на монументах

По умолчанию банкоматы спавнятся в Outpost и Bandit Camp:

```json
"ATM Settings": {
 "Spawn Settings": {
 "Monuments": {
 "compound": {
 "Enabled": true,
 "Display Name": "ATM",
 "Position": { "x": -3.5, "y": 1.15, "z": 2.7 },
 "Rotation": -90.0,
 "Deposit Fee": 0.0,
 "Withdraw Fee": 0.0
 },
 "bandit": {
 "Enabled": true,
 "Display Name": "ATM",
 "Position": { "x": 34.2, "y": 2.35, "z": -24.7 },
 "Rotation": 135.0,
 "Deposit Fee": 0.0,
 "Withdraw Fee": 0.0
 }
 }
 }
}
```

### Банкоматы, размещённые игроками

Игроки могут получить банкомат через команду `bank.giveatm` и разместить его самостоятельно. Такие банкоматы принадлежат игроку, и он может настраивать комиссии.

## Шаг 5: Настройка прав доступа

Если вы хотите ограничить доступ к банку определённым группам:

```
oxide.grant group default banksystem.use
```

Если поле `Permission` в конфиге оставить пустым, банк будет доступен всем.

## Шаг 6: Настройка NPC-банкиров (опционально)

Вы можете привязать интерфейс банка к NPC:

```json
"NPC Settings": {
 "NPCs": [
 "1234567890"
 ]
}
```

Замените значение на Steam ID вашего NPC. При взаимодействии с этим NPC у игрока откроется интерфейс банка.

## Шаг 7: Перезагрузка

После настройки конфигурации выполните:

```
o.reload BankSystem
```

:::tip Готово!
Теперь игроки могут использовать команду `/bank` для открытия банка и взаимодействовать с банкоматами на монументах.
:::

## Настройка Auto-Wipe

Плагин поддерживает автоматическую очистку данных при вайпе сервера:

```json
"Wipe Settings": {
 "Wipe Players?": false,
 "Wipe Logs?": true,
 "Wipe ATMs?": true
}
```

- **Wipe Players** -- очистить балансы и карты игроков
- **Wipe Logs** -- очистить историю транзакций
- **Wipe ATMs** -- очистить данные банкоматов

:::caution Внимание
Если включить `Wipe Players`, все игроки потеряют свои балансы и карты при каждом вайпе сервера.
:::

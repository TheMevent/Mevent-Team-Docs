# Linking System

## Что делает этот плагин?

**Linking System** позволяет игрокам привязать свой Steam-аккаунт к Discord. Игрок генерирует код в игре, вводит его в Discord-боте, и привязка подтверждается. За привязку игрок может получить награды: дополнительные группы, бонусы и предметы.

## Ключевые возможности

- Привязка Steam-аккаунта к Discord через генерацию кода
- Автоматическое добавление игрока в группу после привязки
- Красивый интерфейс с отображением статуса привязки
- Отображение информации Discord-аккаунта (имя, аватар)
- Настраиваемые награды за привязку (отображаются в интерфейсе)
- Возможность отключения привязки
- Кулдаун на генерацию нового кода
- Команды чата для управления привязкой
- Интеграция с Discord-ботом через RCON-команды
- Интеграция с ServerPanel, Notify
- Поддержка нескольких шаблонов интерфейса (V1, V2)

## Требования

| Зависимость | Обязательность | Описание |
|---|---|---|
| **ImageLibrary** | Обязательно | Для изображений в интерфейсе |
| **Discord-бот (LinkBot)** | Обязательно | Для обработки привязки на стороне Discord |
| Notify | Опционально | Для графических уведомлений |
| ServerPanel | Опционально | Для интеграции в серверное меню |
| Oxide / uMod / Carbon | Обязательно | Совместим с Oxide, uMod и Carbon |

## Быстрый старт

1. Установите плагин **ImageLibrary**.
2. Скопируйте файл `LinkingSystem.cs` в папку `oxide/plugins/`.
3. Настройте Discord-бота (LinkBot) для вашего сервера.
4. Перезагрузите сервер или выполните `oxide.reload LinkingSystem`.
5. Выдайте право:
 ```
 oxide.grant group default linkingsystem.use
 ```
6. Зайдите на сервер и введите `/link` для открытия интерфейса.

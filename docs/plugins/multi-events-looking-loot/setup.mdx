# Установка

## Требования

Перед установкой убедитесь, что у вас установлены следующие компоненты:

| Компонент | Обязательность | Описание |
|---|---|---|
| Multi Events | Обязательно | Основной плагин системы ивентов |
| Image Library | Рекомендуется | Загрузка и кэширование изображений для UI |
| Notify / UINotify | Опционально | Всплывающие уведомления для игроков |
| Multi Events Inventory | Опционально | Выдача наград через инвентарь ивентов |
| Multi Events Leaderboard | Опционально | Глобальная статистика по ивентам |

## Пошаговая установка

### Шаг 1. Скачайте файл плагина

Скачайте файл режима и поместите его в папку плагинов вашего сервера:

- **Oxide**: `oxide/plugins/`
- **Carbon**: `carbon/plugins/`

### Шаг 2. Загрузите плагин

Перезагрузите сервер или загрузите плагин через консоль сервера. После первой загрузки будет автоматически создан конфигурационный файл.

### Шаг 3. Настройте конфигурацию

Настройте таблицу ценности предметов и базовый множитель очков. Подробнее см. в разделе [Конфигурация](./configuration).

### Шаг 4. Проверьте работу

Используйте команду `lookingloot.start` в консоли сервера, чтобы вручную запустить ивент и убедиться, что всё работает корректно.

## Обновление

При обновлении плагина просто замените файл в папке плагинов. Конфигурация будет обновлена автоматически с сохранением ваших настроек.

## Порядок загрузки

Плагин автоматически регистрируется в Multi Events при загрузке. Если Multi Events загрузится позже, регистрация произойдёт автоматически при его появлении.

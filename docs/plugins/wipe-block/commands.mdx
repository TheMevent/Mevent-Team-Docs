# Команды Wipe Block

## Команды игроков

### Открытие меню блокировок

| Команда | Описание |
|---|---|
| `/block` | Открывает главное меню с информацией о заблокированных и разблокированных предметах |
| `/wipeblock` | Альтернативная команда для открытия того же меню |

:::info
Эти команды настраиваются в конфигурации. Администратор может изменить их на любые другие.
:::

### Навигация в меню

В интерфейсе доступны три вкладки:

| Вкладка | Описание |
|---|---|
| Заблокированные | Показывает только заблокированные предметы с таймерами обратного отсчёта |
| Разблокированные | Показывает уже доступные предметы |
| Все предметы | Показывает полный список предметов из всех категорий |

## Команды администратора

### Смещение времени блокировки

```
/wb.indent [секунды]
```

Изменяет точку отсчёта блокировки, сдвигая её на указанное количество секунд.

**Параметры:**
- `секунды` -- целое число (положительное для ускорения, отрицательное для замедления)

**Примеры:**

| Пример | Эффект |
|---|---|
| `/wb.indent 3600` | Ускоряет разблокировку на 1 час |
| `/wb.indent 7200` | Ускоряет разблокировку на 2 часа |
| `/wb.indent -1800` | Замедляет разблокировку на 30 минут |
| `/wb.indent 86400` | Ускоряет разблокировку на 24 часа |

:::warning Внимание
Команда `/wb.indent` доступна только администраторам сервера. Значение сохраняется в конфигурации и суммируется при повторных вызовах.
:::

## Экранный виджет

Если виджет включен в конфигурации, на экране каждого игрока отображается индикатор текущей блокировки:

- **Заголовок:** "ВАЙП БЛОК" (или "BLOCK AFTER WIPE" для английского языка)
- **Описание:** "Есть запрещенные предметы"
- **Расположение:** По умолчанию -- правый верхний угол экрана

Виджет автоматически исчезает, когда все предметы разблокированы.

## Уведомления

При разблокировке предмета все игроки с правом `wipeblock.unlocknotify` получают уведомление:

> "'{название предмета}' был разблокирован!"

Если установлен плагин Notify или UINotify, уведомление отображается через него. В противном случае -- через стандартный чат.

# Установка и настройка

Это руководство проведёт вас через процесс установки режима Helicopter Pet для системы MultiEvents.

## Системные требования

- Сервер Rust с Oxide / uMod / Carbon
- Установленный плагин **MultiEvents** (обязательная зависимость)
- Установленный плагин **ImageLibrary** (обязательная зависимость)

## Шаг 1: Установите зависимости

### MultiEvents (обязательно)

Плагин Helicopter Pet является режимом расширения и требует основной плагин MultiEvents.

1. Установите **MultiEvents** согласно его документации.
2. Убедитесь, что плагин загружен без ошибок.

### ImageLibrary (обязательно)

1. Скачайте **ImageLibrary** с [umod.org](https://umod.org/plugins/image-library).
2. Поместите файл `ImageLibrary.cs` в папку `oxide/plugins/`.

:::warning Важно
Без MultiEvents и ImageLibrary плагин Helicopter Pet **не будет работать**.
:::

## Шаг 2: Установите плагин

1. Скопируйте файл `MultiEventsHelicopterPet.cs` в папку `oxide/plugins/` вашего сервера.
2. Плагин загрузится автоматически и зарегистрируется в MultiEvents.
3. Проверьте консоль на наличие ошибок.

Файл конфигурации будет создан автоматически:
```
oxide/config/MultiEventsHelicopterPet.json
```

## Шаг 3: Настройте права доступа

Выдайте права администраторам:

```
oxide.grant group admin MultiEventsHelicopterPet.start
oxide.grant group admin MultiEventsHelicopterPet.stop
```

## Шаг 4: Настройте вертолёт

В конфигурации настройте ключевые параметры вертолёта:

- **Heli Init Position** -- позиция спавна вертолёта (координаты X, Y, Z)
- **Heli Height above ground** -- высота полёта над землёй
- **Helicopter Health** -- здоровье вертолёта

:::tip Совет
Если позиция спавна не указана (`0, 0, 0`), вертолёт появится в центре карты. Рекомендуется установить позицию вблизи основных баз игроков.
:::

## Шаг 5: Проверьте работу

Запустите ивент:

```
helicopterpet.start
```

На карте появится патрульный вертолёт, и на экране игроков появится панель ивента. Привлекайте внимание вертолёта для набора очков.

## Шаг 6: Настройте конфигурацию (по желанию)

Откройте файл `oxide/config/MultiEventsHelicopterPet.json` для настройки:

- **Длительность ивента** -- время проведения события
- **Параметры вертолёта** -- здоровье, позиция, высота
- **Награды** -- настройка наград за места в рейтинге
- **Интеграция с Notify** -- включение/отключение уведомлений

Подробнее в разделе [Конфигурация](./configuration).

## Шаг 7: Перезагрузите плагин

После изменения конфигурации:

```
oxide.reload MultiEventsHelicopterPet
```

:::warning Внимание
Ивент Helicopter Pet опасен для игроков -- вертолёт будет атаковать! Убедитесь, что игроки осведомлены об этом. Рекомендуется установить достаточно высокое здоровье вертолёта, чтобы он не был уничтожен до конца ивента.
:::

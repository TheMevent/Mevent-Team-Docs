# Команды и права

## Команды

### Команды администратора

| Команда | Описание | Право |
|---|---|---|
| `botpurge.start` | Запустить ивент Bot Purge | `MultiEventsBotPurge.start` |
| `botpurge.stop` | Остановить текущий ивент | `MultiEventsBotPurge.stop` |

:::info Примечание
Ивент также может быть запущен и остановлен автоматически через систему MultiEvents по расписанию или через интерфейс управления.
:::

### Консольные команды (UI)

| Команда | Описание |
|---|---|
| `UI_BotPurge hide` | Скрыть панель ивента с экрана (вызывается через UI) |

## Права (Permissions)

| Право | Описание |
|---|---|
| `MultiEventsBotPurge.start` | Право на запуск ивента |
| `MultiEventsBotPurge.stop` | Право на остановку ивента |

### Настройка прав

Для выдачи прав администраторам:

```
oxide.grant group admin MultiEventsBotPurge.start
oxide.grant group admin MultiEventsBotPurge.stop
```

## Команды игроков

У обычных игроков нет специальных команд. Участие в ивенте происходит автоматически -- достаточно убивать NPC-учёных во время активного ивента.

## Использование через MultiEvents

Ивент регистрируется в MultiEvents с командами:
- **Команда запуска:** `botpurge.start`
- **Команда остановки:** `botpurge.stop`

Это позволяет MultiEvents автоматически управлять ивентом через расписание и интерфейс.

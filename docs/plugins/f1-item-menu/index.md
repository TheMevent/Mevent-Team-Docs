# F1 Item Menu

## Что делает этот плагин?

**F1 Item Menu** предоставляет администраторам удобное визуальное меню для выдачи любых предметов из игры. Вместо запоминания shortname предметов, администратор открывает графический интерфейс с иконками всех предметов, выбирает нужный, указывает количество и получает его в инвентарь. Плагин поддерживает поиск предметов, пагинацию и интеграцию с Discord для логирования выдачи.

## Ключевые возможности

- Администраторы видят все предметы игры в удобном графическом меню
- Поиск предметов по названию
- Пагинация для навигации по большому списку предметов
- Выбор количества предметов при выдаче
- Чёрный список предметов (транспорт исключён по умолчанию)
- Отправка уведомлений в Discord при выдаче предметов
- Настраиваемый формат сообщений для Discord

## Требования

| Зависимость | Обязательность | Описание |
|---|---|---|
| Oxide / uMod / Carbon | Обязательно | Совместим с Oxide, uMod и Carbon |

## Быстрый старт

1. Скопируйте файл `F1ItemMenu.cs` в папку `oxide/plugins/`.
2. Перезагрузите сервер или выполните `oxide.reload F1ItemMenu`.
3. Выдайте право администраторам: `oxide.grant group admin f1itemmenu.f1`.
4. Введите `/f1menu` в чат для открытия меню.

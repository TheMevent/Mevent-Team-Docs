# Конфигурация Daily Rewards

Файл конфигурации расположен по пути: `oxide/config/DailyRewards.json`

## Основные параметры

| Параметр | По умолчанию | Описание |
|---|---|---|
| Commands | `["rewards", "daily"]` | Команды чата для открытия интерфейса |
| Work with Notify? | `true` | Использовать плагин Notify для уведомлений |
| Work with LangAPI? | `true` | Использовать LangAPI для мультиязычности |
| Enable Offline Image Mode | `false` | Режим оффлайн-изображений |
| Wipe on new map save | `false` | Сбрасывать прогресс при новом сохранении карты |
| Permission (ex: dailyrewards.use) | `dailyrewards.use` | Право на использование плагина |
| Show UI with rewards when player connects to server? | `false` | Показывать интерфейс наград при подключении игрока |

## Настройки кулдауна

| Параметр | По умолчанию | Описание |
|---|---|---|
| Enabled | `true` | Включить систему кулдауна |
| DefaultCooldown | `3600` | Время кулдауна по умолчанию (в секундах) |
| Cooldowns | см. ниже | Индивидуальные кулдауны для прав |
| CheckAFK | `false` | Не учитывать время AFK (требуется AFKAPI) |

### Пример настройки VIP-кулдаунов

```json
{
 "Cooldown Settings": {
 "Enabled": true,
 "DefaultCooldown": 3600,
 "Cooldowns": {
 "dailyrewards.vip": 3100,
 "dailyrewards.premium": 2600
 },
 "CheckAFK": false
 }
}
```

## Настройки сброса дня

| Параметр | По умолчанию | Описание |
|---|---|---|
| TimeZone | `Europe/London` | Часовой пояс для расчёта сброса |
| Time | `05:00` | Время ежедневного сброса |

## Настройки инвентаря

| Параметр | По умолчанию | Описание |
|---|---|---|
| Enabled | `true` | Включить инвентарь для невостребованных наград |
| RetainItemsOnWipe | `true` | Сохранять предметы в инвентаре после вайпа |
| UseCooldownAfterWipe | `false` | Использовать кулдаун после вайпа |
| CooldownAfterWipe | `0` | Время кулдауна после вайпа (в секундах) |
| UseBuildingBlocked | `false` | Блокировать получение предметов в зоне чужого шкафа |
| UseCombatBlocked | `false` | Блокировать во время комбат-блока |
| UseRaidBlocked | `false` | Блокировать во время рейд-блока |

## Настройки ежедневных наград

Награды настраиваются в секции `"Daily awards"`. Каждый день имеет уникальный номер (1, 2, 3...) и содержит:

| Параметр | Описание |
|---|---|
| Enabled | Включена ли награда для данного дня |
| Image | URL изображения для карточки дня |
| Title | Заголовок карточки дня |
| Awards | Список наград (предметы, команды, плагины) |
| IsSpecialDay | Является ли день специальным (визуально выделяется) |

### Типы наград

**Предмет (Item):**
```json
{
 "Type": "Item",
 "ShortName": "rifle.ak",
 "Amount": 1,
 "Skin": 0,
 "DisplayName": ""
}
```

**Команда (Command):**
```json
{
 "Type": "Command",
 "Command": "deposit %steamid% 100",
 "Amount": 1,
 "Image": "https://example.com/image.png"
}
```

## Настройки уведомлений

| Параметр | Описание |
|---|---|
| On Cooldown Ended | Уведомление при окончании кулдауна |
| Has Cooldown | Уведомление при активном кулдауне |
| Picked up daily reward | Уведомление при получении награды |

Каждое уведомление имеет параметры:
- `Type` -- тип уведомления Notify (числовой ID)
- `Show...` -- когда показывать уведомление

:::tip Рекомендация
Настройте часовой пояс в соответствии с основной аудиторией вашего сервера, чтобы сброс дня происходил в удобное для игроков время.
:::

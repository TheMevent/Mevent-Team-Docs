# Команды Kits

## Команды для игроков

Эти команды доступны всем игрокам на сервере.

| Команда | Описание |
|---|---|
| `/kit` | Открывает интерфейс со списком доступных наборов |
| `/kits` | Аналог команды `/kit` |
| `/kit <название>` | Получить конкретный набор по его названию |

:::info Примечание
Команды `/kit` и `/kits` настраиваются в конфигурации. Администратор сервера может изменить их на любые другие.
:::

### Примеры

```
/kit — открывает меню наборов
/kit starter — получить набор «starter»
/kit vip — получить набор «vip»
```

## Команды для администраторов

### Визуальный редактор

| Команда | Описание |
|---|---|
| `/editkit` | Открывает визуальный редактор наборов в игре (требуется право `Kits.admin` или флаг администратора) |

### Консольные команды

Следующие команды выполняются из серверной консоли или RCON. Администраторы с правом `Kits.admin` также могут выполнять их из игровой консоли.

| Команда | Описание |
|---|---|
| `kits.give <игрок> <набор>` | Выдать предметы набора игроку напрямую (минуя кулдаун и лимиты) |
| `kits.givekit <игрок> <набор> [кол-во]` | Начислить использования набора игроку (он сможет забрать его из меню) |
| `kits.reset data all` | Сбросить данные наборов всех игроков |
| `kits.reset data <steamid>` | Сбросить данные наборов конкретного игрока |
| `kits.reset data <steamid> <набор>` | Сбросить данные конкретного набора у игрока |
| `kits.reset kits` | Удалить все созданные наборы (необратимо!) |
| `kits.template <параметры>` | Переключить шаблон интерфейса (только из консоли сервера) |
| `kits.convert` | Конвертация данных из других плагинов |

### Примеры консольных команд

#### Выдача наборов
```
kits.give PlayerName vip_kit
kits.give 76561198000000000 starter
```

#### Начисление использований
```
kits.givekit PlayerName premium 3
kits.givekit 76561198000000000 daily 5
```

#### Сброс данных
```
kits.reset data all
kits.reset data 76561198000000000
kits.reset data 76561198000000000 starter
kits.reset kits
```

#### Переключение шаблона интерфейса

Доступные шаблоны:

| Режим | Шаблон | Варианты |
|---|---|---|
| Fullscreen | old | Классический стиль |
| Fullscreen | rust | normal, content, categories, content_categories |
| InMenu | rust | normal (V1/V2), content (V1/V2), categories (V1/V2), content_categories (V1/V2) |

```
kits.template fullscreen old
kits.template fullscreen rust normal
kits.template fullscreen rust content
kits.template fullscreen rust categories
kits.template inmenu rust normal v1
kits.template inmenu rust content v2
```

## Различия между kits.give и kits.givekit

| | `kits.give` | `kits.givekit` |
|---|---|---|
| **Что делает** | Выдаёт предметы набора сразу в инвентарь | Добавляет возможность забрать набор из меню |
| **Обходит кулдаун** | Да | Нет (игрок забирает сам) |
| **Игрок должен быть онлайн** | Да | Нет (может забрать позже) |
| **Сохраняется при вайпе** | Настраивается | Настраивается |

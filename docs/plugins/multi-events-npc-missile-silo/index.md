# NPC Missile Silo

## Что делает этот режим?

**NPC Missile Silo** — это режим ивента для Multi Events, в котором игроки отправляются к ракетной шахте и уничтожают элитных учёных, охраняющих комплекс. Каждый убитый учёный приносит очки, а игрок с наибольшим количеством очков по окончании таймера получает эксклюзивные награды.

## Ключевые возможности

- Соревновательная охота на NPC-учёных в зоне ракетной шахты
- Таблица лидеров в режиме реального времени
- Гибкая система наград для нескольких призовых мест
- Поддержка уведомлений через Notify и UINotify
- Встроенный UI-виджет с обратным отсчётом
- Интеграция с Multi Events Inventory и Leaderboard

## Требования

| Зависимость | Обязательно? |
|---|---|
| **Multi Events** | Да |
| **ImageLibrary** | Да |
| Multi Events Inventory | Рекомендуется |
| Multi Events Leaderboard | Рекомендуется |
| Notify / UINotify | Опционально |

## Быстрый старт

1. Установите **Multi Events** на сервер
2. Поместите файл режима NPC Missile Silo в папку `oxide/plugins`
3. Перезагрузите сервер или загрузите плагин
4. Настройте награды в конфигурации при необходимости
5. Ивент готов к запуску через Multi Events или вручную командой

## Как проходит ивент

1. Ивент запускается автоматически через Multi Events или вручную
2. На экране всех игроков появляется UI-виджет с информацией и таймером
3. Игроки отправляются к ракетной шахте и уничтожают NPC-учёных
4. За каждого убитого учёного начисляются очки
5. Таблица лидеров обновляется в реальном времени
6. По истечении таймера победители получают награды

## Настройки

| Параметр | Описание | По умолчанию |
|---|---|---|
| Work with Notify? | Использовать ли плагин Notify для уведомлений | `true` |
| Event Duration | Длительность ивента в секундах | `600` |
| Awards Settings | Настройки наград для каждого места | Предметы для 1-3 мест |

## Устранение неполадок

| Проблема | Решение |
|---|---|
| Ивент не запускается | Убедитесь, что Multi Events установлен и загружен |
| Очки не начисляются | Проверьте, что на карте присутствует ракетная шахта с учёными |
| Нет UI-уведомлений | Проверьте установку ImageLibrary |
| Награды не выдаются | Проверьте корректность настроек наград в конфигурации |

# Установка Daily Rewards

Это руководство проведёт вас через весь процесс установки плагина ежедневных наград.

## Шаг 1: Установите зависимости

### ImageLibrary (обязательно)

1. Скачайте **ImageLibrary** с [umod.org](https://umod.org/plugins/image-library).
2. Поместите файл `ImageLibrary.cs` в папку `oxide/plugins/`.
3. Убедитесь, что плагин загрузился без ошибок.

:::warning Важно
Без ImageLibrary плагин Daily Rewards **не будет корректно работать**.
:::

### Дополнительные зависимости (опционально)

- **Notify** -- для графических уведомлений
- **WipeBlock** -- для блокировки наград после вайпа
- **NoEscape** -- для блокировки наград во время рейда/комбата
- **AFKAPI** -- для проверки AFK-статуса (время AFK не учитывается в кулдауне)

## Шаг 2: Установите плагин

1. Скопируйте файл `DailyRewards.cs` в папку `oxide/plugins/` вашего сервера.
2. Плагин загрузится автоматически.
3. Проверьте консоль на наличие ошибок.

Файл конфигурации будет автоматически создан по пути:
```
oxide/config/DailyRewards.json
```

## Шаг 3: Настройте права доступа

Выдайте право на использование плагина:

```
oxide.grant group default dailyrewards.use
```

Для VIP-групп с уменьшенным кулдауном:

```
oxide.group add vip
oxide.grant group vip dailyrewards.vip
oxide.group add premium
oxide.grant group premium dailyrewards.premium
```

Для права на редактирование наград в интерфейсе:

```
oxide.grant user <steamid> dailyrewards.edit
```

## Шаг 4: Настройте часовой пояс и время сброса

В конфигурации установите нужный часовой пояс и время ежедневного сброса:

```json
{
 "Daily reset settings": {
 "TimeZone": "Europe/Moscow",
 "Time": "05:00"
 }
}
```

## Шаг 5: Настройте награды

В секции `"Daily awards"` конфигурации задайте награды для каждого дня. Каждый день может содержать:
- Предметы (по ShortName)
- Команды (с подстановкой `%steamid%`)
- Плагинные награды

## Шаг 6: Проверьте работу

1. Зайдите на сервер.
2. Введите в чат `/daily` или `/rewards`.
3. Должен открыться интерфейс с ежедневными наградами.

## Шаг 7: Перезагрузите плагин

После изменения конфигурации перезагрузите плагин:

```
oxide.reload DailyRewards
```

## Обновление плагина

При обновлении плагина:

1. Замените файл `DailyRewards.cs` новой версией.
2. Плагин автоматически обновит конфигурацию.
3. В консоли появится сообщение: `Config update detected! Updating config values...`

:::note Примечание
Рекомендуется сохранять резервную копию конфигурации перед обновлением.
:::

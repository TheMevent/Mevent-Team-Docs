# Команды Notify

## Обзор

Плагин Notify предоставляет три команды для отправки уведомлений. Все команды предназначены для администраторов и выполняются через чат или серверную консоль.

:::info Примечание
Обычные игроки не используют команды напрямую. Они автоматически получают уведомления от других плагинов, которые интегрированы с Notify через API.
:::

## Команды администратора

### Отправить уведомление себе

```
notify.show [тип] [сообщение]
```

**Требуемое право:** `notify.notify`

Отправляет уведомление текущему игроку (самому себе). Полезно для тестирования настроек.

**Параметры:**
- `[тип]` -- числовой идентификатор типа уведомления (0, 1 и т.д.)
- `[сообщение]` -- текст уведомления (поддерживает пробелы)

**Примеры:**

```
notify.show 0 Добро пожаловать на сервер!
```
Показывает информационное уведомление (тип 0, синий).

```
notify.show 1 Вы нарушили правила сервера!
```
Показывает уведомление об ошибке (тип 1, красный).

---

### Отправить уведомление конкретному игроку

```
notify.player [steamid/имя] [тип] [сообщение]
```

**Требуемое право:** `notify.player`

Отправляет уведомление указанному игроку. Игрок должен быть онлайн.

**Параметры:**
- `[steamid/имя]` -- SteamID64 или имя целевого игрока
- `[тип]` -- числовой идентификатор типа уведомления
- `[сообщение]` -- текст уведомления

**Примеры:**

```
notify.player 76561198012345678 0 Ваша заявка одобрена!
```
Отправляет информационное уведомление игроку с указанным SteamID.

```
notify.player PlayerName 1 Предупреждение: нарушение правил!
```
Отправляет уведомление об ошибке игроку по имени.

:::warning Внимание
Поиск по имени может найти неправильного игрока, если несколько игроков имеют похожие имена. Рекомендуется использовать SteamID для точной адресации.
:::

---

### Отправить уведомление всем игрокам

```
notify.allplayers [тип] [сообщение]
```

**Требуемое право:** `notify.allplayer`

Отправляет уведомление всем игрокам, которые в данный момент находятся на сервере.

**Параметры:**
- `[тип]` -- числовой идентификатор типа уведомления
- `[сообщение]` -- текст уведомления

**Примеры:**

```
notify.allplayers 0 Вайп через 30 минут! Готовьтесь!
```
Отправляет информационное уведомление всем игрокам.

```
notify.allplayers 1 Сервер будет перезагружен через 5 минут!
```
Отправляет предупреждение всем игрокам.

## Использование из серверной консоли

Все команды можно выполнять из серверной (RCON) консоли. При выполнении из консоли проверка прав доступа не требуется.

```
notify.allplayers 0 Техническое обслуживание через 10 минут
```

## Использование другими плагинами (API)

Другие плагины могут отправлять уведомления, вызывая следующие функции:

### Отправить уведомление одному игроку

```
Notify.Call("SendNotify", player, type, message);
```

- `player` -- целевой игрок
- `type` -- числовой тип уведомления (int)
- `message` -- текст сообщения (string)

### Отправить уведомление всем игрокам

```
Notify.Call("SendNotifyAllPlayers", type, message);
```

### Включить/отключить тип уведомления

```
Notify.Call("TryToggleType", type, newValue);
```

- `type` -- числовой тип (int)
- `newValue` -- `true` (включить) или `false` (отключить)

:::tip Совет
Многие популярные плагины (магазины, рейды, события) уже поддерживают Notify. Достаточно включить опцию "Work with Notify" в их настройках.
:::

# Конфигурация Necronomicon

Файл конфигурации расположен по пути: `oxide/config/Necronomicon.json`

## Основные параметры

| Параметр | По умолчанию | Описание |
|---|---|---|
| Work with Notify? | `true` | Использовать Notify для уведомлений |
| Use auto-wipe? | `true` | Автоматическая очистка данных при вайпе |
| Delete BPs from the player when creating a book? | `false` | Удалять чертежи у игрока при создании книги |
| Cost | `100` | Стоимость изучения одного чертежа |
| Permission | `""` | Право для использования (пустое -- для всех) |

## Настройки книги (Book Item Settings)

| Параметр | По умолчанию | Описание |
|---|---|---|
| Display Name | `"Necromonicon"` | Отображаемое название книги |
| ShortName | `"xmas.present.small"` | ShortName предмета-основы |
| Skin | `2537078809` | Скин предмета |

## Настройки экономики (Economy)

| Параметр | По умолчанию | Описание |
|---|---|---|
| Enabled | `true` | Включить оплату за изучение |
| Type | `Plugin` | Тип экономики: `Plugin` или `Item` |
| Plugin name | `"Economics"` | Название плагина экономики |
| Balance add hook | `"Deposit"` | Функция пополнения |
| Balance remove hook | `"Withdraw"` | Функция списания |
| Balance show hook | `"Balance"` | Функция показа баланса |
| ShortName | `"scrap"` | ShortName предмета (для типа Item) |
| Skin | `0` | Скин предмета (для типа Item) |

## Настройки интерфейса

| Параметр | Описание |
|---|---|
| Active Color | Цвет активных (доступных) чертежей |
| Disactive Color | Цвет неактивных (уже изученных) чертежей |
| Effect | Путь к звуковому эффекту при изучении (пустая строка -- отключить) |
| Enroll Button Settings | Позиция кнопки изучения (AnchorMin, AnchorMax, OffsetMin, OffsetMax) |

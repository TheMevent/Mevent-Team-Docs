# King Of Hill

## Что делает этот режим?

**King Of Hill** — это режим ивента для Multi Events, в котором игроки соревнуются за звание того, кто заберётся выше всех. На протяжении всего ивента система отслеживает максимальную высоту каждого игрока и ведёт таблицу лидеров в реальном времени. По истечении времени игроки с наивысшими достигнутыми точками получают награды.

## Ключевые возможности

- Автоматическое отслеживание максимальной достигнутой высоты каждого игрока
- Таблица лидеров в реальном времени с UI-уведомлениями
- Настраиваемая максимальная допустимая высота (для исключения читеров)
- Гибкая система наград с поддержкой предметов, команд, китов и сторонних плагинов
- Интеграция с Multi Events Inventory и Multi Events Leaderboard
- Поддержка уведомлений через Notify / UINotify
- Совместимость с Oxide и Carbon

## Требования

- **Multi Events** (основной плагин) — обязательно
- **Image Library** — рекомендуется (для отображения иконок)
- **Notify** или **UINotify** — опционально (для всплывающих уведомлений)
- **Multi Events Inventory** — опционально (для выдачи наград через инвентарь ивентов)
- **Multi Events Leaderboard** — опционально (для ведения глобальной статистики)

## Быстрый старт

1. Установите плагин **Multi Events** (если ещё не установлен)
2. Поместите файл режима в папку `oxide/plugins` (или `carbon/plugins`)
3. Перезагрузите сервер или загрузите плагин командой в консоли
4. Режим автоматически зарегистрируется в Multi Events
5. Настройте конфигурацию по желанию (см. [Конфигурация](./configuration))

## Как проходит ивент

1. **Начало** — Multi Events запускает режим King Of Hill. Всем игрокам на сервере отображается UI-уведомление
2. **Соревнование** — каждую секунду система проверяет позицию всех активных игроков. Если текущая высота игрока выше его предыдущего максимума (и не превышает установленный лимит), рекорд обновляется
3. **Таблица лидеров** — в правой части экрана отображается актуальный рейтинг с именами игроков и их максимальной высотой
4. **Окончание** — когда время ивента истекает, система определяет победителей и раздаёт награды согласно настройкам

## Устранение неполадок

| Проблема | Решение |
|---|---|
| Ивент не запускается | Убедитесь, что основной плагин Multi Events установлен и загружен. Проверьте консоль на наличие ошибок |
| UI не отображается | Проверьте, установлен ли Image Library. Убедитесь, что игрок не скрыл уведомления |
| Награды не выдаются | Проверьте настройки наград в конфигурации. Если используется Multi Events Inventory, убедитесь, что он установлен |
| Игроки набирают подозрительную высоту | Уменьшите значение максимальной высоты в конфигурации |

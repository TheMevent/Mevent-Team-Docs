# Установка NPC Missile Silo

## Предварительные требования

Перед установкой убедитесь, что на сервере установлены:

- **Multi Events** — основной плагин (обязательно)
- **ImageLibrary** — для отображения UI-элементов (обязательно)

### Рекомендуемые плагины

- **Multi Events Inventory** — для хранения наград в инвентаре ивентов
- **Multi Events Leaderboard** — для расширенной таблицы лидеров
- **Notify** или **UINotify** — для внутриигровых уведомлений

## Пошаговая установка

### Шаг 1. Загрузка

Загрузите файл режима NPC Missile Silo из комплекта Expansion Pack 1.

### Шаг 2. Размещение файла

Скопируйте файл плагина в папку:

```
oxide/plugins/
```

### Шаг 3. Загрузка плагина

Плагин загрузится автоматически. Если сервер уже запущен:

```
oxide.reload MultiEventsNpcMissileSilo
```

### Шаг 4. Проверка

После загрузки:

1. Файл конфигурации появится в `oxide/config/`
2. Режим зарегистрируется в Multi Events автоматически
3. Ивент станет доступен для запуска

### Шаг 5. Настройка

Откройте конфигурацию и настройте награды и длительность ивента. После изменений перезагрузите плагин.

## Проверка работоспособности

Запустите ивент вручную:

```
npcmissilesilo.start
```

Убедитесь, что UI-виджет отображается и очки начисляются за убийство учёных в зоне ракетной шахты.

Для остановки:

```
npcmissilesilo.stop
```

# Конфигурация

Файл конфигурации расположен по пути:
```
oxide/config/MultiEvents.json
```

## Основные параметры

| Параметр | Тип | По умолчанию | Описание |
|---|---|---|---|
| Info Commands | Массив строк | `["events"]` | Чат-команды для открытия панели ивентов |
| ServerPanel Template (V1, V2) | Строка | `Fullscreen` | Шаблон отображения для интеграции с ServerPanel |
| Enable Offline Image Mode | bool | `false` | Режим офлайн-изображений |
| Work with Notify? | bool | `true` | Использовать Notify/UINotify для уведомлений |
| Enable notify when event start | bool | `true` | Показывать уведомление при запуске ивента |
| Enable timer to start random event | bool | `true` | Включить автозапуск случайных ивентов по таймеру |
| Min Events Duration | Число | `3600` | Минимальный интервал таймера (в секундах) |
| Max Events Duration | Число | `7200` | Максимальный интервал таймера (в секундах) |

## Настройки уведомлений

| Параметр | Тип | Описание |
|---|---|---|
| Work with Notify? | bool | Включить интеграцию с Notify/UINotify |
| Enable notify when event start | bool | Показывать уведомление при старте ивента |
| BlackList of notify to start event | Массив строк | Чёрный список плагинов, для которых не показывать уведомления |

## Настройки инвентаря

| Параметр | Тип | Описание |
|---|---|---|
| Use Inventory? | bool | Включить модуль инвентаря наград |
| Permission | Строка | Право доступа для инвентаря (оставьте пустым для доступа всем) |

## Настройки таблицы лидеров

| Параметр | Тип | Описание |
|---|---|---|
| Use Leaderboard? | bool | Включить модуль таблицы лидеров |
| Permission | Строка | Право доступа для таблицы (оставьте пустым для доступа всем) |

## Настройки UI

| Параметр | Описание |
|---|---|
| No Members Image | Изображение для отображения, когда нет участников |
| Square Image | Вспомогательное квадратное изображение |
| Default Event Icon | Иконка ивента по умолчанию |
| Arrow Left Image | Изображение стрелки "влево" |
| Arrow Right Image | Изображение стрелки "вправо" |
| Leaderboard Background Image | Фоновое изображение для таблицы лидеров |

## Настройка ивентов

Каждый ивент в массиве "Events" содержит следующие параметры:

| Параметр | Описание |
|---|---|
| Enabled | Включён ли ивент |
| Plugin Name | Название плагина для отслеживания статуса |
| Banner Url | URL баннера ивента |
| Title | Заголовок ивента |
| Icon image | URL иконки |
| Short Description | Краткое описание |
| Background | Настройки фона (слой, изображение, закрытие по клику) |
| Content | Настройки контента (фон, элементы, прокрутка) |
| Close Button | Настройки кнопки закрытия |
| Banner | Настройки элемента баннера |

## Визуальный редактор

Для каждого ивента доступен визуальный редактор, позволяющий менять:

- Позицию и размеры элементов (якоря, смещения)
- Цвета фона и текста
- Шрифты и размеры текста
- Изображения и спрайты
- Прокручиваемые области

Откройте редактор командой `/ShowUiEditor`.

## Пример минимальной конфигурации ивента

```json
{
 "Enabled": true,
 "Plugin Name": "YourPluginName",
 "Banner Url": "https://your-server.com/banner.png",
 "Title": "Название ивента",
 "Icon image": "https://your-server.com/icon.png",
 "Short Description": "Краткое описание вашего ивента"
}
```

:::tip Совет
Используйте визуальный редактор для тонкой настройки отображения каждого ивента — это намного удобнее, чем правка JSON-файла вручную.
:::

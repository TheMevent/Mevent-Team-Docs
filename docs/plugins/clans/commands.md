# Команды Clans

## Команды для всех игроков

Эти команды доступны каждому игроку на сервере.

### Открытие интерфейса

```
/clan
```

Открывает графический интерфейс управления кланом. Если игрок состоит в клане, откроется страница рейтингов. Если нет -- страница информации о клане.

```
/clans
```

Альтернативная команда, делает то же самое.

### Создание клана

```
/clan create <тег>
```

Создаёт новый клан с указанным тегом. Тег должен содержать от 2 до 6 символов (настраивается) и не содержать запрещённых слов.

**Пример:**
```
/clan create WOLF
```

:::note
Если в настройках включена плата за создание клана, с баланса игрока будет списана указанная сумма.
:::

### Поиск клана

```
/clan join
```

Открывает интерфейс поиска и просмотра доступных кланов.

### Информация о клане

```
/cinfo <тег>
```

Показывает подробную информацию о клане: описание, дату создания, количество участников и их статус.

**Пример:**
```
/cinfo WOLF
```

### Дружественный огонь

```
/clan ff
/cff
```

Переключает дружественный огонь по участникам своего клана. Если включён общий режим, только лидер (или модератор, если разрешено) может управлять этим.

```
/clan allyff
/aff
```

Переключает дружественный огонь по участникам союзных кланов.

### Клановый чат

```
/c <сообщение>
/cchat <сообщение>
```

Отправляет сообщение, которое видят только участники вашего клана.

**Пример:**
```
/c Собираемся на рейд через 10 минут
```

### Чат альянса

```
/a <сообщение>
/achat <сообщение>
```

Отправляет сообщение, которое видят участники вашего клана и всех союзных кланов.

**Пример:**
```
/a Нужна помощь у нефтяной вышки
```

### Выход из клана

```
/clan leave
```

Покидает текущий клан. Если игрок является лидером и включено автоматическое расформирование -- клан будет распущен.

## Команды для модераторов клана

Эти команды доступны модераторам и лидеру клана.

### Приглашение игрока

```
/clan invite <имя или SteamID>
```

Отправляет приглашение указанному игроку в ваш клан.

**Пример:**
```
/clan invite PlayerName
```

### Отзыв приглашения

```
/clan withdraw <имя или SteamID>
```

Отменяет ранее отправленное приглашение.

**Пример:**
```
/clan withdraw PlayerName
```

### Исключение участника

```
/clan kick <имя или SteamID>
```

Исключает указанного участника из клана.

**Пример:**
```
/clan kick ToxicPlayer
```

### Управление альянсами

```
/clan allyinvite <тег>
```

Отправляет предложение альянса другому клану.

```
/clan allywithdraw <тег>
```

Отзывает предложение альянса.

```
/clan allyaccept <тег>
```

Принимает входящее предложение альянса.

```
/clan allycancel <тег>
```

Отклоняет входящее предложение альянса.

```
/clan allyrevoke <тег>
```

Разрывает существующий альянс с кланом.

**Примеры:**
```
/clan allyinvite BEARS
/clan allyaccept BEARS
/clan allyrevoke BEARS
```

## Команды для лидера клана

Эти команды доступны только лидеру (создателю) клана.

### Назначение модератора

```
/clan promote <имя или SteamID>
```

Повышает обычного участника до модератора.

**Пример:**
```
/clan promote TrustedPlayer
```

### Понижение модератора

```
/clan demote <имя или SteamID>
```

Понижает модератора до обычного участника.

**Пример:**
```
/clan demote FormerMod
```

### Описание клана

```
/clan description <текст>
```

Устанавливает описание клана (до 256 символов по умолчанию).

**Пример:**
```
/clan description Дружный клан! Играем каждый вечер. Принимаем опытных игроков.
```

### Цвет тега клана

```
/clan tagcolor <HEX-цвет>
```

Устанавливает цвет клан-тега в чате. Используйте формат HEX (6 символов, без #).

**Пример:**
```
/clan tagcolor FF5500
```

### Расформирование клана

```
/clan disband
```

Полностью расформировывает клан. Все участники будут удалены, альянсы разорваны.

:::danger
Это действие необратимо! Все данные клана, включая статистику и скины, будут удалены.
:::

## Команды администратора

Все административные команды используют формат `clans.manage <действие>` и требуют право `clans.admin`. Можно вводить в консоли сервера или в игровом чате.

### Просмотр информации

```
clans.manage list
```

Выводит список всех кланов с тегами, лидерами и количеством участников.

```
clans.manage listex
```

Расширенный список: все кланы, все участники, роли и статус онлайн.

```
clans.manage show <тег или имя игрока>
```

Подробная информация об одном клане: описание, дата создания, список участников, альянсы.

**Пример:**
```
clans.manage show WOLF
clans.manage show PlayerName
```

### Отправка сообщений

```
clans.manage msg <тег> <сообщение>
```

Отправляет сообщение всем участникам указанного клана.

**Пример:**
```
clans.manage msg WOLF Нарушение правил сервера!
```

### Управление кланами

```
clans.manage create <имя игрока> <тег>
```

Создаёт клан для указанного игрока с заданным тегом.

```
clans.manage rename <старый тег> <новый тег>
```

Переименовывает клан.

```
clans.manage disband <тег>
```

Расформировывает указанный клан.

### Управление участниками

```
clans.manage invite <тег> <имя игрока>
```

Отправляет приглашение в клан.

```
clans.manage join <тег> <имя игрока>
```

Добавляет игрока в клан напрямую, без приглашения.

```
clans.manage kick <тег> <имя игрока>
```

Исключает игрока из клана.

```
clans.manage kick.player <SteamID>
```

Исключает игрока из любого клана по его SteamID.

```
clans.manage owner <тег> <имя игрока>
```

Назначает нового лидера клана.

```
clans.manage promote <тег> <имя игрока>
```

Повышает участника до модератора.

```
clans.manage demote <тег> <имя игрока>
```

Понижает модератора до участника.

### Служебные команды

```
clans.refreshtop
```

Принудительно обновляет таблицы рейтингов.

```
clans.refreshskins
```

Обновляет кэш скинов для всех кланов.

# Конфигурация Clans

Файл конфигурации расположен по пути: `oxide/config/Clans.json`

После изменения конфигурации перезагрузите плагин командой `o.reload Clans`.

## Основные настройки

| Параметр | По умолчанию | Описание |
|---|---|---|
| Работа с Notify | `true` | Использовать ли плагин Notify/UINotify для уведомлений |
| Команды | `["clan", "clans"]` | Список команд для открытия интерфейса |
| Команды информации о клане | `["cinfo"]` | Команды для просмотра информации о клане |
| Максимум символов описания клана | `256` | Максимальная длина описания клана |
| Тег клана в имени игрока | `true` | Отображать тег клана перед именем игрока |
| Автоматическое создание команды | `true` | Автоматически создавать игровую команду Rust для клана |
| Частота обновления топа | `60` | Интервал обновления рейтингов (в секундах) |
| Значение по умолчанию для нормативов ресурсов | `100000` | Стандартное значение для нормативов добычи |
| Автоматическое расформирование при выходе лидера | `false` | Расформировывать ли клан, если лидер покидает его |

## Интеграция с игровыми командами Rust

| Параметр | По умолчанию | Описание |
|---|---|---|
| Выход из клана через кнопку команды | `true` | Позволяет покинуть клан через кнопку "Leave Team" |
| Исключение через кнопку команды | `true` | Позволяет исключать через кнопку "Kick" в игре |
| Приглашение через систему команд | `true` | Позволяет приглашать через игровой интерфейс |
| Повышение через кнопку команды | `true` | Позволяет повышать участников через интерфейс |
| Принятие приглашения через кнопку | `true` | Позволяет принять приглашение через игровой интерфейс |
| Показывать интерфейс создания клана при создании команды | `false` | Открывать ли окно создания клана при создании команды |

## Настройки аватара

| Параметр | По умолчанию | Описание |
|---|---|---|
| Аватар по умолчанию | URL картинки | Изображение по умолчанию для новых кланов |
| Может лидер менять | `true` | Разрешить лидеру менять аватар клана |
| Может модератор менять | `false` | Разрешить модератору менять аватар клана |
| Может участник менять | `false` | Разрешить участнику менять аватар клана |
| Право на изменение аватара | `""` | Дополнительное право для изменения (пустое = не требуется) |
| Steam API Key | `""` | Ключ Steam API для загрузки аватаров игроков |

## Настройки клан-тегов

| Параметр | По умолчанию | Описание |
|---|---|---|
| Минимальная длина тега | `2` | Минимальное количество символов в теге |
| Максимальная длина тега | `6` | Максимальное количество символов в теге |
| Чувствительность к регистру | `true` | Учитывать ли регистр при поиске дубликатов |
| Заблокированные слова | `["admin", "mod", "owner"]` | Список запрещённых слов в тегах |
| Проверка символов | `true` | Проверять ли допустимость символов в теге |
| Разрешённые спецсимволы | `"!"` | Дополнительные разрешённые символы |

### Цвет тега

| Параметр | По умолчанию | Описание |
|---|---|---|
| Включено | `true` | Разрешить цветные теги |
| Цвет по умолчанию | `AAFF55` | HEX-цвет тега по умолчанию |
| Лидер может менять | `true` | Может ли лидер менять цвет тега |
| Модератор может менять | `false` | Может ли модератор менять цвет тега |
| Участник может менять | `false` | Может ли участник менять цвет тега |

## Настройки чата

| Параметр | По умолчанию | Описание |
|---|---|---|
| Теги кланов в чате | `true` | Отображать ли клан-теги в общем чате |
| Формат тега | `<color=#{color}>[{tag}]</color>` | Формат отображения тега |
| Клановый чат | `true` | Включить отдельный чат для клана |
| Команды кланового чата | `["c", "cchat"]` | Команды для отправки сообщений в клан |
| Чат альянса | `true` | Включить чат между союзными кланами |
| Команды чата альянса | `["a", "achat"]` | Команды для чата с союзниками |
| Работа с BetterChat | `true` | Интеграция с плагином BetterChat |
| Работа с IQChat | `true` | Интеграция с плагином IQChat |
| Работа с игровым чатом | `false` | Интеграция с игровым чатом |

## Настройки дружественного огня

| Параметр | По умолчанию | Описание |
|---|---|---|
| Использовать дружественный огонь | `true` | Включить систему дружественного огня |
| Дружественный огонь для турелей | `false` | Турели не стреляют по соклановцам |
| Общий дружественный огонь | `false` | Один переключатель на весь клан (управляет лидер) |
| Модераторы могут переключать общий | `false` | Дать модераторам право управления общим FF |
| Игроки могут переключать общий | `false` | Дать участникам право управления общим FF |
| Значение по умолчанию | `false` | Начальное состояние дружественного огня |
| Игнорировать на арене | `false` | Отключать FF на арене (требуется ArenaTournament) |

## Настройки альянсов

| Параметр | По умолчанию | Описание |
|---|---|---|

| Дружественный огонь альянса | `true` | Разрешить управление FF между союзниками |

| Добавить союзников в игровые команды | `false` | Добавлять участников союзных кланов в команды Rust |

## Настройки очков и рейтингов

Система очков настраивается в секции "Loot Settings". Каждая запись содержит:

| Поле | Описание |
|---|---|
| Включено | Активна ли эта запись |
| Краткое имя | Идентификатор предмета/существа |
| Тип | Категория (Kill, Gather, LootCrate, Craft, HackCrate) |
| Очки | Количество очков за действие |

| clanstop | Рейтинг кланов |
| playerstop | Рейтинг игроков |
| resources | Нормативы добычи ресурсов |
| skins | Скины клана |
| playerslist | Список игроков |
| alianceslist | Список альянсов |

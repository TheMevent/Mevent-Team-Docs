# Scavenger Hunt

## Что делает этот режим?

**Scavenger Hunt** — это режим ивента-охоты за сокровищами для Multi Events. На карте появляется спрятанный сундук с ценными наградами. Все игроки соревнуются за право первым найти и открыть его. Первый игрок, нашедший сундук, получает все награды.

## Ключевые возможности

- Спавн сундука со случайной позицией на карте
- Настраиваемый префаб и скин сундука
- Отображение расстояния до сундука в UI для всех игроков
- Награда только одному победителю — тому, кто первым найдёт сундук
- Поддержка уведомлений через Notify и UINotify
- Встроенный UI-виджет с обратным отсчётом и расстоянием
- Поддержка наград через предметы, команды, наборы (Kit) и сторонние плагины
- Интеграция с инвентарём ивентов (Multi Events Inventory)

## Требования

| Зависимость | Обязательно? |
|---|---|
| **Multi Events** | Да |
| **ImageLibrary** | Да |
| Multi Events Inventory | Рекомендуется |
| Multi Events Leaderboard | Рекомендуется |
| Notify / UINotify | Опционально |

## Быстрый старт

1. Установите **Multi Events** на сервер
2. Поместите файл режима Scavenger Hunt в папку `oxide/plugins`
3. Перезагрузите сервер или загрузите плагин
4. Файл конфигурации будет создан автоматически
5. Настройте награды и параметры сундука в конфигурации
6. Ивент готов к запуску через Multi Events или вручную командой

## Как проходит ивент

1. Ивент запускается автоматически через Multi Events или вручную администратором
2. На карте в случайном месте появляется сундук с сокровищами
3. Всем игрокам на сервере показывается UI-уведомление с описанием, таймером и расстоянием до сундука
4. Игроки ищут сундук, ориентируясь по расстоянию, отображаемому в UI
5. Первый игрок, который попытается открыть сундук, побеждает и получает все награды
6. Ивент завершается автоматически после нахождения сундука или по истечении таймера

## Настройки

| Параметр | Описание | По умолчанию |
|---|---|---|
| Work with Notify? | Использовать ли плагин Notify для уведомлений | `true` |
| Event Duration | Длительность ивента в секундах | `600` |
| Treasure prefab | Префаб сундука | Большой деревянный ящик |
| Treasure skin id | Скин ID сундука | `1394363785` |
| Reward for the winner | Награда для победителя | Предметы |
| UI Settings | Настройки иконок UI | URL иконок |

## Устранение неполадок

| Проблема | Решение |
|---|---|
| Ивент не запускается | Убедитесь, что Multi Events установлен и загружен |
| Сундук не появляется | Проверьте логи сервера на ошибки спавна |
| Нет UI-уведомлений | Проверьте, установлен ли ImageLibrary |
| Расстояние не отображается | Проверьте настройки UI в конфигурации |
| Награды не выдаются | Проверьте корректность настроек наград в конфигурации |

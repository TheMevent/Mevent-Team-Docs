# Команды Car Sharing

## Команды игроков

### Открытие меню каршеринга

| Команда | Описание |
|---|---|
| `/rent` | Открыть меню выбора автомобиля для аренды |
| `/carsharing` | Альтернативная команда для открытия меню |

:::info
Команды настраиваются в конфигурационном файле. Вы можете добавить свои команды или изменить стандартные.
:::

**Пример использования:**
```
/rent
```

После ввода команды откроется графическое меню с доступными машинами. В меню отображается:
- Название машины
- Класс (Luxe, Comfort и т.д.)
- Средняя скорость
- Стоимость аренды за минуту
- Текущий баланс игрока

### Аренда через NPC

Игрок также может открыть меню каршеринга, нажав на NPC-бота. При этом будут доступны только те классы машин, которые привязаны к конкретному NPC.

### Управление арендой

При аренде машины игроку выдаётся специальный предмет вызова. Чтобы вызвать арендованную машину:

1. Возьмите предмет вызова в руку.
2. Машина появится рядом с вами.

Во время поездки на экране отображается спидометр с информацией:
- Текущая скорость (км/ч)
- Оставшееся время аренды (в минутах)

## Команды администратора

### Команды выдачи машин

Каждая машина может иметь собственную команду для прямой выдачи:

| Команда | Описание |
|---|---|
| `/share.ferrari` | Выдать машину "Ferrari" |
| `/share.mazda` | Выдать машину "Mazda" |
| `/share.skoda` | Выдать машину "Skoda" |
| `/share.opel` | Выдать машину "Opel" |

:::note
Названия команд выдачи задаются в конфигурации для каждой машины отдельно. Приведённые выше -- из конфигурации по умолчанию.
:::

**Пример использования:**
```
/share.ferrari
```

### Управление правами

```
oxide.grant user <SteamID> carsharing.use
oxide.revoke user <SteamID> carsharing.use
oxide.grant group default carsharing.use
```

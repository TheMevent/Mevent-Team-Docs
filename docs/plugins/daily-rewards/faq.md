# Часто задаваемые вопросы (FAQ)

## 1. Как сбросить прогресс всех игроков?

Выполните в серверной консоли команду:
```
dailyrewards.wipe
```
Это сбросит прогресс всех игроков. Для автоматического сброса при вайпе включите параметр `"Wipe on new map save": true` в конфигурации.

---

## 2. Как добавить новый день с наградой?

В конфигурации в секции `"Daily awards"` добавьте новую запись с номером дня. Скопируйте структуру существующего дня и измените параметры:

```json
"8": {
 "Enabled": true,
 "Image": "https://example.com/day8.png",
 "Title": "BONUS",
 "Awards": [
 {
 "Type": "Item",
 "ShortName": "sulfur",
 "Amount": 5000,
 "Skin": 0
 }
 ],
 "IsSpecialDay": true
}
```

---

## 3. Как настроить выдачу валюты через команду?

Используйте тип награды "Command" с подстановкой `%steamid%`:

```json
{
 "Type": "Command",
 "Command": "deposit %steamid% 100",
 "Amount": 1,
 "Image": "https://example.com/coins.png"
}
```

`%steamid%` автоматически заменяется на SteamID игрока.

---

## 4. Можно ли изменить время кулдауна для VIP-игроков?

Да. В секции `"Cooldown Settings"` добавьте нужные права и их кулдауны:

```json
{
 "Cooldown Settings": {
 "DefaultCooldown": 3600,
 "Cooldowns": {
 "dailyrewards.vip": 1800,
 "dailyrewards.premium": 900
 }
 }
}
```

Затем выдайте соответствующие права:
```
oxide.grant group vip dailyrewards.vip
```

---

## 5. Игрок не может забрать награду. В чём причина?

Возможные причины:
1. **Кулдаун не завершён** -- игроку нужно провести больше времени на сервере.
2. **Блокировка после вайпа** -- если включена опция `UseCooldownAfterWipe`.
3. **Рейд-блок / комбат-блок** -- если включены соответствующие опции в настройках инвентаря.
4. **Зона чужого шкафа** -- если включена опция `UseBuildingBlocked`.

---

## 6. Как показывать интерфейс при входе на сервер?

Включите параметр в конфигурации:
```json
"Show UI with rewards when player connects to server?": true
```

---

## 7. Можно ли редактировать награды прямо в игре?

Да. Выдайте право `dailyrewards.edit` администратору:
```
oxide.grant user <steamid> dailyrewards.edit
```
После этого в интерфейсе наград появятся дополнительные элементы управления для добавления и изменения наград.

---

## 8. В каком часовом поясе сбрасываются дни?

По умолчанию используется `Europe/London`. Измените часовой пояс в конфигурации:
```json
"Daily reset settings": {
 "TimeZone": "Europe/Moscow",
 "Time": "05:00"
}
```

---

## 9. Что происходит с наградами при вайпе?

Это зависит от настроек:
- Если `"Wipe on new map save": true` -- прогресс сбрасывается.
- Если `"RetainItemsOnWipe": true` -- предметы в инвентаре сохраняются.
- Вы также можете вручную сбросить данные командой `dailyrewards.wipe`.

---

## 10. Как переключить шаблон интерфейса?

Используйте консольную команду:
```
dailyrewards.template fullscreen 1
dailyrewards.template inmenu 2
```
Это переключит шаблон интерфейса между доступными вариантами.

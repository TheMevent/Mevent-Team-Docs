# Команды и права

## Команды чата

| Команда | Описание | Право |
|---|---|---|
| `/voteday` или `/vt` | Проголосовать за пропуск ночи | Для всех (во время голосования) |

:::info Примечание
Команды голосования настраиваются в конфигурации (`"VoteCommands"`). По умолчанию: `["voteday", "vt"]`.
:::

## Команды администратора

| Команда | Описание | Право |
|---|---|---|
| `sn.admin` | Показать список админ-команд | `skipnight.admin` |
| `sn.admin start_manually` | Принудительно запустить голосование | `skipnight.admin` |

## Права (Permissions)

| Право | Описание |
|---|---|
| `skipnight.admin` | Доступ к административным командам |

:::tip Совет
Голосование доступно всем игрокам по умолчанию. Для ограничения доступа используйте настройки в конфигурации.
:::

## Управление через интерфейс

| Элемент | Описание |
|---|---|
| Кнопка голосования | Проголосовать за пропуск ночи |
| Прогресс-бар | Показывает процент проголосовавших |
| Кнопка скрытия | Скрыть интерфейс (если включена) |
| Кнопка закрытия | Закрыть интерфейс (если включена) |

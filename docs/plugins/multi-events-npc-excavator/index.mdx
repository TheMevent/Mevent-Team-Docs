# Npc Excavator

## Что делает этот режим?

**Npc Excavator** — это режим ивента для Multi Events, в котором игроки соревнуются за очки, уничтожая NPC-учёных на монументе Экскаватор. За каждого убитого учёного на Экскаваторе начисляются очки. Игрок с наибольшим количеством очков к окончанию таймера получает награды.

## Ключевые возможности

- Автоматический подсчёт очков за убийство учёных на Экскаваторе
- Гибкая система наград для нескольких мест в таблице лидеров
- Поддержка уведомлений через Notify и UINotify
- Встроенный UI-виджет с таблицей лидеров и обратным отсчётом
- Поддержка наград через предметы, команды, наборы (Kit) и сторонние плагины
- Интеграция с инвентарём ивентов (Multi Events Inventory)

## Требования

| Зависимость | Обязательно? |
|---|---|
| **Multi Events** | Да |
| **ImageLibrary** | Да |
| Multi Events Inventory | Рекомендуется |
| Multi Events Leaderboard | Рекомендуется |
| Notify / UINotify | Опционально |

## Быстрый старт

1. Установите **Multi Events** на сервер
2. Поместите файл режима Npc Excavator в папку `oxide/plugins`
3. Перезагрузите сервер или загрузите плагин
4. Файл конфигурации будет создан автоматически
5. Настройте награды в конфигурации
6. Ивент готов к запуску через Multi Events или вручную командой

## Как проходит ивент

1. Ивент запускается автоматически через Multi Events или вручную администратором
2. Всем игрокам на сервере показывается UI-уведомление с описанием и таймером
3. Игроки отправляются на монумент Экскаватор и уничтожают NPC-учёных
4. За каждого убитого учёного начисляются очки
5. В режиме реального времени обновляется таблица лидеров с текущими результатами
6. По истечении таймера ивент завершается
7. Победители получают настроенные награды в соответствии с их местом

## Настройки

| Параметр | Описание | По умолчанию |
|---|---|---|
| Work with Notify? | Использовать ли плагин Notify для уведомлений | `true` |
| Event Duration | Длительность ивента в секундах | `600` |
| Awards Settings | Настройки наград для каждого места | Предметы для 1-2 мест |

## Устранение неполадок

| Проблема | Решение |
|---|---|
| Ивент не запускается | Убедитесь, что Multi Events установлен и загружен |
| Нет UI-уведомлений | Проверьте, установлен ли ImageLibrary |
| Очки не начисляются | Убедитесь, что на карте присутствует монумент Экскаватор с NPC-учёными |
| Награды не выдаются | Проверьте корректность настроек наград в конфигурации |

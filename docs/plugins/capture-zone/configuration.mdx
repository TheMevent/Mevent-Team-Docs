# Конфигурация

Файл конфигурации расположен по пути:
```
oxide/config/CaptureZone.json
```

## Основные параметры

| Параметр | Тип | По умолчанию | Описание |
|---|---|---|---|
| Work with Notify? | bool | `true` | Использовать Notify/UINotify для уведомлений |
| Zone Radius | float | `40.0` | Радиус зоны захвата (в метрах) |
| Permission | Строка | `""` | Право доступа для установки зоны (пустое = для всех) |

## Настройки предмета

| Параметр | Тип | По умолчанию | Описание |
|---|---|---|---|
| DisplayName | Строка | `"Zone Capture"` | Отображаемое название предмета |
| ShortName | Строка | `"cupboard.tool"` | Базовый предмет |
| SkinID | Число | `2767790029` | Уникальный скин предмета |

## Настройки маркера

| Параметр | Тип | По умолчанию | Описание |
|---|---|---|---|
| Enabled | bool | `true` | Включить маркеры на карте |
| Display Name | Строка | `"Zone by {owner}"` | Текст маркера (`{owner}` заменяется на имя владельца) |
| Radius | float | `0.3` | Размер маркера |
| Alpha | float | `1.0` | Прозрачность маркера |
| Refresh Rate | float | `3.0` | Частота обновления (секунды) |
| Duration | int | `0` | Длительность (0 = бесконечно) |
| Color 1 | Строка | `"#EA9999"` | Основной цвет маркера |
| Color 2 | Строка | `"#A73636FF"` | Дополнительный цвет маркера |

### Видимость маркера

| Параметр | Тип | По умолчанию | Описание |
|---|---|---|---|
| All Players | bool | `true` | Все игроки видят маркеры |
| Owner | bool | `true` | Владелец видит свои маркеры |
| Teammates | bool | `true` | Тиммейты видят маркер |
| Friends | bool | `true` | Друзья видят маркер |
| Clanmates | bool | `true` | Кланмейты видят маркер |
| Authorized in the cupboard | bool | `true` | Авторизованные видят маркер |
| Permission | Строка | `""` | Право на просмотр маркера |

## Настройки дропа

| Параметр | Тип | По умолчанию | Описание |
|---|---|---|---|
| Enabled | bool | `true` | Включить дроп из контейнеров |

Для каждого контейнера:

| Параметр | Описание |
|---|---|
| Prefab name | Путь к префабу контейнера |
| Min amount | Минимальное количество |
| Max amount | Максимальное количество |
| Chance | Вероятность дропа (0-100) |

## Настройки уведомлений о добыче

| Параметр | Тип | По умолчанию | Описание |
|---|---|---|---|
| Enabled | bool | `true` | Показывать уведомление при добыче на чужой территории |
| Cooldown | float | `5.0` | Задержка между уведомлениями (секунды) |

## Настройки процента налога

| Параметр | Тип | По умолчанию | Описание |
|---|---|---|---|
| Default | float | `20` | Процент налога по умолчанию |

Дополнительные права и процент:

| Право | Процент |
|---|---|
| `capturezone.vip` | 25% |
| `capturezone.deluxe` | 30% |

:::info
Если у игрока несколько прав, используется максимальное значение процента.
:::

## Настройки лимитов

| Параметр | Тип | По умолчанию | Описание |
|---|---|---|---|
| Enabled | bool | `true` | Включить ограничение количества зон |
| Default | int | `3` | Максимальное количество зон по умолчанию |

Дополнительные права и лимиты:

| Право | Лимит зон |
|---|---|
| `capturezone.vip` | 5 |
| `capturezone.deluxe` | 7 |

## Настройки обхода налога

| Параметр | Тип | По умолчанию | Описание |
|---|---|---|---|
| Teammates | bool | `true` | Обход для тиммейтов |
| Friends | bool | `true` | Обход для друзей (плагин Friends) |
| Clanmates | bool | `true` | Обход для кланмейтов (плагин Clans) |
| Authorized in the cupboard | bool | `true` | Обход для авторизованных в шкафу |

## Пример конфигурации

```json
{
 "Work with Notify?": true,
 "Zone Radius": 40.0,
 "Permission": "",
 "Item Settings": {
 "DisplayName": "Zone Capture",
 "ShortName": "cupboard.tool",
 "SkinID": 2767790029
 },
 "Percent Settings": {
 "Default": 20,
 "Permissions": {
 "capturezone.vip": 25,
 "capturezone.deluxe": 30
 }
 },
 "Limits Settings": {
 "Enabled": true,
 "Default": 3,
 "Permissions": {
 "capturezone.vip": 5,
 "capturezone.deluxe": 7
 }
 },
 "Bypass Settings": {
 "Teammates (Rust in-game system)": true,
 "Friends": true,
 "Clanmates": true,
 "Authorized in the cupboard": true
 }
}
```

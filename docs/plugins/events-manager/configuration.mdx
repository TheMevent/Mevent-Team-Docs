# Конфигурация

Файл конфигурации расположен по пути:
```
oxide/config/EventsManager.json
```

## Основные параметры

| Параметр | Тип | По умолчанию | Описание |
|---|---|---|---|
| Commands | Массив строк | `["em"]` | Список команд для открытия панели управления |
| Disable vanilla auto events? | bool | `true` | Отключить стандартные ванильные события Rust |
| Allowed list of vanilla events | Массив строк | `["event_f15e"]` | Белый список ванильных событий, которые не будут отключены |
| Discord web hook | Строка | `""` | URL Discord-вебхука для отправки уведомлений о запуске ивентов |
| Enable event logging to file | bool | `false` | Включить запись журнала запуска ивентов в файл |

## Пример конфигурации

```json
{
 "Commands": [
 "em"
 ],
 "Disable vanilla auto events?": true,
 "Allowed list of vanilla events": [
 "event_f15e"
 ],
 "Discord web hook": "",
 "Enable event logging to file": false
}
```

## Описание параметров

### Команды (Commands)

Массив чат-команд, по которым открывается панель управления. Вы можете добавить несколько команд:

```json
"Commands": ["em", "events", "eventmanager"]
```

### Отключение ванильных событий

Когда параметр `Disable vanilla auto events?` включён, плагин отключает автоматический запуск всех стандартных событий Rust (патрульный вертолёт, аирдроп, грузовой корабль, Chinook 47 и другие).

События из белого списка `Allowed list of vanilla events` не будут затронуты.

### Discord-уведомления

Укажите URL вебхука Discord в параметре `Discord web hook`, чтобы получать уведомления при каждом запуске ивента. Уведомления отправляются в виде Embed-сообщений с названием запущенного события.

### Журнал событий

При включении параметра `Enable event logging to file` все запуски ивентов будут записываться в лог-файл с указанием времени и названия события.

## Настройка через интерфейс

Большинство настроек (расписание, ивенты, категории) выполняется через визуальный интерфейс в игре (команда `/em`). Конфигурационный файл содержит только глобальные параметры сервера.

:::info Автообновление
При обновлении версии плагина конфигурация автоматически обновляется. Новые ивенты и категории добавляются автоматически.
:::

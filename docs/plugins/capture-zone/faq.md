# Часто задаваемые вопросы

## 1. Как получить предмет захвата зоны?

Есть два способа:

1. **Через команду администратора**: `/give.capturezone [steamid] [количество]`
2. **Через дроп из контейнеров**: если включено в конфигурации, предмет может выпадать из лутовых ящиков и бочек с настраиваемой вероятностью

## 2. Как выглядит предмет захвата зоны?

Предмет выглядит как обычный шкаф (Tool Cupboard), но с уникальным скином. Он отображается с особым названием -- по умолчанию "Zone Capture". Устанавливается точно так же, как обычный шкаф.

## 3. Какой радиус у зоны захвата?

По умолчанию радиус составляет 40 метров. Вы можете изменить это значение в конфигурации (параметр "Zone Radius").

## 4. Как работает налог на ресурсы?

Когда игрок добывает ресурсы внутри зоны (дерево, руда, камень, растения), процент от добычи автоматически переносится в шкаф владельца зоны. Стандартный процент -- 20%. Если шкаф заполнен, ресурсы теряются.

## 5. Как освободить друзей от налога?

В конфигурации есть секция "Bypass Settings". Включите нужные параметры:
- **Teammates** -- тиммейты по игровой системе Rust
- **Friends** -- друзья (через плагин Friends)
- **Clanmates** -- члены клана (через плагин Clans)
- **Authorized in the cupboard** -- авторизованные в шкафу

## 6. Можно ли поставить несколько зон?

Да, но количество ограничено. По умолчанию -- 3 зоны на игрока. С правом `capturezone.vip` -- 5 зон, с `capturezone.deluxe` -- 7 зон. Лимиты настраиваются в конфигурации.

## 7. Маркер на карте видят все игроки?

По умолчанию -- да. Но вы можете настроить видимость для разных групп: только владелец, тиммейты, друзья, кланмейты, авторизованные в шкафу или игроки с определённым правом.

## 8. Можно ли ставить зоны рядом друг с другом?

Нет. Между зонами должно быть расстояние не менее двойного радиуса зоны (по умолчанию 80 метров). При попытке установить шкаф слишком близко к другой зоне игрок получит предупреждение.

## 9. Что происходит с зоной при вайпе?

Данные о зонах захвата хранятся в файле данных плагина. При вайпе сервера с удалением данных все зоны будут сброшены. Если вайп проводится без очистки данных, зоны восстановятся.

## 10. Как полностью отключить дроп предмета из контейнеров?

В конфигурации в секции "Drop Settings" установите параметр "Enabled" в `false`. После этого предмет можно будет получить только через команду администратора.

## 11. Совместим ли плагин с PvE-серверами?

Да. Capture Zone работает на любых серверах. На PvE-серверах налог на ресурсы -- это единственный "агрессивный" механизм, при этом шкаф нельзя уничтожить без PvP.

## 12. Как изменить процент налога для разных уровней доступа?

В конфигурации в секции "Percent Settings" измените значения в "Permissions":

```json
"Percent Settings": {
 "Default": 20,
 "Permissions": {
 "capturezone.vip": 25,
 "capturezone.deluxe": 30,
 "capturezone.premium": 40
 }
}
```

Не забудьте зарегистрировать новые права через конфигурацию.

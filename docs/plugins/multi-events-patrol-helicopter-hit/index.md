# Patrol Helicopter Hit

## Что делает этот режим?

**Patrol Helicopter Hit** — это режим ивента для Multi Events, в котором на карте появляется патрульный вертолёт с настраиваемым здоровьем, а игроки соревнуются за очки, нанося ему урон. Каждое попадание по вертолёту приносит очки. Игрок с наибольшим количеством попаданий к моменту уничтожения вертолёта или окончанию таймера получает награды.

## Ключевые возможности

- Спавн патрульного вертолёта с настраиваемым здоровьем и позицией
- Автоматический подсчёт попаданий по вертолёту
- Настройка начальной позиции и высоты вертолёта
- Гибкая система наград для нескольких мест в таблице лидеров
- Поддержка уведомлений через Notify и UINotify
- Встроенный UI-виджет с таблицей лидеров и обратным отсчётом
- Автоматическое завершение ивента при уничтожении вертолёта
- Интеграция с инвентарём ивентов (Multi Events Inventory)

## Требования

| Зависимость | Обязательно? |
|---|---|
| **Multi Events** | Да |
| **ImageLibrary** | Да |
| Multi Events Inventory | Рекомендуется |
| Multi Events Leaderboard | Рекомендуется |
| Notify / UINotify | Опционально |

## Быстрый старт

1. Установите **Multi Events** на сервер
2. Поместите файл режима Patrol Helicopter Hit в папку `oxide/plugins`
3. Перезагрузите сервер или загрузите плагин
4. Файл конфигурации будет создан автоматически
5. Настройте здоровье вертолёта, позицию и награды в конфигурации
6. Ивент готов к запуску через Multi Events или вручную командой

## Как проходит ивент

1. Ивент запускается автоматически через Multi Events или вручную администратором
2. На карте появляется патрульный вертолёт с заданным здоровьем
3. Всем игрокам на сервере показывается UI-уведомление с описанием и таймером
4. Игроки стреляют по вертолёту, зарабатывая очки за каждое попадание
5. В режиме реального времени обновляется таблица лидеров с текущими результатами
6. Ивент завершается при уничтожении вертолёта или по истечении таймера
7. Победители получают настроенные награды в соответствии с их местом

## Настройки

| Параметр | Описание | По умолчанию |
|---|---|---|
| Work with Notify? | Использовать ли плагин Notify для уведомлений | `true` |
| Event Duration | Длительность ивента в секундах | `600` |
| Heli Init Position | Начальная позиция вертолёта | `(0, 0, 0)` |
| Heli Height above ground | Высота вертолёта над землёй | `50` |
| Helicopter Health | Здоровье вертолёта | `10000` |
| Awards Settings | Настройки наград для каждого места | Предметы для 1-2 мест |

## Устранение неполадок

| Проблема | Решение |
|---|---|
| Ивент не запускается | Убедитесь, что Multi Events установлен и загружен |
| Вертолёт не появляется | Проверьте настройки позиции и высоты в конфигурации |
| Нет UI-уведомлений | Проверьте, установлен ли ImageLibrary |
| Очки не начисляются | Убедитесь, что попадания регистрируются по ивентовому вертолёту |
| Награды не выдаются | Проверьте корректность настроек наград в конфигурации |

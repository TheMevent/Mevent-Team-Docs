# Genie

## Что делает этот плагин?

**Genie** добавляет на сервер систему волшебных ламп (джиннов). Игроки находят специальные лампы как предметы, а при использовании запускается анимация открытия и случайная выдача наград. Каждая лампа имеет свой набор наград с разными шансами выпадения.

## Ключевые возможности

- Лампы появляются как обычные игровые предметы с кастомным скином
- Анимированный процесс открытия с прогресс-баром
- Система шансов для каждой награды
- Несколько типов джиннов с разными наборами наград
- Награды: предметы, команды, валюта плагинов
- Звуковые эффекты при открытии, прогрессе и завершении
- Настраиваемое время открытия и частота обновления анимации
- Лампы могут выпадать из лут-контейнеров (опционально)
- Контроль стакания ламп в инвентаре
- Права доступа для каждого типа лампы
- Интеграция с Notify для отправки уведомлений
- Админ-команда для выдачи ламп игрокам

## Требования

| Зависимость | Обязательность | Описание |
|---|---|---|
| **ImageLibrary** | Обязательно | Необходима для загрузки и отображения изображений |
| Notify | Опционально | Для отправки графических уведомлений |
| Oxide / uMod / Carbon | Обязательно | Совместим с Oxide, uMod и Carbon |

## Быстрый старт

1. Установите плагин **ImageLibrary**, если он ещё не установлен.
2. Скопируйте файл `Genie.cs` в папку `oxide/plugins/` вашего сервера.
3. Перезагрузите сервер или выполните команду `oxide.reload Genie`.
4. Выдайте лампу игроку командой:
 ```
 genie.give <steamid> default
 ```
5. Игрок получит лампу в инвентарь. При использовании предмета откроется интерфейс с анимацией.

# Команды Bank System

## Команды игроков

### Открытие банка

```
/bank
```

Открывает главный интерфейс банковской системы. В нём игрок может:
- Создать банковскую карту
- Просмотреть баланс
- Просмотреть историю транзакций и добычи
- Совершить перевод другому игроку

:::info Информация
Команда настраивается в конфигурации (поле `Commands`). По умолчанию -- `/bank`. Можно добавить несколько вариантов, например `["bank", "b", "balance"]`.
:::

### Использование банкомата

Банкоматы не требуют чат-команд -- достаточно подойти к банкомату и взаимодействовать с ним (кнопка `E`). В интерфейсе банкомата доступны:

- **Пополнение** -- обменять банкноты из инвентаря на электронную валюту
- **Снятие** -- получить банкноты обратно в инвентарь
- **Перевод** -- отправить средства другому игроку по номеру карты
- **Настройки** (для владельца банкомата) -- установить комиссии за пополнение и снятие
- **Ремонт** (для владельца банкомата) -- восстановить состояние банкомата за ресурсы

## Команды администратора

:::caution Внимание
Все команды ниже доступны только администраторам сервера или из серверной консоли.
:::

### Выдача банкнот

```
bank.givenote [steamId] [количество]
```

Выдаёт указанное количество банкнот игроку прямо в инвентарь.

**Примеры:**
```
bank.givenote 76561198000000000 100
bank.givenote 76561198000000000 500
```

### Выдача банкомата

```
bank.giveatm [steamId]
```

Выдаёт банкомат-предмет (торговый автомат с особым скином) игроку. Игрок может разместить его и стать владельцем.

**Пример:**
```
bank.giveatm 76561198000000000
```

### Управление балансом

#### Установить баланс

```
bank.setbalance [steamId] [сумма]
```

Устанавливает точный баланс для указанного игрока. Текущий баланс будет полностью заменён.

**Пример:**
```
bank.setbalance 76561198000000000 5000
```

#### Пополнить баланс

```
bank.deposit [steamId] [сумма]
```

Добавляет указанную сумму к текущему балансу игрока.

**Пример:**
```
bank.deposit 76561198000000000 1000
```

#### Снять с баланса

```
bank.withdraw [steamId] [сумма]
```

Вычитает указанную сумму из баланса игрока.

**Пример:**
```
bank.withdraw 76561198000000000 500
```

#### Перевод между игроками

```
bank.transfer [steamId отправителя] [steamId получателя] [сумма]
```

Принудительно переводит средства от одного игрока другому.

**Пример:**
```
bank.transfer 76561198000000001 76561198000000002 300
```

### Очистка данных

```
bank.wipe
```

Полностью очищает все данные банкоматов. Используйте с осторожностью.

### Конвертация данных

#### Из Economics

```
bank.convert.economics
```

Конвертирует балансы игроков из плагина Economics в Bank System.

#### Из ServerRewards

```
bank.convert.serverrewards
```

Конвертирует очки игроков из плагина ServerRewards в Bank System.

:::danger Важно
Конвертация данных -- необратимая операция. Рекомендуется сделать резервную копию данных перед выполнением.
:::

## Интерфейсные операции

Следующие действия выполняются через графический интерфейс (без чат-команд):

| Действие | Где доступно | Описание |
|---|---|---|
| Создание карты | Интерфейс банка | Создать новую банковскую карту |
| Пополнение | Банкомат | Обменять банкноты на электронные средства |
| Снятие | Банкомат | Получить банкноты в инвентарь |
| Перевод | Банкомат / Интерфейс банка | Отправить деньги другому игроку |
| Настройка комиссий | Банкомат (владелец) | Установить % комиссии за пополнение/снятие |
| Ремонт банкомата | Банкомат (владелец) | Восстановить состояние за ресурсы |
| Вывод накопленных комиссий | Банкомат (владелец) | Забрать заработанные комиссии на свой счёт |

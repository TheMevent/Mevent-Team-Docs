# Necronomicon

## Что делает этот плагин?

**Necronomicon** добавляет на сервер систему книг чертежей. Игрок может создать книгу (Necronomicon), в которую записываются все его изученные чертежи. Книгу можно передать другому игроку, который сможет изучить все содержащиеся в ней чертежи через стол исследования. Это позволяет делиться чертежами между игроками, продавать их или передавать тиммейтам.

## Ключевые возможности

- Создание книги с записью всех изученных чертежей игрока
- Изучение чертежей из книги через стол исследования
- Стоимость изучения настраивается через экономику (Economics, ServerRewards или предметы)
- Визуальный интерфейс для выбора чертежей
- Опция удаления чертежей у создателя книги
- Автоматическая очистка данных при вайпе
- Интеграция с Notify для уведомлений

## Требования

| Зависимость | Обязательность | Описание |
|---|---|---|
| Oxide / uMod / Carbon | Обязательно | Совместим с Oxide, uMod и Carbon |
| **Notify** | По желанию | Красивые уведомления |
| Плагин экономики | По желанию | Для стоимости создания книги |

## Быстрый старт

1. Скопируйте файл `Necronomicon.cs` в папку `oxide/plugins/`.
2. Перезагрузите сервер или выполните `oxide.reload Necronomicon`.
3. Выдайте книгу игроку: `necronomicon.give ник`.
4. Игрок помещает книгу в стол исследования и изучает чертежи.

# Referrals

## Что делает этот плагин?

**Referrals** -- это реферальная система, которая позволяет игрокам приглашать других игроков на сервер через промокоды. Каждый игрок получает уникальный промокод, которым может поделиться. Когда другой игрок активирует промокод, оба получают награды. Чем больше игроков приглашено, тем больше наград получает реферер.

## Ключевые возможности

- Автоматическая генерация уникальных промокодов для каждого игрока
- Настраиваемые награды за определённое количество приглашений (1, 2, 5, 7, 10...)
- Награды для игрока, активировавшего промокод (опционально)
- Визуальный интерфейс с отображением промокода, приглашённых игроков и прогресса наград
- Активация промокода через чат или интерфейс
- Минимальное время игры для активации промокода (защита от мультиаккаунтов)
- Блокировка наград после вайпа, во время рейда и комбата
- Автоматический вайп данных (опционально)
- Поддержка нескольких языков (EN, RU)
- Интеграция с Notify, NoEscape

## Требования

| Зависимость | Обязательность | Описание |
|---|---|---|
| **ImageLibrary** | Обязательно | Необходима для загрузки изображений в интерфейсе |
| Notify | Опционально | Для отправки графических уведомлений |
| NoEscape | Опционально | Для блокировки наград во время рейда/комбата |
| Oxide / uMod / Carbon | Обязательно | Совместим с Oxide, uMod и Carbon |

## Быстрый старт

1. Установите плагин **ImageLibrary**, если он ещё не установлен.
2. Скопируйте файл `Referrals.cs` в папку `oxide/plugins/` вашего сервера.
3. Перезагрузите сервер или выполните команду `oxide.reload Referrals`.
4. Зайдите на сервер и введите в чат `/ref` для открытия интерфейса.
5. Скопируйте свой промокод и поделитесь с друзьями.
6. Другой игрок может активировать промокод командой `/promo <код>`.

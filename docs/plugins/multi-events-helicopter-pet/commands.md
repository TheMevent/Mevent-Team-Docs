# Команды и права

## Команды

### Команды администратора

| Команда | Описание | Право |
|---|---|---|
| `helicopterpet.start` | Запустить ивент Helicopter Pet | `MultiEventsHelicopterPet.start` |
| `helicopterpet.stop` | Остановить текущий ивент | `MultiEventsHelicopterPet.stop` |

:::info Примечание
Ивент также может быть запущен автоматически через систему MultiEvents по расписанию.
:::

### Консольные команды (UI)

| Команда | Описание |
|---|---|
| `UI_HelicopterPet hide` | Скрыть панель ивента с экрана (вызывается через UI) |

## Права (Permissions)

| Право | Описание |
|---|---|
| `MultiEventsHelicopterPet.start` | Право на запуск ивента |
| `MultiEventsHelicopterPet.stop` | Право на остановку ивента |

### Настройка прав

```
oxide.grant group admin MultiEventsHelicopterPet.start
oxide.grant group admin MultiEventsHelicopterPet.stop
```

## Команды игроков

У обычных игроков нет специальных команд. Участие в ивенте автоматическое -- достаточно привлечь внимание вертолёта во время активного ивента.

## Механика начисления очков

Очки начисляются в зависимости от состояния ИИ вертолёта по отношению к игроку:

| Состояние вертолёта | Очки |
|---|---|
| **Орбита** (ORBIT) -- вертолёт кружит вокруг игрока | 1 очко |
| **Атака** (STRAFE) -- вертолёт атакует игрока | 2 очка |

:::warning Внимание
Привлечение внимания вертолёта опасно -- он будет стрелять по игроку! Игрокам нужно балансировать между набором очков и выживанием.
:::

## Завершение ивента

Ивент завершается, когда:
- Истекает время ивента
- Вертолёт уничтожен (здоровье снижено до 0)
- Администратор останавливает ивент командой `helicopterpet.stop`

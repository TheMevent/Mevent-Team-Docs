# Установка Kits

## Шаг 1: Установите зависимости

Перед установкой Kits необходимо установить обязательный плагин:

- **ImageLibrary** — загрузите и поместите в папку плагинов. Без него интерфейс наборов не будет работать.

Опциональные плагины для расширенного функционала:

| Плагин | Для чего нужен |
|---|---|
| ServerPanel | Интеграция с серверным меню |
| CopyPaste | Выдача построек через наборы |
| Notify / UINotify | Красивые всплывающие уведомления |
| NoEscape | Блокировка наборов во время рейда/боя |

## Шаг 2: Загрузите плагин

1. Скопируйте файл плагина в папку:
 - **Oxide**: `server/НАЗВАНИЕ_СЕРВЕРА/oxide/plugins/`
 - **Carbon**: `server/НАЗВАНИЕ_СЕРВЕРА/carbon/plugins/`

2. Плагин загрузится автоматически. Если сервер уже запущен, используйте:
 ```
 o.reload Kits
 ```

3. После первой загрузки будет создан конфигурационный файл:
 - `oxide/config/Kits.json` (или `carbon/config/Kits.json`)

## Шаг 3: Настройте основные параметры

Откройте конфигурационный файл и настройте:

### Команды
По умолчанию плагин регистрирует команды `/kit` и `/kits`. Вы можете изменить их:
```json
"Commands": ["kit", "kits", "набор"]
```

### Экономика
Настройте систему оплаты для платных наборов:
- **Plugin** — использовать плагин экономики (Economics, ServerRewards и др.)
- **Item** — оплата внутриигровыми предметами (например, скрап)

### Автовайп
Включите `Automatic wipe on wipe: true`, если хотите сбрасывать данные игроков при каждом вайпе сервера.

## Шаг 4: Создайте первый набор

1. Зайдите в игру с правами администратора
2. Введите команду `/editkit`
3. В визуальном редакторе создайте новый набор:
 - Укажите название (уникальный идентификатор)
 - Добавьте предметы из инвентаря
 - Настройте кулдаун, стоимость и права доступа
4. Сохраните набор

## Шаг 5: Выдайте права доступа

Выдайте необходимые права игрокам или группам:

```
oxide.grant group default kits.наборимя
oxide.grant user <steamid> Kits.admin
```

## Шаг 6: Проверка

Зайдите за обычного игрока (без прав администратора) и проверьте:
1. Команда `/kit` открывает интерфейс
2. Доступные наборы отображаются корректно
3. Получение набора работает

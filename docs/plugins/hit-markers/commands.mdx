# Команды и права

## Команды чата

| Команда | Описание | Право |
|---|---|---|
| `/marker` или `/hits` | Открыть окно настроек маркеров попаданий | Настраиваемое (по умолчанию для всех) |

:::info Примечание
Команды настраиваются в конфигурации в параметре `"Commands"`. По умолчанию: `["marker", "hits"]`.
:::

## Настройки в интерфейсе

Через интерфейс игрок может управлять следующими параметрами:

| Настройка | Описание |
|---|---|
| Text | Включить/отключить числовое отображение урона |
| Icon | Включить/отключить иконку попадания (с изменением цвета при хедшоте) |
| Health Line | Включить/отключить полоску здоровья противника |
| Buildings | Включить/отключить отображение урона по строениям |
| Шрифт | Выбор шрифта для числового урона |
| Размер шрифта | Настройка размера шрифта |

## Права (Permissions)

| Право | Описание |
|---|---|
| Настраиваемое | Общее право для использования маркеров (задаётся в конфигурации, пустое -- для всех) |
| Права шрифтов | Индивидуальные права для каждого шрифта |
| Права режимов | Индивидуальные права для каждого режима (Text, Icon, Health Line, Buildings) |

:::tip Совет
Если поле `Permission` оставить пустым, функция будет доступна всем игрокам.
:::

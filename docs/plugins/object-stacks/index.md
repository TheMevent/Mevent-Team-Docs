# Object Stacks

## Что делает этот плагин?

**Object Stacks** позволяет игрокам ставить размещаемые объекты (ящики, бочки, генераторы, печи, батареи и др.) друг на друга, создавая вертикальные стопки. Игрок смотрит на уже размещённый объект и нажимает правую кнопку мыши с таким же предметом в руках -- новый объект ставится поверх существующего. Максимальное количество объектов в стопке определяется правами доступа.

## Ключевые возможности

- Стакание объектов друг на друга правой кнопкой мыши
- Настраиваемый список поддерживаемых объектов с индивидуальными смещениями
- Система прав для ограничения количества объектов в стопке
- Ограничение стакания только в зоне шкафа (TC)
- Чёрный список скинов для запрета стакания определённых объектов
- Работа на буксирах (Tugboat)
- Автоматическое отслеживание стабильности и столкновений
- Логирование в консоль и файл

## Требования

| Зависимость | Обязательность | Описание |
|---|---|---|
| Oxide / uMod / Carbon | Обязательно | Совместим с Oxide, uMod и Carbon |

## Быстрый старт

1. Скопируйте файл `ObjectStacks.cs` в папку `oxide/plugins/`.
2. Перезагрузите сервер или выполните `oxide.reload ObjectStacks`.
3. Выдайте право для стакания: `oxide.grant group default objectstacks.box_wooden.use`.
4. Возьмите ящик в руки, наведите на размещённый ящик и нажмите правую кнопку мыши.

# Конфигурация

Конфигурационный файл создаётся автоматически при первом запуске плагина. Он находится по пути `oxide/config/Stacks.json`.

## Параметры конфигурации

```json
{
 "Commands": ["stacks"],
 "Work with Notify?": true,
 "Work with LangAPI?": true,
 "Changing multiplies in containers using a hammer": false,
 "Default Multiplier for new containers": 1.0,
 "Disallow overstack items?": true,
 "Blocked List": {
 "Items": ["item", "short name"],
 "Skins": [111111111111, 222222222222]
 }
}
```

## Описание параметров

### Commands

Список чат-команд для открытия панели управления. По умолчанию: `["stacks"]`.

Вы можете добавить дополнительные алиасы:
```json
"Commands": ["stacks", "stack", "st"]
```

### Work with Notify?

Включает интеграцию с плагинами Notify/UINotify для отправки красивых уведомлений. Если плагин уведомлений не установлен, сообщения будут отправлены в чат.

- `true` -- использовать Notify (по умолчанию)
- `false` -- всегда отправлять сообщения в чат

### Work with LangAPI?

Включает интеграцию с плагином LangAPI для автоматической локализации названий предметов.

- `true` -- использовать LangAPI (по умолчанию)
- `false` -- использовать стандартные английские названия

### Changing multiplies in containers using a hammer

Позволяет администраторам изменять множитель контейнера, ударив по нему киянкой (hammer).

- `true` -- включить изменение через киянку
- `false` -- выключено (по умолчанию)

### Default Multiplier for new containers

Стандартный множитель стака для новых контейнеров. Значение `1.0` означает стандартные размеры стаков.

**Примеры:**
- `1.0` -- стандартные стаки
- `2.0` -- двойные стаки во всех контейнерах
- `0.5` -- половинные стаки

### Disallow overstack items?

Запрещает перемещение предметов с количеством, превышающим максимальный стак, в инвентарь игрока.

- `true` -- оверстак запрещён, предметы автоматически разделяются (по умолчанию)
- `false` -- разрешить перемещение оверстакнутых предметов

### Blocked List

Список предметов и скинов, для которых не будут применяться изменения стаков.

#### Items

Массив shortname-ов предметов, исключённых из системы стаков:

```json
"Items": ["explosive.timed", "ammo.rocket.basic"]
```

#### Skins

Массив ID скинов, исключённых из системы стаков:

```json
"Skins": [123456789, 987654321]
```

## Файлы данных

Плагин сохраняет данные в двух файлах:

| Файл | Описание |
| ---- | -------- |
| `oxide/data/Stacks/Items.json` | Настройки стаков для каждого предмета |
| `oxide/data/Stacks/Containers.json` | Настройки множителей для контейнеров |

:::warning Важно
Не редактируйте файлы данных вручную при работающем сервере. Используйте панель управления или консольные команды.
:::

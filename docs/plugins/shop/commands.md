# Команды Shop

## Команды игроков

### Открытие магазина

```
/shop
```

Открывает главный интерфейс магазина. Игрок может:
- Просматривать категории и товары
- Покупать предметы
- Продавать предметы из инвентаря
- Переводить деньги другим игрокам
- Добавлять товары в избранное
- Использовать повторную покупку

:::info Информация
По умолчанию доступны команды `/shop` и `/shops`. Можно настроить свои команды в конфигурации.
:::

### Взаимодействие с NPC-магазином

Подойдите к NPC-продавцу и нажмите кнопку взаимодействия (`E`). Откроется интерфейс магазина с категориями, привязанными к этому NPC.

### Взаимодействие с торговым автоматом

Подойдите к настроенному торговому автомату и нажмите кнопку взаимодействия (`E`). Откроется интерфейс магазина с привязанными категориями.

## Команды администратора

:::caution Внимание
Все административные команды требуют соответствующих прав доступа.
:::

### Установка шаблона магазина

```
/shop.install
```

**Требуемое право:** `shop.admin`

Запускает процесс установки шаблона магазина. Шаблон содержит предустановленные категории и товары с ценами. Можно выбрать из нескольких доступных шаблонов.

**Пример использования с указанием индекса:**
```
/shop.install 0 0
```

Первый параметр -- индекс шаблона, второй -- индекс шаблона в меню.

:::danger Важно
Установка шаблона может перезаписать существующие данные магазина. Убедитесь, что вы сделали резервную копию перед выполнением.
:::

### Привязка к торговому автомату

```
/shop.setvm
```

**Требуемое право:** `shop.setvm`

Привязывает магазин к торговому автомату, на который вы смотрите. После привязки при взаимодействии с этим автоматом будет открываться интерфейс магазина.

**Как использовать:**
1. Разместите торговый автомат
2. Встаньте перед ним и наведите на него прицел
3. Введите `/shop.setvm`
4. Настройте категории в конфигурации

### Привязка к NPC

```
/shop.setnpc
```

**Требуемое право:** `shop.setnpc`

Привязывает магазин к NPC, на которого вы смотрите.

**Как использовать:**
1. Создайте NPC на сервере
2. Наведите на него прицел
3. Введите `/shop.setnpc`
4. Настройте категории в конфигурации

### Редактирование товаров

Администраторы с правом `shop.admin` могут редактировать товары прямо в игровом интерфейсе:
- Изменять цены покупки и продажи
- Устанавливать лимиты покупок (общие и ежедневные)
- Включать/отключать товары
- Настраивать изображения

Для входа в режим редактирования откройте магазин с правами администратора -- кнопки редактирования появятся автоматически.

## Консольные команды

| Команда | Описание |
|---|---|
| `shop.install` | Установить шаблон магазина (из серверной консоли) |

## Интерфейсные действия

Следующие действия выполняются через графический интерфейс:

| Действие | Описание |
|---|---|
| Покупка | Выберите товар, укажите количество, подтвердите покупку |
| Продажа | Перейдите на вкладку продажи, выберите предмет из инвентаря |
| Перевод | Откройте раздел переводов, укажите игрока и сумму |
| Избранное | Нажмите на иконку звезды рядом с товаром |
| Повторная покупка | Перейдите в раздел истории покупок (если включено) |
| Смена валюты | Выберите валюту из списка доступных (если настроено несколько) |
| Редактирование (админ) | Нажмите на кнопку редактирования рядом с товаром |

## Управление правами

### Выдача прав через консоль

```
oxide.grant user НикИгрока shop.admin
oxide.grant user НикИгрока shop.setvm
oxide.grant user НикИгрока shop.setnpc
oxide.grant group vip shop.vip
oxide.grant group default shop.use
```

### Снятие прав

```
oxide.revoke user НикИгрока shop.admin
oxide.revoke group vip shop.vip
```

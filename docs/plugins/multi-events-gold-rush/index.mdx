# Gold Rush

## Описание

**Gold Rush** -- это режим расширения для плагина MultiEvents, при котором во время добычи руды и деревьев игроки получают шанс на дополнительные предметы. Каждый удар по ресурсу (руда или дерево) даёт шанс получить бонусный предмет из настраиваемого списка. Руда и деревья имеют отдельные таблицы наград с разными шансами.

## Ключевые возможности

- Бонусные предметы при добыче руды и деревьев
- Раздельные таблицы наград для руды и деревьев
- Настраиваемые шансы выпадения для каждого предмета
- Система множителей шансов по правам доступа (VIP)
- Поддержка Notify / UINotify для уведомлений
- Настраиваемая длительность ивента
- UI-панель с таймером обратного отсчёта и описанием ивента

## Требования

| Зависимость | Обязательность | Описание |
|---|---|---|
| **MultiEvents** | Обязательно | Основной плагин системы событий |
| **ImageLibrary** | Обязательно | Загрузка и отображение изображений в интерфейсе |
| Notify / UINotify | Опционально | Для отправки визуальных уведомлений игрокам |

## Быстрый старт

1. Убедитесь, что плагины **MultiEvents** и **ImageLibrary** установлены и работают.
2. Скопируйте файл `MultiEventsGoldRush.cs` в папку `oxide/plugins/` вашего сервера.
3. Плагин автоматически зарегистрируется в системе MultiEvents.
4. Настройте таблицы наград для руды и деревьев в конфигурации.
5. Запустите ивент через MultiEvents или командой:
 ```
 goldrush.start
 ```
6. Начните добывать руду или деревья для получения бонусных предметов.

# Команды и права

## Команды чата

| Команда | Описание | Право |
|---|---|---|
| `/stats` | Открыть профиль статистики | `statistics.use` |
| `/statistics` | Открыть профиль статистики (альтернативная) | `statistics.use` |
| `/leaders` | Открыть таблицу лидеров | `statistics.use` |
| `/leaderboard` | Открыть таблицу лидеров (альтернативная) | `statistics.use` |
| `/stats <steamid>` | Посмотреть профиль другого игрока | `statistics.use` + доп. право |

:::tip Совет
Команды настраиваются в конфигурации: `Statistics Commands` для профиля и `Leaderboard Commands` для таблицы лидеров.
:::

## Консольные команды

Внутренние команды интерфейса обрабатываются через консольную команду `UI_Stats`.

## Права (Permissions)

| Право | Описание | По умолчанию |
|---|---|---|
| `statistics.use` | Доступ к статистике и таблице лидеров | Не выдано |
| `statistics.hide` | Скрытие из таблицы лидеров | Не выдано |

### Дополнительные настраиваемые права

| Параметр конфигурации | Описание |
|---|---|
| Permission to open player profile when clicked on leaderboard | Право для просмотра профиля при клике на игрока в таблице лидеров |
| Permission to view the other player's profile | Право для просмотра профиля другого игрока через команду |
| Permission to collect statistics | Право для сбора статистики (если включено ограничение) |

### Настройка прав

```
oxide.grant group default statistics.use
oxide.grant group admin statistics.hide
```

## Отображаемая статистика

### Профиль игрока

- Убийства (Kills)
- Смерти (Deaths)
- K/D-соотношение
- Любимое оружие
- Общий счёт (Score)
- Время на сервере
- Добытые ресурсы (с разбивкой по типам)

### Таблица лидеров

- Рейтинг игроков по очкам
- Топ-позиции с возможностью получения наград
- Поиск игрока по имени

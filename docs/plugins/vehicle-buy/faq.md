# Часто задаваемые вопросы (FAQ)

## 1. Какие виды транспорта поддерживает плагин?

Плагин поддерживает 19+ типов транспорта: миникоптер, транспортный вертолёт, боевой вертолёт, модульные машины (2/3/4 модуля), буксир, лодка, RHIB, соло- и дуо-подлодки, лошадь, снегоход, Tomaha, воздушный шар, переработчик, педальный велосипед, мотоцикл и мотоцикл с коляской. Вы можете добавлять свои виды транспорта через конфигурацию.

## 2. Как добавить новый вид транспорта?

Добавьте новую запись в секцию `Vehicles` конфигурации. Укажите уникальный ключ, название, путь к объекту (prefab), изображение, цену и другие параметры. Полный список настроек описан в документации по конфигурации.

## 3. Как настроить оплату разными валютами?

Для каждого транспорта можно задать свой тип валюты через параметр `Currency`:
- `0` -- внутриигровой предмет (указывается shortname)
- `1` -- Economics
- `2` -- Server Rewards

Также можно использовать BankSystem.

## 4. Как работает система отзыва (recall)?

Игрок вводит команду `/callback <ключ>`, и купленный транспорт телепортируется к его местоположению. Опционально можно настроить стоимость отзыва. Отзыв невозможен, если транспорт занят другим игроком.

## 5. Как работает подбор молотком?

Если для транспорта `Pickup type` установлен в `1`, игрок может подобрать его ударами молотка (3 удара за 2 секунды). Транспорт превращается в предмет в инвентаре. Способ подбора можно изменить на команду (`/pickup`), установив значение `0`.

## 6. Как настроить торговые автоматы в мирных зонах?

В секции `Vending machines` конфигурации включите автоматы для нужных зон (Bandit Camp, Outpost, Fishing Village). Для каждого автомата задайте список товаров с указанием ключа транспорта, предмета оплаты и цены.

## 7. Можно ли ограничить количество покупок одного транспорта?

Да. Параметр `Maximum number of purchases` позволяет установить лимит покупок для каждого транспорта. Значение `0` означает отсутствие ограничений.

## 8. Как работает кулдаун на покупку?

Параметр `Cooldown to buy (in seconds)` задаёт время ожидания между покупками. Например, значение `300` означает, что после покупки транспорта игрок должен подождать 5 минут перед следующей покупкой этого же транспорта.

## 9. Данные сохраняются при вайпе?

По умолчанию данные сохраняются. Чтобы очищать данные при смене карты, установите параметр `Wipe on new map save` в `true`.

## 10. Как отключить урон купленному транспорту?

Установите параметр `Disable vehicles damage?` в `true`. Это сделает весь купленный через плагин транспорт неуязвимым.

## 11. Что делать, если транспорт не спавнится?

Убедитесь, что:
- Рядом с игроком достаточно свободного места (дистанция спавна настраивается)
- Инвентарь игрока не полон (при подборе/выдаче)
- Для водного транспорта игрок находится рядом с водой

Если транспорт не может быть размещён, он возвращается в инвентарь игрока в виде предмета.

## 12. Плагин работает с Carbon?

Да, плагин полностью совместим с Carbon. При использовании Carbon автоматически используется встроенная система изображений вместо ImageLibrary.

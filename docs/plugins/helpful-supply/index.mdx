# Helpful Supply

## Что делает этот плагин?

**Helpful Supply** позволяет создавать кастомные сигнальные шашки (Supply Signal) с уникальным содержимым аирдропов. Каждый тип сигнальной шашки имеет собственный скин, название и список лута. Например, можно создать "Медицинскую сигнальную шашку" с аптечками или "Оружейную сигнальную шашку" с оружием. Также поддерживается выпадение кастомных шашек из контейнеров на карте.

## Ключевые возможности

- Создание неограниченного количества типов сигнальных шашек с уникальным скином
- Настройка содержимого аирдропа для каждого типа (предметы, количество, шанс выпадения)
- Контроль доступа к аирдропу (все игроки, тиммейты, друзья, кланы)
- Настройка скорости сброса (скорость падения, высота самолёта)
- Выпадение кастомных шашек из бочек и ящиков на карте
- Консольная команда для выдачи шашек игрокам
- Интеграция с плагинами Friends, Clans и Loottable

## Требования

| Зависимость | Обязательность | Описание |
|---|---|---|
| Oxide / uMod / Carbon | Обязательно | Совместим с Oxide, uMod и Carbon |
| **Friends** | По желанию | Доступ к аирдропу для друзей |
| **Clans** | По желанию | Доступ к аирдропу для членов клана |
| **Loottable** | По желанию | Интеграция с таблицами лута |

## Быстрый старт

1. Скопируйте файл `HelpfulSupply.cs` в папку `oxide/plugins/`.
2. Перезагрузите сервер или выполните `oxide.reload HelpfulSupply`.
3. Настройте типы сигнальных шашек в `oxide/config/HelpfulSupply.json`.
4. Выдайте шашку игроку: `hsupply.give ник heal 1`.

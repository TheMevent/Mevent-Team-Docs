# Конфигурация Crafts

Файл конфигурации расположен по пути: `oxide/config/Crafts.json`

## Основные параметры

| Параметр | По умолчанию | Описание |
|---|---|---|
| Commands | `["craft", "crafts"]` | Чат-команды для открытия меню |
| Close UI when reusing a command? | `false` | Закрывать интерфейс при повторном вводе команды |
| Work with Notify? | `true` | Использовать Notify для уведомлений |
| Work with LangAPI? | `true` | Использовать LangAPI для мультиязычности |
| Permission | `""` | Право доступа к меню (пустое -- для всех) |

## Настройки экономики

| Параметр | Описание |
|---|---|
| Show | Показывать баланс в интерфейсе |
| Type | Тип экономики: `Plugin` или `Item` |
| Plugin name | Название плагина экономики (например, `Economics`) |
| Balance add hook | Функция пополнения баланса |
| Balance remove hook | Функция списания баланса |
| Balance show hook | Функция показа баланса |
| ShortName | ShortName предмета (для типа Item) |
| Skin | Скин предмета (для типа Item) |

## Категории

Каждая категория содержит:

| Параметр | Описание |
|---|---|
| Enabled | Включена ли категория |
| Permission | Право доступа к категории (пустое -- для всех) |
| Title | Название категории |
| Color | Цвет фона категории (HEX + прозрачность) |
| Crafts | Список рецептов крафта в категории |

## Настройки рецепта

| Параметр | Описание |
|---|---|
| Enabled | Включён ли рецепт |
| Image | URL изображения рецепта |
| Title | Название рецепта |
| Description | Описание рецепта |
| Permission | Право доступа к рецепту |
| Type | Тип крафта: `Command`, `Vehicle`, `Item`, `Recycler`, `ModularCar` |
| Prefab | Путь к префабу (для транспорта) |
| CmdToGive | Команда для выдачи |
| WorkbenchLevel | Требуемый уровень верстака: `None`, `One`, `Two`, `Three` |
| ReturnItem | Возвращать ли предмет при поднятии |
| CustomizeRotation | Разрешить поворот при размещении |

## Типы крафтов

| Тип | Описание |
|---|---|
| `Command` | Выполняет команду при крафте |
| `Vehicle` | Спавнит транспортное средство |
| `Item` | Создаёт предмет в инвентаре |
| `Recycler` | Размещает переработчик |
| `ModularCar` | Спавнит модульный автомобиль |

## Кастомный верстак

Плагин поддерживает размещение кастомных верстаков, которые можно привязать к определённым категориям и правам. Настройка выполняется в секции `"Custom Workbench"`.

:::tip Совет
Используйте право `crafts.bypass` для администраторов, чтобы они могли крафтить без ресурсов и верстака.
:::

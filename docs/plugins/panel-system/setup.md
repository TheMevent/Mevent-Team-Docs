# Установка Panel System

Это руководство проведёт вас через процесс установки информационной панели на вашем сервере.

## Шаг 1: Установите зависимости

### ImageLibrary (обязательно)

1. Скачайте **ImageLibrary** с [umod.org](https://umod.org/plugins/image-library).
2. Поместите файл `ImageLibrary.cs` в папку `oxide/plugins/`.
3. Убедитесь, что плагин загрузился без ошибок.

:::warning Важно
Без ImageLibrary плагин Panel System **не будет работать**. В консоли появится ошибка: `IMAGE LIBRARY IS NOT INSTALLED.`
:::

### Дополнительные зависимости (по желанию)

| Плагин | Назначение |
|---|---|
| **Notify** | Красивые уведомления вместо сообщений в чате |
| **UINotify** | Альтернатива Notify |
| **WipeBlock** | Отображение статуса вайп-блока на панели событий |
| **Economics** | Отображение баланса валюты на панели |
| **ServerRewards** | Отображение баланса очков наград |

## Шаг 2: Установите плагин

1. Скопируйте файл `PanelSystem.cs` в папку `oxide/plugins/`.
2. Плагин загрузится автоматически.
3. Проверьте консоль на наличие ошибок.

Файл конфигурации будет создан автоматически:
```
oxide/config/PanelSystem.json
```

Файл данных будет создан по пути:
```
oxide/data/PanelSystem/Main.json
```

## Шаг 3: Проверьте работу

1. Зайдите на сервер.
2. Вы должны увидеть информационную панель в правом верхнем углу экрана.
3. На панели должны отображаться:
 - Количество игроков онлайн
 - Текущее игровое время
 - Индикаторы событий (если настроены)

:::info Примечание
При первом запуске используются настройки по умолчанию. По умолчанию включены: онлайн, время, вертолёт, танк, аирдроп и CH47.
:::

## Шаг 4: Настройте элементы панели

Откройте конфигурацию `oxide/config/PanelSystem.json`.

### Включение/отключение элементов

Для каждого элемента панели есть параметр `"Enabled"`:

- **Панель онлайна** -- секция "Players Settings"
- **Панель спящих** -- секция "Sleepers Settings" (по умолчанию отключена)
- **Панель времени** -- секция "Time Settings"
- **Кнопки** -- секция "Buttons Setting"
- **Логотип** -- секция "Logotype"
- **Кнопка настроек** -- секция "Settings Button"
- **События** -- каждое событие в массиве "Events"
- **Экономика** -- каждая валюта в массиве "Economics"

### Пример: включить панель спящих

В секции "Sleepers Settings" измените:
```json
"Enabled": true
```

## Шаг 5: Настройте быстрые кнопки

Быстрые кнопки позволяют игрокам выполнять команды одним кликом. Настройте их в секции "Buttons Setting":

```json
"Buttons": [
 {
 "Enabled": true,
 "Image": "https://ссылка-на-иконку.png",
 "Permission": "",
 "Lang Key (oxide/lang/**/PanelSystem.json)": "BtnShop",
 "Command": "chat.say /shop"
 }
]
```

- **Image** -- URL иконки кнопки
- **Permission** -- право для видимости (пустое -- для всех)
- **Lang Key** -- ключ в языковом файле для текста кнопки
- **Command** -- команда, выполняемая при нажатии

:::tip Совет
Для выполнения чат-команды используйте формат `chat.say /команда`. Для консольной команды просто укажите её имя.
:::

## Шаг 6: Настройте события

Каждое событие в массиве "Events" можно включить или отключить:

```json
{
 "Enabled": true,
 "Image": "https://ссылка-на-иконку.png",
 "Permission": "",
 "Type": "Helicopter",
 ...
}
```

Индикатор события автоматически меняет цвет при появлении/исчезновении объекта на карте.

## Шаг 7: Настройте функцию скрытия (по желанию)

Чтобы игроки могли полностью скрывать панель, включите функцию в секции "Hide Settings":

```json
"Hide Settings": {
 "Enabled": true,
 "Command": "panel",
 "Show Logo?": true
}
```

После этого игроки смогут использовать:
- `/panel hide` -- скрыть панель
- `/panel show` -- показать панель

## Шаг 8: Перезагрузите плагин

После изменения конфигурации:
```
oxide.reload PanelSystem
```

## Обновление плагина

При обновлении:

1. Замените файл `PanelSystem.cs` новой версией.
2. Плагин автоматически обновит конфигурацию.
3. В консоли появятся сообщения:
 ```
 Config update detected! Updating config values...
 Config update completed!
 ```
4. Новые события и настройки будут добавлены автоматически.

:::note Примечание
Плагин сохраняет ваши пользовательские настройки при обновлении. Новые параметры добавляются со значениями по умолчанию.
:::

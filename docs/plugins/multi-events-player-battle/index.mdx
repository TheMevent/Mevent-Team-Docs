# Player Battle

## Что делает этот режим?

**Player Battle** — это PvP-режим ивента для Multi Events, в котором игроки соревнуются за очки, убивая других игроков. За каждое убийство игрока начисляется одно очко. Игрок с наибольшим количеством фрагов к окончанию таймера получает награды.

## Ключевые возможности

- PvP-соревнование с автоматическим подсчётом убийств
- Защита от фарма очков: повторное убийство одного и того же игрока не засчитывается
- Опциональное отключение TruePve на время ивента
- Гибкая система наград для нескольких мест в таблице лидеров
- Поддержка уведомлений через Notify и UINotify
- Встроенный UI-виджет с таблицей лидеров и обратным отсчётом
- Поддержка наград через предметы, команды, наборы (Kit) и сторонние плагины
- Интеграция с инвентарём ивентов (Multi Events Inventory)

## Требования

| Зависимость | Обязательно? |
|---|---|
| **Multi Events** | Да |
| **ImageLibrary** | Да |
| Multi Events Inventory | Рекомендуется |
| Multi Events Leaderboard | Рекомендуется |
| Notify / UINotify | Опционально |

## Быстрый старт

1. Установите **Multi Events** на сервер
2. Поместите файл режима Player Battle в папку `oxide/plugins`
3. Перезагрузите сервер или загрузите плагин
4. Файл конфигурации будет создан автоматически
5. Настройте параметры PvP и награды в конфигурации
6. Ивент готов к запуску через Multi Events или вручную командой

## Как проходит ивент

1. Ивент запускается автоматически через Multi Events или вручную администратором
2. При необходимости отключается TruePve для PvP-сражений
3. Всем игрокам на сервере показывается UI-уведомление с описанием и таймером
4. Игроки убивают других игроков, зарабатывая очки за каждый фраг
5. Повторные убийства одного и того же игрока не засчитываются (если включена проверка)
6. В режиме реального времени обновляется таблица лидеров с текущими результатами
7. По истечении таймера ивент завершается, TruePve восстанавливается
8. Победители получают настроенные награды в соответствии с их местом

## Настройки

| Параметр | Описание | По умолчанию |
|---|---|---|
| Work with Notify? | Использовать ли плагин Notify для уведомлений | `true` |
| Event Duration | Длительность ивента в секундах | `600` |
| Disable TruePve | Отключить TruePve на время ивента | `true` |
| Check player kills | Проверять уникальность убийств (1 жертва = 1 очко) | `true` |
| Awards Settings | Настройки наград для каждого места | Предметы для 1-2 мест |

## Устранение неполадок

| Проблема | Решение |
|---|---|
| Ивент не запускается | Убедитесь, что Multi Events установлен и загружен |
| Игроки не могут наносить урон друг другу | Проверьте, включена ли опция Disable TruePve |
| Нет UI-уведомлений | Проверьте, установлен ли ImageLibrary |
| Очки не начисляются | Проверьте, что включена опция Check player kills и убийства уникальны |
| Награды не выдаются | Проверьте корректность настроек наград в конфигурации |

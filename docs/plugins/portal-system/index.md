# Portal System

## Что делает этот плагин?

**Portal System** добавляет на сервер систему порталов, которые игроки могут размещать в проёмах дверей и использовать для мгновенной телепортации между своими базами. Игрок покупает или получает предмет-портал, размещает его в дверном проёме, после чего связывает с другим порталом. Поддерживается настройка стоимости входа, кулдаунов, лимитов порталов и визуальных скинов.

## Ключевые возможности

- Размещение порталов в дверных проёмах
- Телепортация между связанными порталами
- Покупка порталов за ресурсы (через чат-команду)
- Ограничение количества порталов на игрока (с VIP-правами)
- Стоимость использования портала (ресурсы или экономика)
- Переименование порталов
- Шаринг порталов с тиммейтами, друзьями, кланом
- Визуальные скины порталов с разными смещениями
- Маркеры порталов на карте
- Кулдаун между телепортациями (с VIP-перезаписью)
- Полоска здоровья портала
- Интеграция с NoEscape, Clans, Economics, ServerRewards

## Требования

| Зависимость | Обязательность | Описание |
|---|---|---|
| **ImageLibrary** | По желанию | Иконки в интерфейсе |
| **Economics** | По желанию | Экономическая стоимость покупки |
| **ServerRewards** | По желанию | Стоимость через очки наград |
| **NoEscape** | По желанию | Блокировка телепортации при рейде |
| **Clans** | По желанию | Шаринг порталов для клана |
| Oxide / uMod / Carbon | Обязательно | Совместим с Oxide, uMod и Carbon |

## Быстрый старт

1. Скопируйте файл `PortalSystem.cs` в папку `oxide/plugins/`.
2. Перезагрузите сервер или выполните `oxide.reload PortalSystem`.
3. Выдайте право на использование: `oxide.grant group default portalsystem.use`.
4. Выдайте портал игроку: `giveportal ник 2`.
5. Игрок размещает порталы в дверных проёмах и связывает их через меню.

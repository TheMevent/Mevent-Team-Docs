# Конфигурация Car Sharing

Конфигурационный файл расположен по пути:
```
oxide/config/CarSharing.json
```

## Основные параметры

| Параметр | Тип | По умолчанию | Описание |
|---|---|---|---|
| Commands | массив строк | `["rent", "carsharing"]` | Чат-команды для открытия меню каршеринга |
| Permission | строка | `""` | Право доступа для использования плагина (пустое -- доступно всем) |
| Work with Notify? | bool | `true` | Использовать плагин Notify для уведомлений |

## Настройки экономики

| Параметр | Тип | Описание |
|---|---|---|
| Type | `Plugin` / `Item` | Тип валюты: плагин экономики или внутриигровой предмет |
| Plugin name | строка | Название плагина экономики (например, `Economics`) |
| Balance add hook | строка | Хук для добавления баланса |
| Balance remove hook | строка | Хук для снятия баланса |
| Balance show hook | строка | Хук для отображения баланса |
| ShortName | строка | Shortname предмета (если тип -- предмет) |
| Display Name | строка | Отображаемое имя предмета (пустое -- по умолчанию) |
| Skin | число | ID скина предмета |

## Предмет вызова транспорта

Игрок получает специальный предмет, который позволяет вызвать арендованную машину.

| Параметр | Тип | По умолчанию | Описание |
|---|---|---|---|
| Display Name | строка | `"RUSTSharing {carname}"` | Название предмета (поддерживается `{carname}` для подстановки имени машины) |
| ShortName | строка | `"box.wooden.large"` | Базовый предмет |
| Skin | число | `2591806406` | Скин предмета |

## Настройки машин

Каждая машина задаётся в массиве `Cars` и имеет следующие параметры:

| Параметр | Тип | Описание |
|---|---|---|
| Permission | строка | Право доступа к этой машине (пустое -- доступна всем) |
| Command to give | строка | Чат-команда для выдачи машины |
| Image | строка | URL изображения машины для меню |
| Name | строка | Название машины |
| Cost per minute | число | Стоимость аренды за минуту |
| Car class | строка | Класс машины (например, `Luxe`, `Comfort`) |
| Average Speed | строка | Отображаемая средняя скорость |

### Настройки модульной машины

| Параметр | Тип | Описание |
|---|---|---|
| CodeLock | bool | Установить кодовый замок |
| KeyLock | bool | Установить ключевой замок |
| Engine Parts Tier | число | Уровень деталей двигателя (1-3) |
| Fresh Water Amount | число | Количество пресной воды |
| Fuel Amount | число | Количество топлива при создании |
| Modules | массив строк | Модули машины (например: `vehicle.1mod.engine`, `vehicle.1mod.cockpit.armored`) |

## Настройки NPC-ботов

Боты настраиваются в секции `Bots`, где ключ -- это ID NPC-бота:

| Параметр | Тип | Описание |
|---|---|---|
| Car Classes | массив строк | Доступные классы машин (`*` -- все классы) |

### Пример конфигурации бота

```json
"Bots": {
 "5154454": {
 "Car Classes (* - all)": [
 "Luxe",
 "Comfort"
 ]
 }
}
```

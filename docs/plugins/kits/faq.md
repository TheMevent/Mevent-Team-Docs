# Часто задаваемые вопросы — Kits

### 1. Как создать новый набор?

Зайдите в игру с правами администратора и введите команду `/editkit`. Откроется визуальный редактор, в котором вы можете создать новый набор, добавить предметы, настроить кулдаун, стоимость и права доступа. Все изменения сохраняются автоматически.

### 2. Как сделать набор доступным только для VIP-игроков?

При создании набора в редакторе укажите право доступа (permission), например `kits.vip`. Затем выдайте это право нужной группе:
```
oxide.grant group vip kits.vip
```

### 3. Как настроить платный набор?

В редакторе наборов укажите стоимость набора. Валюта настраивается в конфигурации плагина — можно использовать Economics, ServerRewards или внутриигровые предметы (скрап, дерево и т.д.).

### 4. Можно ли выдавать наборы офлайн-игрокам?

Да, используйте команду `kits.givekit <steamid> <название> [кол-во]`. Это начислит использования набора игроку, и он сможет забрать его из меню наборов при следующем входе на сервер.

### 5. Как настроить автонабор при входе на сервер?

Добавьте название набора в список автонаборов в конфигурации:
```json
"Auto Kits": ["autokit", "autokit_vip", "autokit_premium"]
```
Наборы проверяются снизу вверх — игрок получит первый набор, на который у него есть права.

### 6. Данные наборов сбрасываются при вайпе?

По умолчанию данные не сбрасываются. Чтобы включить автоматический сброс, установите в конфигурации:
```json
"Automatic wipe on wipe": true
```
Вы также можете сбросить данные вручную командой `kits.reset data all`.

### 7. Как привязать наборы к NPC на карте?

В конфигурации добавьте ID NPC и список наборов:
```json
"NPC Kits": {
 "ID_ВАШЕГО_NPC": {
 "Description": "Бесплатные наборы",
 "Kits": ["starter", "daily"]
 }
}
```
При взаимодействии с NPC игрок увидит привязанные наборы.

### 8. Как разрешить получение определённых наборов во время рейд-блока?

Добавьте названия наборов в белый список NoEscape в конфигурации:
```json
"Whitelist for NoEscape": ["medical_kit", "emergency_kit"]
```

### 9. Как изменить внешний вид интерфейса?

Выполните из серверной консоли команду `kits.template` для просмотра доступных шаблонов. Например:
```
kits.template fullscreen rust content
```
Доступны полноэкранные и встроенные в меню шаблоны в различных стилях.

### 10. Плагин совместим с Carbon?

Да, плагин Kits полностью совместим как с Oxide (uMod), так и с Carbon. Никаких дополнительных настроек для работы с Carbon не требуется.

### 11. Как включить логирование в Discord?

В конфигурации найдите раздел "Discord Logging Settings" и настройте:
1. Установите `Enabled: true` для нужного типа (получение/выдача)
2. Укажите URL вебхука Discord
3. Настройте формат сообщения

### 12. Как удалить все наборы и начать с нуля?

Выполните команду `kits.reset kits` из серверной консоли. Это удалит все созданные наборы. Данные игроков при этом сохранятся — для их сброса используйте `kits.reset data all`.

# Operation Fortress

## Что делает этот плагин?

**Operation Fortress** добавляет на сервер масштабное PvP-событие захвата крепости. На карте появляется укреплённая база с NPC-охранниками, вышками и боссом-командором. Игроки должны активировать определённое количество вышек для захвата крепости. Победители получают награды (экономика, команды, предметы). Событие поддерживает автоматический запуск, маркеры на карте и интеграцию с PvE-системами.

## Ключевые возможности

- Автоматический запуск события по таймеру
- Настраиваемые пресеты NPC-охранников (экипировка, оружие, поведение)
- Несколько типов NPC: охранники территории, снайперы на вышках, босс-командор
- Система активации вышек для захвата крепости
- Фаза подготовки с обратным отсчётом
- Маркеры на карте (Vending + Radius)
- Система наград (Economics, ServerRewards, команды)
- Интеграция с PveMode для зон PvP/PvE
- Блокировка телепортации и команд во время события
- Пресеты зданий (RustEdit)
- Объявление победителя всем игрокам

## Требования

| Зависимость | Обязательность | Описание |
|---|---|---|
| **NpcSpawn** | Обязательно | Спавн NPC-охранников |
| **PveMode** | По желанию | Управление PvP/PvE зонами |
| **Notify** / **UINotify** | По желанию | Красивые уведомления |
| **BetterNpc** | По желанию | Интеграция с улучшенными NPC |
| **AlphaLoot** | По желанию | Управление лутом |
| Oxide / uMod / Carbon | Обязательно | Совместим с Oxide, uMod и Carbon |

## Быстрый старт

1. Установите **NpcSpawn** на сервер.
2. Скопируйте файл `OperationFortress.cs` в папку `oxide/plugins/`.
3. Перезагрузите сервер или выполните `oxide.reload OperationFortress`.
4. Настройте пресеты зданий и NPC в конфигурации.
5. Запустите событие: `operation.start`.

# Servers

## Что делает этот плагин?

**Servers** добавляет на сервер визуальный интерфейс для просмотра всех серверов вашей сети. Игроки открывают меню через чат-команду и видят карточки серверов с баннерами, описаниями, текущим онлайном и кнопкой быстрого подключения. Информация обновляется автоматически через настраиваемый интервал. Поддерживается интеграция с BattleMetrics для получения подробной информации.

## Ключевые возможности

- Визуальный список серверов с баннерами и описаниями
- Автоматическое обновление информации об онлайне
- Интеграция с BattleMetrics (название, описание, онлайн, размер карты)
- Кнопка быстрого подключения к серверу
- Настраиваемый интервал обновления информации
- Два шаблона отображения (V1, V2)
- Интеграция с ServerPanel
- Поддержка ImageLibrary для баннеров
- Совместимость с Oxide и Carbon

## Требования

| Зависимость | Обязательность | Описание |
|---|---|---|
| **ImageLibrary** | По желанию | Загрузка баннеров серверов |
| **ServerPanel** | По желанию | Интеграция с панелью сервера |
| Oxide / uMod / Carbon | Обязательно | Совместим с Oxide, uMod и Carbon |

## Быстрый старт

1. Скопируйте файл `Servers.cs` в папку `oxide/plugins/`.
2. Перезагрузите сервер или выполните `oxide.reload Servers`.
3. Настройте список серверов в `oxide/config/Servers.json`.
4. Выдайте право: `oxide.grant group default servers.use`.
5. Введите `/servers` в чат для просмотра списка.

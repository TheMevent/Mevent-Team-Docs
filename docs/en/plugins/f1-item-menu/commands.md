# Commands and Permissions

## Chat Commands

| Command | Description | Permission |
|---|---|---|
| `/f1menu` | Open the visual item menu | `f1itemmenu.f1` |

## Permissions

| Permission | Description |
|---|---|
| `f1itemmenu.f1` | Allows using the `/f1menu` command to open the item give menu |

:::warning Warning
This permission should only be granted to trusted administrators, as it allows giving any items in unlimited quantities.
:::

## Interface Controls

| Action | Description |
|---|---|
| Click on an item | Opens the quantity selection menu |
| Search field | Filters items by name |
| `<` / `>` buttons | Page navigation |
| `Close` button | Close the menu |

:::info Note
Items added to the blacklist (e.g., vehicles) are displayed in red and are unavailable for giving.
:::

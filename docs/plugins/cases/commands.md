# Команды Cases

## Команды для игроков

| Команда | Описание |
|---|---|
| `/cases` | Открывает главное меню кейсов |
| `/opencase` | Аналог команды `/cases` |

:::info Примечание
Команды настраиваются в конфигурации. Администратор может изменить их на любые другие (например, `/case`, `/open` и т.д.).
:::

### Как открыть кейс

1. Введите `/cases` в чат
2. Выберите кейс из списка
3. Убедитесь, что у вас достаточно валюты
4. Нажмите кнопку открытия
5. Наблюдайте за анимацией рулетки
6. Получите свой выигрыш!

## Команды для администраторов

### Выдача кейсов

| Команда | Описание |
|---|---|
| `givecase <steamid> <id_кейса> <кол-во>` | Выдать определённое количество кейсов конкретному игроку |
| `givecase all <id_кейса> <кол-во>` | Выдать кейс(ы) всем игрокам, находящимся на сервере |

:::tip Примеры
Выдать 5 кейсов (ID: 1) конкретному игроку:
```
givecase 76561198000000000 1 5
```

Выдать 1 кейс (ID: 2) всем игрокам:
```
givecase all 2 1
```
:::

### Параметры команды givecase

| Параметр | Описание |
|---|---|
| `steamid` | Steam ID игрока или `all` для всех игроков |
| `id_кейса` | Числовой ID кейса из конфигурации |
| `кол-во` | Количество выдаваемых кейсов |

:::caution Важно
Команда `givecase` доступна только администраторам и из серверной консоли. Обычные игроки не могут её использовать.
:::

## Внутриигровой редактор

Администраторы с правом `Cases.edit` могут редактировать кейсы и предметы через встроенный игровой интерфейс, без необходимости редактирования конфигурационного файла вручную.

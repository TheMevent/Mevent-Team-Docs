# Конфигурация

Файл конфигурации расположен по пути: `oxide/config/MultiEventsBotPurge.json`

## Основные параметры

| Параметр | По умолчанию | Описание |
|---|---|---|
| Work with Notify? | `true` | Использовать плагин Notify / UINotify для отправки уведомлений |
| Event Duration | `600` | Длительность ивента в секундах (по умолчанию 10 минут) |

## Таблица очков за ботов

Параметр `"Bots point"` определяет количество очков, начисляемых за уничтожение каждого типа NPC-бота.

| Тип бота (ShortPrefabName) | Очки по умолчанию | Описание |
|---|---|---|
| `scientistnpc_bradley` | 2 | Обычный учёный из танка Брэдли |
| `scientistnpc_bradley_heavy` | 3 | Тяжёлый учёный из танка Брэдли |

:::tip Совет
Вы можете добавить любые типы NPC-ботов в таблицу, используя их ShortPrefabName. Чем опаснее бот, тем больше очков рекомендуется назначать.
:::

## Настройки наград

Секция `"Awards Settings"` определяет награды для игроков по местам в рейтинге.

| Параметр | Описание |
|---|---|
| Put the reward in the MultiEventsInventory? | Выдавать награду через инвентарь MultiEventsInventory |
| Player Awards (top - award) | Список наград по местам (1 -- первое место, 2 -- второе и т.д.) |

### Параметры каждой награды

| Параметр | Описание |
|---|---|
| Award IDs from MultiEventsInventory | ID наград из инвентаря MultiEventsInventory |
| Awards | Список предметов-наград |
| Amount (Minimal / Maximum) | Количество наград для выдачи |

### Типы наград

Каждая награда может быть одного из типов:

| Тип | Описание |
|---|---|
| `Item` | Игровой предмет (указывается ShortName, количество, скин) |
| `Command` | Консольная команда (поддерживает %steamid% и %username%) |
| `Plugin` | Вызов хука другого плагина (например, Economics) |
| `Kit` | Выдача набора через плагин Kits |

### Параметры предмета

| Параметр | Описание |
|---|---|
| ShortName | Короткое имя предмета |
| Amount Min / Amount Max | Диапазон количества |
| DisplayName | Пользовательское имя (пусто -- стандартное) |
| Skin | ID скина |
| Chance | Шанс выпадения (0-100) |

## Пример конфигурации

```json
{
 "Work with Notify?": true,
 "Event Duration": 600,
 "Bots point": {
 "scientistnpc_bradley": 2.0,
 "scientistnpc_bradley_heavy": 3.0
 },
 "Awards Settings": {
 "Put the reward in the MultiEventsInventory?": true,
 "Player Awards (top - award)": {
 "1": {
 "Award IDs from MultiEventsInventory": [0, 1, 2, 3],
 "Awards": [
 {
 "Type": "Item",
 "Command (%steamid%)": "",
 "Kit": "",
 "Plugin": { "Hook": "", "Plugin Name": "", "Amount": 0 },
 "Item": {
 "DisplayName (empty - default)": "",
 "ShortName": "wood",
 "Skin": 0,
 "Amount Min": 3500,
 "Amount Max": 3500
 },
 "Chance": 100.0
 }
 ],
 "Amount": { "Minimal": 1, "Maximum": 1 }
 }
 }
 }
}
```

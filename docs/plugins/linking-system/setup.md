# Установка Linking System

Это руководство проведёт вас через процесс установки системы привязки Discord.

## Шаг 1: Установите зависимости

### ImageLibrary (обязательно)

1. Скачайте **ImageLibrary** с [umod.org](https://umod.org/plugins/image-library).
2. Поместите файл `ImageLibrary.cs` в папку `oxide/plugins/`.
3. Убедитесь, что плагин загрузился без ошибок.

### Discord-бот (LinkBot) (обязательно)

Для работы плагина необходим Discord-бот, который будет обрабатывать привязку на стороне Discord. Бот взаимодействует с сервером через RCON-команды.

1. Настройте бота согласно инструкциям к LinkBot.
2. Убедитесь, что бот подключен к RCON вашего сервера.
3. Создайте канал `#verify` в вашем Discord-сервере.

## Шаг 2: Установите плагин

1. Скопируйте файл `LinkingSystem.cs` в папку `oxide/plugins/`.
2. Плагин загрузится автоматически.
3. Проверьте консоль -- при подключении бота появится сообщение: `LinkBot is connected`.

Файл конфигурации:
```
oxide/config/LinkingSystem.json
```

## Шаг 3: Настройте права доступа

```
oxide.grant group default linkingsystem.use
```

## Шаг 4: Настройте группу для привязанных игроков

В конфигурации укажите группу, в которую будут добавляться привязанные игроки:

```json
"What group for linked players": "discord"
```

Создайте группу:
```
oxide.group add discord
```

Настройте привилегии для этой группы по вашему усмотрению.

## Шаг 5: Настройте награды

В секции `"Rewards UI elements"` конфигурации настройте отображаемые бонусы за привязку:

```json
{
 "text": "MORE HEALTH",
 "textSize": 11,
 "textColor": "#DCDCDC"
}
```

:::info Примечание
Секция `Rewards` определяет только визуальное отображение наград в интерфейсе. Сами бонусы настраиваются через группу и другие плагины.
:::

## Шаг 6: Проверьте работу

1. Зайдите на сервер.
2. Введите `/link` -- откроется интерфейс.
3. Нажмите "CONNECT DISCORD" и сгенерируйте код.
4. Вставьте код в Discord-канал `#verify`.
5. Нажмите "CHECK CONNECTION STATUS" для подтверждения.

## Шаг 7: Перезагрузите плагин

```
oxide.reload LinkingSystem
```

## Обновление плагина

1. Замените файл `LinkingSystem.cs`.
2. Плагин автоматически обновит конфигурацию.
3. В консоли появится: `Config update detected! Updating config values...`

:::note Примечание
Рекомендуется сохранять резервную копию данных перед обновлением.
:::

# Часто задаваемые вопросы

## 1. Сколько участников может быть в одном клане?

По умолчанию максимальное количество участников -- **8 человек**. Вы можете изменить это значение в конфигурации, параметр "Member Limit" в секции "Limit Settings". Лимит модераторов настраивается отдельно (по умолчанию 2).

## 2. Можно ли сохранять кланы между вайпами?

Да. По умолчанию данные кланов сохраняются при вайпе карты. Если кланы пропадают, проверьте настройки "Purge Settings" в конфигурации -- параметры "Wipe clans on new map save" и "Wipe players on new map save" должны быть установлены в `false`.

## 3. Как работает система альянсов?

Альянс -- это союз между двумя кланами. Модератор или лидер клана отправляет предложение другому клану командой `/clan allyinvite <тег>`. Если второй клан принимает его командой `/clan allyaccept <тег>`, альянс устанавливается. Участники союзных кланов могут общаться через чат альянса (`/a`), а дружественный огонь между ними можно отключить. Максимальное количество альянсов по умолчанию -- 2.

## 4. Как настроить плату за создание клана?

Включите параметр "Charge a fee to create a clan?" в секции "Paid Functionality Settings" и укажите стоимость в поле "Cost". Также настройте секцию "Economy" -- укажите плагин экономики (например, Economics), или тип оплаты предметами (scrap, metal.fragments и т.д.). Убедитесь, что указанный плагин экономики установлен на сервере.

## 5. Что произойдёт, если лидер клана перестанет играть?

Если параметр "Automatic disbanding of the clan when the leader leaves the clan" установлен в `true`, клан будет расформирован при выходе лидера. По умолчанию эта опция отключена. Также работает система автоочистки: если клан неактивен более 14 дней (настраивается), он будет автоматически удалён.

Администратор может вручную назначить нового лидера командой:
```
clans.manage owner <тег клана> <имя нового лидера>
```

## 6. Как работает система рейтингов?

Рейтинг рассчитывается на основе очков, которые начисляются за различные действия: убийства, добычу ресурсов, открытие ящиков, уничтожение танка и вертолёта. Каждое действие имеет свой вес в очках, который настраивается в секции "Loot Settings". Например, убийство игрока даёт 1 очко, а уничтожение вертолёта -- 15. Рейтинги обновляются автоматически с заданным интервалом (по умолчанию каждые 60 секунд).

## 7. Плагин поддерживает BetterChat / IQChat?

Да, плагин имеет встроенную интеграцию с обоими чат-плагинами. Включите соответствующие опции в секции "Chat Settings" конфигурации. После этого клан-теги будут автоматически отображаться перед именами игроков в чате. Формат тега настраивается через параметр "Tag format".

## 8. Можно ли ограничить создание кланов определённой группе?

Да. Включите параметр "Use permission to create a clan" в секции "Permission Settings". После этого создавать кланы смогут только игроки с правом `clans.cancreate`. Выдайте его нужной группе:
```
o.grant group vip clans.cancreate
```

Аналогичные ограничения доступны для вступления, выхода, исключения, расформирования и использования скинов.

## 9. Как работает дружественный огонь?

Есть два режима:
- **Индивидуальный** (по умолчанию) -- каждый игрок сам решает, включить ли дружественный огонь по соклановцам командой `/cff`.
- **Общий** -- один переключатель на весь клан, управляемый лидером (или модератором, если разрешено в настройках).

То же самое работает для альянсов: команда `/aff` переключает дружественный огонь по участникам союзных кланов.

Дополнительно можно включить дружественный огонь для турелей, чтобы они не стреляли по соклановцам.

## 10. Как переместить данные кланов на другой сервер?

Данные кланов хранятся в папке `oxide/data/Clans/`. Скопируйте все файлы из этой папки на новый сервер, установите плагин и перезагрузите сервер. Также убедитесь, что версия плагина на новом сервере совпадает или новее.

## 11. Какие символы разрешены в теге клана?

По умолчанию разрешены латинские буквы и цифры. Дополнительные символы указываются в параметре "Allowed Characters" секции "Clan Tag Settings". Символы из списка "Blocked Words" (по умолчанию: admin, mod, owner) запрещены в любой части тега.

## 12. Как работают скины клана?

Лидер или модератор клана может установить единый набор скинов для снаряжения через интерфейс (страница "Скины"). Скины применяются к определённым предметам (маска, нагрудник, штаны, ботинки, оружие). Каждый участник может отключить клановые скины для себя, если параметр "Disable Skins" включён в конфигурации. Интеграция с плагинами Skinner и SkinBox расширяет доступные скины.

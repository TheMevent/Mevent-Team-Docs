# Команды и права

## Команды чата

| Команда | Описание | Право |
|---|---|---|
| `/servers` | Открыть список серверов | Настраиваемое (по умолчанию `servers.use`) |

:::info Примечание
Команды настраиваются в конфигурации в параметре `"Open UI Commands"`. По умолчанию: `["servers"]`.
:::

## Права (Permissions)

| Право | Описание |
|---|---|
| `servers.use` | Разрешает использование команды для просмотра списка серверов |

:::tip Совет
Право настраивается в конфигурации (`"Permission (ex: servers.use)"`). Вы можете изменить его на любое другое.
:::

## Управление в интерфейсе

| Действие | Описание |
|---|---|
| Клик по серверу | Выбрать сервер для просмотра подробной информации |
| Кнопка "Connect" | Подключиться к выбранному серверу |
| Прокрутка | Прокрутка описания сервера |

# FAQ и решение проблем

## Частые вопросы

### 1. Как выдать предмет через меню?

Введите `/f1menu` в чат, найдите нужный предмет (можно использовать поиск), нажмите на него и выберите количество. Предмет будет выдан в ваш инвентарь.

---

### 2. Почему некоторые предметы отображаются красным?

Красным цветом отмечены предметы из чёрного списка (транспортные средства). Они недоступны для выдачи через данное меню.

---

### 3. Можно ли изменить чёрный список?

Чёрный список определён в исходном коде плагина. Для его изменения необходимо отредактировать файл `F1ItemMenu.cs` и перекомпилировать плагин.

---

### 4. Как настроить уведомления в Discord?

Включите `"Enable Discord Webhook"` в конфигурации, укажите URL вебхука и настройте формат сообщения. Каждый раз при выдаче предмета в указанный канал будет отправлено сообщение.

---

## Решение проблем

### Команда `/f1menu` не работает

**Причина:** У игрока нет права `f1itemmenu.f1`.

**Решение:** Выдайте право: `oxide.grant user ник f1itemmenu.f1` или `oxide.grant group admin f1itemmenu.f1`.

---

### Уведомления не приходят в Discord

**Причина:** Вебхук отключён или URL указан неверно.

**Решение:** Проверьте, что `"Enable Discord Webhook"` установлен в `true` и URL вебхука корректный. Убедитесь, что сервер имеет доступ к интернету.

---

### Меню не показывает все предметы

**Причина:** Предметы распределены по страницам (132 предмета на страницу).

**Решение:** Используйте кнопки навигации `<` и `>` для переключения страниц, или воспользуйтесь полем поиска.

# Часто задаваемые вопросы (FAQ)

## 1. Как изменить длину промокода?

В конфигурации измените параметр `"Promo Code Length"`. По умолчанию используется 8 символов. Вы также можете изменить набор допустимых символов в `"Promo Code Chars"`.

---

## 2. Игрок не может активировать промокод. Почему?

Возможные причины:
1. **Недостаточно времени на сервере** -- игроку нужно наиграть минимум `"Minimum play time (seconds)"` секунд (по умолчанию 1 час). Сообщение: "Вам необходимо наиграть ещё..."
2. **Собственный промокод** -- игрок пытается активировать свой промокод.
3. **Уже активирован** -- промокод можно активировать только один раз.
4. **Промокод не найден** -- указан неверный промокод.

---

## 3. Как добавить новую награду за приглашения?

В секции `"Awards"` конфигурации добавьте новый объект с указанием количества приглашений:

```json
{
 "InvitesAmount": 15,
 "ID": 6,
 "Type": "Item",
 "ShortName": "rifle.ak",
 "Amount": 1,
 "Skin": 0
}
```

---

## 4. Можно ли выдать награду игроку, который активировал промокод?

Да. Включите параметр `"Give an award to a player who activates a promo code?"` в `true` и настройте массив `"Award for the player who activates the promo code"`. Каждая награда имеет параметр `Chance` для указания шанса выпадения.

---

## 5. Как защититься от мультиаккаунтов?

Увеличьте параметр `"Minimum play time (seconds)"`. Например, установите 7200 (2 часа), чтобы игроку нужно было провести на сервере 2 часа перед активацией промокода.

---

## 6. Данные рефералов сбрасываются при вайпе?

По умолчанию нет. Чтобы включить автоматический сброс, установите `"Use auto-wipe?"` в `true`.

---

## 7. Как заблокировать получение наград после вайпа?

Включите в секции `"Rewards blocks"`:
```json
{
 "Use Block Rewards After Wipe?": true,
 "Block After Wipe Time": 7200
}
```

Это заблокирует получение наград на 2 часа после вайпа.

---

## 8. Можно ли использовать команды как награды?

Да. Используйте тип `"Command"`:

```json
{
 "InvitesAmount": 3,
 "Type": "Command",
 "Command": "addgroup %steamid% vip 7d",
 "Amount": 1,
 "Image": "https://example.com/vip.png",
 "Title": "VIP на 7 дней"
}
```

---

## 9. Как ограничить доступ к реферальной системе?

Укажите право в конфигурации:
```json
"Permission (example: referrals.use)": "referrals.use"
```

Затем выдайте право нужным группам:
```
oxide.grant group default referrals.use
```

---

## 10. Игрок не может получить награду из-за рейд-блока. Как отключить?

В секции `"Rewards blocks"` установите:
```json
"Block Rewards During Raid Block?": false,
"Block Rewards During Combat Block?": false
```

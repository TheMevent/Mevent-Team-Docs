# Ultimate Cases

## Что делает этот плагин?

**Ultimate Cases** -- это продвинутая система кейсов для Rust. Игроки могут открывать кейсы с помощью ключей и получать случайные награды через анимацию рулетки. Плагин поддерживает обмен валюты на ключи, NPC-торговцев на монументах, систему раритета, бесплатные кейсы и инвентарь для хранения наград.

## Ключевые возможности

- Настраиваемые кейсы с рулеткой и анимацией открытия
- Система ключей для открытия кейсов
- Обмен валюты (игровых предметов или плагинов экономики) на ключи
- NPC-торговцы на монументах для обмена
- Система раритета с визуальным отображением
- Бесплатные кейсы с таймером
- Множественное открытие кейсов
- Режим быстрого открытия (Quick Unbox)
- Инвентарь для хранения выигранных предметов
- Настраиваемые ограничения: рейд-блок, комбат-блок, вайп-блок
- Дневные и общие лимиты на открытие
- Диалоговая система с NPC
- Режим редактирования прямо в игре (для администраторов)
- Скидки за объём покупки ключей
- VIP-привилегии с индивидуальными скидками
- Discord-баннер в интерфейсе
- Интеграция с ServerPanel, Notify, LangAPI, Kits, WipeBlock, NoEscape
- Поддержка пользовательских предметов (CustomItemDefinitions)

## Требования

| Зависимость | Обязательность | Описание |
|---|---|---|
| **ImageLibrary** | Обязательно | Для изображений в интерфейсе |
| Notify | Опционально | Для графических уведомлений |
| ServerPanel | Опционально | Для интеграции в серверное меню |
| LangAPI | Опционально | Для мультиязычности |
| WipeBlock | Опционально | Для блокировки после вайпа |
| NoEscape | Опционально | Для блокировки во время рейда/комбата |
| MonumentFinder | Опционально | Для NPC-торговцев на монументах |
| Oxide / uMod / Carbon | Обязательно | Совместим с Oxide, uMod и Carbon |

## Быстрый старт

1. Установите плагин **ImageLibrary**.
2. Скопируйте файл `UltimateCases.cs` в папку `oxide/plugins/`.
3. Перезагрузите сервер или выполните `oxide.reload UltimateCases`.
4. Введите в чат `/cases` или `/opencases` для открытия интерфейса.
5. Выдайте ключи игрокам командой:
 ```
 cases.give <steamid> keys <amount>
 ```

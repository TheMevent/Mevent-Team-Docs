# Часто задаваемые вопросы

## 1. Как открыть панель управления ивентами?

Введите `/em` в игровом чате. Убедитесь, что у вас есть право доступа `EventsManager.use`. Для выдачи прав используйте:
```
o.grant user ваш_steamid EventsManager.use
```

## 2. Какие ивенты поддерживаются из коробки?

Плагин поддерживает десятки ивентов от различных разработчиков:

- **Ванильные события**: вертолёт, аирдроп, Chinook 47, грузовой корабль, сани Санты
- **Mevent**: Operation Fortress, Collection Resources, Foundation Drop, Gold Rush, Helicopter Pet, Hunt Animal, King Of Hill, Looking Loot, Special Cargo
- **KpucTaJl**: Air Event, Water Event, Arctic Base Event, Satellite Dish Event, Junkyard Event, Power Plant Event, Harbor Event и другие
- **Adem**: Sputnik, Armored Train, Convoy, Shipwreck, Caravan и другие
- **Nivex**: Dangerous Treasures, Raidable Bases (Easy/Medium/Hard/Nightmare)
- И многие другие от Mercury, DezLife, Cashr, Bazz3l, Fruster и т.д.

## 3. Как добавить свой кастомный ивент?

1. Откройте панель через `/em`
2. Нажмите **"Добавить ивент"**
3. Укажите:
 - **Название** — отображаемое имя ивента
 - **Команда** — консольная команда для запуска
 - **Плагин** — название плагина (для проверки, загружен ли он)
 - **Создатель** — категория для группировки
4. Сохраните ивент

## 4. Ивент стоит в расписании, но не запускается. Что делать?

Проверьте следующее:
- Количество игроков онлайн соответствует условиям (мин./макс. игроков)
- Текущий день недели входит в список активных дней
- Плагин ивента установлен и загружен (статус загрузки отображается в панели)
- Расписание сохранено правильно

## 5. Как отключить стандартные ванильные события Rust?

В конфигурации установите параметр `"Disable vanilla auto events?"` в `true`. Плагин отключит автоматический запуск всех стандартных событий, кроме тех, что указаны в белом списке `"Allowed list of vanilla events"`.

## 6. Как настроить Discord-уведомления?

1. Создайте вебхук в настройках вашего Discord-канала
2. Скопируйте URL вебхука
3. Вставьте URL в параметр `"Discord web hook"` в конфигурации плагина
4. Перезагрузите плагин

Теперь при каждом автоматическом запуске ивента в Discord-канал будет приходить уведомление.

## 7. Можно ли назначить несколько ивентов на один слот расписания?

Да. При создании расписания вы можете назначить несколько ивентов. Система будет случайно выбирать один из них при каждом запуске. Это полезно для разнообразия — игроки не будут знать заранее, какой именно ивент начнётся.

## 8. Как полностью сбросить все настройки к значениям по умолчанию?

Используйте консольную команду сервера:
```
eventmanager.reset.default
```
Это удалит все пользовательские ивенты, расписания и категории, и загрузит стандартный список.

## 9. Совместим ли плагин с Carbon?

Да, плагин полностью совместим как с **Oxide**, так и с **Carbon**. На Carbon не требуется ImageLibrary — плагин использует встроенную систему изображений.

## 10. Как узнать, какие ивенты были запущены?

Включите параметр `"Enable event logging to file"` в конфигурации. После этого все запуски ивентов будут записываться в лог-файл с датой и временем.

# Bow Master

## Описание

**Bow Master** -- это режим расширения для плагина MultiEvents, в котором игроки соревнуются в меткости стрельбы из лука и арбалета. Очки начисляются за каждое убийство, совершённое стрелами (тип урона Arrow). Учитываются убийства как игроков, так и животных. Игрок с наибольшим количеством очков к концу ивента побеждает.

## Ключевые возможности

- Соревновательный режим по стрельбе из лука и арбалета
- Учитываются только убийства стрелами (тип урона Arrow)
- Засчитываются убийства игроков и животных
- Таблица лидеров в реальном времени
- Настраиваемые награды за места в рейтинге
- Интеграция с MultiEventsInventory для выдачи наград
- Интеграция с MultiEventsLeaderboard для глобальной статистики
- Поддержка Notify / UINotify для уведомлений
- Настраиваемая длительность ивента
- UI-панель с таймером и текущим счётом

## Требования

| Зависимость | Обязательность | Описание |
|---|---|---|
| **MultiEvents** | Обязательно | Основной плагин системы событий |
| **ImageLibrary** | Обязательно | Загрузка и отображение изображений в интерфейсе |
| MultiEventsInventory | Опционально | Для выдачи наград через инвентарь событий |
| MultiEventsLeaderboard | Опционально | Для ведения глобальной таблицы лидеров |
| Notify / UINotify | Опционально | Для отправки визуальных уведомлений игрокам |

## Быстрый старт

1. Убедитесь, что плагины **MultiEvents** и **ImageLibrary** установлены и работают.
2. Скопируйте файл `MultiEventsBowMaster.cs` в папку `oxide/plugins/` вашего сервера.
3. Плагин автоматически зарегистрируется в системе MultiEvents.
4. Запустите ивент через MultiEvents или командой:
 ```
 bowmaster.start
 ```
5. Начните убивать игроков или животных с помощью лука или арбалета для набора очков.

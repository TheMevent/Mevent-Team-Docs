# Конфигурация

Конфигурационный файл создаётся автоматически при первом запуске плагина и находится по пути `oxide/config/AutoRecycler.json`.

## Основные параметры

```json
{
 "Virtual recycler commands": ["virtualrec", "vrec", "recvirtual"],
 "Auto recycler UI commands": ["autorec", "ar", "autorecycler"],
 "Crate-recycler settings": {
 "Shortname": "box.wooden.large",
 "DisplayName": "Crate-recycler",
 "SkinID": 3100475476
 },
 "Recbox cost": [
 {
 "Shortname": "scrap",
 "Amount": 100
 }
 ],
 "Items to Recycle": { ... }
}
```

## Описание параметров

### Virtual recycler commands

Список чат-команд для открытия виртуального переработчика. По умолчанию: `["virtualrec", "vrec", "recvirtual"]`.

### Auto recycler UI commands

Список чат-команд для открытия интерфейса автопереработки. По умолчанию: `["autorec", "ar", "autorecycler"]`.

### Crate-recycler settings

Настройки предмета ящика-переработчика:

| Параметр | Описание | По умолчанию |
| ----------- | ------------------------------------- | --------------------- |
| Shortname | Короткое имя базового предмета | `box.wooden.large` |
| DisplayName | Отображаемое имя | `Crate-recycler` |
| SkinID | Идентификатор скина | `3100475476` |

### Recbox cost

Стоимость покупки ящика-переработчика. Массив предметов, которые нужно оплатить:

```json
"Recbox cost": [
 {
 "Shortname": "scrap",
 "Amount": 100
 }
]
```

Вы можете добавить несколько предметов:

```json
"Recbox cost": [
 { "Shortname": "scrap", "Amount": 200 },
 { "Shortname": "metal.fragments", "Amount": 500 }
]
```

### Items to Recycle

Словарь всех перерабатываемых предметов с настройками. Заполняется автоматически при первом запуске.

Каждый предмет имеет следующую структуру:

```json
"wood.armor.jacket": {
 "IsEnabled": true,
 "ItemAmount": [
 {
 "Shortname": "wood",
 "Amount": 100,
 "Enabled": true,
 "Chance": 100
 },
 {
 "Shortname": "rope",
 "Amount": 1,
 "Enabled": true,
 "Chance": 50
 }
 ]
}
```

| Поле | Описание |
| --------- | -------------------------------------------------- |
| IsEnabled | Доступен ли предмет для автопереработки |
| Shortname | Короткое имя ресурса-результата |
| Amount | Количество получаемого ресурса |
| Enabled | Включён ли данный результат |
| Chance | Шанс выпадения (0--100). 100 = гарантировано |

## Настройка шансов переработки

Для некоторых предметов (например, оружия) компоненты выпадают не всегда. Шанс указывается в процентах:

- `100` -- компонент выпадает всегда
- `50` -- компонент выпадает в 50% случаев
- `25` -- компонент выпадает в 25% случаев
- `0` -- компонент никогда не выпадает

## Файлы данных

| Файл | Описание |
| ---- | -------- |
| `oxide/data/AutoRecycler.json` | Данные о настройках автопереработки игроков |

:::warning Важно
Раздел "Items to Recycle" заполняется автоматически при первом запуске на основе игровых рецептов. Не удаляйте его -- вносите изменения в существующие записи.
:::

# Команды и права

## Команды чата

| Команда | Описание | Право |
|---|---|---|
| `/cases` | Открыть интерфейс кейсов | Настраивается |
| `/opencases` | Открыть интерфейс кейсов (альтернативная) | Настраивается |
| `/cases.modal.open <id>` | Открыть конкретный модал кейса | Настраивается |

:::tip Совет
Команды настраиваются в конфигурации в параметре `Open UI Commands`. По умолчанию: `["opencases", "cases"]`.
:::

## Консольные команды

| Команда | Описание | Кто может использовать |
|---|---|---|
| `cases.give <steamid> keys <amount>` | Выдать ключи игроку | Администраторы / RCON |
| `cases.give <steamid> case <caseId> <amount>` | Выдать кейс игроку | Администраторы / RCON |

### Примеры использования

Выдать 10 ключей игроку:
```
cases.give 76561198012345678 keys 10
```

Выдать кейс игроку:
```
cases.give 76561198012345678 case 1 5
```

## Права (Permissions)

| Право | Описание | По умолчанию |
|---|---|---|
| `ultimatecases.edit` | Режим редактирования кейсов в игре | Не выдано |
| `ultimatecases.vip1` | VIP-1 скидка на обмен (5%) | Не выдано |
| `ultimatecases.vip2` | VIP-2 скидка на обмен (10%) | Не выдано |
| `ultimatecases.vip3` | VIP-3 скидка на обмен (15%) | Не выдано |
| `ultimatecases.vip4` | VIP-4 скидка на обмен (20%) | Не выдано |

:::info Примечание
VIP-права и скидки полностью настраиваются в конфигурации в секции `"Exchange Settings" -> "PrivilegeSettings"`.
:::

### Настройка прав

```
oxide.grant group admin ultimatecases.edit
oxide.grant group vip ultimatecases.vip1
oxide.grant group premium ultimatecases.vip2
```

## Механика работы

### Открытие кейса

1. Игрок открывает интерфейс кейсов (`/cases`).
2. Выбирает кейс из списка.
3. Просматривает возможные награды (Potential Drops).
4. Если у игрока достаточно ключей, нажимает "OPEN".
5. Запускается анимация рулетки.
6. Выигранный предмет попадает в хранилище наград.
7. Игрок забирает предмет из хранилища.

### Обмен валюты на ключи

1. В интерфейсе выбирается валюта для обмена.
2. Указывается количество ключей.
3. Применяются скидки за объём и VIP-привилегии.
4. Ключи зачисляются на баланс.

### Ограничения открытия

Кейсы могут быть заблокированы в следующих ситуациях:
- Комбат-блок
- Рейд-блок
- Зона чужого шкафа
- Плавание
- Ранение
- Дуэль
- Полёт на вертолёте
- Сон

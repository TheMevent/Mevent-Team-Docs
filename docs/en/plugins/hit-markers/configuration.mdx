# Hit Markers Configuration

The configuration file is located at: `oxide/config/HitMarkers.json`

## Main Parameters

| Parameter | Default | Description |
|---|---|---|
| Commands | `["marker", "hits"]` | Chat commands to open settings |
| Permission | `""` | Access permission (empty -- for everyone) |
| Work with Notify? | `true` | Use Notify for notifications |
| Show damage to NPC | `true` | Display damage to NPCs |
| Show damage to animals | `false` | Display damage to animals |
| Marker removal time | `0.25` | Hit marker display duration (seconds) |

## Font Settings (Fonts)

Available fonts for numerical damage:

| ID | Font | Default |
|---|---|---|
| 0 | `robotocondensed-bold.ttf` | Bold |
| 1 | `robotocondensed-regular.ttf` | Regular |
| 2 | `permanentmarker.ttf` | Marker |
| 3 | `droidsansmono.ttf` | Monospaced |

Each font can have its own access permission.

## Font Size Limits

| Parameter | Default | Description |
|---|---|---|
| Min Font Size | `8` | Minimum font size |
| Max Font Size | `18` | Maximum font size |

## Display Modes (Buttons)

| Mode | Description | Default |
|---|---|---|
| Text | Numerical damage display | Enabled |
| Icon | Hit icon with headshot color | Enabled |
| Health Line | Enemy health bar | Enabled |
| Buildings | Building damage | Enabled |

Each mode can have its own access permission.

## Default Values

| Parameter | Default | Description |
|---|---|---|
| Font ID | `0` | Default font |
| Font Size | `14` | Default font size |
| Text | `true` | Numerical damage enabled |
| Icon | `false` | Hit icon disabled |
| Health Line | `false` | Health bar disabled |
| Buildings | `false` | Building damage disabled |

## Line Settings

| Parameter | Default | Description |
|---|---|---|
| Show Line? | `true` | Show the health bar line |
| Show Text? | `false` | Show the health text value |

# Команды и права

## Команды чата

| Команда | Описание | Право |
|---|---|---|
| `/daily` | Открыть интерфейс ежедневных наград | `dailyrewards.use` |
| `/rewards` | Открыть интерфейс ежедневных наград (альтернативная) | `dailyrewards.use` |

:::tip Совет
Команды чата настраиваются в конфигурации в параметре `Commands`. По умолчанию доступны `daily` и `rewards`.
:::

## Консольные команды

| Команда | Описание | Кто может использовать |
|---|---|---|
| `dailyrewards.wipe` | Сбросить прогресс всех игроков | Только серверная консоль |
| `dailyrewards.manage playtime set <steam_id> <amount>` | Установить время игры для игрока | Только серверная консоль |
| `dailyrewards.top` | Показать топ игроков по времени игры | Только серверная консоль |
| `dailyrewards.template [fullscreen/inmenu] [номер]` | Переключить шаблон интерфейса | Только серверная консоль |

### Примеры использования

Сброс прогресса всех игроков:
```
dailyrewards.wipe
```

Установка времени игры для конкретного игрока:
```
dailyrewards.manage playtime set 76561198012345678 3600
```

Переключение шаблона на полноэкранный:
```
dailyrewards.template fullscreen 1
```

## Права (Permissions)

| Право | Описание | По умолчанию |
|---|---|---|
| `dailyrewards.use` | Доступ к интерфейсу ежедневных наград | Не выдано |
| `dailyrewards.edit` | Редактирование наград через интерфейс | Не выдано |
| `dailyrewards.vip` | VIP-группа с уменьшенным кулдауном (3100 сек.) | Не выдано |
| `dailyrewards.premium` | Premium-группа с минимальным кулдауном (2600 сек.) | Не выдано |

:::info Примечание
Названия VIP-прав и соответствующие кулдауны настраиваются в конфигурации в секции `Cooldown Settings`. Вы можете создать любое количество групп с разными временами ожидания.
:::

### Настройка прав

```
oxide.grant group default dailyrewards.use
oxide.grant group admin dailyrewards.edit
oxide.grant group vip dailyrewards.vip
oxide.grant group premium dailyrewards.premium
```

## Уведомления (Notify)

Плагин поддерживает отправку уведомлений через Notify при следующих событиях:

| Событие | Тип уведомления | Описание |
|---|---|---|
| Кулдаун окончен | `2130354` | Награда доступна для получения |
| Кулдаун активен | `2130355` | Показывает оставшееся время до получения |
| Награда получена | `2130356` | Подтверждение получения награды |

Уведомления настраиваются в секции `"Notification settings"` конфигурации.

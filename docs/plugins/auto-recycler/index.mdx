# AutoRecycler

## Что делает этот плагин?

**AutoRecycler** позволяет вашему серверу предоставить игрокам автоматическую переработку предметов при их получении, виртуальный переработчик, доступный по команде, а также специальный ящик-переработчик, который автоматически перерабатывает помещённые в него предметы. Администраторы могут настраивать результаты переработки, шансы выпадения компонентов и включать/отключать переработку для каждого предмета.

## Ключевые возможности

- Автоматическая переработка предметов при их попадании в инвентарь игрока
- Виртуальный переработчик, открываемый командой из любого места
- Ящик-переработчик (Recbox) -- специальный ящик, автоматически перерабатывающий предметы
- Покупка ящика-переработчика за внутриигровые ресурсы
- Графический интерфейс для настройки автопереработки по категориям
- Индивидуальная настройка результатов переработки и шансов для каждого предмета
- Система прав для разделения доступа к функциям
- Администраторы могут включать/отключать переработку для отдельных предметов

## Требования

| Зависимость | Обязательна? | Описание |
| ----------- | ------------ | -------- |
| Oxide/uMod | Да | Платформа для плагинов |

## Быстрый старт

1. Поместите файл плагина в папку `oxide/plugins`.
2. Перезагрузите сервер или выполните `oxide.reload AutoRecycler`.
3. Выдайте игрокам необходимые права доступа (см. таблицу ниже).
4. Игроки могут использовать `/vrec` для виртуального переработчика или `/autorec` для настройки автопереработки.
5. Настройте результаты переработки и стоимость ящика в конфигурации.

## Права доступа

| Право | Описание |
| ---------------------------------- | --------------------------------------------------------- |
| `autorecycler.admin` | Полный доступ ко всем функциям плагина |
| `autorecycler.autorec` | Доступ к интерфейсу автопереработки |
| `autorecycler.recvirtual` | Доступ к виртуальному переработчику |
| `autorecycler.recyclercratetest` | Доступ к тестированию ящика-переработчика |
| `autorecycler.characteristics` | Доступ к настройке характеристик переработки (вкл/выкл) |
| `autorecycler.recboxbuy` | Право на покупку ящика-переработчика |

## Команды игроков

| Команда | Право | Описание |
| --------------------- | --------------------------- | ------------------------------------------------ |
| `/vrec` (`/virtualrec`, `/recvirtual`) | `autorecycler.recvirtual` | Открывает виртуальный переработчик |
| `/autorec` (`/ar`, `/autorecycler`) | `autorecycler.autorec` | Открывает интерфейс настройки автопереработки |
| `/autorectoggle` | -- | Включает/выключает автопереработку |
| `/recbox` | -- | Показывает стоимость ящика-переработчика |
| `/recboxbuy` | `autorecycler.recboxbuy` | Покупает ящик-переработчик |

**Примеры:**

```
/vrec
/autorec
/autorectoggle
/recbox
/recboxbuy
```

## Команды администратора

| Команда | Описание |
| -------------------------------------- | ------------------------------------------------ |
| `/recboxtest` | Выдаёт 1000 ящиков-переработчиков (только админ) |
| `recyclercrate.give [имя_игрока]` | Выдаёт ящик-переработчик указанному игроку (консоль) |

**Примеры:**

```
/recboxtest
recyclercrate.give PlayerName
```

## Устранение неполадок

| Проблема | Решение |
| -------- | ------- |
| Виртуальный переработчик не открывается | Проверьте наличие права `autorecycler.recvirtual` |
| Автопереработка не работает | Убедитесь, что автопереработка включена (`/autorectoggle`) и нужные предметы отмечены в интерфейсе `/autorec` |
| Ящик-переработчик не перерабатывает предметы | Проверьте, что предмет включён в конфигурации (поле `IsEnabled`) |
| Игрок не может купить ящик | Проверьте наличие права `autorecycler.recboxbuy` и достаточность ресурсов |
| Предметы пропадают при закрытии виртуального переработчика | Предметы автоматически возвращаются в инвентарь или сбрасываются на землю, если инвентарь полон |
| Ошибка при использовании команд | Убедитесь, что у игрока есть соответствующие права доступа |

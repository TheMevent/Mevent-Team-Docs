# Установка и настройка

Это руководство проведёт вас через весь процесс установки режима Bot Purge для системы MultiEvents.

## Системные требования

- Сервер Rust с Oxide / uMod / Carbon
- Установленный плагин **MultiEvents** (обязательная зависимость)
- Установленный плагин **ImageLibrary** (обязательная зависимость)

## Шаг 1: Установите зависимости

### MultiEvents (обязательно)

Плагин Bot Purge является режимом расширения и не может работать без основного плагина MultiEvents.

1. Установите **MultiEvents** согласно его документации.
2. Убедитесь, что плагин загружен без ошибок.

### ImageLibrary (обязательно)

1. Скачайте **ImageLibrary** с [umod.org](https://umod.org/plugins/image-library).
2. Поместите файл `ImageLibrary.cs` в папку `oxide/plugins/`.

:::warning Важно
Без MultiEvents и ImageLibrary плагин Bot Purge **не будет работать**.
:::

## Шаг 2: Установите плагин

1. Скопируйте файл `MultiEventsBotPurge.cs` в папку `oxide/plugins/` вашего сервера.
2. Плагин загрузится автоматически и зарегистрируется в MultiEvents.
3. Проверьте консоль на наличие ошибок.

Файл конфигурации будет создан автоматически:
```
oxide/config/MultiEventsBotPurge.json
```

## Шаг 3: Настройте права доступа

Выдайте права администраторам для управления ивентом:

```
oxide.grant group admin MultiEventsBotPurge.start
oxide.grant group admin MultiEventsBotPurge.stop
```

## Шаг 4: Проверьте работу

Запустите ивент командой:

```
botpurge.start
```

На экране игроков должна появиться панель ивента с таймером и таблицей лидеров. Убивайте NPC-учёных для набора очков.

## Шаг 5: Настройте конфигурацию (по желанию)

Откройте файл `oxide/config/MultiEventsBotPurge.json` для настройки:

- **Длительность ивента** -- время проведения события
- **Таблица очков за ботов** -- количество очков за каждый тип NPC
- **Награды** -- настройка наград за места в рейтинге
- **Интеграция с Notify** -- включение/отключение визуальных уведомлений

Подробнее о каждом параметре читайте в разделе [Конфигурация](./configuration).

## Шаг 6: Перезагрузите плагин

После изменения конфигурации:

```
oxide.reload MultiEventsBotPurge
```

:::tip Совет
Плагин автоматически регистрируется в MultiEvents при загрузке. Если MultiEvents был перезагружен, Bot Purge перерегистрируется автоматически.
:::

## Обновление плагина

При обновлении:

1. Замените файл `MultiEventsBotPurge.cs` новой версией.
2. Плагин перезагрузится автоматически.
3. Проверьте конфигурацию на наличие новых параметров.

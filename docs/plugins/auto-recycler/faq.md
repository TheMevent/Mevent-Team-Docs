# Часто задаваемые вопросы (FAQ)

## 1. Чем отличается автопереработка от виртуального переработчика?

**Автопереработка** -- предметы перерабатываются мгновенно при попадании в инвентарь игрока. Игрок выбирает, какие предметы перерабатывать автоматически, через интерфейс `/autorec`.

**Виртуальный переработчик** -- это обычный переработчик, который открывается командой `/vrec` из любого места. Игрок вручную кладёт в него предметы, как в обычный переработчик.

## 2. Как включить автопереработку для игрока?

1. Выдайте игроку право: `oxide.grant user <steamid> autorecycler.autorec`
2. Игрок вводит `/autorec` и отмечает нужные предметы.
3. Автопереработка начнёт работать при получении отмеченных предметов.

## 3. Как работает ящик-переработчик (Recbox)?

Ящик-переработчик -- это специальный большой деревянный ящик с уникальным скином. Когда игрок помещает в него предметы, они автоматически перерабатываются. Ящик можно разместить на земле как обычный ящик для хранения.

## 4. Как игрок может купить ящик-переработчик?

1. Убедитесь, что у игрока есть право `autorecycler.recboxbuy`.
2. Игрок вводит `/recbox` для просмотра стоимости.
3. Игрок вводит `/recboxbuy` для покупки.
4. Ресурсы списываются из инвентаря, и ящик выдаётся игроку.

## 5. Можно ли настроить, какие ресурсы выпадают при переработке?

Да, в конфигурации в разделе "Items to Recycle" вы можете изменить результаты переработки для каждого предмета: количество ресурсов, шанс выпадения, а также полностью включить или отключить конкретный результат.

## 6. Что происходит с предметами при закрытии виртуального переработчика?

Все предметы, находящиеся в виртуальном переработчике, сбрасываются на землю рядом с игроком при закрытии. Это предотвращает потерю предметов.

## 7. Можно ли быстро включить/выключить автопереработку?

Да, используйте команду `/autorectoggle`. Она мгновенно включает или выключает всю автопереработку без необходимости открывать интерфейс.

## 8. Как администратор может отключить переработку определённого предмета для всех?

Администратор с правом `autorecycler.characteristics` может открыть `/autorec` и нажать кнопку "ENABLED/DISABLED" на конкретном предмете. Это глобально включает/отключает возможность переработки этого предмета для всех игроков.

## 9. Список предметов для переработки не содержит некоторых предметов. Почему?

Список генерируется автоматически на основе игровых рецептов при первом запуске. Предметы без рецепта крафта (транспорт, некоторые специальные предметы) не включаются в список. Транспортные средства (вертолёты, лодки, машины) намеренно исключены из списка переработки.

## 10. Можно ли изменить стоимость ящика-переработчика?

Да, измените раздел "Recbox cost" в конфигурации. Вы можете указать любое количество предметов и их количество. После изменения перезагрузите плагин.

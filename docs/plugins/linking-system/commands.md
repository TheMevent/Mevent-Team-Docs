# Команды и права

## Команды чата

| Команда | Описание | Право |
|---|---|---|
| `/link` | Открыть интерфейс привязки Discord | `linkingsystem.use` |
| `/link status` | Показать текущий статус привязки | `linkingsystem.use` |
| `/link code` | Получить код для привязки | `linkingsystem.use` |
| `/link remove` | Удалить привязку Discord | `linkingsystem.use` |

:::tip Совет
Команды настраиваются в конфигурации в параметре `Open UI Commands`. По умолчанию: `["link"]`.
:::

### Примеры использования

Открыть интерфейс:
```
/link
```

Проверить статус привязки:
```
/link status
```

Удалить привязку:
```
/link remove
```

## Консольные команды (внутренние)

Следующие команды используются Discord-ботом (LinkBot) для взаимодействия с плагином и не предназначены для ручного использования:

| Команда | Описание |
|---|---|
| `LinkBot_BotOnline` | Бот сообщает о подключении |
| `LinkBot_UpdatePlayer` | Обновление данных привязки игрока |
| `LinkBot_RemovePlayer` | Удаление привязки игрока |

:::warning Внимание
Эти команды используются исключительно Discord-ботом через RCON. Не вводите их вручную.
:::

## Права (Permissions)

| Право | Описание | По умолчанию |
|---|---|---|
| `linkingsystem.use` | Доступ к системе привязки | Не выдано |

### Настройка прав

```
oxide.grant group default linkingsystem.use
```

## Процесс привязки

### Для игрока

1. Введите `/link` в чат на сервере.
2. В интерфейсе нажмите "CONNECT DISCORD".
3. Сгенерируйте уникальный код.
4. Скопируйте код (CTRL+C).
5. Перейдите в Discord-канал `#verify` вашего сервера.
6. Вставьте код в чат или используйте Discord-бота.
7. Вернитесь в игру и нажмите "CHECK CONNECTION STATUS".
8. При успешной привязке статус изменится на "Account linked".

### Отвязка

1. Введите `/link` для открытия интерфейса.
2. Нажмите "DISCONNECT ACCOUNT".
3. Или используйте команду `/link remove`.

### Статусы привязки

| Статус | Описание |
|---|---|
| Not Linked | Аккаунт не привязан |
| Wait Confirm | Код сгенерирован, ожидание подтверждения |
| Linked | Аккаунт успешно привязан |

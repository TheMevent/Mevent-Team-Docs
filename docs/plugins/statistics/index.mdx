# Statistics

## Что делает этот плагин?

**Statistics** -- это система статистики, которая отслеживает действия игроков на сервере и отображает их в красивом интерфейсе. Плагин ведёт подсчёт убийств, смертей, добытых ресурсов, любимого оружия и других показателей. Включает профиль игрока и таблицу лидеров.

## Ключевые возможности

- Подробный профиль игрока с полной статистикой
- Таблица лидеров (Leaderboard) с рейтингом
- Отслеживание убийств, смертей, добычи ресурсов
- Определение любимого оружия игрока
- Система очков (Score) с настраиваемыми таблицами
- Время игры на сервере (интеграция с PlayTimeRewards / PlaytimeTracker)
- Награды за топовые позиции в рейтинге
- Возможность скрыть себя из таблицы лидеров
- Просмотр профиля другого игрока
- Настраиваемый учёт убийств ботов, самоубийств, тиммейтов
- Автоматический вайп статистики (опционально)
- Интеграция с ServerPanel, Notify, PlayerDatabase
- Поддержка сбора статистики только для игроков с правом

## Требования

| Зависимость | Обязательность | Описание |
|---|---|---|
| **ImageLibrary** | Обязательно | Необходима для изображений в интерфейсе |
| Notify | Опционально | Для графических уведомлений |
| PlayTimeRewards / PlaytimeTracker | Опционально | Для отслеживания времени игры |
| ServerPanel | Опционально | Для интеграции в серверное меню |
| PlayerDatabase | Опционально | Для хранения данных |
| Oxide / uMod / Carbon | Обязательно | Совместим с Oxide, uMod и Carbon |

## Быстрый старт

1. Установите плагин **ImageLibrary**.
2. Скопируйте файл `Statistics.cs` в папку `oxide/plugins/`.
3. Перезагрузите сервер или выполните `oxide.reload Statistics`.
4. Выдайте право:
 ```
 oxide.grant group default statistics.use
 ```
5. Введите в чат `/stats` для просмотра профиля или `/leaders` для таблицы лидеров.

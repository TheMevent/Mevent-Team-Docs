# Команды и права

## Команды чата

| Команда | Описание | Право |
|---|---|---|
| `/spawnjeep` | Заспавнить джип перед игроком | Только администраторы (IsAdmin) |

:::warning Важно
Команда `/spawnjeep` доступна только игрокам с правами администратора сервера (IsAdmin).
:::

## Права (Permissions)

Плагин Ultimate Cars использует встроенную проверку `player.IsAdmin` для определения доступа к команде спавна. Отдельные права Oxide/uMod не регистрируются.

## Управление транспортом

| Клавиша | Действие |
|---|---|
| W | Ускорение (вперёд) |
| S | Торможение / задний ход |
| A | Поворот влево |
| D | Поворот вправо |
| Space | Ручной тормоз |

## Взаимодействие

- **Посадка:** Подойдите к автомобилю и нажмите E для посадки в водительское сиденье.
- **Высадка:** Нажмите E для выхода из автомобиля.
- **Уничтожение:** Автомобиль уничтожается при разрушении базовой сущности.

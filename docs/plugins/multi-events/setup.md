# Установка и настройка

## Системные требования

- Сервер Rust с установленным **Oxide** или **Carbon**
- Плагин **ImageLibrary** (только для Oxide)

## Пошаговая установка

### Шаг 1: Загрузка плагина

Скопируйте файл `MultiEvents.cs` в директорию плагинов:

```
oxide/plugins/MultiEvents.cs
```

### Шаг 2: Установка зависимостей

**Обязательные** (для Oxide):
- ImageLibrary

**Опциональные:**
- Notify или UINotify — для всплывающих уведомлений
- ServerPanel — для интеграции с панелью сервера
- MultiEventsInventory — для инвентаря наград
- MultiEventsLeaderboard — для таблицы лидеров

### Шаг 3: Перезагрузка

Перезагрузите сервер или выполните:
```
o.reload MultiEvents
```

### Шаг 4: Настройка ивентов

Отредактируйте конфигурацию и добавьте ваши серверные ивенты в секцию "Events". Для каждого ивента необходимо указать:

| Параметр | Описание |
|---|---|
| Enabled | Включён ли ивент |
| Plugin Name | Название плагина ивента |
| Banner Url | URL изображения баннера |
| Title | Название ивента |
| Icon image | URL иконки ивента |
| Short Description | Краткое описание ивента |

### Шаг 5: Настройка прав

```
o.grant user SteamID64 multievents.edit
o.grant group admin multievents.edit
```

### Шаг 6: Визуальная настройка

Войдите в игру и используйте `/ShowUiEditor` для визуальной настройки интерфейса ивентов.

## Настройка модулей

### Инвентарь наград

Для активации инвентаря наград:

1. Установите плагин **MultiEventsInventory**
2. В конфигурации Multi Events включите параметр "Use Inventory?"
3. При необходимости задайте право доступа для инвентаря

### Таблица лидеров

Для активации таблицы лидеров:

1. Установите плагин **MultiEventsLeaderboard**
2. В конфигурации Multi Events включите параметр "Use Leaderboard?"
3. При необходимости задайте право доступа для таблицы

## Шаблоны отображения

Плагин поддерживает два шаблона отображения при интеграции с ServerPanel:

| Шаблон | Описание |
|---|---|
| V1 | Стандартный вид |
| V2 (Fullscreen) | Полноэкранный режим |

Выбор шаблона осуществляется через параметр "ServerPanel Template" в конфигурации.

## Автозапуск ивентов

Для настройки автоматического запуска случайных ивентов:

1. Включите параметр "Enable timer to start random event"
2. Задайте минимальную и максимальную длительность таймера (в секундах)
3. По истечении случайного интервала система запустит случайный ивент из списка включённых

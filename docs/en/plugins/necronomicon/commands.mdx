# Commands & Permissions

## Console Commands

| Command | Description | Example |
|---|---|---|
| `necronomicon.give [nickname/steamid]` | Give a Necronomicon book to a player | `necronomicon.give Player1` |

:::info
The `necronomicon.give` command runs from the server console or RCON. The book is created with the specified player's current blueprints.
:::

## Permissions

| Permission | Description |
|---|---|
| Configurable | Permission for using the book (set in config, empty = everyone) |

:::tip
If the `"Permission"` field in the config is left empty, any player can use the book.
:::

## How It Works

1. An admin gives a Necronomicon book to a player via the console command.
2. The player opens a Research Table.
3. The player places the book into the table.
4. A visual interface appears showing the blueprints contained in the book.
5. The player can learn blueprints for the cost specified in the config.
6. After learning, the blueprint is added to the player's collection.

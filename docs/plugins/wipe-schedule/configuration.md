# Конфигурация Wipe Schedule

Конфигурационный файл расположен по пути: `oxide/config/WipeSchedule.json`

## Основные параметры

| Параметр | Тип | По умолчанию | Описание |
|---|---|---|---|
| Work with Notify? | булево | `true` | Использовать Notify/UINotify для уведомлений |
| Commands to open the calendar | массив | `["wipe", "wipedata"]` | Чат-команды для открытия календаря |
| Time Zone | число | `2` | Часовой пояс (смещение от UTC в часах) |
| Start calendar week on Sunday | булево | `false` | Начинать неделю с воскресенья (если `false` -- с понедельника) |
| ServerPanel Template (V1, V2) | строка | `V1` | Шаблон для интеграции с серверным меню |

## Пользовательские ключи перевода

Раздел "Your keys in the translator" позволяет добавлять собственные тексты для пользовательских событий.

Формат записи:

```json
{
 "Custom Event": {
 "ru": "Событие сервера",
 "en": "Custom Event"
 },
 "Custom Event Description": {
 "ru": "Описание события сервера",
 "en": "Custom Event Description"
 }
}
```

Каждый ключ содержит два поля:
- `ru` -- текст на русском языке
- `en` -- текст на английском языке

## Настройка часового пояса

Параметр "Time Zone" определяет, какое время используется для расчёта событий. Указывается как целое число -- смещение от UTC.

Примеры:
- UTC+0 (Лондон): `0`
- UTC+1 (Берлин): `1`
- UTC+2 (Киев): `2`
- UTC+3 (Москва, Минск): `3`

## Настройка событий через интерфейс

Создание и редактирование событий выполняется через визуальный интерфейс внутри игры. Для каждого события доступны:

### Общие настройки

| Параметр | Описание |
|---|---|
| Тип события | Глобальный вайп, вайп карты, вайп блок, глобальное событие, пользовательское событие |
| Название | Отображаемое название события в календаре |
| Описание | Дополнительная информация о событии |
| Цвет | HEX-цвет маркера события в календаре (с поддержкой прозрачности) |
| Иконка | Ссылка на пользовательское изображение иконки события |

### Настройка даты

| Режим | Описание |
|---|---|
| Один раз | Указываются конкретные дата и время |
| Повторять | Настраивается расписание повтора по дням, неделям, месяцам или годам |

### Настройка повтора

При выборе режима "Повторять" доступны:

| Параметр | Описание |
|---|---|
| Месяцы | Выбор месяцев, в которых происходит событие |
| Дни месяца | Выбор конкретных чисел |
| Дни недели | Выбор дней недели для повтора |
| Интервал недель | Выбор номера недели в месяце (первая, вторая и т.д.) |
| Время | Часы и минуты проведения события |

## Рекомендации

- Всегда проверяйте корректность часового пояса, чтобы даты в календаре совпадали с реальным временем сервера.
- Используйте пользовательские события для анонсов рейд-найтов, турниров и других серверных ивентов.
- Для каждого типа события рекомендуется задать уникальный цвет, чтобы игроки легко различали их на календаре.

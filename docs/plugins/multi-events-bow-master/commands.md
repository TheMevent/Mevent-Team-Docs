# Команды и права

## Команды

### Команды администратора

| Команда | Описание | Право |
|---|---|---|
| `bowmaster.start` | Запустить ивент Bow Master | `MultiEventsBowMaster.start` |
| `bowmaster.stop` | Остановить текущий ивент | `MultiEventsBowMaster.stop` |

:::info Примечание
Ивент также может быть запущен автоматически через систему MultiEvents по расписанию.
:::

### Консольные команды (UI)

| Команда | Описание |
|---|---|
| `UI_BowMaster hide` | Скрыть панель ивента с экрана (вызывается через UI) |

## Права (Permissions)

| Право | Описание |
|---|---|
| `MultiEventsBowMaster.start` | Право на запуск ивента |
| `MultiEventsBowMaster.stop` | Право на остановку ивента |

### Настройка прав

```
oxide.grant group admin MultiEventsBowMaster.start
oxide.grant group admin MultiEventsBowMaster.stop
```

## Команды игроков

У обычных игроков нет специальных команд. Участие в ивенте автоматическое -- достаточно убивать сущностей (игроков или животных) с помощью лука или арбалета во время активного ивента.

## Механика начисления очков

- Учитываются только убийства с типом урона **Arrow** (стрелы)
- Засчитываются убийства **игроков** (BasePlayer) и **животных** (BaseAnimalNPC)
- Каждое убийство стрелой приносит 1 очко

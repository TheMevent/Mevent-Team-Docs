# Часто задаваемые вопросы (FAQ)

## 1. Как создать новый тип лампы?

В конфигурации в массиве `"Genies"` добавьте новый объект с уникальным `"ID"`. Скопируйте структуру существующей лампы и измените параметры:

```json
{
 "ID": "vip_lamp",
 "Enabled": true,
 "Permission": "genie.vip",
 "Lamp": {
 "DisplayName": "VIP Lamp",
 "ShortName": "xmas.present.small",
 "Skin": 2540200362
 },
 "Awards": [...]
}
```

---

## 2. Как выдать лампу игроку?

Используйте консольную команду:
```
genie.give <steamid> <genieId>
```

Например:
```
genie.give 76561198012345678 default
```

---

## 3. Как настроить шансы выпадения наград?

Каждая награда имеет параметр `"Chance"` от 0 до 100. Чем выше значение, тем больше шанс выпадения:
- `70` -- высокий шанс (70%)
- `50` -- средний шанс (50%)
- `10` -- редкий предмет (10%)
- `1` -- очень редкий предмет (1%)

---

## 4. Можно ли сделать так, чтобы лампы выпадали из контейнеров?

Да. Лампы могут быть настроены для выпадения из лут-контейнеров через параметр `"Drop"` в конфигурации. Это позволяет лампам появляться в ящиках, бочках и других контейнерах.

---

## 5. Как ограничить использование лампы только VIP-игрокам?

Укажите право в настройках лампы:
```json
{
 "Permission": "genie.vip"
}
```

Затем выдайте право:
```
oxide.grant group vip genie.vip
```

Игроки без этого права не смогут использовать лампу данного типа.

---

## 6. Интерфейс не открывается при использовании лампы. Что делать?

Возможные причины:
1. **ImageLibrary не установлен** -- установите плагин ImageLibrary.
2. **Лампа выключена** -- проверьте `"Enabled": true` в конфигурации.
3. **Нет права** -- если указан Permission, убедитесь, что у игрока есть соответствующее право.
4. **Неправильный скин** -- убедитесь, что скин предмета совпадает с настройками.

---

## 7. Как изменить время открытия лампы?

Измените параметр `"Delay"` в настройках лампы. Значение указывается в секундах:
- `3` -- быстрое открытие
- `5` -- стандартное (по умолчанию)
- `10` -- медленное, более драматичное

---

## 8. Можно ли выдавать команды как награду?

Да. Используйте тип награды `"Command"`:

```json
{
 "Type": "Command",
 "Command": "deposit %steamid% 500",
 "Amount": 1,
 "Image": "https://example.com/coin.png",
 "Title": "500 монет",
 "Chance": 30
}
```

---

## 9. Как отключить звуковые эффекты?

Установите пустые строки для параметров эффектов:
```json
{
 "OpeningEffect": "",
 "ProgressEffect": "",
 "FinishEffect": ""
}
```

---

## 10. Лампы стакаются в инвентаре. Как это отключить?

В настройках лампы установите `"ControlStack": false`. По умолчанию `"ControlStack": true`, что позволяет плагину контролировать стакание ламп.

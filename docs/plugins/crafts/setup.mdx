# Установка Crafts

## Шаг 1: Установите зависимости

### ImageLibrary (обязательно)

1. Скачайте **ImageLibrary** с [umod.org](https://umod.org/plugins/image-library).
2. Поместите файл `ImageLibrary.cs` в папку `oxide/plugins/`.
3. Убедитесь, что плагин загрузился без ошибок.

:::warning Важно
Без ImageLibrary плагин Crafts **не будет работать**. В консоли появится ошибка: `IMAGE LIBRARY IS NOT INSTALLED.`
:::

### Дополнительные зависимости (по желанию)

| Плагин | Назначение |
|---|---|
| **Notify** | Красивые уведомления вместо сообщений в чате |
| **SpawnModularCar** | Спавн модульных автомобилей через крафт |
| **LangAPI** | Мультиязычная поддержка интерфейса |

## Шаг 2: Установите плагин

1. Скопируйте файл `Crafts.cs` в папку `oxide/plugins/`.
2. Плагин загрузится автоматически.
3. Проверьте консоль на наличие ошибок.

Файл конфигурации будет создан автоматически:
```
oxide/config/Crafts.json
```

## Шаг 3: Настройте права доступа

Выдайте права администраторам для редактирования крафтов:

```
oxide.grant group admin crafts.edit
oxide.grant group admin crafts.setworkbench
```

Если в конфигурации указано право для использования меню, выдайте его всем игрокам:

```
oxide.grant group default crafts.use
```

## Шаг 4: Проверьте работу

1. Зайдите на сервер.
2. Введите `/craft` в чат.
3. Должно открыться меню крафта с категориями и рецептами.

## Шаг 5: Настройте категории и рецепты

Откройте файл `oxide/config/Crafts.json`.

### Настройка категорий

Каждая категория содержит список крафтов и может иметь своё право доступа:

```json
"Categories": [
 {
 "Enabled": true,
 "Permission": "",
 "Title": "Vehicles",
 "Crafts": [...]
 }
]
```

### Настройка рецепта

Каждый рецепт включает:
- **Название и изображение** -- для отображения в интерфейсе
- **Тип крафта** -- Command, Vehicle, Item, Recycler, ModularCar
- **Требуемые ресурсы** -- список предметов для крафта
- **Уровень верстака** -- None, One, Two, Three
- **Право доступа** -- индивидуальное для каждого рецепта

## Шаг 6: Перезагрузите плагин

```
oxide.reload Crafts
```

## Обновление плагина

1. Замените файл `Crafts.cs` новой версией.
2. Плагин автоматически обновит конфигурацию.
3. Проверьте консоль на наличие ошибок.

:::note Примечание
Рекомендуется сохранять резервную копию конфигурации перед обновлением.
:::

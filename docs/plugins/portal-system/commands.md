# Команды и права

## Команды чата

| Команда | Описание | Право |
|---|---|---|
| `/portal` | Открыть меню управления порталами | `portalsystem.use` |
| `/buyportal` | Купить портал за ресурсы/валюту | `portalsystem.buy` |
| `/giveportal [ник] [количество]` | Выдать порталы игроку | `portalsystem.give` |
| `/spawnportal [ник] [количество]` | Заспавнить порталы (админ) | `portalsystem.spawn` |

## Консольные команды

| Команда | Описание |
|---|---|
| `giveportal [ник/steamid] [количество]` | Выдать порталы из серверной консоли |

## Интерфейс портала

При подходе к порталу доступны действия через визуальное меню:

| Действие | Описание |
|---|---|
| Переименовать | Изменить название портала |
| Связать | Связать портал с другим порталом |
| Шаринг | Настроить доступ (тиммейты, друзья, клан) |
| Маркер на карте | Включить/отключить маркер |
| Цвет маркера | Выбрать цвет маркера на карте |
| Скин портала | Сменить визуальный скин |
| Удалить | Удалить портал |
| Показывать текст | Включить/выключить текст имени |

## Права (Permissions)

| Право | Описание |
|---|---|
| `portalsystem.use` | Базовое использование порталов |
| `portalsystem.place` | Размещение порталов |
| `portalsystem.buy` | Покупка порталов |
| `portalsystem.give` | Выдача порталов другим |
| `portalsystem.spawn` | Спавн порталов (админ) |
| `portalsystem.admin` | Администрирование |
| `portalsystem.free` | Бесплатная телепортация |
| `portalsystem.freeuse` | Бесплатное использование |
| `portalsystem.limit.N` | Лимит порталов (N = 4, 6, 10) |
| `portalsystem.vip1` | VIP-кулдаун (5 секунд) |
| `portalsystem.vip2` | VIP-кулдаун (2 секунды) |

:::tip Совет
Лимиты порталов задаются в конфигурации (`"Extra portal limits from permissions"`). По умолчанию: 2, 4, 6, 10.
:::

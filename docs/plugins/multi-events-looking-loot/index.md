# Looking Loot

## Что делает этот режим?

**Looking Loot** — это режим ивента для Multi Events, в котором игроки соревнуются в поиске и сборе лута из контейнеров на карте. Каждый собранный предмет приносит очки, причём разные предметы могут иметь разную ценность. Игрок, набравший наибольшее количество очков за отведённое время, побеждает.

## Ключевые возможности

- Автоматический подсчёт очков за каждый собранный предмет
- Настраиваемая таблица ценности предметов (каждый предмет может иметь свой множитель очков)
- Базовый множитель очков по умолчанию для предметов, не указанных в таблице
- Таблица лидеров в реальном времени
- Гибкая система наград по местам в рейтинге
- Интеграция с Multi Events Inventory и Multi Events Leaderboard
- Поддержка уведомлений через Notify / UINotify
- Совместимость с Oxide и Carbon

## Требования

- **Multi Events** (основной плагин) — обязательно
- **Image Library** — рекомендуется (для отображения иконок)
- **Notify** или **UINotify** — опционально (для всплывающих уведомлений)
- **Multi Events Inventory** — опционально (для выдачи наград через инвентарь ивентов)
- **Multi Events Leaderboard** — опционально (для ведения глобальной статистики)

## Быстрый старт

1. Установите плагин **Multi Events** (если ещё не установлен)
2. Поместите файл режима в папку `oxide/plugins` (или `carbon/plugins`)
3. Перезагрузите сервер или загрузите плагин командой в консоли
4. Режим автоматически зарегистрируется в Multi Events
5. Настройте таблицу ценности предметов и награды в конфигурации (см. [Конфигурация](./configuration))

## Как проходит ивент

1. **Начало** — Multi Events запускает режим Looking Loot. Всем игрокам отображается UI-уведомление о начале ивента
2. **Соревнование** — игроки ищут и собирают лут из ящиков, бочек и других контейнеров по всей карте. Каждый предмет приносит очки в зависимости от его ценности
3. **Таблица лидеров** — в правой части экрана отображается актуальный рейтинг с именами игроков и набранными очками
4. **Окончание** — по истечении времени система определяет победителей и раздаёт награды

## Устранение неполадок

| Проблема | Решение |
|---|---|
| Ивент не запускается | Убедитесь, что основной плагин Multi Events установлен и загружен |
| Очки не начисляются | Проверьте, что предмет есть в таблице ценности или установлен ненулевой базовый множитель (`Default Score`) |
| UI не отображается | Проверьте, установлен ли Image Library. Убедитесь, что игрок не скрыл уведомления |
| Награды не выдаются | Проверьте настройки наград в конфигурации. Если включён Multi Events Inventory, убедитесь, что он установлен |

# Команды и права

## Команды чата

У плагина Genie нет команд чата для обычных игроков. Игроки используют лампу как обычный предмет в инвентаре.

## Консольные команды

| Команда | Описание | Кто может использовать |
|---|---|---|
| `genie.give <steamid> <genieId>` | Выдать лампу указанному игроку | Только администраторы |

### Примеры использования

Выдать лампу с ID "default" игроку:
```
genie.give 76561198012345678 default
```

Выдать лампу с ID "vip_lamp" игроку:
```
genie.give 76561198012345678 vip_lamp
```

:::warning Внимание
Если указан несуществующий ID лампы или игрок не найден, в консоли появится сообщение об ошибке.
:::

## Права (Permissions)

Права настраиваются индивидуально для каждого типа лампы в конфигурации:

| Параметр | Описание |
|---|---|
| Permission | Право, необходимое для использования данного типа лампы |

Если поле `Permission` оставлено пустым (`""`), лампа доступна всем игрокам.

### Пример настройки прав

Если в конфигурации указано:
```json
{
 "Permission": "genie.vip"
}
```

Выдайте право:
```
oxide.grant group vip genie.vip
```

## Игровой процесс

1. Игрок получает лампу (через админ-команду, лут-контейнер или другие плагины).
2. Лампа выглядит как обычный предмет с кастомным скином.
3. При использовании предмета открывается интерфейс.
4. Игрок нажимает "Потри лампу" для начала открытия.
5. Отображается анимация прогресса с эффектами.
6. По завершении игрок получает случайную награду с учётом шансов.
7. Награда выдаётся автоматически.

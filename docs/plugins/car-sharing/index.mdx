# Car Sharing

## Что делает этот плагин?

**Car Sharing** позволяет вашему серверу добавить полноценную систему каршеринга, в которой игроки могут арендовать автомобили за внутриигровую валюту с поминутной оплатой. Плагин поддерживает настраиваемые классы машин, NPC-ботов для взаимодействия, спидометр и красивый пользовательский интерфейс.

## Ключевые возможности

- Аренда модульных автомобилей с поминутной тарификацией
- Настраиваемые классы машин (Luxe, Comfort и др.)
- Интеграция с NPC-ботами (HumanNPC) для точек аренды
- Встроенный спидометр с отображением скорости и оставшегося времени
- Поддержка экономических плагинов (Economics, ServerRewards, BankSystem) и предметов в качестве валюты
- Гибкая система прав доступа для каждой машины
- Автоматическая выдача предмета вызова транспорта с настраиваемым скином
- Поддержка уведомлений через Notify / UINotify

## Требования

| Зависимость | Обязательна | Описание |
|---|---|---|
| ImageLibrary | Да | Загрузка и отображение изображений в интерфейсе |
| SpawnModularCar | Да | Создание модульных автомобилей с заданными модулями |
| Economics / ServerRewards | Нет | Плагин экономики для оплаты аренды |
| Notify / UINotify | Нет | Уведомления для игроков |
| HumanNPC | Нет | NPC-боты для точек аренды |

## Быстрый старт

1. Установите плагин `CarSharing.cs` в папку `oxide/plugins`.
2. Убедитесь, что установлены зависимости **ImageLibrary** и **SpawnModularCar**.
3. Перезагрузите сервер или выполните `oxide.reload CarSharing`.
4. Отредактируйте конфигурационный файл `oxide/config/CarSharing.json`, добавив нужные машины и NPC-ботов.
5. Используйте команду `/rent` или `/carsharing` для открытия меню аренды.

## Права доступа

| Право | Описание |
|---|---|
| `carsharing.use` (настраиваемое) | Общее право на использование системы каршеринга |
| `carsharing.vip` (настраиваемое) | Право доступа к определённым классам машин (настраивается для каждой машины) |

:::info
Название права задаётся в конфигурации. Если поле `Permission` оставлено пустым, машина доступна всем игрокам.
:::

## Команды игроков

| Команда | Описание |
|---|---|
| `/rent` | Открыть меню каршеринга |
| `/carsharing` | Открыть меню каршеринга (альтернативная команда) |

**Пример использования:**
```
/rent
```
Откроется меню с выбором доступных машин, где можно увидеть класс, среднюю скорость и стоимость аренды.

## Команды администратора

| Команда | Описание |
|---|---|
| Индивидуальные команды выдачи (напр. `share.ferrari`) | Выдать конкретную машину игроку |

**Пример:**
```
share.ferrari
```
Каждая машина может иметь собственную команду выдачи, которая настраивается в конфигурации.

## Устранение неполадок

| Проблема | Решение |
|---|---|
| Плагин не работает, ошибка в консоли | Убедитесь, что установлены **ImageLibrary** и **SpawnModularCar** |
| Меню не открывается | Проверьте, есть ли у игрока необходимое право доступа (если настроено в конфигурации) |
| Машина не спавнится | Убедитесь, что модули автомобиля корректно указаны в настройках модульной машины |
| Баланс не списывается | Проверьте настройки экономики: плагин, хуки и имя предмета должны быть указаны корректно |
| NPC не открывает меню | Проверьте, что ID NPC-бота корректно указан в секции `Bots` конфигурации |
| Спидометр не отображается | Плагин автоматически показывает спидометр при аренде; убедитесь, что игрок сидит в арендованной машине |

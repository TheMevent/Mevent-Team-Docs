# Ultimate Leaderboard Configuration

The configuration file is located at: `oxide/config/UltimateLeaderboard.json`

## Main Parameters

| Parameter | Default | Description |
|---|---|---|
| Permission to use plugin | `""` | Access permission (empty -- available to all) |
| Commands | `["leaderboard", "stats"]` | Chat commands |
| Work with Notify? | `true` | Use Notify |
| Enable Offline Image Mode | `false` | Offline image mode |
| ServerPanel Template | `Fullscreen` | ServerPanel template |
| Wipe data on new save | `true` | Wipe data on new map save |
| Discord webhook for log rewards | `""` | Webhook for reward logging |

## Performance Settings

| Parameter | Default | Description |
|---|---|---|
| Leaderboard Cache Interval (seconds) | `600` | Cache update interval (seconds) |
| Leaderboard Cache Batch Size | `10` | Batch size for caching |
| Cooldown between actions (in seconds) | `0.2` | Cooldown between actions |
| Save all player stats on server save | `false` | Save all data on server save |
| Enable real-time stats updates to database | `false` | Update database in real time |
| Real-time played time stats update interval | `10` | Play time update interval (seconds) |
| Show only online players in search | `true` | Show only online players in search |

## Tracking Settings

| Parameter | Default | Description |
|---|---|---|
| Count NPC kills as player kills | `false` | Count NPC kills as player kills |
| Count damage/kills to teammates | `true` | Track damage/kills to teammates |
| Count damage/kills to clan members | `true` | Track damage/kills to clan members |
| Count damage/kills to friends | `true` | Track damage/kills to friends |

## Naked Player Settings

| Parameter | Default | Description |
|---|---|---|
| Enabled | `false` | Enable naked player settings |
| AllowedItems | Clothing list | Allowed items for "naked" status |
| MaxAliveTimeSeconds | `0` | Max alive time (0 -- no limit) |

## Data Storage

| Parameter | Default | Description |
|---|---|---|
| Storage Type | `SQLite` | Storage type: `SQLite` or `MySQL` |

### SQLite

```json
{
 "SQLite": {
 "DatabaseName": "players.db"
 }
}
```

### MySQL

```json
{
 "MySQL": {
 "Host": "localhost",
 "Port": 3306,
 "DatabaseName": "rust",
 "Username": "root",
 "Password": ""
 }
}
```

## Loot Settings

Point accrual system for actions:

| Parameter | Description |
|---|---|
| Enabled | Enable the loot system |
| Loots | List of entries with types and points |

Loot types:
- `Kill` -- kill
- `Death` -- death
- `Gather` -- resource gathering
- `LootItems` -- looting containers

## Tabs

Tabs are configured in the `Tabs` array. Each tab contains:

| Parameter | Description |
|---|---|
| Name | Tab name |
| Blocks | List of statistics blocks |

Block types: `Profile`, `Leaderboard`, `HitRate`, `Building`, and others.

## Profile Settings

| Parameter | Default | Description |
|---|---|---|
| Enable click to open profile in leaderboard table | `false` | Open profile on click |
| Delay between opening player profiles (in seconds) | `5` | Delay between opening profiles |

## Privacy

| Parameter | Default | Description |
|---|---|---|
| Disable IP address logging | `false` | Disable IP logging for GDPR |
| Show debug information | `false` | Show debug information in the console |

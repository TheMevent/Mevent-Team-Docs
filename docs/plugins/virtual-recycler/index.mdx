# Virtual Recycler

## Что делает этот плагин?

**Virtual Recycler** позволяет игрокам открывать переработчик (Recycler) через чат-команду в любом месте, без необходимости искать стационарный переработчик на карте. Игрок вводит команду, перед ним появляется виртуальный переработчик, который можно использовать как обычный. Скорость переработки настраивается через систему прав, что позволяет создавать VIP-уровни с ускоренной переработкой.

## Ключевые возможности

- Открытие виртуального переработчика через чат-команду
- Настраиваемая скорость переработки для каждого права
- VIP-система с ускоренной переработкой
- Автозапуск переработчика при открытии (опционально)
- Автоматический возврат предметов при закрытии
- Отдельные скорости для стационарных переработчиков
- Поддержка Notify для уведомлений
- Интеграция с RaidableBases (право для использования в RaidableBases)

## Требования

| Зависимость | Обязательность | Описание |
|---|---|---|
| **Notify** / **UINotify** | По желанию | Уведомления |
| **RaidableBases** | По желанию | Интеграция с рейдовыми базами |
| Oxide / uMod / Carbon | Обязательно | Совместим с Oxide, uMod и Carbon |

## Быстрый старт

1. Скопируйте файл `VirtualRecycler.cs` в папку `oxide/plugins/`.
2. Перезагрузите сервер или выполните `oxide.reload VirtualRecycler`.
3. Выдайте право: `oxide.grant group default virtualrecycler.use`.
4. Введите `/vrec` в чат для открытия виртуального переработчика.

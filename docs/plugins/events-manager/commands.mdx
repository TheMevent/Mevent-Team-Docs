# Команды

## Команды чата

| Команда | Право доступа | Описание |
|---|---|---|
| `/em` | `EventsManager.use` | Открыть визуальную панель управления |

:::note Настраиваемые команды
Команда `/em` является командой по умолчанию. Вы можете изменить или добавить другие команды в конфигурации (параметр "Commands").
:::

## Консольные команды

| Команда | Описание |
|---|---|
| `eventmanager.reset.default` | Сбросить все данные (ивенты, расписание, категории) к значениям по умолчанию. Только из серверной консоли. |

## Управление через визуальную панель

После открытия панели через `/em` вам доступны следующие действия:

### Управление ивентами

- **Добавить ивент** — создайте новый ивент с указанием названия, команды запуска, плагина и категории создателя
- **Редактировать ивент** — измените параметры существующего ивента
- **Удалить ивент** — полностью удалите ивент из системы
- **Запустить вручную** — немедленно запустите ивент из панели

### Управление расписанием

- **Создать расписание** — добавьте новое расписание с выбором типа (фиксированное время, интервал или случайный)
- **Редактировать расписание** — измените параметры существующего расписания
- **Удалить расписание** — удалите расписание из списка

### Управление категориями

- **Добавить категорию** — создайте новую категорию создателей
- **Редактировать категорию** — переименуйте существующую категорию

## Права доступа

| Право | Описание |
|---|---|
| `EventsManager.use` | Полный доступ к панели управления ивентами |

## Пример использования

### Создание ивента с автозапуском

1. Откройте панель командой `/em`
2. Перейдите на вкладку нужной категории
3. Нажмите **"Добавить ивент"**
4. Укажите название, команду запуска и плагин
5. Сохраните ивент
6. Нажмите на ивент для редактирования
7. Создайте расписание (например: каждые 60 минут, при 10+ игроках онлайн)
8. Сохраните расписание

### Запуск ивента вручную

1. Откройте панель командой `/em`
2. Найдите нужный ивент
3. Нажмите кнопку **"Запустить"** рядом с расписанием

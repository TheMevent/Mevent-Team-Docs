# Конфигурация Helpful Supply

Файл конфигурации расположен по пути: `oxide/config/HelpfulSupply.json`

## Структура конфигурации

Конфигурация содержит массив `"Supplies"` с настройками каждого типа сигнальной шашки.

## Настройки типа шашки

| Параметр | Описание |
|---|---|
| Name | Уникальное имя типа (используется в командах) |
| Item | Настройки предмета (название, shortname, скин) |
| Loot | Настройки содержимого аирдропа |
| Drop | Настройки выпадения из контейнеров |

## Настройки предмета (Item)

| Параметр | Описание |
|---|---|
| Display Name | Отображаемое название предмета |
| ShortName | ShortName предмета (всегда `supply.signal`) |
| Skin | Уникальный скин для визуального различия |

## Настройки лута (Loot)

### Доступ (Access)

| Параметр | По умолчанию | Описание |
|---|---|---|
| AllPlayers | `true` | Все игроки могут открыть аирдроп |
| Teammates | `true` | Тиммейты могут открыть аирдроп |
| Friends | `true` | Друзья (плагин Friends) могут открыть |
| Clanmates | `true` | Сокланы (плагин Clans) могут открыть |

### Скорость сброса (Speed)

| Параметр | По умолчанию | Описание |
|---|---|---|
| Enabled | `false` | Использовать кастомную скорость сброса |
| DropVelocity | `120` | Скорость падения аирдропа |
| DropGroundDistance | `120` | Дистанция от земли для раскрытия парашюта |
| PlaneFlightHeight | `250` | Высота полёта самолёта |
| PlaneSpeedMultiplier | `1` | Множитель скорости самолёта |

### Содержимое (Items)

Каждый предмет в содержимом имеет:

| Параметр | Описание |
|---|---|
| ShortName | ShortName предмета |
| Skin | Скин предмета (0 -- стандартный) |
| Amount | Количество (Min/Max) |
| Chance | Шанс выпадения (0-100) |

### Количество предметов (Amount)

| Параметр | Описание |
|---|---|
| Min | Минимальное количество предметов в аирдропе |
| Max | Максимальное количество предметов в аирдропе |

## Настройки выпадения из контейнеров (Drop)

| Параметр | Описание |
|---|---|
| Enabled | Включить выпадение из контейнеров |
| Containers | Список контейнеров с шансами |

### Настройки контейнера

| Параметр | Описание |
|---|---|
| Prefab | Путь к префабу контейнера |
| Chance | Шанс выпадения (в процентах) |
| Amount | Количество (Min/Max) |

# Команды и права

## Команды

| Команда | Описание | Право |
|---|---|---|
| `operation.start` | Запустить событие вручную | `operationfortress.start` |
| `operation.stop` | Остановить текущее событие | `operationfortress.stop` |

:::info Примечание
Команды выполняются через чат или серверную консоль. При выполнении из консоли проверка прав не требуется.
:::

## Права (Permissions)

| Право | Описание |
|---|---|
| `operationfortress.start` | Разрешает запуск события |
| `operationfortress.stop` | Разрешает остановку события |

## Механика события

1. **Подготовка** -- объявление о начале события с обратным отсчётом
2. **Активная фаза** -- игроки штурмуют крепость и активируют вышки
3. **Захват** -- после активации нужного количества вышек событие завершается
4. **Награды** -- победитель получает настроенные награды

## Блокировка команд

Во время события определённые команды блокируются для участников:

- `tpr` -- запрос телепортации
- `home` -- телепортация домой
- `tpa` -- принятие телепортации

Список заблокированных команд настраивается в конфигурации.

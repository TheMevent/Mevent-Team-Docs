# Конфигурация Wipe Block

Конфигурационный файл расположен по пути: `oxide/config/WipeBlock.json`

## Основные параметры

| Параметр | Тип | По умолчанию | Описание |
|---|---|---|---|
| ServerPanel Template (V1, V2) | строка | `V1` | Шаблон отображения в серверном меню (V1 или V2) |
| Commands | массив | `["block", "wipeblock"]` | Чат-команды для открытия меню блокировок |
| Work with Notify? | булево | `true` | Использовать Notify/UINotify для уведомлений |
| Time Indent (seconds) | число | `0` | Смещение времени блокировки в секундах (позволяет ускорить или замедлить разблокировку) |
| Prevent to use blocked items | булево | `true` | Запретить использование заблокированных предметов (надевание, перезарядка, установка в турели) |
| Prevent to craft blocked items | булево | `false` | Запретить крафт заблокированных предметов |

## Настройка категорий и предметов

Раздел "Settings" содержит массив категорий. Каждая категория включает:

| Параметр | Описание |
|---|---|
| Lang Key | Ключ перевода для названия категории (например, "Weapons", "Explosives", "Attire") |
| Items | Словарь, где ключ -- время блокировки в секундах, значение -- список предметов |

### Формат предмета

| Параметр | Описание |
|---|---|
| ShortName | Системное название предмета в Rust (например, `rifle.ak`, `explosive.timed`) |
| Skin | ID скина предмета (0 для стандартного) |

### Пример категории

```json
{
 "Lang Key": "Weapons",
 "Items": {
 "3600": [
 { "ShortName": "pistol.revolver", "Skin": 0 },
 { "ShortName": "shotgun.double", "Skin": 0 }
 ],
 "7200": [
 { "ShortName": "pistol.semiauto", "Skin": 0 },
 { "ShortName": "pistol.python", "Skin": 0 }
 ],
 "86400": [
 { "ShortName": "lmg.m249", "Skin": 0 }
 ]
 }
}
```

В этом примере:
- Револьвер и двустволка разблокируются через 1 час после вайпа
- Полуавтомат и Питон -- через 2 часа
- М249 -- через 24 часа

## Настройка интерфейса

### Раздел Interface

| Параметр | Описание |
|---|---|
| Gradients | Массив из 101 HEX-цвета для отображения прогресс-бара разблокировки. Зелёный -- разблокировано, жёлтый -- частично, красный -- заблокировано |

### Раздел OnScreen Settings

| Параметр | Тип | По умолчанию | Описание |
|---|---|---|---|
| Show on screen? | булево | `true` | Показывать виджет блокировки на экране |
| AnchorMin | строка | `"1 1"` | Минимальная точка привязки виджета |
| AnchorMax | строка | `"1 1"` | Максимальная точка привязки виджета |
| OffsetMin | строка | `"-150 -40"` | Минимальное смещение виджета |
| OffsetMax | строка | `"-20 -10"` | Максимальное смещение виджета |

## Языковые настройки

Плагин поддерживает русский и английский языки. Ключи переводов для категорий задаются в поле "Lang Key" каждой категории. Для добавления нового языка категории отредактируйте файл языка плагина в `oxide/lang/`.

## Рекомендации по настройке

- Для тематических серверов с короткими вайпами используйте таймеры от 1 до 6 часов.
- Для серверов с длинными вайпами (неделя+) можно выставлять блокировки до 48-72 часов.
- Включите запрет крафта (`Prevent to craft blocked items`), чтобы игроки не тратили ресурсы на недоступные предметы.
- Настройте уведомления через Notify для лучшего пользовательского опыта.

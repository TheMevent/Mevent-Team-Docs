# Команды Vehicle Buy

## Команды игроков

### Открытие магазина

| Команда | Описание |
|---|---|
| `/vehiclebuy` | Открыть магазин транспорта |
| `/vb` | Краткая команда для открытия магазина |
| `/vehicle` | Альтернативная команда |

**Пример:**
```
/vehiclebuy
```

После ввода команды откроется графическое меню с доступным транспортом. Игрок может выбрать нужный транспорт и нажать кнопку покупки.

:::info
Команды настраиваются в конфигурации. Администратор может изменить или добавить свои команды.
:::

### Отзыв транспорта (Recall)

| Команда | Описание |
|---|---|
| `/callback <ключ_транспорта>` | Отозвать купленный транспорт к текущему местоположению |

**Пример:**
```
/callback copter
```

Транспорт телепортируется к вашему местоположению. Обратите внимание:
- Транспорт не должен быть занят другим игроком
- Необходимо право `vehiclebuy.recall`
- Может потребоваться оплата за отзыв (настраивается)
- Отзыв невозможен в определённых локациях

### Подбор транспорта (Pickup)

| Команда | Описание |
|---|---|
| `/pickup` | Подобрать транспорт рядом с вами в инвентарь |

**Пример:**
```
/pickup
```

Подобранный транспорт превращается в предмет, который можно разместить позже. Условия:
- Необходимо находиться рядом с транспортом (дистанция настраивается, по умолчанию 5 метров)
- Необходимо право `vehiclebuy.pickup`
- В инвентаре должно быть свободное место
- Может потребоваться оплата за подбор

### Подбор молотком

Некоторые виды транспорта можно подобрать ударом молотка (если настроено). Для этого:
1. Возьмите молоток в руку.
2. Ударьте по транспорту 3 раза в течение 2 секунд.
3. Транспорт будет подобран в инвентарь.

:::note
Способ подбора (команда или молоток) настраивается для каждого типа транспорта отдельно параметром `Pickup type`.
:::

## Команды администратора

### Управление правами

```
oxide.grant user <SteamID> vehiclebuy.use
oxide.grant user <SteamID> vehiclebuy.free
oxide.grant user <SteamID> vehiclebuy.pickup
oxide.grant user <SteamID> vehiclebuy.recall
oxide.grant group default vehiclebuy.use
```

### Индивидуальные права транспорта

Если для конкретного транспорта указано индивидуальное право доступа:
```
oxide.grant user <SteamID> vehiclebuy.vip
oxide.grant group vip vehiclebuy.vip
```

:::tip
Право `vehiclebuy.free` позволяет покупать весь транспорт бесплатно -- полезно для тестирования или VIP-привилегий.
:::

## Покупка через торговые автоматы

Игроки также могут покупать транспорт через торговые автоматы, которые автоматически размещаются в мирных зонах:
- Bandit Camp
- Outpost
- Fishing Village (A, B, C)

Для покупки подойдите к торговому автомату в мирной зоне и выберите нужный товар. Оплата производится внутриигровыми предметами.

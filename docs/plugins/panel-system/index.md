# Panel System

## Что делает этот плагин?

**Panel System** позволяет вашему серверу отображать компактную информационную панель в углу экрана каждого игрока. На панели показываются количество игроков онлайн, игровое время, статус серверных событий (вертолёт, танк, аирдроп, карго и другие), баланс экономики и быстрые кнопки для выполнения команд. Игроки могут самостоятельно настраивать панель: менять размер, цвета и скрывать ненужные элементы.

## Ключевые возможности

- Игроки видят количество онлайн-игроков в реальном времени
- Игроки видят текущее игровое (или реальное) время
- Игроки видят статус серверных событий: вертолёт, танк Брэдли, аирдроп, CH47, карго, конвой, бронепоезд и другие
- Игроки видят свой баланс экономики (Economics, ServerRewards и др.)
- Игроки могут использовать быстрые кнопки (магазин, статистика, банк и др.)
- Игроки могут настраивать панель: менять размер элементов, цвета активных/неактивных индикаторов
- Игроки могут скрывать отдельные элементы панели или всю панель целиком
- Администраторы настраивают все элементы через конфигурацию
- Поддержка пользовательских событий от сторонних плагинов
- Совместимость с Oxide и Carbon

## Требования

| Зависимость | Обязательность | Описание |
|---|---|---|
| **ImageLibrary** | Обязательно | Загрузка и отображение иконок в панели |
| **Notify** / **UINotify** | По желанию | Красивые уведомления вместо чата |
| **WipeBlock** | По желанию | Отображение статуса вайп-блока на панели |
| Oxide / uMod / Carbon | Обязательно | Совместим с Oxide, uMod и Carbon |

## Быстрый старт

1. Установите **ImageLibrary** на сервер.
2. Скопируйте файл `PanelSystem.cs` в папку `oxide/plugins/`.
3. Перезагрузите сервер или выполните `oxide.reload PanelSystem`.
4. Панель автоматически появится у всех подключённых игроков.
5. Откройте файл `oxide/config/PanelSystem.json` для настройки элементов.

## Права доступа

Права доступа в Panel System настраиваются гибко -- каждый элемент панели может иметь собственное право. По умолчанию все права пустые, что означает доступ для всех.

| Элемент | Где задаётся право | Что контролирует |
|---|---|---|
| Логотип | Настройки логотипа | Кто видит логотип сервера |
| Онлайн | Настройки панели игроков | Кто видит количество онлайн |
| Время | Настройки панели времени | Кто видит игровое время |
| Спящие | Настройки панели спящих | Кто видит количество спящих |
| Кнопки | Настройки панели кнопок | Кто видит быстрые кнопки |
| Каждая кнопка | Настройки конкретной кнопки | Кто видит данную кнопку |
| Каждое событие | Настройки конкретного события | Кто видит индикатор события |
| Каждая экономика | Настройки конкретной экономики | Кто видит баланс данной валюты |
| Настройки панели | Кнопка настроек | Кто может открыть окно настройки панели |

:::info Примечание
Если поле права оставлено пустым, элемент виден всем игрокам. Чтобы ограничить доступ, укажите нужное право и зарегистрируйте его через Oxide.
:::

## Команды игроков

| Команда | Описание | Пример |
|---|---|---|
| `/panelsystem.buttons` | Показать/скрыть панель быстрых кнопок | `/panelsystem.buttons` |
| `/(команда скрытия)` | Скрыть или показать всю панель | `/panel hide` или `/panel show` |

:::info Примечание
Команда скрытия панели настраивается в конфигурации (параметр "Command" в секции "Hide Settings"). По умолчанию: `panel`. Функция должна быть включена (`"Enabled": true`).
:::

## Команды администратора

Плагин Panel System не имеет специальных команд для администраторов. Вся настройка выполняется через конфигурационный файл `oxide/config/PanelSystem.json`.

## Отслеживаемые события

Panel System автоматически отслеживает следующие серверные события:

| Событие | Описание |
|---|---|
| Вертолёт (Helicopter) | Патрульный вертолёт на карте |
| Танк Брэдли (Bradley) | Танк Брэдли на карте |
| Аирдроп (Airdrop) | Сброс припасов с самолёта |
| CH47 (Chinook) | Транспортный вертолёт CH47 |
| Карго (CargoShip) | Грузовое судно |
| Конвой (Convoy) | Событие "Конвой" |
| Бронепоезд (ArmoredTrain) | Бронированный поезд |
| Спутник (Sputnik) | Событие "Спутник" |
| Космическое событие (SpaceEvent) | Космическое событие |
| Воздушное событие (AirEvent) | Воздушное событие |
| Вайп-блок (WipeBlock) | Статус блокировки после вайпа |
| Спящие (Sleepers) | Количество спящих игроков |

Также поддерживаются пользовательские события от сторонних плагинов.

## Устранение неполадок

### Панель не отображается

**Симптомы:** Игроки не видят информационную панель.

**Причина:** ImageLibrary не установлена, или панель скрыта игроком.

**Решение:** Убедитесь, что ImageLibrary установлена. Попросите игрока ввести команду показа панели (например, `/panel show`). Проверьте, что элементы включены в конфигурации.

---

### Индикаторы событий не меняют цвет

**Симптомы:** Индикаторы всегда показывают один цвет, не реагируя на появление/исчезновение событий.

**Причина:** Плагин не получает информацию о событиях.

**Решение:** Убедитесь, что соответствующие события включены в конфигурации (`"Enabled": true`). Для пользовательских событий (Convoy, ArmoredTrain и др.) убедитесь, что плагин события установлен и вызывает нужные хуки.

---

### Баланс экономики не отображается

**Симптомы:** Индикатор экономики не показывает баланс или показывает 0.

**Причина:** Экономический плагин не установлен или неправильно настроен.

**Решение:** Убедитесь, что указанный плагин экономики установлен. Проверьте правильность названия плагина и функции баланса в конфигурации.

---

### Ошибка "IMAGE LIBRARY IS NOT INSTALLED."

**Симптомы:** В консоли многократно повторяется сообщение об ошибке.

**Причина:** ImageLibrary не установлена на сервере.

**Решение:** Скачайте и установите ImageLibrary. Перезагрузите Panel System.

---

### Панель мешает другим элементам интерфейса

**Симптомы:** Панель перекрывает другие плагины или элементы HUD.

**Причина:** Позиция панели совпадает с другими элементами.

**Решение:** Измените позицию панели через параметр "Panel Anchor" (верхний правый или верхний левый угол). Также можно изменить "Events Panel Anchor" для панели событий.

# Установка и настройка

## Системные требования

- Сервер Rust с установленным **Oxide** или **Carbon**

## Пошаговая установка

### Шаг 1: Загрузка плагина

Скопируйте файл `CaptureZone.cs` в директорию плагинов:

```
oxide/plugins/CaptureZone.cs
```

### Шаг 2: Перезагрузка

Перезагрузите сервер или выполните:
```
o.reload CaptureZone
```

### Шаг 3: Настройка конфигурации

Отредактируйте файл конфигурации при необходимости:
```
oxide/config/CaptureZone.json
```

### Шаг 4: Выдача предметов

Выдайте предметы захвата зоны игрокам:
```
give.capturezone SteamID64 количество
```

## Как работает захват территории

### Установка зоны

1. Игрок получает специальный предмет (шкаф с уникальным скином)
2. Игрок устанавливает его на карте, как обычный шкаф
3. Вокруг шкафа автоматически создаётся зона захвата с заданным радиусом
4. На карте появляется маркер (если включено)

### Сбор налога

Когда другой игрок добывает ресурсы внутри зоны захвата:
1. Плагин определяет, находится ли игрок в зоне
2. Проверяет, не является ли он другом, тиммейтом, кланмейтом или авторизованным в шкафу
3. Если проверки не пройдены, часть ресурсов (заданный процент) переносится в шкаф владельца
4. Игрок получает уведомление о потере ресурсов

### Обход налога

Налог не взимается с:
- **Тиммейтов** (встроенная система команд Rust)
- **Друзей** (через плагин Friends)
- **Кланмейтов** (через плагин Clans)
- **Авторизованных в шкафу** (игроки, добавленные в шкаф)
- **Игроков с правом игнорирования** (`capturezone.ignore`)

Каждый тип обхода можно включить или отключить в конфигурации.

## Настройка маркеров на карте

Маркеры зон отображаются на игровой карте. Настраиваемые параметры:

| Параметр | Описание |
|---|---|
| Enabled | Включить/отключить маркеры |
| Display Name | Текст маркера (поддерживает `{owner}` для имени владельца) |
| Radius | Размер маркера на карте |
| Alpha | Прозрачность маркера |
| Refresh Rate | Частота обновления маркера (в секундах) |
| Duration | Длительность отображения (0 = бесконечно) |
| Color 1 / Color 2 | Цвета маркера в формате HEX |

### Кто видит маркеры?

Видимость маркеров настраивается отдельно:

| Параметр | Описание |
|---|---|
| All Players | Все игроки видят маркеры |
| Owner | Только владелец видит свои маркеры |
| Teammates | Тиммейты видят маркер |
| Friends | Друзья видят маркер |
| Clanmates | Кланмейты видят маркер |
| Authorized in the cupboard | Авторизованные в шкафу видят маркер |
| Permission | Только игроки с указанным правом видят маркер |

## Настройка дропа из контейнеров

Специальный предмет захвата зоны может выпадать из обычных контейнеров с настраиваемой вероятностью:

| Параметр | Описание |
|---|---|
| Enabled | Включить/отключить дроп |
| Prefab name | Название контейнера |
| Min amount | Минимальное количество предметов |
| Max amount | Максимальное количество предметов |
| Chance | Вероятность дропа (0-100%) |

По умолчанию предмет может выпасть из:
- Стандартных ящиков (50% шанс)
- Бочек с лутом (5% шанс)

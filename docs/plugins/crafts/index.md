# Crafts

## Что делает этот плагин?

**Crafts** добавляет на сервер полноценную систему крафта с визуальным интерфейсом. Игроки могут открывать меню крафта через чат-команду и создавать транспорт (миникоптер, лодки, вертолёты), предметы, модульные автомобили, а также использовать встроенный переработчик. Крафт требует определённые ресурсы и может быть привязан к уровню верстака. Администраторы гибко настраивают категории, рецепты, права доступа и экономику.

## Ключевые возможности

- Игроки открывают визуальное меню крафта через чат-команду
- Поддержка крафта транспорта (миникоптер, лодки, RHIB, вертолёты и др.)
- Поддержка крафта предметов, модульных автомобилей и переработчиков
- Система категорий с правами доступа для каждой категории
- Привязка крафта к уровню верстака (1, 2, 3)
- Поддержка кастомных верстаков (Custom Workbench)
- Редактирование крафтов прямо из интерфейса (для администраторов)
- Интеграция с экономическими плагинами (Economics, ServerRewards)
- Поддержка Notify для уведомлений
- Совместимость с Oxide и Carbon

## Требования

| Зависимость | Обязательность | Описание |
|---|---|---|
| **ImageLibrary** | Обязательно | Загрузка и отображение изображений в интерфейсе |
| **Notify** / **UINotify** | По желанию | Красивые уведомления вместо чата |
| **SpawnModularCar** | По желанию | Спавн модульных автомобилей |
| **LangAPI** | По желанию | Мультиязычная поддержка |
| Oxide / uMod / Carbon | Обязательно | Совместим с Oxide, uMod и Carbon |

## Быстрый старт

1. Установите **ImageLibrary** на сервер.
2. Скопируйте файл `Crafts.cs` в папку `oxide/plugins/`.
3. Перезагрузите сервер или выполните `oxide.reload Crafts`.
4. Используйте команду `/craft` в чате для открытия меню крафта.
5. Настройте категории и рецепты в файле `oxide/config/Crafts.json`.

## Права доступа

| Право | Кому нужно | Что открывает |
|---|---|---|
| `crafts.edit` | Администраторам | Позволяет редактировать крафты через интерфейс |
| `crafts.setworkbench` | Администраторам | Позволяет устанавливать кастомные верстаки |
| `crafts.bypass` | Администраторам | Обходит проверки прав и верстака при крафте |
| Настраиваемое право | Игрокам | Право на использование меню (задаётся в конфигурации) |
| Права категорий | Игрокам | Индивидуальные права для каждой категории |
| Права крафтов | Игрокам | Индивидуальные права для каждого рецепта |

## Устранение неполадок

### Меню не открывается

**Симптомы:** Команда `/craft` не отображает интерфейс.

**Причина:** ImageLibrary не установлена или у игрока нет права доступа.

**Решение:** Установите ImageLibrary. Проверьте, что право доступа выдано или поле Permission в конфигурации пустое.

---

### Крафт недоступен -- "нет права"

**Симптомы:** Игрок видит рецепт, но не может скрафтить.

**Причина:** У игрока нет права на конкретный крафт или категорию.

**Решение:** Выдайте нужное право через `oxide.grant`.

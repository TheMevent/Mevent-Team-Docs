# Collection Resources

## Что делает этот режим?

**Collection Resources** — это режим ивента для Multi Events, в котором игроки соревнуются в добыче ресурсов. Каждый добытый ресурс приносит очки, а система множителей позволяет балансировать ценность разных инструментов, предметов и привилегий. Игрок, набравший наибольшее количество очков за отведённое время, побеждает.

## Ключевые возможности

- Автоматический подсчёт очков за добычу любых ресурсов (руда, дерево, камень, растения, сбор предметов)
- Множители для инструментов — разные инструменты дают разное количество очков
- Множители для типов ресурсов — настройка ценности конкретных предметов с учётом скина
- Множители по правам доступа — VIP-игроки могут получать больше очков
- Таблица лидеров в реальном времени
- Гибкая система наград по местам в рейтинге
- Интеграция с Multi Events Inventory и Multi Events Leaderboard
- Поддержка уведомлений через Notify / UINotify
- Право для настройки ивента через UI

## Требования

- **Multi Events** (основной плагин) — обязательно
- **Image Library** — рекомендуется (для отображения иконок)
- **Notify** или **UINotify** — опционально (для всплывающих уведомлений)
- **Multi Events Inventory** — опционально (для выдачи наград через инвентарь ивентов)
- **Multi Events Leaderboard** — опционально (для ведения глобальной статистики)

## Быстрый старт

1. Установите плагин **Multi Events** (если ещё не установлен)
2. Поместите файл режима в папку `oxide/plugins` (или `carbon/plugins`)
3. Перезагрузите сервер или загрузите плагин командой в консоли
4. Режим автоматически зарегистрируется в Multi Events
5. Настройте множители и награды в конфигурации (см. [Конфигурация](./configuration))

## Права доступа

| Право | Описание |
|---|---|
| `multieventscollectionresources.start` | Разрешение на запуск ивента вручную |
| `multieventscollectionresources.stop` | Разрешение на остановку ивента вручную |
| `multieventscollectionresources.edit` | Разрешение на редактирование настроек ивента через UI |
| Настраиваемые права множителей | Права для увеличенных множителей очков (например, VIP) |

Подробнее о командах см. в разделе [Команды](./commands).

## Как проходит ивент

1. **Начало** — Multi Events запускает режим Collection Resources. Всем игрокам отображается UI-уведомление
2. **Соревнование** — игроки добывают ресурсы по всей карте. Засчитывается добыча руды, камня, дерева, сбор растений и подбор предметов. Количество очков зависит от множителей инструмента, типа ресурса и привилегий игрока
3. **Таблица лидеров** — в правой части экрана отображается актуальный рейтинг с именами игроков и набранными очками
4. **Окончание** — по истечении времени система определяет победителей и раздаёт награды

## Устранение неполадок

| Проблема | Решение |
|---|---|
| Ивент не запускается | Убедитесь, что основной плагин Multi Events установлен и загружен |
| Очки не начисляются за определённый ресурс | Проверьте множители в конфигурации. Убедитесь, что множитель предмета не равен 0 |
| Множители прав не работают | Убедитесь, что права зарегистрированы и выданы игрокам. Проверьте правильность написания прав в конфигурации |
| UI не отображается | Проверьте, установлен ли Image Library. Убедитесь, что игрок не скрыл уведомления |

# Ultimate Cars

## Что делает этот плагин?

**Ultimate Cars** добавляет на сервер систему кастомных транспортных средств (джипов) с реалистичной физикой Unity. Транспорт использует Rigidbody и WheelCollider для симуляции движения. Администраторы могут спавнить джипы через чат-команду. Водитель управляет машиной через стандартные клавиши ввода (WASD), автомобиль поддерживает ускорение, торможение, поворот, стабилизацию и физику подвески.

## Ключевые возможности

- Кастомные транспортные средства на базе физики Unity (Rigidbody + WheelCollider)
- Реалистичное управление: ускорение, торможение, рулевое управление
- Настраиваемые параметры двигателя (крутящий момент, тормозной момент)
- Настраиваемая подвеска (пружина, демпфирование, ход)
- Стабилизация автомобиля (антикрен, антиопрокидывание)
- Конфигурация колёсных коллайдеров (позиции, радиус, трение)
- Сиденье водителя с точкой крепления
- Загрузка конфигурации из JSON-файла

## Требования

| Зависимость | Обязательность | Описание |
|---|---|---|
| Oxide / uMod / Carbon | Обязательно | Совместим с Oxide, uMod и Carbon |

## Быстрый старт

1. Скопируйте файл `UltimateCars.cs` в папку `oxide/plugins/`.
2. Создайте файл конфигурации `oxide/data/UltimateCars/JeepJP.json` с параметрами автомобиля.
3. Перезагрузите сервер или выполните `oxide.reload UltimateCars`.
4. Используйте `/spawnjeep` для спавна автомобиля (только администраторы).

# FAQ и решение проблем

## Частые вопросы

### 1. Как добавить новый сервер в список?

Откройте конфигурацию и добавьте новый объект в массив `"Servers"`. Укажите IP, Port и BmID (если есть). Перезагрузите плагин.

---

### 2. Что такое BmID и зачем он нужен?

BmID -- это ID вашего сервера на BattleMetrics. Он позволяет автоматически получать название, описание, онлайн и баннер сервера. Найдите ваш сервер на [battlemetrics.com](https://www.battlemetrics.com) и скопируйте ID из URL.

---

### 3. Как изменить интервал обновления информации?

Измените параметр `"Interval updates servers info"` в конфигурации. Значение указывается в секундах (минимум 30).

---

### 4. Как использовать кастомный баннер?

Укажите URL баннера в настройках сервера. Если баннер не указан, используется стандартный из параметра `"Default server banner"`.

---

## Решение проблем

### Список серверов пуст

**Причина:** Массив `"Servers"` в конфигурации пуст или серверы не отвечают.

**Решение:** Добавьте серверы в конфигурацию. Проверьте, что IP и Port указаны правильно.

---

### Информация о сервере не обновляется

**Причина:** BattleMetrics API недоступен или BmID указан неправильно.

**Решение:** Проверьте BmID. Убедитесь, что сервер имеет доступ к интернету.

---

### Баннер не загружается

**Причина:** URL баннера недоступен или ImageLibrary не установлена.

**Решение:** Проверьте URL баннера. Установите ImageLibrary для загрузки изображений.

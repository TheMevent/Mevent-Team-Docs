# Конфигурация Quarry Levels

Файл конфигурации расположен по пути: `oxide/config/QuarryLevels.json`

## Настройки сервера (Server Settings)

| Параметр | По умолчанию | Описание |
|---|---|---|
| Chat icon ID | `0` | ID аватара для сообщений в чате |

## Настройки уровней (Quarry Level Settings)

| Параметр | По умолчанию | Описание |
|---|---|---|
| Quarry max level | `5` | Максимальный уровень карьера |
| Pumpjack max level | `5` | Максимальный уровень насосной |
| Quarry capacity per level | `5` | Прибавка слотов хоппера за уровень |
| Pumpjack capacity per level | `5` | Прибавка слотов за уровень (насосная) |
| Enable Economics? | `false` | Использовать Economics для оплаты |
| Economics cost | `5000` | Стоимость улучшения (Economics) |
| Economics currency | `credits` | Валюта Economics |
| Enable ServerRewards? | `false` | Использовать ServerRewards для оплаты |
| ServerRewards cost | `5000` | Стоимость улучшения (ServerRewards) |
| ServerRewards currency | `RP` | Валюта ServerRewards |

## Настройки ресурсов (Deposit Settings)

| Параметр | По умолчанию | Описание |
|---|---|---|
| Metal ore unlock at | `3` | Уровень разблокировки металлической руды |
| Sulfure ore unlock at | `4` | Уровень разблокировки серной руды |
| HQM ore unlock at | `5` | Уровень разблокировки HQM руды |

## Настройки производительности (Quarry Production Settings)

| Параметр | Описание |
|---|---|
| Stones Production | Производительность камня |
| Metal Production | Производительность металлической руды |
| Sulfur Production | Производительность серной руды |
| HQM Production | Производительность HQM руды |
| Oil Production | Производительность нефти (насосная) |

## Настройки топлива (Quarry Options)

| Параметр | Описание |
|---|---|
| ProcessRateDissel | Скорость обработки на дизеле |
| WorkPointDissel | Рабочие очки на дизеле |
| ProcessRateFuel | Скорость обработки на Low Grade Fuel |
| WorkPointFuel | Рабочие очки на Low Grade Fuel |

## Настройки игроков (Player Settings)

| Параметр | По умолчанию | Описание |
|---|---|---|
| Can an unauthorized player toggling? | `false` | Запретить неавторизованным переключать карьер |
| Can an unauthorized player looting? | `false` | Запретить неавторизованным открывать хоппер |

## Настройки Survey (Survey Settings)

| Параметр | По умолчанию | Описание |
|---|---|---|
| Enable oil craters? | `false` | Включить нефтяные кратеры от Survey Charge |
| Oil craters chance | `10` | Шанс появления нефтяного кратера (%) |

## Настройки интерфейса

### Кнопка (Button Settings)

| Параметр | Описание |
|---|---|
| ButtonColor | Цвет кнопки (HEX) |
| ButtonOpacity | Прозрачность кнопки |
| ButtonFontColor | Цвет текста кнопки |

### Панель (Panel Settings)

| Параметр | Описание |
|---|---|
| PanelColor | Цвет фона панели |
| PanelOpacity | Прозрачность панели |
| PanelFontColor | Цвет текста панели |

# Часто задаваемые вопросы — Cases

### 1. Как добавить новый кейс?

Откройте конфигурационный файл `Cases.json` и добавьте новый кейс в массив "Cases". Укажите уникальный ID, название, изображение, стоимость и список предметов с шансами выпадения. После сохранения перезагрузите плагин.

### 2. Как настроить шансы выпадения предметов?

Каждый предмет в кейсе имеет параметр `Chance`. Чем выше значение — тем чаще предмет выпадает. Например, предмет с шансом 70 будет выпадать значительно чаще, чем предмет с шансом 5. Шансы не обязаны составлять 100% в сумме — они определяют относительную вероятность.

### 3. Как сделать бесплатный ежедневный кейс?

Настройте бонусный кейс в конфигурации:
- `Enabled: true`
- `Cost: 0`
- `CooldownTime: 86400` (24 часа в секундах)

Игроки смогут открывать его бесплатно один раз в сутки.

### 4. Как начислять кейсы за время, проведённое на сервере?

1. Установите плагин **PlaytimeTracker**
2. В конфигурации включите `Playtime Tracker Settings`
3. Настройте привязку времени к ID кейсов:
```json
"Cases": {
 "3600": 1,
 "14400": 2
}
```
В этом примере: после 1 часа игры игрок получает кейс ID 1, после 4 часов — кейс ID 2.

### 5. Можно ли использовать разные валюты для разных кейсов?

Да, каждый кейс поддерживает параметр Custom Currency. Включите его и настройте индивидуальную валюту для конкретного кейса, независимо от основной экономики.

### 6. Как выдать кейсы всем игрокам на сервере?

Используйте команду из серверной консоли:
```
givecase all <id_кейса> <количество>
```
Например: `givecase all 1 3` — выдаст 3 кейса с ID 1 всем онлайн-игрокам.

### 7. Как изменить внешний вид интерфейса?

В конфигурации измените параметр `Template`:
- `Fullscreen` — полноэкранный режим
- `InMenu` — встроенный в серверное меню

Также вы можете настроить цвета редкости для визуального отображения ценности предметов.

### 8. Можно ли добавить в кейс не только предметы, но и команды?

Да, каждый элемент кейса поддерживает тип `Command`. Укажите тип `"Command"` и в поле команды укажите консольную команду сервера. Используйте `%steamid%` для подстановки Steam ID игрока.

### 9. Как отключить анимацию рулетки?

В конфигурации установите `"Enable item scrolling?": false`. Предметы будут выдаваться мгновенно, без анимации прокрутки.

### 10. Данные кейсов сбрасываются при вайпе?

По умолчанию данные сохраняются между вайпами. Сами кейсы и их содержимое настраиваются в конфигурации и не зависят от вайпа. Данные о купленных кейсах игроков хранятся в файле данных.

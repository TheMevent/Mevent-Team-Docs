# Установка и настройка

## Системные требования

- Сервер Rust с установленным **Oxide** или **Carbon**
- Плагин **ImageLibrary** (только для Oxide)

## Пошаговая установка

### Шаг 1: Загрузка плагина

Скопируйте файл `EventsManager.cs` в директорию плагинов вашего сервера:

```
oxide/plugins/EventsManager.cs
```

### Шаг 2: Установка зависимостей

Если вы используете **Oxide**, убедитесь, что плагин **ImageLibrary** установлен и работает. Для **Carbon** это не требуется.

### Шаг 3: Загрузка плагина

Перезагрузите сервер или выполните команду в консоли:

```
o.reload EventsManager
```

### Шаг 4: Первоначальная настройка

При первом запуске плагин автоматически:
- Создаст файл конфигурации
- Загрузит список ивентов по умолчанию с поддержкой десятков популярных плагинов
- Загрузит список создателей (разработчиков ивентов)

### Шаг 5: Настройка прав доступа

Выдайте права доступа администраторам:

```
o.grant user SteamID64 EventsManager.use
o.grant group admin EventsManager.use
```

### Шаг 6: Открытие панели управления

Введите `/em` в игровом чате для доступа к визуальной панели управления.

## Настройка расписания

Через панель управления вы можете создавать расписания трёх типов:

### Фиксированное время (Static)

Ивент запускается в точное время (например, каждый день в 14:00).

- **Час** — от 0 до 24
- **Минута** — от 0 до 59
- **Дни недели** — можно выбрать конкретные дни (если не выбрано — ежедневно)
- **Смещение** — случайное смещение в минутах (от мин. до макс.)

### Повтор каждые N минут (Every)

Ивент запускается с фиксированным интервалом от последнего запуска.

- **Интервал** — количество минут между запусками

### Случайный интервал (Random)

Ивент запускается через случайное количество минут из заданного диапазона.

- **Минимум** — минимальный интервал в минутах
- **Максимум** — максимальный интервал в минутах

## Дополнительные условия запуска

Для каждого расписания можно задать:

| Параметр | Описание |
|---|---|
| Мин. игроков | Ивент не запустится, если на сервере меньше указанного числа игроков |
| Макс. игроков | Ивент не запустится, если на сервере больше указанного числа игроков |
| Дни недели | Ивент будет запускаться только в выбранные дни |

## Управление ванильными событиями

Плагин позволяет отключить стандартные ванильные события Rust (вертолёт, аирдроп и т.д.) и управлять ими через расписание.

В конфигурации доступен белый список — события из этого списка не будут отключены.

## Файлы данных

Плагин хранит данные в нескольких файлах:

| Файл | Описание |
|---|---|
| `EventManager/CreatorsSettings` | Список категорий создателей |
| `EventManager/EventsSettings` | Список зарегистрированных ивентов |
| `EventManager/TimesSettings` | Настройки расписания |

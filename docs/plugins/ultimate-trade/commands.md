# Команды и права

## Команды чата

| Команда | Описание | Право |
|---|---|---|
| `/trade [ник]` | Отправить запрос на обмен указанному игроку | `ultimatetrade.use` |
| `/trade accept` или `/trade yes` | Принять входящий запрос на обмен | `ultimatetrade.use` |
| `/trade cancel` или `/trade no` | Отклонить входящий запрос на обмен | `ultimatetrade.use` |
| `/t [ник]` | Короткая команда для обмена | `ultimatetrade.use` |

:::info Примечание
Команды настраиваются в конфигурации в параметре `"Commands to open trade"`. По умолчанию: `["trade", "t"]`.
:::

## Права (Permissions)

### Основное право

| Право | Описание |
|---|---|
| `ultimatetrade.use` | Базовое право для использования системы обмена |

### Пользовательские права

Дополнительные права настраиваются в секции `"Permission Settings"` конфигурации. Каждое право может иметь:

| Параметр | Описание |
|---|---|
| Slot size | Количество слотов обмена |
| Cooldown | Кулдаун после обмена (секунды) |

### Пример настройки прав

```json
"Permission Settings": {
 "ultimatetrade.vip": {
 "Slot size for this permission": 12,
 "Cooldown after trade for this permission": 30
 },
 "ultimatetrade.premium": {
 "Slot size for this permission": 16,
 "Cooldown after trade for this permission": 10
 }
}
```

### Как работают множественные права

- **Слоты:** Используется наибольшее количество среди всех прав игрока
- **Кулдаун:** Используется наименьшее значение среди всех прав игрока
- **Базовые значения:** 8 слотов, 60 секунд кулдауна (если нет дополнительных прав)

### Пример выдачи прав

```
oxide.grant group default ultimatetrade.use
oxide.grant group vip ultimatetrade.vip
oxide.grant group premium ultimatetrade.premium
```

## Процесс обмена

1. **Игрок A** вводит `/trade НикИгрокаB`
2. **Игрок B** получает уведомление и вводит `/trade accept` или `/trade yes`
3. Открывается интерфейс торговой витрины для обоих игроков
4. Каждый игрок размещает предметы в своей половине витрины
5. Оба игрока подтверждают сделку (блокируют свою сторону)
6. Предметы автоматически обмениваются
7. Витрина закрывается

:::caution Внимание
Запрос на обмен истекает через указанное в конфигурации время (по умолчанию 15 секунд). Если получатель не ответит, запрос отменяется.
:::

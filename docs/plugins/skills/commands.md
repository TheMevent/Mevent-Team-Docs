# Команды Skills

## Команды для игроков

### Открытие меню навыков

```
/skills
```
или
```
/skill
```

Открывает интерфейс навыков, где можно просматривать категории, уровни и прокачивать умения.

**Требование:** право доступа, указанное в конфигурации (по умолчанию — доступно всем).

:::info
Команды можно переименовать в конфигурации в поле «Commands».
:::

---

### Открытие модального окна по ID

```
/skills.modal [id]
```

Открывает конкретное модальное окно навыка по его идентификатору. Используется для быстрого доступа к определённому навыку.

---

## Команды администратора

Все административные команды выполняются из серверной консоли или RCON.

### Управление XP игроков

#### Начислить XP
```
skills.manage addxp [игрок/all] [количество]
```
**Примеры:**
```
skills.manage addxp 76561198012345678 1000
```
Начислит 1000 XP указанному игроку.

```
skills.manage addxp all 500
```
Начислит 500 XP всем игрокам онлайн.

---

#### Забрать XP
```
skills.manage takexp [игрок/all] [количество]
```
**Пример:**
```
skills.manage takexp 76561198012345678 500
```
Снимет 500 XP с баланса указанного игрока. Баланс не может стать меньше 0.

---

#### Установить XP
```
skills.manage setxp [игрок/all] [количество]
```
**Пример:**
```
skills.manage setxp 76561198012345678 5000
```
Установит баланс XP игрока ровно в 5000.

---

## Навигация в интерфейсе

| Действие | Описание |
|---|---|
| Выбор категории | Нажмите на категорию для фильтрации навыков |
| Просмотр навыка | Нажмите на навык для просмотра уровней и описания |
| Прокачка уровня | Нажмите кнопку изучения на нужном уровне |
| Сброс навыка | Нажмите кнопку сброса для возврата XP |

:::tip Редактор интерфейса
Администраторы с правом `skills.edit` могут использовать встроенный визуальный редактор для настройки внешнего вида интерфейса навыков прямо в игре. Редактор позволяет менять расположение элементов, цвета и изображения.
:::

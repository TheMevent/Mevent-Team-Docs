# Команды Wipe Schedule

## Команды игроков

### Открытие календаря

| Команда | Описание |
|---|---|
| `/wipe` | Открывает интерактивный календарь событий сервера |
| `/wipedata` | Альтернативная команда для открытия календаря |

:::info
Команды настраиваются в конфигурации и могут быть изменены администратором.
:::

### Навигация в календаре

Внутри интерфейса календаря доступны следующие элементы:

| Элемент | Описание |
|---|---|
| Стрелки навигации (< >) | Переключение между месяцами |
| Дни с маркерами | Клик по дню показывает подробную информацию о событиях |
| Типы событий | Цветовые маркеры обозначают разные типы событий |

## Команды администратора

### Управление через интерфейс

Основное управление событиями осуществляется через визуальный интерфейс:

1. Откройте календарь командой `/wipe`.
2. Нажмите кнопку **"Изменить ивенты"** (видна только администраторам с правом `wipeschedule.admin`).
3. В открывшемся редакторе доступны следующие действия:

| Действие | Описание |
|---|---|
| Добавить событие | Создание нового события с выбором типа, названия, цвета и расписания |
| Удалить | Удаление выбранного события |
| Сохранить | Сохранение изменений в текущем событии |

### Консольные команды

| Команда | Описание |
|---|---|
| `wipeschedule.time` | Показывает текущее серверное время с учётом настроенного часового пояса |

:::tip Пример
Используйте `wipeschedule.time` в серверной консоли, чтобы убедиться, что часовой пояс настроен правильно.
:::

### Создание события: пошаговая инструкция

1. Откройте календарь (`/wipe`).
2. Нажмите "Изменить ивенты".
3. Нажмите "Добавить событие".
4. Выберите тип события (глобальный вайп, вайп карты, вайп блок, событие).
5. Введите название и описание.
6. Выберите режим даты:
 - **Один раз** -- укажите конкретную дату и время.
 - **Повторять** -- настройте интервал повтора (дни, недели, месяцы, годы).
7. При необходимости задайте пользовательский цвет и иконку.
8. Нажмите "Сохранить".

### Настройка повтора: примеры

| Сценарий | Настройка |
|---|---|
| Вайп каждый четверг | Режим "Повторять", дни недели: четверг |
| Вайп 1-го числа каждого месяца | Режим "Повторять", дни месяца: 1 |
| Рейд-найт каждую субботу | Режим "Повторять", дни недели: суббота |
| Глобальный вайп в первый четверг месяца | Режим "Повторять", дни недели: четверг, интервал: первая неделя |

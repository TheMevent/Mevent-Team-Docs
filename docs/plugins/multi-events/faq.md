# Часто задаваемые вопросы

## 1. В чём разница между Multi Events и Events Manager?

**Events Manager** — это планировщик расписания ивентов. Он управляет тем, *когда* и *какие* ивенты будут запущены автоматически.

**Multi Events** — это фреймворк отображения. Он отвечает за *как* ивенты выглядят для игроков: красивый интерфейс, баннеры, описания, таблица лидеров и инвентарь наград.

Оба плагина дополняют друг друга и могут использоваться вместе.

## 2. Как добавить свой ивент в панель?

Отредактируйте конфигурацию (`oxide/config/MultiEvents.json`) и добавьте новую запись в секцию "Events". Укажите название плагина, баннер, заголовок, иконку и описание. После этого перезагрузите плагин.

## 3. Как изменить внешний вид панели ивентов?

Используйте встроенный визуальный редактор:

1. Выдайте себе право: `o.grant user ваш_steamid multievents.edit`
2. Введите `/ShowUiEditor` в чате
3. Редактируйте элементы перетаскиванием, меняйте цвета и размеры
4. Изменения сохраняются автоматически

## 4. Как настроить автозапуск случайных ивентов?

1. В конфигурации включите `"Enable timer to start random event": true`
2. Задайте минимальный и максимальный интервал в секундах:
 - `"Min Events Duration": 3600` (1 час)
 - `"Max Events Duration": 7200` (2 часа)
3. Убедитесь, что хотя бы один ивент имеет `"Enabled": true`

## 5. Уведомления не появляются при старте ивента. Что делать?

Проверьте:
- Установлен ли плагин Notify или UINotify
- Включён ли параметр `"Work with Notify?"` в конфигурации
- Включён ли параметр `"Enable notify when event start"`
- Ивент не добавлен в чёрный список уведомлений

## 6. Как скрыть определённые ивенты из уведомлений?

Добавьте название плагина ивента в массив `"BlackList of notify to start event"` в конфигурации. Ивенты из этого списка не будут вызывать всплывающие уведомления.

## 7. Совместим ли плагин с Carbon?

Да, Multi Events полностью совместим как с **Oxide**, так и с **Carbon**. На Carbon не требуется ImageLibrary — плагин использует встроенную систему изображений Carbon.

## 8. Как подключить таблицу лидеров и инвентарь?

1. Установите соответствующие модули: **MultiEventsLeaderboard** и/или **MultiEventsInventory**
2. В конфигурации Multi Events включите:
 - `"Use Leaderboard?": true` — для таблицы лидеров
 - `"Use Inventory?": true` — для инвентаря
3. При необходимости задайте права доступа для каждого модуля
4. Перезагрузите плагин

## 9. Можно ли использовать свои изображения?

Да. Все изображения (баннеры, иконки, фоны) загружаются по URL. Укажите URL ваших изображений в конфигурации. Рекомендуемый формат — PNG. Изображения кэшируются через ImageLibrary.

## 10. Как локализовать интерфейс?

Multi Events поддерживает мультиязычность через систему языковых файлов Oxide. Русский и английский языки включены по умолчанию. Для изменения текстов отредактируйте языковой файл:

```
oxide/lang/ru/MultiEvents.json
```

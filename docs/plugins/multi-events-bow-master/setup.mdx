# Установка и настройка

Это руководство проведёт вас через процесс установки режима Bow Master для системы MultiEvents.

## Системные требования

- Сервер Rust с Oxide / uMod / Carbon
- Установленный плагин **MultiEvents** (обязательная зависимость)
- Установленный плагин **ImageLibrary** (обязательная зависимость)

## Шаг 1: Установите зависимости

### MultiEvents (обязательно)

Плагин Bow Master является режимом расширения и требует основной плагин MultiEvents.

1. Установите **MultiEvents** согласно его документации.
2. Убедитесь, что плагин загружен без ошибок.

### ImageLibrary (обязательно)

1. Скачайте **ImageLibrary** с [umod.org](https://umod.org/plugins/image-library).
2. Поместите файл `ImageLibrary.cs` в папку `oxide/plugins/`.

:::warning Важно
Без MultiEvents и ImageLibrary плагин Bow Master **не будет работать**.
:::

## Шаг 2: Установите плагин

1. Скопируйте файл `MultiEventsBowMaster.cs` в папку `oxide/plugins/` вашего сервера.
2. Плагин загрузится автоматически и зарегистрируется в MultiEvents.
3. Проверьте консоль на наличие ошибок.

Файл конфигурации будет создан автоматически:
```
oxide/config/MultiEventsBowMaster.json
```

## Шаг 3: Настройте права доступа

Выдайте права администраторам:

```
oxide.grant group admin MultiEventsBowMaster.start
oxide.grant group admin MultiEventsBowMaster.stop
```

## Шаг 4: Проверьте работу

Запустите ивент:

```
bowmaster.start
```

На экране игроков появится панель ивента. Убивайте сущностей стрелами для набора очков.

## Шаг 5: Настройте конфигурацию (по желанию)

Откройте файл `oxide/config/MultiEventsBowMaster.json` для настройки:

- **Длительность ивента** -- время проведения события
- **Награды** -- настройка наград за места в рейтинге
- **Интеграция с Notify** -- включение/отключение уведомлений

Подробнее в разделе [Конфигурация](./configuration).

## Шаг 6: Перезагрузите плагин

После изменения конфигурации:

```
oxide.reload MultiEventsBowMaster
```

:::tip Совет
Плагин автоматически перерегистрируется в MultiEvents при перезагрузке основного плагина.
:::

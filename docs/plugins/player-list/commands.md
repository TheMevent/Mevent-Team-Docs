# Команды и права

## Команды чата

| Команда | Описание | Право |
|---|---|---|
| `/players` или `/plist` | Открыть список онлайн-игроков | Настраиваемое (по умолчанию для всех) |

:::info Примечание
Команды настраиваются в конфигурации в параметре `"Commands"`. По умолчанию: `["players", "plist"]`.
:::

## Права (Permissions)

| Право | Описание |
|---|---|
| `playerlist.hide` | Скрывает игрока из списка (для администраторов, модераторов) |
| Настраиваемое | Общее право для открытия списка (задаётся в конфигурации) |
| Права полей | Индивидуальные права для каждого поля профиля |
| Права кнопок | Индивидуальные права для каждой кнопки (например, `playerlist.admin` для кнопки кика) |

## Кнопки действий

По умолчанию доступны следующие кнопки в профиле игрока:

| Кнопка | Команда | Описание |
|---|---|---|
| TP | `tpr %steamid%` | Запрос телепортации к игроку |
| Trade | `trade %username%` | Запрос обмена с игроком |
| Stats | `stats %steamid%` | Просмотр статистики игрока |
| Kick | `kick %steamid%` | Кикнуть игрока (только с правом `playerlist.admin`) |

### Переменные для команд

| Переменная | Описание |
|---|---|
| `%steamid%` | SteamID64 выбранного игрока |
| `%username%` | Имя выбранного игрока |

:::tip Совет
Вы можете добавлять собственные кнопки с любыми командами. Используйте переменные `%steamid%` и `%username%` для подстановки данных выбранного игрока.
:::

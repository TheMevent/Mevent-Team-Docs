# Установка и настройка

## Системные требования

- Сервер Rust с Oxide/uMod или Carbon
- Установленный плагин **ImageLibrary** (обязательно)

## Пошаговая установка

### Шаг 1: Установите зависимости

Скачайте и установите плагин **ImageLibrary**, поместив его в папку `oxide/plugins`. Это обязательная зависимость, без которой плагин не будет работать.

### Шаг 2: Установите плагин

Поместите файл плагина в папку `oxide/plugins` на вашем сервере.

### Шаг 3: Перезапуск или перезагрузка

Перезапустите сервер или выполните команду в консоли:

```
oxide.reload Stacks
```

### Шаг 4: Выдайте права доступа

Выдайте администратору право на использование плагина:

```
oxide.grant user <steamid> stacks.admin
oxide.grant group admin stacks.admin
```

### Шаг 5: Первоначальная настройка

1. Зайдите на сервер и введите `/stacks` в чат.
2. В открывшейся панели вы увидите две вкладки: **Предметы** и **Контейнеры**.
3. На вкладке **Предметы** вы можете искать и изменять размер стака для каждого предмета.
4. На вкладке **Контейнеры** вы можете задать множители для каждого типа контейнера.

## Опциональные зависимости

| Плагин | Назначение |
| --------------- | -------------------------------------------------------- |
| Notify / UINotify | Красивые всплывающие уведомления вместо сообщений в чат |
| LangAPI | Автоматическая локализация названий предметов |
| FurnaceSplitter | Корректная работа Shift+ПКМ при разделении предметов в печи |

## Проверка работоспособности

После установки выполните следующие шаги для проверки:

1. Введите `/stacks` в чат -- должна открыться панель управления.
2. Найдите любой предмет (например, "wood") и измените его размер стака.
3. Возьмите этот предмет в инвентарь и убедитесь, что он стакается до нового значения.

## Обновление плагина

Для обновления просто замените файл плагина на новую версию и выполните:

```
oxide.reload Stacks
```

Данные о настройках стаков и контейнеров сохраняются в дата-файлах и не будут потеряны при обновлении.

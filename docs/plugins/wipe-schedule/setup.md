# Установка Wipe Schedule

## Системные требования

- Сервер Rust с установленным Oxide/uMod
- **ImageLibrary** -- обязательная зависимость для отображения иконок
- Рекомендуемые плагины (опционально):
 - **Notify** или **UINotify** -- для уведомлений игроков
 - **ServerPanel** -- для интеграции с серверным меню

## Пошаговая установка

### Шаг 1: Установка зависимостей

Убедитесь, что на сервере установлен плагин ImageLibrary. Без него иконки событий не будут отображаться.

### Шаг 2: Загрузка плагина

Скопируйте файл плагина в директорию:

```
oxide/plugins/
```

### Шаг 3: Автоматическая настройка

После первой загрузки плагин создаст:
- Конфигурационный файл: `oxide/config/WipeSchedule.json`
- Файл данных расписания
- Языковые файлы для русского и английского языков

### Шаг 4: Настройка часового пояса

Откройте конфигурационный файл и установите ваш часовой пояс в параметре "Time Zone". Значение указывается как смещение от UTC в часах:

- Москва: `3`
- Киев: `2`
- Минск: `3`
- Лондон: `0`
- Берлин: `1`

### Шаг 5: Создание событий

1. Зайдите на сервер с правами администратора.
2. Введите в чат `/wipe`.
3. Нажмите кнопку "Изменить ивенты".
4. Нажмите "Добавить событие" и настройте его через визуальный интерфейс.

## Настройка прав

Выдайте право администратора для управления расписанием:

```
oxide.grant user <имя_игрока> wipeschedule.admin
oxide.grant group admin wipeschedule.admin
```

## Типы событий

При создании событий доступны следующие типы:

| Тип | Описание |
|---|---|
| Глобальный вайп | Полный вайп сервера (карта + чертежи) |
| Вайп карты | Вайп только карты |
| Вайп блок | Период блокировки после вайпа |
| Глобальное событие | Особые серверные ивенты |
| Событие | Пользовательские события (рейд-найты, турниры и т.д.) |

## Режимы повтора

Для каждого события можно настроить повтор:

| Режим | Описание |
|---|---|
| Один раз | Событие происходит однократно в указанную дату и время |
| Дни | Повтор через указанное количество дней |
| Недели | Повтор в определённые дни недели |
| Месяцы | Повтор в указанные месяцы и числа |
| Годы | Ежегодный повтор |

## Интеграция с ServerPanel

Плагин автоматически регистрируется в ServerPanel. Выберите нужный шаблон (V1 или V2) в конфигурации.

## Проверка установки

1. Зайдите на сервер.
2. Введите `/wipe` в чате.
3. Убедитесь, что календарь открывается и текущий день подсвечен.
4. Проверьте отображение иконок событий.

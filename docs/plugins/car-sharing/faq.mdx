# Часто задаваемые вопросы (FAQ)

## 1. Какие плагины необходимы для работы Car Sharing?

Обязательные зависимости:
- **ImageLibrary** -- для загрузки и отображения изображений в интерфейсе
- **SpawnModularCar** -- для создания модульных автомобилей

Опциональные плагины: Economics / ServerRewards / BankSystem (экономика), Notify / UINotify (уведомления), HumanNPC (NPC-боты).

## 2. Как изменить команды для открытия меню?

Откройте файл `oxide/config/CarSharing.json` и измените массив `Commands`:
```json
"Commands": ["rent", "carsharing", "mycommand"]
```
После сохранения перезагрузите плагин командой `oxide.reload CarSharing`.

## 3. Как настроить поминутную оплату?

В настройках каждой машины есть параметр `Cost per minute`. Установите нужное значение. Плагин будет автоматически списывать указанную сумму за каждую минуту аренды. Если у игрока заканчиваются средства, аренда прекращается.

## 4. Как добавить новую машину?

Откройте конфигурацию и в массиве `Cars` добавьте новый объект с настройками:
- Укажите название, изображение и класс
- Настройте стоимость аренды за минуту
- Задайте модули автомобиля в секции модульной машины
- При необходимости укажите право доступа и команду выдачи

## 5. Как привязать каршеринг к NPC?

1. Создайте NPC-бота с помощью плагина **HumanNPC**.
2. Запомните ID бота.
3. В конфигурации добавьте ID в секцию `Bots`:
```json
"Bots": {
 "ID_ВАШЕГО_NPC": {
 "Car Classes (* - all)": ["Luxe", "Comfort"]
 }
}
```
Игрок, нажав на NPC, увидит только машины указанных классов.

## 6. Можно ли использовать предметы вместо плагина экономики?

Да. В секции `Economy` измените тип на `Item` и укажите shortname предмета. Например, для оплаты скрапом:
```json
"Economy": {
 "Type (Plugin/Item)": "Item",
 "ShortName": "scrap",
 "Skin": 0
}
```

## 7. Как работает предмет вызова транспорта?

После аренды игрок получает специальный предмет в инвентарь. Этот предмет имеет настраиваемый скин и название (например, "RUSTSharing Ferrari"). Используя предмет, игрок вызывает арендованную машину рядом с собой.

## 8. Как настроить кодовый замок на арендованных машинах?

В настройках модульной машины установите параметр `CodeLock` в значение `true`:
```json
"Modular Car Settings": {
 "CodeLock": true,
 "KeyLock": false
}
```
Код замка генерируется автоматически и будет известен только арендатору.

## 9. Машина не создаётся -- что делать?

- Убедитесь, что плагин **SpawnModularCar** установлен и загружен.
- Проверьте, что модули в конфигурации указаны корректно (например, `vehicle.1mod.engine`).
- Посмотрите консоль сервера на наличие ошибок.

## 10. Как изменить уровень деталей двигателя?

В параметре `Engine Parts Tier` укажите значение от 1 до 3:
- `1` -- детали низкого качества
- `2` -- детали среднего качества
- `3` -- детали высокого качества

# Команды и права

## Консольные команды

| Команда | Описание | Пример |
|---|---|---|
| `hsupply.give [ник/steamid/all] [имя шашки] [количество]` | Выдать кастомную сигнальную шашку игроку | `hsupply.give Player1 heal 3` |

### Параметры команды

- `[ник/steamid/all]` -- имя игрока, SteamID64 или `all` для выдачи всем
- `[имя шашки]` -- название типа шашки (например, `heal`, `guns`, `explosive`)
- `[количество]` -- количество шашек для выдачи

### Примеры

```
hsupply.give Player1 heal 1
```
Выдать одну медицинскую шашку игроку Player1.

```
hsupply.give 76561198012345678 guns 2
```
Выдать две оружейные шашки игроку по SteamID.

```
hsupply.give all explosive 1
```
Выдать по одной взрывной шашке всем игрокам на сервере.

## Права (Permissions)

У плагина Helpful Supply нет специальных прав доступа. Консольная команда `hsupply.give` выполняется из серверной консоли или RCON.

:::info Примечание
Любой игрок может использовать кастомную сигнальную шашку, если она есть в его инвентаре. Контроль доступа осуществляется через выдачу шашек.
:::

# Часто задаваемые вопросы -- Shop

## 1. Как установить начальный набор товаров в магазин?

Используйте команду `/shop.install` (требуется право `shop.admin`). Плагин предложит выбрать один из готовых шаблонов, содержащих предметы, категории и цены. После установки вы можете редактировать все параметры.

## 2. Можно ли использовать несколько валют одновременно?

Да. В разделе `Economy` настраивается основная валюта (например, Economics), а в разделе `Additional Economics` можно добавить дополнительные валюты (например, ServerRewards или предметы). Игроки смогут выбирать, в какой валюте совершать покупки.

## 3. Как настроить скидки для VIP-игроков?

В разделе `Discount` конфигурации добавьте записи вида `"право": процент`. Например:
- `"shop.default": 0` -- без скидки
- `"shop.vip": 10` -- скидка 10%

Затем выдайте соответствующее право группе или игроку через `oxide.grant`. Применяется наибольшая скидка из всех имеющихся у игрока прав.

## 4. Как привязать магазин к NPC на сервере?

1. Разместите NPC (через плагин CustomNPC или другой).
2. Наведите на него прицел.
3. Введите `/shop.setnpc`.
4. В конфигурации появится новая запись с ID этого NPC.
5. Укажите категории, которые будут доступны у этого NPC.

Можно создать несколько NPC-продавцов с разными категориями товаров для разных локаций.

## 5. Почему после обновления плагина магазин пустой?

Это может произойти, если обновление изменило формат данных. Выполните `/shop.install` для переустановки шаблона. Рекомендуется делать резервные копии данных магазина перед обновлением плагина.

## 6. Как настроить лимиты покупок?

Лимиты настраиваются для каждого товара индивидуально через интерфейс администратора:
- **Общий лимит** -- максимальное количество покупок за весь вайп
- **Ежедневный лимит** -- максимальное количество покупок в день

Лимиты сбрасываются при вайпе, если включена настройка `Limits` в разделе `Auto-Wipe Settings`.

## 7. Как включить логирование покупок в Discord?

1. Создайте Webhook в вашем Discord-канале.
2. Откройте конфигурацию Shop.
3. В разделе `Discord Logging Settings` включите нужные типы логирования (Buy, Sell, Transfer).
4. Вставьте URL Webhook.
5. Настройте текст сообщений с помощью переменных (`{username}`, `{item}`, `{amount}`, `{price}`).

## 8. Можно ли запретить продажу определённых предметов или скинов?

Да. Используйте раздел `Blocked skins for sell` в конфигурации. Укажите короткое имя предмета и список скинов, которые нельзя продавать. Это полезно для защиты от продажи уникальных предметов.

## 9. Как работает блокировка магазина после вайпа?

Если включена настройка `Wipe Block`, магазин будет недоступен в течение указанного времени после вайпа (`Wipe Cooldown`, по умолчанию 3600 секунд = 1 час). Это даёт игрокам время на начальное развитие перед доступом к магазину.

## 10. Как ограничить магазин только для определённых игроков?

Укажите право доступа в параметре `Permission to use plugin` (например, `shop.use`). Затем выдайте это право нужным группам или игрокам:
```
oxide.grant group default shop.use
```

Для NPC-магазинов можно задать отдельное право в параметре `Permission` каждого NPC.

## 11. Работает ли магазин с предметами как валютой (без Economics)?

Да. Установите тип экономики на `Item` вместо `Plugin`:
```json
"Economy": {
 "Type (Plugin/Item)": "Item",
 "ShortName": "scrap",
 "Skin": 0
}
```

В этом случае валютой будет предмет из инвентаря игрока (в данном примере -- scrap).

## 12. Как отключить продажу предметов игроками?

Установите параметр `Enable selling items?` в `false`. После этого игроки смогут только покупать товары.

## 13. Почему магазин не открывается сразу после респавна?

По умолчанию включена блокировка магазина на 60 секунд после респавна (`Respawn Block`). Это предотвращает мгновенную покупку снаряжения после смерти. Чтобы отключить, установите `Respawn Block` в `false`.

## 14. Как настроить магазин для разных торговых автоматов?

1. Разместите торговые автоматы на карте.
2. Наведите на каждый прицел и выполните `/shop.setvm`.
3. В конфигурации для каждого автомата укажите свой набор категорий.

Это позволяет создать специализированные торговые точки (оружейный магазин, продуктовый и т.д.).

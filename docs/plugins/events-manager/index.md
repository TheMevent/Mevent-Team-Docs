# Events Manager

## Что делает этот плагин?

**Events Manager** позволяет вашему серверу централизованно управлять всеми серверными событиями через удобный визуальный интерфейс. Плагин автоматически запускает ивенты по расписанию, поддерживает десятки популярных событий от разных разработчиков, а также позволяет отключать стандартные ванильные события Rust. Все настройки расписания выполняются прямо в игре через графическую панель администратора.

## Ключевые возможности

- **Визуальный редактор расписания** — настраивайте запуск ивентов прямо в игре через интерфейс
- **Три типа расписания** — фиксированное время (Static), повтор каждые N минут (Every), случайный интервал (Random)
- **Поддержка 50+ ивентов** — встроенная интеграция с плагинами от Mevent, KpucTaJl, Adem, Nivex, и многих других
- **Ванильные события** — управление вертолётом, аирдропом, грузовым кораблём, Chinook 47, санями Санты
- **Условия запуска** — минимальное и максимальное количество игроков онлайн для старта ивента
- **Привязка к дням недели** — запускайте ивенты только в определённые дни
- **Случайный выбор ивентов** — для одного слота расписания можно назначить несколько ивентов, и система случайно выберет один
- **Discord-интеграция** — уведомления о запуске ивентов через вебхуки

## Требования

| Требование | Описание |
|---|---|
| Oxide / Carbon | Фреймворк для модификаций сервера |
| ImageLibrary | Обязательно для Oxide (не требуется для Carbon) |

## Быстрый старт

1. Установите плагин `EventsManager.cs` в папку `oxide/plugins` вашего сервера
2. Убедитесь, что установлен **ImageLibrary** (для Oxide)
3. Перезагрузите сервер или используйте `o.reload EventsManager`
4. Выдайте себе право доступа: `o.grant user ваш_steamid EventsManager.use`
5. Введите команду `/em` в чате для открытия панели управления

## Права доступа

| Право | Описание |
|---|---|
| `EventsManager.use` | Доступ к панели управления ивентами (команда `/em`) |

## Команды игроков

У данного плагина нет команд для обычных игроков. Все команды предназначены для администраторов.

## Команды администратора

| Команда | Описание |
|---|---|
| `/em` | Открыть визуальную панель управления ивентами |
| `eventmanager.reset.default` | Сбросить все данные к значениям по умолчанию (только из консоли сервера) |

:::tip Пример
Откройте чат и введите `/em`, чтобы открыть панель управления. Там вы сможете:
- Добавлять новые ивенты
- Редактировать расписание
- Запускать ивенты вручную
- Управлять категориями создателей
:::

## Устранение неполадок

| Проблема | Решение |
|---|---|
| Команда `/em` не работает | Проверьте, что у вас есть право `EventsManager.use`. Используйте `o.grant user ваш_steamid EventsManager.use` |
| Ивенты не запускаются автоматически | Убедитесь, что расписание настроено, и количество игроков на сервере соответствует условиям (мин./макс. игроков) |
| Ванильные события всё ещё появляются | Проверьте настройку "Disable vanilla auto events?" в конфигурации. Некоторые события можно оставить в белом списке |
| Ивент показан как "не загружен" | Убедитесь, что плагин ивента установлен и загружен на сервере |
| Ошибка при загрузке конфигурации | Проверьте JSON-файл конфигурации на ошибки синтаксиса. Плагин выведет ошибку в консоль |
| Discord-уведомления не приходят | Убедитесь, что URL вебхука указан правильно в настройке "Discord web hook" |

# Quarry Levels Installation

## Step 1: Install the Plugin

1. Copy the `QuarryLevels.cs` file into the `oxide/plugins/` folder.
2. The plugin will load automatically.

Configuration file:
```
oxide/config/QuarryLevels.json
```

## Step 2: Configure Permissions

```
oxide.grant group default quarrylevels.use
```

## Step 3: Install Dependencies (optional)

| Plugin | Purpose |
|---|---|
| **Economics** | Paying for upgrades with currency |
| **ServerRewards** | Paying for upgrades with reward points |

If you want to use economy instead of items:

```json
"Enable Economics?": true,
"Economics cost": 5000,
"Economics currency": "credits"
```

## Step 4: Verify Operation

1. Place a quarry (Mining Quarry).
2. Open it (press E).
3. You will see an "Upgrade" button in the interface.
4. Click "Upgrade" -- an information panel about the level will open.
5. Confirm the upgrade (cost: 1 quarry or currency).

## Step 5: Configure the Settings

Open `oxide/config/QuarryLevels.json`:

### Main Parameters

- **Quarry max level** -- maximum quarry level (default: 5)
- **Pumpjack max level** -- maximum pumpjack level
- **Quarry capacity per level** -- capacity increase per level
- **Deposit Settings** -- resource unlock levels

## Step 6: Reload the Plugin

```
oxide.reload QuarryLevels
```

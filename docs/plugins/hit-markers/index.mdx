# Hit Markers

## Что делает этот плагин?

**Hit Markers** добавляет на сервер визуальные маркеры попаданий. Игроки видят числовое значение нанесённого урона, изменение цвета иконки попадания при хедшоте, полоску здоровья противника и индикатор урона по строениям. Каждый игрок может настроить маркеры по своему вкусу: включить/отключить отдельные элементы, изменить шрифт и размер текста.

## Ключевые возможности

- Отображение числового урона при попадании (Text)
- Иконка попадания с изменением цвета при хедшоте (Icon)
- Полоска здоровья противника над слотами (Health Line)
- Отображение урона по строениям (Buildings)
- Настройка шрифта, размера и вида для каждого игрока
- Поддержка урона по NPC и животным
- Интеграция с Notify для уведомлений
- Совместимость с Oxide и Carbon

## Требования

| Зависимость | Обязательность | Описание |
|---|---|---|
| **ImageLibrary** | Обязательно | Загрузка и отображение изображений |
| **Notify** / **UINotify** | По желанию | Красивые уведомления |
| Oxide / uMod / Carbon | Обязательно | Совместим с Oxide, uMod и Carbon |

## Быстрый старт

1. Установите **ImageLibrary** на сервер.
2. Скопируйте файл `HitMarkers.cs` в папку `oxide/plugins/`.
3. Перезагрузите сервер или выполните `oxide.reload HitMarkers`.
4. Введите `/marker` или `/hits` в чат для открытия настроек маркеров.

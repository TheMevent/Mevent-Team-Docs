# Skills

## Что делает этот плагин?

**Skills** позволяет вашему серверу добавить полноценную RPG-систему навыков для игроков. Игроки зарабатывают очки опыта (XP) за убийства, добычу ресурсов, лут ящиков и другие действия, а затем тратят их на прокачку разнообразных умений: от увеличения добычи руды до сопротивления урону, ускорения крафта и многого другого. Навыки организованы по категориям, каждый имеет несколько уровней с возрастающей стоимостью и эффектом.

## Ключевые возможности

- Более 35 различных типов навыков: добыча, выживание, бой, крафт, верховая езда, транспорт и другие
- Гибкая XP-система с начислением за убийства, лут, добычу ресурсов, смерти (штраф) и многое другое
- Многоуровневая прокачка каждого навыка (до 10+ уровней)
- Категории навыков для удобной навигации
- Возврат XP при сбросе навыка (настраиваемый процент)
- VIP-множители на заработок XP по правам доступа
- Передача XP при убийстве — забирайте часть XP у убитого игрока
- Сброс навыков при смерти (опционально)
- Автоматический вайп данных при новом сейве
- Поддержка пользовательской экономики (ServerRewards, Economics и др.)
- Встроенный редактор интерфейса для администраторов (прямо в игре)
- Уведомления о получении XP и изучении навыков (чат, GameTip, Discord)
- Защита от вайпа (специальный навык WipeProtection)
- Интеграция с ServerPanel, Notify, Clans, Friends, EventHelper

## Требования

| Зависимость | Обязательно? | Описание |
|---|---|---|
| ImageLibrary | Да | Загрузка и отображение изображений в интерфейсе |
| ServerPanel | Нет | Интеграция с серверным меню |
| Notify / UINotify | Нет | Всплывающие уведомления |
| Clans | Нет | Учёт клановых связей при начислении XP |
| Friends | Нет | Учёт друзей при начислении XP |
| EventHelper / Battles / Duel / Duelist / ArenaTournament | Нет | Отключение начисления XP на аренах и ивентах |

## Быстрый старт

1. Установите **ImageLibrary** на сервер.
2. Поместите файл `Skills.cs` в папку `oxide/plugins`.
3. Перезагрузите сервер или выполните `o.reload Skills`.
4. Настройте категории и навыки в `oxide/config/Skills.json`.
5. Проверьте работу: введите `/skills` в игровом чате.

## Права доступа

| Право | Кому нужно | Что открывает |
|---|---|---|
| Настраиваемое (пусто по умолчанию) | Игрокам | Доступ к меню навыков (если задано в конфигурации) |
| `skills.edit` | Администраторам | Доступ к встроенному редактору интерфейса |
| `skills.xprate.vip` | VIP-игрокам | Множитель заработка XP x1.5 |
| `skills.xprate.premium` | Премиум-игрокам | Множитель заработка XP x2 |
| `skills.xptransfer.vip` | VIP-игрокам | Передача 75% XP при убийстве |
| `skills.xptransfer.premium` | Премиум-игрокам | Передача 100% XP при убийстве |
| Права для уровней навыков | VIP-игрокам | Доступ к высокоуровневым навыкам (например, `skills.gathering.level6`) |

## Команды игроков

| Команда | Что делает | Кто может использовать |
|---|---|---|
| `/skills` или `/skill` | Открывает меню навыков | Все игроки (или с настроенным правом) |

## Команды администратора

| Команда | Что делает | Кто может использовать |
|---|---|---|
| `skills.manage addxp [игрок/all] [кол-во]` | Начисляет XP игроку или всем | Администратор / RCON |
| `skills.manage takexp [игрок/all] [кол-во]` | Забирает XP у игрока или всех | Администратор / RCON |
| `skills.manage setxp [игрок/all] [кол-во]` | Устанавливает точное количество XP | Администратор / RCON |

**Пример:**
```
skills.manage addxp 76561198012345678 1000
```
Начислит 1000 XP указанному игроку.

```
skills.manage addxp all 500
```
Начислит 500 XP всем онлайн-игрокам.

## Устранение неполадок

### Навыки не отображаются в меню
- **Симптом:** Игрок открывает `/skills`, но список пуст.
- **Причина:** Все навыки отключены в конфигурации.
- **Решение:** Убедитесь, что у навыков параметр `"Enabled"` установлен в `true`.

### XP не начисляется
- **Симптом:** Игрок выполняет действия, но XP не растёт.
- **Причина:** Система начисления XP отключена или действие не настроено в разделе XP Rewards.
- **Решение:** Проверьте, что `"XP Rewards Settings" > "Enabled"` установлено в `true` и нужное действие есть в списке наград.

### Навык не прокачивается — «Недостаточно XP»
- **Симптом:** Игрок нажимает на уровень навыка, но получает отказ.
- **Причина:** У игрока недостаточно XP, или для этого уровня требуется специальное право доступа.
- **Решение:** Проверьте баланс XP игрока. Если у уровня указано право доступа (например, `skills.gathering.level6`), его нужно выдать.

### Интерфейс не открывается
- **Симптом:** Команда `/skills` не работает.
- **Причина:** Не установлен ImageLibrary, или у игрока нет права доступа.
- **Решение:** Установите ImageLibrary. Если настроено право доступа, выдайте его.

### Навыки сбрасываются при вайпе
- **Симптом:** После вайпа карты все навыки обнуляются.
- **Причина:** Включена настройка авто-вайпа при новом сейве.
- **Решение:** Установите `"Auto Wipe on New Save Settings" > "Enabled"` в `false`, чтобы сохранять прогресс.

### Игрок на арене получает XP
- **Симптом:** XP начисляется во время дуэлей или ивентов.
- **Причина:** Плагины Duel/EventHelper не установлены или не интегрированы.
- **Решение:** Установите нужные плагины — Skills автоматически определит их и отключит начисление XP на аренах.

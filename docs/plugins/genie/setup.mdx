# Установка Genie

Это руководство проведёт вас через процесс установки плагина волшебных ламп.

## Шаг 1: Установите зависимости

### ImageLibrary (обязательно)

1. Скачайте **ImageLibrary** с [umod.org](https://umod.org/plugins/image-library).
2. Поместите файл `ImageLibrary.cs` в папку `oxide/plugins/`.
3. Убедитесь, что плагин загрузился без ошибок.

:::warning Важно
Без ImageLibrary плагин Genie **не будет корректно работать**. Игрокам будет показано сообщение об ошибке.
:::

## Шаг 2: Установите плагин

1. Скопируйте файл `Genie.cs` в папку `oxide/plugins/` вашего сервера.
2. Плагин загрузится автоматически.
3. Проверьте консоль на наличие ошибок.

Файл конфигурации будет автоматически создан по пути:
```
oxide/config/Genie.json
```

## Шаг 3: Настройте права доступа (если нужны)

Если для конкретного типа лампы указано право (Permission), выдайте его соответствующим игрокам:

```
oxide.grant group vip genie.special
```

:::info Примечание
Если поле `Permission` для лампы оставлено пустым, она будет доступна всем игрокам без ограничений.
:::

## Шаг 4: Настройте лампы

Откройте файл `oxide/config/Genie.json` и настройте параметры ламп:

- **ID** -- уникальный идентификатор типа лампы
- **Lamp** -- настройки предмета (ShortName, скин, имя)
- **Awards** -- список наград с шансами
- **Delay** -- время открытия в секундах

## Шаг 5: Проверьте работу

Выдайте лампу командой:
```
genie.give <ваш_steamid> default
```

Используйте полученный предмет -- должен открыться интерфейс с анимацией.

## Шаг 6: Перезагрузите плагин

После изменения конфигурации:

```
oxide.reload Genie
```

## Обновление плагина

При обновлении:

1. Замените файл `Genie.cs` новой версией.
2. Плагин автоматически обновит конфигурацию.
3. Перезагрузите плагин.

:::note Примечание
Рекомендуется сохранять резервную копию конфигурации перед обновлением.
:::

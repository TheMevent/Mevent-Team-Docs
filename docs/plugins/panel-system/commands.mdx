# Команды и права

## Команды чата

| Команда | Описание | Право |
|---|---|---|
| `/panelsystem.buttons` | Показать/скрыть панель быстрых кнопок | Для всех (если кнопки настроены) |
| `/(команда скрытия)` | Скрыть или показать всю панель | Настраивается в конфигурации |

:::info Примечание
Команда скрытия панели настраивается в конфигурации в секции `"Hide Settings"`. По умолчанию: `panel`. Функция должна быть включена (`"Enabled": true`).
:::

### Пример использования скрытия

| Команда | Описание |
|---|---|
| `/panel hide` | Скрыть всю панель |
| `/panel show` | Показать панель |

## Консольные команды

| Команда | Описание |
|---|---|
| `panelsystem.sendcmd` | Внутренняя команда для выполнения действий через кнопки панели |

## Права (Permissions)

### Права элементов панели

В Panel System права настраиваются гибко -- каждый элемент может иметь собственное право. По умолчанию все права пустые, что означает доступ для всех.

| Элемент | Где задаётся | Описание |
|---|---|---|
| Логотип | `"Logotype" > "Permission"` | Кто видит логотип сервера |
| Онлайн | `"Players Settings" > "Permission"` | Кто видит количество онлайн |
| Спящие | `"Sleepers Settings" > "Permission"` | Кто видит количество спящих |
| Время | `"Time Settings" > "Permission"` | Кто видит игровое время |
| Кнопки (панель) | `"Buttons Setting" > "Permission"` | Кто видит панель кнопок |
| Каждая кнопка | `"Buttons" > [n] > "Permission"` | Кто видит конкретную кнопку |
| Каждое событие | `"Events" > [n] > "Permission"` | Кто видит индикатор события |
| Каждая экономика | `"Economics" > [n] > "Permission"` | Кто видит баланс |
| Настройки | `"Settings Button" > "Permission"` | Кто может открыть окно настройки панели |

### Права размеров

Каждый элемент панели может иметь несколько размеров (Big, Medium, Small), и для каждого размера можно указать право доступа.

### Пример настройки прав

```
oxide.grant group default panelsystem.online
oxide.grant group vip panelsystem.buttons
oxide.grant group admin panelsystem.settings
```

:::tip Совет
Если поле `"Permission"` оставлено пустым, элемент виден всем игрокам. Чтобы ограничить доступ, укажите нужное право и выдайте его через Oxide.
:::

## Быстрые кнопки

Быстрые кнопки выполняют команды при нажатии. Команды задаются в конфигурации:

| Формат | Описание | Пример |
|---|---|---|
| `chat.say /команда` | Выполнить чат-команду | `chat.say /shop` |
| `консольная_команда` | Выполнить консольную команду | `inventory.open` |

### Пример конфигурации кнопки

```json
{
 "Enabled": true,
 "Image": "https://example.com/icon.png",
 "Permission": "",
 "Lang Key (oxide/lang/**/PanelSystem.json)": "BtnShop",
 "Command": "chat.say /shop"
}
```

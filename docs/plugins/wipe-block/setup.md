# Установка Wipe Block

## Системные требования

- Сервер Rust с установленным фреймворком Oxide/uMod
- Рекомендуемые плагины (опционально):
 - **Notify** или **UINotify** -- для красивых уведомлений о разблокировке предметов
 - **ServerPanel** -- для отображения информации о блокировках в серверном меню

## Пошаговая установка

### Шаг 1: Загрузка плагина

Скопируйте файл плагина в директорию:

```
oxide/plugins/
```

### Шаг 2: Загрузка на сервер

После размещения файла плагин загрузится автоматически. Если автозагрузка отключена, используйте консольную команду:

```
o.load WipeBlock
```

### Шаг 3: Первоначальная настройка

После первой загрузки плагин создаст конфигурационный файл:

```
oxide/config/WipeBlock.json
```

Откройте его и настройте под свой сервер.

### Шаг 4: Настройка предметов

В конфигурации задайте категории и предметы с таймерами блокировки. Время указывается **в секундах** от момента вайпа:

- `3600` = 1 час
- `7200` = 2 часа
- `14400` = 4 часа
- `21600` = 6 часов
- `86400` = 24 часа (1 день)

### Шаг 5: Перезагрузка

После внесения изменений в конфигурацию перезагрузите плагин:

```
o.reload WipeBlock
```

## Настройка прав

Выдайте нужные права игрокам или группам:

```
oxide.grant user <имя_игрока> wipeblock.ignore
oxide.grant group <группа> wipeblock.unlocknotify
```

- `wipeblock.ignore` -- полное игнорирование блокировок (подходит для администраторов)
- `wipeblock.unlocknotify` -- получение уведомлений при разблокировке

## Интеграция с другими плагинами

### Notify / UINotify

Если на сервере установлен Notify или UINotify, включите параметр "Work with Notify?" в конфигурации (`true` по умолчанию). Уведомления о разблокировке предметов будут отображаться через систему уведомлений.

### ServerPanel

Плагин автоматически регистрирует категорию в ServerPanel. Выберите шаблон интерфейса (V1 или V2) в параметре "ServerPanel Template".

## Проверка установки

1. Зайдите на сервер.
2. Введите в чат `/block`.
3. Должно открыться меню с категориями предметов и таймерами разблокировки.
4. Убедитесь, что виджет на экране отображает информацию о текущей блокировке.

# Команды и права

## Консольные команды

| Команда | Описание | Пример |
|---|---|---|
| `necronomicon.give [ник/steamid]` | Выдать книгу Necronomicon игроку | `necronomicon.give Player1` |

:::info Примечание
Команда `necronomicon.give` выполняется из серверной консоли или RCON. При выдаче книга создаётся с текущими чертежами указанного игрока.
:::

## Права (Permissions)

| Право | Описание |
|---|---|
| Настраиваемое | Право для использования книги (задаётся в конфигурации, пустое -- для всех) |

:::tip Совет
Если поле `"Permission"` в конфигурации оставить пустым, любой игрок сможет использовать книгу.
:::

## Как работает система

1. Администратор выдаёт книгу Necronomicon игроку через консольную команду.
2. Игрок открывает стол исследования (Research Table).
3. Игрок помещает книгу в стол.
4. Появляется визуальный интерфейс с чертежами, содержащимися в книге.
5. Игрок может изучить чертежи за указанную в конфигурации стоимость.
6. После изучения чертёж добавляется игроку.

# Kits

## Что делает этот плагин?

**Kits** позволяет вашему серверу предоставлять игрокам заранее настроенные наборы предметов (киты). Администраторы создают наборы с определённым содержимым, задают кулдауны, ограничения по правам и стоимость. Игроки могут получать наборы через удобный игровой интерфейс, через чат-команды или у NPC-торговцев.

## Ключевые возможности

- Создание неограниченного количества наборов с любыми предметами
- Настраиваемые кулдауны и лимиты использования для каждого набора
- Система прав доступа — разные наборы для разных групп игроков
- Покупка наборов за внутриигровую валюту (Economics, ServerRewards или предметы)
- Автонаборы при входе на сервер (с возможностью отключения игроком)
- Привязка наборов к NPC на карте
- Полностью настраиваемый интерфейс (полноэкранный и встроенный в меню)
- Поддержка категорий наборов
- Интеграция с CopyPaste для выдачи построек
- Блокировка получения во время рейда или боя (NoEscape)
- Автоматический вайп данных наборов при вайпе сервера
- Логирование в Discord
- Редактирование наборов прямо в игре (визуальный редактор)
- Поддержка Oxide и Carbon

## Требования

| Зависимость | Обязательная? | Описание |
|---|---|---|
| **ImageLibrary** | Да | Требуется для отображения изображений в интерфейсе |
| ServerPanel | Нет | Интеграция с панелью серверного меню |
| CopyPaste | Нет | Для выдачи построек через наборы |
| Notify / UINotify | Нет | Красивые уведомления для игроков |
| NoEscape | Нет | Блокировка наборов во время рейда/боя |

## Быстрый старт

1. Установите плагин **ImageLibrary** (обязательно)
2. Поместите файл плагина в папку `oxide/plugins` (или `carbon/plugins`)
3. Перезагрузите сервер или выполните `o.reload Kits`
4. Откройте конфигурационный файл и настройте команды, экономику и интерфейс
5. Используйте команду `/editkit` в игре для создания наборов через визуальный редактор
6. Выдайте необходимые права доступа игрокам или группам

## Права доступа

| Право | Описание |
|---|---|
| `Kits.admin` | Полный доступ к управлению наборами и административным командам |
| `Kits.dlc` | Обход ограничений на DLC-предметы |
| `Kits.bypasscooldown` | Обход кулдауна при получении наборов |
| `kits.changeautokit` | Возможность включать/выключать автонаборы (настраивается) |
| Индивидуальные права для каждого набора | Задаются при создании набора в редакторе |

## Команды игроков

| Команда | Описание |
|---|---|
| `/kit` или `/kits` | Открывает главное меню наборов |
| `/kit <название>` | Получить конкретный набор по названию |

:::tip Пример
Чтобы получить стартовый набор, введите в чат:
```
/kit starter
```
:::

## Команды администратора

| Команда | Описание |
|---|---|
| `/editkit` | Открывает визуальный редактор наборов в игре |
| `kits.give <имя/steamid> <название_набора>` | Выдать игроку предметы набора напрямую |
| `kits.givekit <имя/steamid> <название_набора> [кол-во]` | Начислить использования набора игроку |
| `kits.reset data all` | Сбросить данные наборов всех игроков |
| `kits.reset data <steamid>` | Сбросить данные наборов конкретного игрока |
| `kits.reset data <steamid> <название_набора>` | Сбросить данные конкретного набора у игрока |
| `kits.reset kits` | Удалить все созданные наборы |
| `kits.template <шаблон>` | Переключить шаблон интерфейса (только из консоли сервера) |
| `kits.convert` | Конвертация данных из других плагинов |

:::tip Примеры
```
kits.give 76561198000000000 vip_kit
kits.givekit PlayerName starter 5
kits.reset data 76561198000000000 starter
kits.template fullscreen rust normal
```
:::

## Устранение неполадок

### Интерфейс не открывается
Убедитесь, что плагин **ImageLibrary** установлен и загружен. Без него плагин не сможет отобразить меню наборов.

### Игрок не видит набор в списке
Проверьте, что у игрока есть необходимое право доступа для данного набора. Если включена опция "Показывать все наборы", наборы будут видны, но недоступны для получения без прав.

### Набор не выдаётся — сообщение о кулдауне
Кулдаун ещё не истёк. Игрок увидит оставшееся время в интерфейсе. Администраторы с правом `Kits.bypasscooldown` могут обходить кулдауны.

### Набор недоступен во время рейда или боя
Если включена интеграция с NoEscape, наборы блокируются во время рейд-блока или комбат-блока. Добавьте название набора в белый список (Whitelist), если хотите разрешить его получение во время блокировки.

### Предметы не помещаются в инвентарь
Если инвентарь игрока заполнен, предметы будут выброшены на землю рядом с игроком.

### Автонабор не выдаётся при входе
Проверьте, что название набора добавлено в список автонаборов в конфигурации, и что у игрока есть право на получение данного набора.

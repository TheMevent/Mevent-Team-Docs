# Команды

## Команды администратора

| Команда | Описание |
|---|---|
| `/give.capturezone [steamid/ник] [количество]` | Выдать предмет захвата зоны указанному игроку |

:::note Только для администраторов
Команда доступна только игрокам с правами администратора сервера.
:::

### Примеры использования

Выдать 1 предмет игроку по SteamID:
```
/give.capturezone 76561198012345678 1
```

Выдать 5 предметов игроку по нику:
```
/give.capturezone PlayerName 5
```

Если количество не указано, выдаётся 1 предмет:
```
/give.capturezone 76561198012345678
```

## Команды игроков

У Capture Zone нет чат-команд для игроков. Взаимодействие происходит через игровую механику:

### Установка зоны захвата

1. Получите специальный предмет (шкаф с уникальным скином)
2. Выберите место для установки
3. Установите шкаф, как обычный предмет
4. Зона автоматически создаётся вокруг шкафа

### Уничтожение зоны

Уничтожьте шкаф любым способом (разрушение, разложение) для удаления зоны.

## Права доступа

| Право | Описание |
|---|---|
| `capturezone.ignore` | Ресурсы игрока не облагаются налогом на чужих территориях |

### Настраиваемые права (по умолчанию)

| Право | Процент налога | Лимит зон |
|---|---|---|
| (без прав) | 20% | 3 |
| `capturezone.vip` | 25% | 5 |
| `capturezone.deluxe` | 30% | 7 |

### Дополнительные настраиваемые права

| Назначение | Описание |
|---|---|
| Право на установку | Если указано в конфигурации -- только игроки с этим правом могут устанавливать зоны |
| Право на просмотр маркера | Если указано -- только игроки с этим правом видят маркеры зон на карте |

## Управление правами

### Выдача прав

```
o.grant user SteamID64 capturezone.ignore
o.grant user SteamID64 capturezone.vip
o.grant user SteamID64 capturezone.deluxe
```

### Выдача прав группе

```
o.grant group vip capturezone.vip
o.grant group premium capturezone.deluxe
```

### Отзыв прав

```
o.revoke user SteamID64 capturezone.vip
```

## Сообщения игрокам

| Сообщение | Описание |
|---|---|
| Потеря ресурсов | Уведомление при добыче на чужой территории с указанием процента |
| Слишком близко к зоне | Ошибка при установке шкафа рядом с другой зоной |
| Нет прав | Ошибка при отсутствии необходимого права на установку |
| Достигнут лимит | Ошибка при превышении максимального количества зон |

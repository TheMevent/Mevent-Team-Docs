# Команды и права

## Команды чата

| Команда | Описание | Право |
|---|---|---|
| `/ref` | Открыть интерфейс реферальной системы | Настраивается |
| `/referal` | Открыть интерфейс (альтернативная) | Настраивается |
| `/promo <код>` | Активировать промокод | Настраивается |
| `/code <код>` | Активировать промокод (альтернативная) | Настраивается |

:::tip Совет
Команды настраиваются в конфигурации. `Commands` -- для открытия интерфейса, `Commands to activate the promo code` -- для активации промокодов.
:::

### Примеры использования

Открыть интерфейс:
```
/ref
```

Активировать промокод:
```
/promo ABC12345
```

## Консольные команды

У плагина нет специальных консольных команд для администраторов. Управление осуществляется через конфигурацию.

## Права (Permissions)

| Право | Описание | По умолчанию |
|---|---|---|
| Настраивается | Доступ к реферальной системе | Пустое (доступно всем) |

:::info Примечание
Право доступа настраивается в конфигурации в параметре `"Permission (example: referrals.use)"`. Если оставить пустым, плагин доступен всем игрокам без ограничений.
:::

## Механика работы

### Для реферера (приглашающего)

1. Откройте интерфейс командой `/ref`.
2. Скопируйте свой уникальный промокод.
3. Поделитесь промокодом с другими игроками.
4. При активации промокода другим игроком вы получите уведомление.
5. За определённое количество приглашений открываются награды.

### Для приглашённого

1. Получите промокод от друга.
2. Введите команду `/promo <код>`.
3. Если вы наиграли достаточно времени -- промокод активируется.
4. Вы получите награду за активацию (если настроено).

### Ограничения

- Игрок не может активировать собственный промокод.
- Промокод можно активировать только один раз.
- Необходимо минимальное время игры на сервере (по умолчанию 1 час).
- Награды могут быть заблокированы во время рейда, комбата или после вайпа.

# Конфигурация Panel System

Файл конфигурации расположен по пути:
- **Oxide:** `oxide/config/PanelSystem.json`
- **Carbon:** `carbon/configs/PanelSystem.json`

## Общие параметры

| Параметр | По умолчанию | Описание |
|---|---|---|
| Work with Notify? | `true` | Использовать Notify для уведомлений |
| Display type | `"Overlay"` | Тип отображения: `Overlay` или `Hud` |
| Panel Anchor | `UpperRight` | Позиция основной панели |
| Events Panel Anchor | `UpperLeft` | Позиция панели событий |

## Логотип (Logotype)

| Параметр | Описание |
|---|---|
| Enabled | Включить логотип |
| Permission | Право для видимости |
| Image | URL изображения логотипа |
| Command | Команда при клике (пусто -- не кликабельно) |

## Онлайн (Players Settings)

| Параметр | Описание |
|---|---|
| Enabled | Включить панель онлайна |
| Permission | Право для видимости |
| Image | URL иконки |
| Settings | Настройки пользовательской кастомизации |
| Size | Доступные размеры (Big, Medium, Small) |

## Спящие (Sleepers Settings)

| Параметр | По умолчанию | Описание |
|---|---|---|
| Enabled | `false` | Включить панель спящих (по умолчанию отключена) |

## Время (Time Settings)

| Параметр | Описание |
|---|---|
| Enabled | Включить панель времени |
| GameUpdate | Показывать серверное время вместо реального |
| Permission | Право для видимости |

## Кнопка настроек (Settings Button)

| Параметр | Описание |
|---|---|
| Enabled | Включить кнопку настроек |
| Permission | Право для видимости |
| Image | URL иконки |
| Color | Цвет кнопки (HEX) |

## Быстрые кнопки (Buttons Setting)

| Параметр | Описание |
|---|---|
| Enabled | Включить панель кнопок |
| Permission | Право для видимости панели |
| Commands | Чат-команды для переключения панели кнопок |
| Buttons | Массив кнопок |

### Настройки кнопки

| Параметр | Описание |
|---|---|
| Enabled | Включить кнопку |
| Image | URL иконки |
| Permission | Право для видимости |
| Lang Key | Ключ текста в языковом файле |
| Command | Команда при нажатии |

## События (Events)

Массив событий для отслеживания:

| Параметр | Описание |
|---|---|
| Enabled | Включить индикатор |
| Image | URL иконки |
| Permission | Право для видимости |
| Type | Тип события (Helicopter, Bradley, Airdrop, CH47, CargoShip, Convoy, ArmoredTrain, Sputnik, SpaceEvent, AirEvent) |
| Settings | Настройки пользовательской кастомизации |
| Size | Доступные размеры |

## Экономика (Economics)

Массив экономических индикаторов:

| Параметр | Описание |
|---|---|
| Enabled | Включить индикатор |
| Permission | Право для видимости |
| Plugin | Название плагина экономики |
| Hook | Функция для получения баланса |
| Image | URL иконки |

## Скрытие панели (Hide Settings)

| Параметр | По умолчанию | Описание |
|---|---|---|
| Enabled | `false` | Включить возможность скрытия |
| Command | `"panel"` | Команда для скрытия/показа |
| Show Logo? | `true` | Показывать логотип при скрытой панели |

## Настройки пользовательской кастомизации

Каждый элемент панели может содержать объект `"Settings"`:

| Параметр | Описание |
|---|---|
| Enabled | Разрешить пользовательскую настройку |
| Permission | Право для настройки |
| CanChangeColor1/Color2 | Разрешить изменение цветов |
| CanChangeVisibility | Разрешить скрытие элемента |

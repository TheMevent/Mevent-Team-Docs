# Ultimate Leaderboard

## Что делает этот плагин?

**Ultimate Leaderboard** -- это комплексная система таблиц лидеров и профилей игроков для Rust. Плагин отслеживает убийства, смерти, добычу ресурсов, строительство, попадания по частям тела и множество других показателей. Данные хранятся в SQLite или MySQL базе данных, что обеспечивает высокую производительность даже на больших серверах.

## Ключевые возможности

- Полнофункциональная таблица лидеров с множеством категорий
- Подробные профили игроков с расширенной статистикой
- Вкладки (Tabs) с настраиваемыми блоками и полями
- Карта попаданий (Hit Rate) -- статистика по частям тела
- Статистика строительства (Building)
- Система наград за топ-позиции
- Хранение данных в SQLite или MySQL
- Кэширование таблицы лидеров с настраиваемым интервалом
- Поиск игроков по имени или SteamID
- Возможность скрыть себя из таблицы
- Кланы и клановый рейтинг
- Автосообщения в чат со статистикой
- Вайп данных при новом сохранении карты
- Система лута (Loot) для начисления очков
- Настройка учёта тиммейтов, кланов, друзей
- Настройки для "голых" игроков (naked players)
- Интеграция с ServerPanel, Notify, LangAPI, BetterChat, Clans, Friends
- Интеграция с экономикой: Economics, ServerRewards, BankSystem
- Discord webhook для логирования наград
- Поддержка Harmony-патчей

## Требования

| Зависимость | Обязательность | Описание |
|---|---|---|
| **ImageLibrary** | Обязательно | Для изображений в интерфейсе |
| Notify | Опционально | Для графических уведомлений |
| ServerPanel | Опционально | Для интеграции в серверное меню |
| LangAPI | Опционально | Для мультиязычности |
| Clans / Friends | Опционально | Для клановой статистики и фильтрации |
| Oxide / uMod / Carbon | Обязательно | Совместим с Oxide, uMod и Carbon |

## Быстрый старт

1. Установите плагин **ImageLibrary**.
2. Скопируйте файл `UltimateLeaderboard.cs` в папку `oxide/plugins/`.
3. Перезагрузите сервер или выполните `oxide.reload UltimateLeaderboard`.
4. Введите в чат `/leaderboard` или `/stats` для открытия интерфейса.

# Часто задаваемые вопросы (FAQ)

## 1. Как перейти с SQLite на MySQL?

1. Настройте параметры MySQL в конфигурации.
2. Используйте консольную команду для миграции:
 ```
 leaderboard.migrate
 ```
3. Измените `"Storage Type"` на `"MySQL"`.
4. Перезагрузите плагин.

---

## 2. Как скрыть себя из таблицы лидеров?

Игроку нужно право `ultimateleaderboard.hide.self`:
```
oxide.grant user <steamid> ultimateleaderboard.hide.self
```

Затем игрок использует команду `/leaderboard.hide` для переключения видимости.

---

## 3. Как сбросить все данные?

Выполните в серверной консоли:
```
leaderboard.wipe
```

Для автоматического сброса при вайпе включите `"Wipe data on new save": true`.

---

## 4. Таблица лидеров обновляется медленно. Что делать?

Уменьшите интервал кэширования:
```json
"Leaderboard Cache Interval (seconds)": 300
```

Увеличьте размер пакета:
```json
"Leaderboard Cache Batch Size": 20
```

:::warning Внимание
Слишком частое обновление может увеличить нагрузку на сервер.
:::

---

## 5. Как отключить подсчёт убийств тиммейтов?

Установите в конфигурации:
```json
"Count damage/kills to teammates": false,
"Count damage/kills to clan members": false,
"Count damage/kills to friends": false
```

---

## 6. Как настроить награды за топ-позиции?

В секции `Awards` конфигурации настройте категории наград и места. Каждое место может содержать список предметов, команд или плагинных наград.

---

## 7. Как включить логирование наград в Discord?

Укажите Discord webhook URL:
```json
"Discord webhook for log rewards": "https://discord.com/api/webhooks/..."
```

---

## 8. Как добавить новую вкладку?

В массиве `"Tabs"` добавьте новый объект с уникальным именем и блоками. Блоки определяют, какая статистика отображается на вкладке.

---

## 9. Данные статистики пропадают после перезапуска. Почему?

Возможные причины:
1. `"Wipe data on new save"` установлен в `true` -- данные сбрасываются при вайпе.
2. Проблемы с базой данных -- проверьте логи сервера.
3. Файл базы данных повреждён -- восстановите из резервной копии.

---

## 10. Как включить отладочную информацию?

Установите `"Show debug information": true`. Это покажет все обновления статистики в консоли, что полезно для настройки и диагностики.

---

## 11. Поддерживается ли GDPR?

Да. Включите параметр `"Disable IP address logging": true` для отключения логирования IP-адресов игроков.

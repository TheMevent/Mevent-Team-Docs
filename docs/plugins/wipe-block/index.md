# Wipe Block

## Что делает этот плагин?

**Wipe Block** позволяет вашему серверу автоматически блокировать определённые предметы после вайпа на заданное время. Игроки не смогут использовать, надевать или крафтить заблокированные предметы до истечения установленного таймера. Плагин предоставляет красивый интерфейс с таймерами обратного отсчёта, чтобы игроки всегда знали, когда откроется доступ к нужным вещам.

## Ключевые возможности

- Гибкая блокировка любых предметов с разделением по категориям (оружие, взрывчатка, броня и другие)
- Индивидуальные таймеры разблокировки для каждого предмета (от минут до дней)
- Красивый внутриигровой интерфейс со списком заблокированных и разблокированных предметов
- Виджет на экране с информацией о текущей блокировке
- Уведомления при разблокировке предметов (поддержка Notify и UINotify)
- Запрет крафта заблокированных предметов (опционально)
- Запрет на установку заблокированных предметов в турели, РСЗО и дроны
- Интеграция с ServerPanel для отображения в серверном меню
- Градиентная цветовая шкала прогресса разблокировки

## Требования

- Сервер Rust с Oxide/uMod
- Рекомендуемые плагины (не обязательно): Notify или UINotify (для уведомлений), ServerPanel (для серверного меню)

## Быстрый старт

1. Скопируйте файл плагина в папку `oxide/plugins` на вашем сервере.
2. Перезагрузите сервер или загрузите плагин командой `o.load WipeBlock`.
3. Откройте конфигурационный файл `oxide/config/WipeBlock.json` и настройте категории предметов и время блокировки.
4. Перезагрузите плагин после внесения изменений в конфигурацию.
5. Проверьте работу, введя в чат `/block` или `/wipeblock`.

## Права доступа

| Право | Описание |
|---|---|
| `wipeblock.ignore` | Игрок с этим правом игнорирует все блокировки и может использовать любые предметы |
| `wipeblock.unlocknotify` | Игрок получает уведомления о разблокировке предметов |

## Команды игроков

| Команда | Описание |
|---|---|
| `/block` | Открывает главное меню блокировки после вайпа |
| `/wipeblock` | Альтернативная команда для открытия меню |

:::tip Пример
Введите `/block` в чате, чтобы увидеть список всех заблокированных и разблокированных предметов с таймерами.
:::

## Команды администратора

| Команда | Описание |
|---|---|
| `/wb.indent [секунды]` | Сдвигает время начала блокировки на указанное количество секунд (положительное значение -- вперёд, отрицательное -- назад) |

:::tip Пример
`/wb.indent 3600` -- сдвинет расчёт блокировки на 1 час вперёд, ускорив разблокировку предметов.
:::

## Устранение неполадок

| Проблема | Решение |
|---|---|
| Меню не открывается командой `/block` | Убедитесь, что команды не изменены в конфигурации. Проверьте, что плагин загружен (`o.load WipeBlock`). |
| Предметы не блокируются после вайпа | Проверьте, что в конфигурации указаны предметы с корректными ShortName и временем блокировки больше 0. |
| Игрок может использовать заблокированные предметы | Убедитесь, что у игрока нет права `wipeblock.ignore`. Проверьте, что параметр "Prevent to use blocked items" включен. |
| Крафт заблокированных предметов не запрещён | Включите параметр "Prevent to craft blocked items" в конфигурации. |
| Виджет на экране не отображается | Проверьте, что параметр "Show on screen?" включен в разделе OnScreen Settings конфигурации. |
| Градиенты работают некорректно | Убедитесь, что в списке градиентов задано не менее 101 значения цвета. |

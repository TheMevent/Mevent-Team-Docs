# Команды и права

## Команды

### Команды администратора

| Команда | Описание | Право |
|---|---|---|
| `foundationdrop.start` | Запустить ивент Foundation Drop | `MultiEventsFoundationDrop.start` |
| `foundationdrop.stop` | Остановить текущий ивент | `MultiEventsFoundationDrop.stop` |

:::info Примечание
Ивент также может быть запущен автоматически через систему MultiEvents по расписанию.
:::

### Консольные команды (UI)

| Команда | Описание |
|---|---|
| `UI_FoundationDrop vote` | Присоединиться к ивенту во время фазы голосования |
| `UI_FoundationDrop hide` | Скрыть панель ивента с экрана |

## Права (Permissions)

| Право | Описание |
|---|---|
| `MultiEventsFoundationDrop.start` | Право на запуск ивента |
| `MultiEventsFoundationDrop.stop` | Право на остановку ивента |

### Настройка прав

```
oxide.grant group admin MultiEventsFoundationDrop.start
oxide.grant group admin MultiEventsFoundationDrop.stop
```

## Команды игроков

Игроки присоединяются к ивенту через UI-кнопку голосования во время фазы регистрации. Специальных чат-команд нет.

## Фазы ивента

1. **Голосование** -- игроки нажимают кнопку для участия. Длительность настраивается параметром `Voting`.
2. **Ожидание** -- подготовка арены и телепортация игроков. Длительность настраивается параметром `Waiting`.
3. **Игра** -- фундаменты начинают разрушаться, игроки должны выживать.
4. **Финальная битва** -- когда остаётся один фундамент, включается радиация и выдаётся оружие.
5. **Завершение** -- последний выживший побеждает.

## Блокировки во время ивента

Во время активного ивента для участников блокируются:
- Команды, не указанные в списке `"Allowed commands on event"`
- Телепортация (через NTeleportation)
- Выход из зоны (через NoEscape, если включено)

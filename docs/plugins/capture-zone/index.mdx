# Capture Zone

## Что делает этот плагин?

**Capture Zone** позволяет вашему серверу добавить механику захвата территорий с помощью специального предмета. Игроки устанавливают особый шкаф, который создаёт зону захвата вокруг себя. Все остальные игроки, добывающие ресурсы в этой зоне, теряют часть добычи -- она автоматически переносится в шкаф владельца зоны. Плагин поддерживает маркеры на карте, систему лимитов и настраиваемый дроп из контейнеров.

## Ключевые возможности

- **Захват территории** -- установка специального шкафа для создания зоны контроля
- **Налог на ресурсы** -- процент от добычи других игроков автоматически попадает в шкаф владельца
- **Маркеры на карте** -- зоны захвата отображаются на игровой карте с настраиваемыми цветами
- **Система лимитов** -- ограничение количества зон для каждого игрока с поддержкой VIP-прав
- **Настраиваемый процент** -- разный процент налога для разных уровней доступа
- **Обход для друзей** -- тиммейты, друзья, клан и авторизованные в шкафу не облагаются налогом
- **Дроп из контейнеров** -- специальный предмет захвата может выпадать из лутовых контейнеров
- **Уведомления** -- оповещения игрокам при добыче на чужой территории

## Требования

| Требование | Описание |
|---|---|
| Oxide / Carbon | Фреймворк для модификаций сервера |

### Опциональные зависимости

| Плагин | Описание |
|---|---|
| Notify / UINotify | Всплывающие уведомления |
| Clans | Интеграция с кланами (обход налога для клана) |
| Friends | Интеграция с друзьями (обход налога для друзей) |
| StackSizeController / StackModifier | Совместимость со стаками предметов |

## Быстрый старт

1. Установите `CaptureZone.cs` в папку `oxide/plugins`
2. Перезагрузите сервер
3. Выдайте предмет захвата зоны администратору: `/give.capturezone ваш_steamid 1`
4. Установите специальный шкаф на выбранном месте
5. Зона захвата автоматически создаётся вокруг шкафа

## Права доступа

| Право | Описание |
|---|---|
| `capturezone.ignore` | Игнорирование налога -- ресурсы не забираются на чужих территориях |
| `capturezone.vip` | Увеличенный процент налога (25%) и лимит зон (5) |
| `capturezone.deluxe` | Максимальный процент налога (30%) и лимит зон (7) |

:::note Настраиваемые права
Права `capturezone.vip` и `capturezone.deluxe` задаются в конфигурации. Вы можете изменить названия прав, процент налога и лимиты.
:::

Дополнительные (опциональные) права:

| Право | Описание |
|---|---|
| Право на установку | Если задано в конфигурации, только игроки с этим правом смогут устанавливать зоны |
| Право на видимость маркера | Если задано, только игроки с этим правом видят маркеры на карте |

## Команды администратора

| Команда | Описание |
|---|---|
| `/give.capturezone [steamid] [количество]` | Выдать предмет захвата зоны указанному игроку |

:::tip Пример
```
/give.capturezone 76561198012345678 3
```
Выдаст 3 предмета захвата зоны игроку с указанным SteamID.
:::

## Команды игроков

У данного плагина нет чат-команд для обычных игроков. Игроки взаимодействуют с плагином через установку и уничтожение специального шкафа.

## Устранение неполадок

| Проблема | Решение |
|---|---|
| Шкаф не создаёт зону захвата | Убедитесь, что используется специальный предмет со скином, а не обычный шкаф. Используйте `/give.capturezone` для получения правильного предмета |
| "Вы слишком близко к другой зоне" | Зоны захвата не могут перекрываться. Установите шкаф на расстоянии не менее двойного радиуса зоны от ближайшей |
| "Вы достигли лимита зон" | Игрок превысил допустимое количество зон. Выдайте VIP-права для увеличения лимита |
| Маркер не отображается на карте | Проверьте, что маркер включён в конфигурации и игрок имеет право на его просмотр |
| Ресурсы не забираются | Проверьте, что игрок не является тиммейтом, другом, кланмейтом или авторизованным в шкафу |
| Предмет не стакается | Установите плагин StackSizeController или StackModifier для корректной работы стаков |

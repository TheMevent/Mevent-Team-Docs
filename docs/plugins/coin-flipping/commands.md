# Команды и права

## Команды чата

| Команда | Описание | Право |
|---|---|---|
| `/flip` | Открыть интерфейс подбрасывания монетки | `coinflipping.use` |

:::tip Совет
Команда чата настраивается в конфигурации в параметре `Commands`. По умолчанию используется `flip`, но вы можете добавить любые альтернативные команды.
:::

## Консольные команды

У плагина нет специальных консольных команд для администраторов. Внутренние команды интерфейса обрабатываются через консольную команду `UI_CoinFlip`, которая используется исключительно для навигации в UI.

## Права (Permissions)

| Право | Описание | По умолчанию |
|---|---|---|
| `coinflipping.use` | Открытие интерфейса плагина | Не выдано |
| `coinflipping.npcs` | Игра против бота | Не выдано |
| `coinflipping.players` | Игра против других игроков | Не выдано |
| `coinflipping.createroom` | Создание комнаты для игры с другими игроками | Не выдано |

### Настройка прав

Выдача прав всем игрокам:
```
oxide.grant group default coinflipping.use
oxide.grant group default coinflipping.npcs
oxide.grant group default coinflipping.players
oxide.grant group default coinflipping.createroom
```

Выдача прав только VIP-группе:
```
oxide.grant group vip coinflipping.players
oxide.grant group vip coinflipping.createroom
```

:::info Примечание
Названия прав настраиваются в конфигурации в секции `Permissions Settings`. Вы можете изменить их на любые другие.
:::

## Игровой процесс

### Игра против бота

1. Откройте интерфейс командой `/flip`.
2. Выберите режим **BOT**.
3. Выберите сторону монеты: **HEADS** (орёл) или **TAILS** (решка).
4. Введите сумму ставки или выберите из предложенных вариантов.
5. Нажмите **PLAY**.
6. Монета подбрасывается, и результат определяется случайным образом.

### Игра против игрока

1. Откройте интерфейс командой `/flip`.
2. Выберите режим **PLAYER**.
3. Создайте комнату с указанием ставки или присоединитесь к существующей комнате.
4. Дождитесь, пока оба игрока будут готовы.
5. Монета подбрасывается, и победитель получает обе ставки.

# Установка Statistics

Это руководство проведёт вас через процесс установки системы статистики.

## Шаг 1: Установите зависимости

### ImageLibrary (обязательно)

1. Скачайте **ImageLibrary** с [umod.org](https://umod.org/plugins/image-library).
2. Поместите файл `ImageLibrary.cs` в папку `oxide/plugins/`.
3. Убедитесь, что плагин загрузился без ошибок.

### Дополнительные зависимости (опционально)

- **PlayTimeRewards** или **PlaytimeTracker** -- для отслеживания игрового времени
- **Notify** -- для графических уведомлений
- **ServerPanel** -- для интеграции в серверное меню
- **PlayerDatabase** -- для сохранения данных в базе

## Шаг 2: Установите плагин

1. Скопируйте файл `Statistics.cs` в папку `oxide/plugins/`.
2. Плагин загрузится автоматически.
3. Проверьте консоль на наличие ошибок.

Файл конфигурации:
```
oxide/config/Statistics.json
```

## Шаг 3: Настройте права доступа

Основные права:

```
oxide.grant group default statistics.use
```

Для скрытия из таблицы лидеров:

```
oxide.grant group admin statistics.hide
```

## Шаг 4: Настройте таблицу очков

В конфигурации в секции `"Score Table (shortname - score)"` настройте очки за различные действия:

```json
{
 "kills": 1,
 "deaths": -1,
 "supply_drop": 3,
 "helicopter": 15,
 "bradley": 10
}
```

## Шаг 5: Настройте список оружия и ресурсов

В секциях `"Weapons"` и `"Resources"` укажите, какое оружие и ресурсы отслеживать:

```json
{
 "Weapons": ["rifle.ak", "rifle.bolt", "rifle.l96"],
 "Resources": ["stones", "sulfur.ore", "metal.ore", "wood"]
}
```

## Шаг 6: Проверьте работу

1. Зайдите на сервер.
2. `/stats` -- просмотр профиля.
3. `/leaders` -- таблица лидеров.

## Шаг 7: Перезагрузите плагин

```
oxide.reload Statistics
```

## Обновление плагина

1. Замените файл `Statistics.cs`.
2. Плагин автоматически обновит конфигурацию.

:::note Примечание
Рекомендуется сохранять резервную копию конфигурации и данных перед обновлением.
:::

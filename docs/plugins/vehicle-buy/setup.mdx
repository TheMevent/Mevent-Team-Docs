# Установка Vehicle Buy

## Системные требования

- Сервер Rust с Oxide/uMod или Carbon
- Плагин **KaruzaEntitiesCommon** (обязателен)
- Плагин **ImageLibrary** (обязателен)
- Один из экономических плагинов (Economics, ServerRewards, BankSystem) или использование внутриигровых предметов

## Пошаговая установка

### Шаг 1: Установка зависимостей

1. Установите **KaruzaEntitiesCommon** в папку `oxide/plugins`.
2. Установите **ImageLibrary** в папку `oxide/plugins`.
3. (Опционально) Установите плагин экономики.
4. (Опционально) Установите **Notify** или **UINotify** для уведомлений.

### Шаг 2: Установка плагина

1. Загрузите файл `VehicleBuy.cs`.
2. Поместите его в папку `oxide/plugins`.
3. Перезагрузите сервер или выполните:
 ```
 oxide.reload VehicleBuy
 ```

### Шаг 3: Первоначальная настройка

Конфигурационный файл создастся автоматически:
```
oxide/config/VehicleBuy.json
```

Основные шаги:

1. **Команды** -- задайте команды для магазина (по умолчанию: `vehiclebuy`, `vb`, `vehicle`).
2. **Команды отзыва** -- задайте команды для recall (по умолчанию: `callback`).
3. **Команды подбора** -- задайте команды для pickup (по умолчанию: `pickup`).
4. **Валюта** -- настройте названия валют для Economics, Server Rewards и BankSystem.
5. **Транспорт** -- отредактируйте список доступного транспорта.
6. **Торговые автоматы** -- включите автоматы в нужных мирных зонах.

### Шаг 4: Выдача прав доступа

```
oxide.grant group default vehiclebuy.use
oxide.grant group vip vehiclebuy.free
oxide.grant group default vehiclebuy.pickup
oxide.grant group default vehiclebuy.recall
```

### Шаг 5: Настройка торговых автоматов (опционально)

В конфигурации включите торговые автоматы для нужных мирных зон:
- Bandit Camp
- Outpost
- Fishing Village A, B, C

Для каждого автомата настройте список товаров с указанием ключа транспорта, предмета оплаты и цены.

## Совместимость с Carbon

Плагин полностью совместим с Carbon. При использовании Carbon вместо ImageLibrary автоматически используется встроенная система хранения изображений Carbon.

## Проверка установки

1. Зайдите на сервер.
2. Введите `/vehiclebuy` в чат.
3. Должен открыться магазин с доступным транспортом.
4. Купите транспорт и убедитесь, что он появляется рядом.
5. Проверьте работу торговых автоматов в мирных зонах.

## Вайп данных

Плагин поддерживает автоматический вайп данных при новом сохранении карты. Для включения этой функции установите параметр `Wipe on new map save` в `true`.

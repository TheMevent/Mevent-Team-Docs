# Установка Referrals

Это руководство проведёт вас через процесс установки реферальной системы.

## Шаг 1: Установите зависимости

### ImageLibrary (обязательно)

1. Скачайте **ImageLibrary** с [umod.org](https://umod.org/plugins/image-library).
2. Поместите файл `ImageLibrary.cs` в папку `oxide/plugins/`.
3. Убедитесь, что плагин загрузился без ошибок.

### Дополнительные зависимости (опционально)

- **Notify** -- для графических уведомлений
- **NoEscape** -- для блокировки наград во время рейда/комбата

## Шаг 2: Установите плагин

1. Скопируйте файл `Referrals.cs` в папку `oxide/plugins/` вашего сервера.
2. Плагин загрузится автоматически.
3. Проверьте консоль на наличие ошибок.

Файл конфигурации будет автоматически создан по пути:
```
oxide/config/Referrals.json
```

## Шаг 3: Настройте права доступа (опционально)

По умолчанию право доступа пустое (`""`), что означает доступ для всех игроков. Если вы хотите ограничить доступ:

```json
"Permission (example: referrals.use)": "referrals.use"
```

Затем выдайте право:
```
oxide.grant group default referrals.use
```

## Шаг 4: Настройте награды

Откройте файл `oxide/config/Referrals.json` и настройте:

1. **Awards** -- награды для реферера за количество приглашений
2. **Award for the player who activates the promo code** -- награды для активировавшего промокод

## Шаг 5: Настройте минимальное время игры

Для защиты от мультиаккаунтов настройте минимальное время игры, необходимое для активации промокода:

```json
"Minimum play time (seconds)": 3600
```

По умолчанию: 3600 секунд (1 час).

## Шаг 6: Проверьте работу

1. Зайдите на сервер.
2. Введите в чат `/ref` -- откроется интерфейс с промокодом.
3. С другого аккаунта введите `/promo <код>` для активации.

## Шаг 7: Перезагрузите плагин

После изменения конфигурации:

```
oxide.reload Referrals
```

## Обновление плагина

При обновлении:

1. Замените файл `Referrals.cs` новой версией.
2. Плагин автоматически обновит конфигурацию.
3. Перезагрузите плагин.

:::note Примечание
Рекомендуется сохранять резервную копию конфигурации и данных перед обновлением.
:::

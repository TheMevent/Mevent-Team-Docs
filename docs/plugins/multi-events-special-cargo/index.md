# Special Cargo

## Что делает этот режим?

**Special Cargo** — это уникальный режим ивента для Multi Events, основанный на доставке груза. При запуске ивента случайный игрок становится курьером и должен доставить специальный груз от стартового монумента к конечному. Если курьер погибает, роль переходит другому случайному игроку. Телепортация для курьера заблокирована. Первый курьер, успешно доставивший груз, побеждает.

## Ключевые возможности

- Случайный выбор курьера среди активных игроков
- Маркеры на карте для стартового и конечного монументов
- Отслеживание позиции курьера на карте в реальном времени
- Блокировка телепортации для курьера
- Смена курьера при его гибели
- Настраиваемые списки запрещённых монументов
- Настраиваемый радиус зоны монумента
- Поддержка уведомлений через Notify и UINotify
- Встроенный UI-виджет с обратным отсчётом и расстоянием до цели
- Интеграция с инвентарём ивентов (Multi Events Inventory)

## Требования

| Зависимость | Обязательно? |
|---|---|
| **Multi Events** | Да |
| **ImageLibrary** | Да |
| Multi Events Inventory | Рекомендуется |
| Multi Events Leaderboard | Рекомендуется |
| Notify / UINotify | Опционально |

## Быстрый старт

1. Установите **Multi Events** на сервер
2. Поместите файл режима Special Cargo в папку `oxide/plugins`
3. Перезагрузите сервер или загрузите плагин
4. Файл конфигурации будет создан автоматически
5. Настройте монументы, маркеры и награды в конфигурации
6. Ивент готов к запуску через Multi Events или вручную командой

## Как проходит ивент

1. Ивент запускается автоматически через Multi Events или вручную администратором
2. Случайно выбираются стартовый и конечный монументы на карте
3. На карте появляются маркеры монументов
4. Случайный онлайн-игрок назначается курьером
5. Курьер должен добраться от стартового монумента до конечного
6. Позиция курьера отмечается маркером на карте
7. Телепортация для курьера заблокирована
8. Если курьер погибает, новый курьер выбирается случайным образом
9. Когда курьер достигает конечного монумента, он побеждает и получает награды
10. Ивент также завершается по истечении таймера

## Настройки

| Параметр | Описание | По умолчанию |
|---|---|---|
| Work with Notify? | Использовать ли плагин Notify для уведомлений | `true` |
| Block TP? | Блокировать телепортацию для курьера | `false` |
| Monument Radius | Радиус зоны монумента для засчитывания доставки | `50` |
| Event Duration | Длительность ивента в секундах | `600` |
| Blocked Monuments | Список запрещённых монументов | oilrig, cave, power_sub |
| Monument Marker | Настройки маркера монументов | Цвет, радиус, частота обновления |
| Runner Marker | Настройки маркера курьера | Отображаемое имя |
| Reward for the winner | Награда для победителя | Предметы |

## Устранение неполадок

| Проблема | Решение |
|---|---|
| Ивент не запускается | Убедитесь, что Multi Events установлен и на карте достаточно монументов |
| Маркеры не отображаются | Проверьте настройки маркеров в конфигурации |
| Нет UI-уведомлений | Проверьте, установлен ли ImageLibrary |
| Награды не выдаются | Проверьте корректность настроек наград в конфигурации |

# Установка Notify

Это руководство проведёт вас через весь процесс установки плагина уведомлений на вашем сервере.

## Шаг 1: Установите зависимости

### ImageLibrary (обязательно)

Плагин Notify требует ImageLibrary для работы с изображениями в уведомлениях.

1. Скачайте **ImageLibrary** с [umod.org](https://umod.org/plugins/image-library).
2. Поместите файл `ImageLibrary.cs` в папку `oxide/plugins/`.
3. Убедитесь, что плагин загрузился без ошибок.

:::warning Важно
Без ImageLibrary плагин Notify **не будет работать**. В консоли появится ошибка: `IMAGE LIBRARY IS NOT INSTALLED!`
:::

## Шаг 2: Установите плагин

1. Скопируйте файл `Notify.cs` в папку `oxide/plugins/` вашего сервера.
2. Плагин загрузится автоматически.
3. Проверьте консоль на наличие ошибок.

Файл конфигурации будет автоматически создан по пути:
```
oxide/config/Notify.json
```

## Шаг 3: Настройте права доступа

Выдайте право на просмотр уведомлений всем игрокам:

```
oxide.grant group default notify.see
```

Если вы хотите, чтобы только определённая группа видела уведомления:

```
oxide.group add vip
oxide.grant group vip notify.see
```

:::info Примечание
Игроки без права `notify.see` получат текстовое сообщение в чат вместо графического уведомления (если эта функция включена в настройках).
:::

### Права для администраторов

```
oxide.grant group admin notify.notify
oxide.grant group admin notify.player
oxide.grant group admin notify.allplayer
```

## Шаг 4: Проверьте работу

Зайдите на сервер и выполните команду в консоли:

```
notify.show 0 Тестовое уведомление!
```

Вы должны увидеть синее всплывающее уведомление в правом верхнем углу экрана.

Для проверки уведомления типа "ошибка":

```
notify.show 1 Это тестовая ошибка!
```

## Шаг 5: Настройте конфигурацию (по желанию)

Откройте файл `oxide/config/Notify.json` для тонкой настройки:

- **Позиция на экране** -- верхний правый или нижний левый угол
- **Размеры** -- ширина и высота уведомлений
- **Время отображения** -- как долго уведомление остаётся на экране
- **Максимальное количество** -- сколько уведомлений одновременно показывать
- **Типы уведомлений** -- настройка цветов, иконок и поведения

Подробнее о каждом параметре читайте в разделе [Конфигурация](./configuration).

## Шаг 6: Перезагрузите плагин

После изменения конфигурации перезагрузите плагин:

```
oxide.reload Notify
```

:::tip Совет
Если вы используете Notify как систему уведомлений для других плагинов (например, магазин, рейды, события), убедитесь, что Notify загружается **раньше** этих плагинов, или просто перезагрузите зависимые плагины после установки Notify.
:::

## Обновление плагина

При обновлении плагина:

1. Замените файл `Notify.cs` новой версией.
2. Плагин автоматически обновит конфигурацию, сохранив ваши настройки.
3. В консоли появится сообщение: `Config update detected! Updating config values...`
4. После обновления: `Config update completed!`

:::note Примечание
Резервные копии конфигурации создаются автоматически перед обновлением. Рекомендуется также сохранять копию вручную перед крупными обновлениями.
:::

# Команды Battlepass

## Команды для игроков

Все команды ниже вводятся в игровой чат (начиная с `/`).

### Основная команда

```
/pass
```

Открывает главное меню боевого пропуска с вкладками: миссии, кейсы, инвентарь.

**Требование:** право доступа `battlepass.use`

---

### Прямой доступ к разделам

| Команда | Открывает |
|---|---|
| `/passmissions` или `/pmissions` | Страница миссий |
| `/passcase` | Страница кейсов |
| `/passinv` | Страница инвентаря наград |

:::info
Все команды можно переименовать в конфигурации плагина. Указанные выше — значения по умолчанию.
:::

## Команды администратора

### Управление валютой

#### Начисление первой валюты
```
addfirstcurrency [steamid/имя] [количество]
```
**Пример:**
```
addfirstcurrency 76561198012345678 5000
```
Начисляет 5000 единиц первой валюты указанному игроку.

#### Начисление второй валюты
```
addsecondcurrency [steamid/имя] [количество]
```
**Пример:**
```
addsecondcurrency 76561198012345678 2000
```
Начисляет 2000 единиц второй валюты указанному игроку.

---

### Управление миссиями

#### Сброс миссий (из серверной консоли)
```
battlepass.wipemissions
```
Полностью сбрасывает все текущие миссии и генерирует новые. Используется для принудительного обновления заданий.

**Кто может использовать:** только администратор из серверной консоли или RCON.

---

### Утилиты

#### Конвертация старых данных
```
battlepass.convert.olddata
```
Конвертирует данные игроков из старого формата хранения в новый. Используется при обновлении с более ранних версий плагина.

**Кто может использовать:** только администратор из серверной консоли.

---

## Навигация в интерфейсе

Внутри меню боевого пропуска доступны следующие действия:

| Действие | Описание |
|---|---|
| Переход между вкладками | Используйте кнопки «Миссии», «Кейсы», «Инвентарь» |
| Просмотр кейса | Нажмите «Посмотреть» на нужном кейсе |
| Открытие кейса | Нажмите «Открыть» и подтвердите |
| Выбор валюты | Выберите, какой валютой оплачивать кейс |
| Получение предмета | Нажмите на предмет в инвентаре, чтобы забрать его |
| Пагинация | Используйте стрелки для перехода между страницами |

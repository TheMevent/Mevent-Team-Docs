# Player List

## Что делает этот плагин?

**Player List** добавляет на сервер визуальный список всех онлайн-игроков с красивым интерфейсом. Игроки могут просматривать список, открывать профили других игроков с дополнительной информацией (статистика, клан, рейтинг) и выполнять действия через кнопки (телепортация, обмен, кик). Интерфейс поддерживает пагинацию, поиск и гибкую настройку.

## Ключевые возможности

- Визуальный список всех онлайн-игроков
- Профили игроков с настраиваемыми полями (клан, рейтинг, оружие, K/D)
- Кнопки действий (телепортация, обмен, статистика, кик)
- Интеграция с плагинами Statistics, Clans, Friends
- Настраиваемые права для каждой кнопки и поля
- Скрытие игроков с правом `playerlist.hide`
- Поддержка ImageLibrary для аватаров
- Интеграция с Notify для уведомлений

## Требования

| Зависимость | Обязательность | Описание |
|---|---|---|
| **ImageLibrary** | Обязательно | Загрузка аватаров игроков |
| **Notify** / **UINotify** | По желанию | Уведомления |
| **Friends** | По желанию | Отображение статуса друга |
| **Clans** | По желанию | Отображение клана |
| Oxide / uMod / Carbon | Обязательно | Совместим с Oxide, uMod и Carbon |

## Быстрый старт

1. Установите **ImageLibrary** на сервер.
2. Скопируйте файл `PlayerList.cs` в папку `oxide/plugins/`.
3. Перезагрузите сервер или выполните `oxide.reload PlayerList`.
4. Введите `/players` или `/plist` в чат.
